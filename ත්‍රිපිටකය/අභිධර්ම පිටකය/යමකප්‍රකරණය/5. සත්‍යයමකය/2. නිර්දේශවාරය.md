# 2\. නිර්දේශවාරය

1\. දුඃඛය දුඃඛසත්‍යය ද යත්: එසේ යි. දුඃඛසත්‍යය දුඃඛය ද යත්: කායික දුඃඛය චෛතසික දුඃඛය තබා සෙස්ස දුඃඛ සත්‍යය යි. දුඃඛ නො වේ. කායික දුඃඛය චෛතසික දුඃඛය දුඃඛ ද වූයේ ම දුඃඛ සත්‍ය ද වේ.

සමුදය සමුදයසත්‍ය ද යත්: සමුදය සත්‍යය තබා අවශේෂ වූ සමුදය සමුදය යි. සමුදයසත්‍යය නො වේ. සමුදයසත්‍යය සමුදය වූයේ ම සමුදයසත්‍යය ද වේ. සමුදයසත්‍යය සමුදය ද යත්: එසේ යි.

නිරෝධය නිරෝධසත්‍ය ද යත්: නිරෝධසත්‍යය තබා අවශේෂ වූ නිරෝධය නිරෝධය වෙයි, නිරෝධසත්‍යය නො වේ. නිරෝධසත්‍යය නිරෝධ ද වූයේ ම නිරෝධසත්‍ය ද වේ. නිරෝධසත්‍යය නිරෝධ ද යත්: එසේ යි.

මාර්ගය මාර්ගසත්‍ය ද යත්: මාර්ගසත්‍යය තබා අවශේෂ වූ මාර්ගය මාර්ගය වෙයි, මාර්ගසත්‍යය නො වේ. මාර්ගසත්‍යය මාර්ග ද වූයේ ම මාර්ගසත්‍යය ද වේ. මාර්ගසත්‍යය මාර්ග ද යත්: එසේ යි. (අනුලෝම යි.)

2\. දුඃඛ නො වූයෙක් දුඃඛසත්‍යය නො වේ ද යත්: කායික දුඃඛය චෛතසික දුඃඛය තබා අවශේෂය දුඃඛය නො වෙයි, දුඃඛ සත්‍යය වේ. දුඃඛය ද දුඃඛසත්‍යය ද තබා අවශේෂය දුඃඛ ද නො වූයේ ම දුඃඛසත්‍ය ද නො වේ. දුඃඛසත්‍යය නො වූයෙක් දුඃඛය නො වේ ද යත්: එසේ යි.

සමුදය නො වූයෙක් සමුදයසත්‍යය නො වේ ද යත්: එසේ යි. සමුදයසත්‍යය නො වූයෙක් සමුදය නො වේ ද යත්: සමුදයසත්‍යය තබා අවශේෂ වූ සමුදය සමුදයසත්‍යය නො වෙයි, සමුදය වේ. සමුදය ද සමුදයසත්‍යය ද තබා අවශේෂය සමුදය ද නො වූයේ ම සමුදයසත්‍යය ද නො වේ.

නිරෝධ නො වූයෙක් නිරෝධසත්‍යය නො වේ ද යත්: එසේ යි. නිරෝධසත්‍යය නො වූයෙක් නිරෝධය නො වේ ද යත්: නිරෝධසත්‍යය තබා අවශේෂ වූ නිරෝධය නිරෝධසත්‍යය නො වෙයි. නිරෝධය වේ. නිරෝධය ද නිරෝධසත්‍යය ද තබා අවශේෂය නිරෝධය ද නො වූයේ ම නිරෝධසත්‍යය ද නො වේ.

මාර්ග නො වූයෙක් මාර්ගසත්‍යය නො වේ ද යත්: එසේ යි. මාර්ගසත්‍යය නො වූයෙක් මාර්ගය නො වේ ද යත්: මාර්ග සත්‍යය තබා, අවශේෂ වූ මාර්ගය මාර්ගසත්‍යය නො වෙයි. මාර්ගය වේ. මාර්ගය ද මාර්ගසත්‍යය ද තබා අවශේෂය මාර්ගය ද නො වූයේ ම මාර්ගසත්‍යය ද නො වේ. (ප්‍රත්‍යනීක යි.)

පදශෝධනවාර යි.

3\. දුඃඛය දුඃඛසත්‍යය ද යත්: එසේ යි. සත්‍යයෝ සමුදය සත්‍ය ද යත්: සමුදයසත්‍යය සත්‍ය ද වූයේ ම සමුදයසත්‍ය ද වෙයි, අවශේෂ සත්‍යයෝ සත්‍යයෝ යි. සමුදය සත්‍යය නො වේ.

දුඃඛය දුඃඛසත්‍ය ද යත්: එසේ යි. සත්‍යයෝ නිරෝධසත්‍ය ද යත්: ... සත්‍යයෝ මාර්ගසත්‍ය ද යත්: මාර්ගසත්‍යය සත්‍ය ද වූයේ ම මාර්ගසත්‍ය ද වේ. අවශේෂසත්‍යයෝ සත්‍යයෝ යි, මාර්ගසත්‍යය නො වේ.

සමුදය සමුදයසත්‍යය ද යත්: සමුදයසත්‍යය තබා අවශේෂ වූ සමුදය සමුදය වෙයි. සමුදයසත්‍යය නො වේ. සමුදයසත්‍යය සමුදය ද වූයේ ම සමුදයසත්‍යය ද වේ. සත්‍යයෝ දුඃඛසත්‍ය ද යත්: ... සත්‍යයෝ නිරෝධසත්‍ය ද යත්: ... සත්‍යයෝ මාර්ගසත්‍ය ද යත්: මාර්ගසත්‍යය සත්‍ය ද වූයේ ම මාර්ගසත්‍යයද වේ. අවශේෂ සත්‍යයෝ සත්‍යයෝ යි, මාර්ගසත්‍යය නො වේ.

නිරෝධය නිරෝධසත්‍ය ද යත්: නිරෝධසත්‍යය තබා අවශේෂ වූ නිරෝධය නිරෝධය වෙයි, නිරෝධසත්‍යය නො වේ. නිරෝධසත්‍යය නිරෝධය ද වූයේ ම නිරෝධසත්‍යය ද වේ. සත්‍යයෝ දුඃඛසත්‍ය ද යත්: ... සත්‍යයෝ සමුදයසත්‍ය ද යත්: … සත්‍යයෝ මාර්ගසත්‍ය ද යත්: මාර්ගසත්‍යය සත්‍ය ද වූයේ ම මාර්ගසත්‍ය ද වේ. අවශේෂසත්‍යයෝ සත්‍යයෝ යි, මාර්ගසත්‍යය නො වේ.

මාර්ගය මාර්ගසත්‍ය ද යත්: මාර්ගසත්‍යය තබා අවශේෂ වූ මාර්ගය මාර්ගය වෙයි, මාර්ගසත්‍යය නො වේ. මාර්ගසත්‍යය මාර්ගය ද වූයේ ම මාර්ගසත්‍යය ද වේ. සත්‍යයෝ දුඃඛසත්‍ය ද යත්: ... සත්‍යයෝ සමුදයසත්‍ය ද යත්: ... සත්‍යයෝ නිරෝධසත්‍ය ද යත්: නිරෝධසත්‍යය සත්‍ය වූයේ ම නිරෝධසත්‍යය ද වේ. අවශේෂසත්‍යයෝ සත්‍යයෝ යි, නිරෝධසත්‍යය නො වේ. (අනුලෝම යි.)

4\. දුඃඛ නො වූයෙක් දුඃඛසත්‍යය නො වේ ද යත්: කායික දූබය චෛතසික දුඃඛය තබා අවශේෂය දුඃඛය නො වෙයි. දුඃඛසත්‍යය වේ. දුඃඛය ද දුඃඛසත්‍යය ද තබා අවශේෂය දුඃඛ නො වූයේ ම දුඃඛසත්‍යය ද නො වේ. සත්‍ය නො වූවාහු සමුදයසත්‍යය නො වේ ද යත්: එසේ යි.

දුඃඛ නො වූයෙක් දුඃඛසත්‍යය නො වේ ද යත්: කායික දුඃඛය චෛතසික දුඃඛය තබා අවශේෂය දුඃඛය නො වෙයි. දුඃඛසත්‍යය වේ. දුඃඛය ද දුඃඛසත්‍ය ද තබා අවශේෂය දුඃඛ ද නො වූයේ ම දුඃඛසත්‍ය ද නො වේ. සත්‍ය නො වූවාහු නිරෝධසත්‍යය නො වේ ද යත්: ... සත්‍ය නො වූවාහු මාර්ගසත්‍යය නො වේ ද යත්: එසේ යි.

සමුදය නො වූයෙක් සමුදයසත්‍යය නො වේ ද යත්: එසේ යි. සත්‍ය නො වූවාහු දුඃඛසත්‍යය නො වේ ද යත්: එසේ යි.

සමුදය නො වූයෙක් සමුදයසත්‍යය නො වේ ද යත්: එසේ යි. සත්‍ය නො වූවාහු නිරෝධසත්‍යය නො වේ ද යත්: සත්‍ය නො වූවාහු මාර්ගසත්‍යය නො වේ ද යත්: එසේ යි.

නිරෝධ නො වූයෙක් නිරෝධසත්‍යය නො වේ ද යත්: එසේ යි. සත්‍ය නො වූවාහු දුඃඛසත්‍යය නො වේ ද යත්: ... සත්‍ය නො වූවාහු සමුදය සත්‍යය නො වේ ද යත්: ... සත්‍ය නො වූවාහු මාර්ගසත්‍යය නො වේ ද යත්: එසේ යි.

මාර්ග නො වූයෙක් මාර්ගසත්‍යය නො වේ ද යත්: එසේ යි. සත්‍ය නො වූවාහු දුඃඛසත්‍යය නො වේ ද යත්: එසේ යි. මාර්ග නො වූයෙක් මාර්ගසත්‍යය නො වේ ද යත්: එසේ යි. සත්‍ය නො වූවාහු සමුදයසත්‍යය නො වේ ද යත්: එසේ යි. (ප්‍රත්‍යනීක යි.)

පදශෝධන මූලචක්‍රවාර යි.

5\. දුඃඛය සත්‍ය ද යත්: එසේ යි. සත්‍යයෝ දුඃඛසත්‍ය ද යත්: දුඃඛසත්‍යය සත්‍ය ද වූයේ ම දුඃඛසත්‍යය ද වේ. අවශේෂ සත්‍යයෝ සත්‍යයෝ යි, දුඃඛසත්‍යය නො වේ.

සමුදය සත්‍ය ද යත්: එසේ යි. ... නිරෝධය සත්‍ය ද යත්: එසේ යි. ... මාර්ගය සත්‍ය ද යත්: එසේ යි. සත්‍යයෝ මාර්ගසත්‍ය ද යත්: මාර්ගසත්‍යය සත්‍ය වූයේ ම මාර්ගසත්‍යය ද වේ. අවශේෂ සත්‍යයෝ සත්‍යයෝ යි, මාර්ගසත්‍යය නො වේ. (අනුලෝම යි.)

6\. දුඃඛ නො වූයෙක් සත්‍ය නො වේ ද යත්: දුඃඛය තබා අවශේෂසත්‍යයෝ දුඃඛය නො වෙයි, සත්‍යයහ. දුඃඛය ද සත්‍යය ද තබා අවශේෂය දුඃඛ ද නො වූයේ ම සත්‍යයෝ ද නො වෙත්. සත්‍ය නො වූවාහු දුඃඛසත්‍යය නො වේ ද යත්: එසේ යි.

සමුදය නො වූයෙක් සත්‍ය නො වේ ද යත්: සමුදය තබා ... නිරෝධ නො වූයෙක් සත්‍ය නො වේ ද යත්: නිරෝධය තබා ... මාර්ග නො වූයෙක් සත්‍ය නො වේ ද යත්: මාර්ගය තබා අවශේෂසත්‍යයෝ මාර්ගය නො වෙයි, සත්‍යයහ. මාර්ගය ද සත්‍යය ද තබා අවශේෂය මාර්ග ද නො වූයේ ම සත්‍යයෝ ද නො වෙත්. සත්‍ය නො වූවාහු මාර්ගසත්‍යය නො වේ ද යත්: එසේ යි. (ප්‍රත්‍යනීක යි.)

ශුද්ධසත්‍යවාර යි.

7\. දුඃඛය සත්‍ය ද යත්: එසේ යි. සත්‍යයෝ සමුදයසත්‍යය ද යත්: සමුදයසත්‍යය සත්‍ය වූයේ ම සමුදයසත්‍ය ද වේ. අවශේෂසත්‍යයෝ සත්‍යයෝ යි, සමුදයසත්‍ය නො වේ.

දුඃඛය සත්‍ය ද යත්: එසේ යි. සත්‍යයෝ නිරෝධසත්‍ය ද යත්: ... සත්‍යයෝ මාර්ගසත්‍යය ද යත්: මාර්ගසත්‍යය සත්‍ය වූයේ ම මාර්ගසත්‍යය ද වේ. අවශේෂසත්‍යයෝ සත්‍යයෝ යි, මාර්ගසත්‍යය නො වේ.

සමුදය සත්‍ය ද යත්: එසේ යි. ... නිරෝධය සත්‍ය ද යත්: එසේ යි. මාර්ගය සත්‍ය ද යත්: එසේ යි. සත්‍යයෝ දුඃඛසත්‍ය ද යත් ... සත්‍යයෝ සමුදයසත්‍ය ද යත්: ... සත්‍යයෝ නිරෝධසත්‍ය ද යත්: නිරෝධසත්‍යය සත්‍ය වූයේ ම නිරෝධසත්‍ය ද වේ. අවශේෂසත්‍යයෝ සත්‍යයෝ යි, නිරෝධසත්‍යය නො වේ. (අනුලෝම යි.)

8\. දුඃඛ නො වූයෙක් සත්‍යය නො වේ ද යත්: දුඃඛය තබා අවශේෂසත්‍යයෝ දුඃඛය නො වෙයි, සත්‍යය හ. දුඃඛය ද සත්‍යය ද තබා අවශේෂය දුඃඛය ද නො වූයේ ම සත්‍යයෝ ද නො වෙත්. සත්‍ය නො වූවාහු සමුදයසත්‍යය නො වේ ද යත්: එසේ යි. දුඃඛ නො වූයෙක් සත්‍යය නො වේ ද යත්: දුඃඛය තබා අවශේෂ සත්‍යයෝ දුඃඛය නො වේ, සත්‍යය හ. දුඃඛය ද සත්‍යය ද තබා අවශේෂය දුඃඛය ද නො වූයේ ම සත්‍යයෝ ද නො වෙත්. සත්‍ය නො වූවාහු නිරෝධසත්‍ය නො වේ ද යත්: ... සත්‍ය නො වූවාහු මාර්ගසත්‍ය නො වේ ද යත්: එසේ යි.

සමුදය නො වූයෙක් සත්‍යය නො වේ ද යත්: සමුදය තබා අවශේෂසත්‍යයෝ සමුදය නො වෙයි, සත්‍යය හ. සමුදය ද සත්‍යය ද තබා අවශේෂය සමුදය ද නො වූයේ ම සත්‍යයෝ ද නො වෙත්. සත්‍ය නො වූවාහු දුඃඛසත්‍යය නො වේ ද යත්: ... නිරෝධ නො වූයෙක් සත්‍යය නො වේ ද යත්: නිරෝධය තබා ... මාර්ග නො වූයෙක් සත්‍යය නො වේ ද යත්: මාර්ගය තබා අවශේෂසත්‍යයෝ මාර්ගය නො වේ, සත්‍යය හ. මාර්ගය ද සත්‍යය ද තබා අවශේෂය මාර්ගය ද නො වූයේ ම සත්‍යයෝ ද නො වෙත්. සත්‍ය නො වූවාහු දුඃඛසත්‍යය නො වේ ද යත්: එසේ යි.

මාර්ග නො වූයෙක් සත්‍යය නො වේ ද යත්: මාර්ගය තබා අවශේෂසත්‍යයෝ මාර්ගය නො වෙයි, සත්‍යය හ. මාර්ගය ද සත්‍යය ද තඛා අවශේෂය මාර්ගය ද නො වූයේ ම සත්‍යයෝ ද නො වෙත්. සත්‍ය නො වූවාහු සමුදයසත්‍යය නො වේ ද යත්: එසේ යි. ... සත්‍ය නො වූවාහු නිරෝධසත්‍යය නො වේ ද යත්: එසේ යි. (ප්‍රත්‍යනීක යි.)

ශුද්ධසත්‍ය මූලචක්‍රවාර යි.
