# ශුන්‍යතය

344\. කවර ධර්මයෝ කුසල්හ යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස ප්‍රථමභූමියට පැමිණීම පිණිස නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමනයන්ගෙන් වෙන් ව ම ... ශුන්‍යත ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්මයෝ කුශලයෝ යි.

345\. කවර ධර්මයෝ කුසල්හ යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස ප්‍රථමභූමියට පැමිණීම පිණිස නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලෝකෝත්තර ධ්‍යානය වඩා ද, විතර්කවිචාරයන්ගේ සන්හිඳීමෙන් ... ද්වීතියධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට … ශුන්‍යතපඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්මයෝ කුශලයෝ යි.

ශුන්‍යත යි.
