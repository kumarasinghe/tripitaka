# අප්‍රණිහිත මූලකප්‍රතිපදා

353\. කවර ධර්මයෝ කුසල්හ යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස ප්‍රථමභූමියට පැමිණීම පිණිස නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලෝකෝත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අප්‍රණිහිත ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්මයෝ කුශලයෝ යි.

354\. කවර ධර්මයෝ කුසල්හ යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස ප්‍රථමභූමියට පැමිණීම පිණිස නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලෝකෝත්තර ධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති ක්ෂීප්‍රාභිඥ වූ අප්‍රණිහිත ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්මයෝ කුශලයෝ යි.

355\. කවර ධර්මයෝ කුසල්හ යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස ප්‍රථමභූමියට පැමිණීම පිණිස නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලෝකෝත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම සුඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අප්‍රණිහිත ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්මයෝ කුශලයෝ යි.

356\. කවර ධර්මයෝ කුසල්හ යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස ප්‍රථමභූමියට පැමිණීම පිණිස නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලෝකෝත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාහිඥ වූ අප්‍රණිහිත ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්මයෝ කුශලයෝ යි.

357\. කවර ධර්මයෝ කුසල්හ යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස ප්‍රථමභූමියට පැමිණීම පිණිස නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලෝකෝත්තරධ්‍යානය වඩා ද, විතර්ක විචාරයන්ගේ සන්හිඳීමෙන් ... ද්විතීයධ්‍යානයට ... තෘතියධ්‍යානයට ... චතුර්ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... දුඃඛප්‍රතිපත්ති ඇති, දන්ධාභිඥ වූ අප්‍රණිහිත පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, දුඃඛප්‍රතිපත් ඇති ක්‍ෂිප්‍රහිඥ වූ අප්‍රණිහිත වූ ... සුඛප්‍රතිපත් ඇති දන්ධාහිඥ වූ අප්‍රණීහිත වූ ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාහිඥ වූ අප්‍රණිහිත වූ (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද,) එසමයෙහි ස්පර්ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්මයෝ කුශලයෝ යි.

අප්‍රණීහිතමූලක ප්‍රතිපදා යි.
