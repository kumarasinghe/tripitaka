# විංශද්මහානය

358\. කවර ධර්මයෝ කුසල්හ යත්:

යම් කලෙකැ ලොකොත්තර වූ මාර්ගය වඩා ද ... ලොකොත්තර වූ සතිපට්ඨානය වඩා ද ලොකොත්තර වූ සම්‍යක්ප්‍රධානය වඩා ද ... ලොකොත්තර වූ ඍද්ධිපාදය වඩා ද ... ලොකොත්තර වූ ඉන්ද්‍රිය වඩා ද ... ලොකොත්තර වූ බලය වඩා ද ... ලොකොත්තර වූ බොධ්‍යඞ්ගය වඩා ද ... ලොකොත්තර වූ සත්‍යය වඩා ද ... ලොකොත්තර වූ ශමථය වඩා ද ... ලොකොත්තර වූ ධර්මය වඩා ද ... ලොකොත්තර වූ ස්කන්ධය වඩා ද ... ලොකොත්තර වූ ආයතනය වඩා ද ... ලොකොත්තර වූ ධාතු වඩා ද ... ලොකොත්තර වූ ආහාරය වඩා ද ... ලොකොත්තර වූ ස්පර්ශය වඩා ද ... ලොකොත්තර වූ වෙදනාව වඩා ද ... ලොකොත්තර වූ සංඥාව වඩා ද ... ලොකොත්තර වූ චෙතනාව වඩා ද ... දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තර චිත්තය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත්ති ඇති, දන්ධාභිඥ වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්මයෝ කුශලයෝ යි.

විංශද් මහානය යි.
