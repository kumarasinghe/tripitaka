# අධිපති ප්‍රථමමාර්ගය

359\. කවර ධර්මයෝ කුසල්හ යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණස ප්‍රථමභූමියට පැමිණීම පිණිස නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලෝකෝත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද ... වීර්ය්‍යය අධිපති කොට ඇති ... චිත්තය අධිපති කොට ඇති ... විමංසාව අධිපති කොට ඇති (ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද) එසමයෙහි ස්පර්ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්මයෝ කුශලයෝ යි.

360\. කවර ධර්මයෝ කුසල්හ යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණස ප්‍රථමභූමියට පැමිණීම පිණිස නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, විතර්කවිචාරයන්ගේ සන්හිඳීමෙන් ... ද්විතීයධ්‍යානයට ... තෘතියධ්‍යානයට ... චතුර්ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ඡන්දය අධිපති කොට ඇති පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද ... වීර්ය්‍යය අධිපති කොට ඇති ... චිත්තය අධිපති කොට ඇති ... විමංසාව අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද,) එසමයෙහි ස්පර්ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්මයෝ කුශලයෝ යි.

361\. කවර ධර්මයෝ කුසල්හ යත්:

යම් කලෙක ලෝකෝත්තර වූ මාර්ගය වඩා ද ... ලෝකෝත්තර වූ සති පට්ඨානය වඩා ද ... ලෝකෝත්තර වූ සම්‍යක්ප්‍රධානය වඩා ද ... ලෝකෝත්තර වූ ඍද්ධිපාදය වඩා ද ... ලෝකෝත්තර වූ ඉන්ද්‍රිය වඩා ද ... ලෝකෝත්තර වූ බලය වඩා ද ... ලෝකෝත්තර වූ බොධ්‍යඞ්ගය වඩා ද ... ලෝකෝත්තර වූ සත්‍යය වඩා ද ... ලෝකෝත්තර වූ ශමථය වඩා ද ... ලෝකෝත්තර වූ ධර්මය වඩා ද ... ලෝකෝත්තර වූ ස්කන්ධය වඩා ද ... ලෝකෝත්තර වූ ආයතනය වඩා ද ... ලෝකෝත්තර වූ ධතු වඩා ද ... ලෝකෝත්තර වූ ආහාරය වඩා ද ... ලෝකෝත්තර වූ ස්පර්ශය වඩා ද ... ලෝකෝත්තර වූ වෙදනාව වඩා ද ... ලෝකෝත්තර වූ සංඥාව වඩා ද ... ලෝකෝත්තර වූ චෙතනාව වඩා ද ... දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස ප්‍රථමභූමියට පැමිණීම පිණිස නෛර්‍ය්‍යාණික වූ අපවයගාමී වු ලොකොත්තර චිත්තය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද ... වීර්ය්‍යය අධිපති කොට ඇත ... චිත්තය අධිපති කොට ඇත ... විමංසාව අධිපති කොට ඇති (ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද) එසමයෙහි ස්පර්ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්මයෝ කුශලයෝ යි.

අධිපති ප්‍රථමමාර්ගය යි.
