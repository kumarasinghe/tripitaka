# අප්පණිහිතය

351\. කවර ධර්මයෝ කුසල්හ යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස ප්‍රථමභූමියට පැමිණීම පිණිස නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලෝකෝත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... අප්‍රණිහිත ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්මයෝ කුශලයෝ යි.

352\. කවර ධර්මයෝ කුසල්හ යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස ප්‍රථමභූමියට පැමිණීම පිණිස නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලෝකෝත්තරධ්‍යානය වඩා ද, විතර්ක විචාරයන්ගේ සන්හිඳීමෙන් ... ද්විතීයධ්‍යානයට ... තෘතියධ්‍යානයට ... චතුර්ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට … අප්‍රණිහිත පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, පැමිණ වෙසේ ද, එසමයෙහි ස්පර්ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්මයෝ කුශලයෝ යි.

අප්‍රණිහිතය යි.
