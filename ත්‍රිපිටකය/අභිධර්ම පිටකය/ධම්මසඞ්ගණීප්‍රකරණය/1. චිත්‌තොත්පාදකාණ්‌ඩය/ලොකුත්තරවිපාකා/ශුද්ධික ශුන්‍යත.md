# ශුද්ධික ශුන්‍යත

511\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථම භූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන්වෙන් ව ම ... ශුන්‍යතප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තර කුශලධ්‍යානය කළ බැවින් වැඩු බැවින් කාමයන්ගෙන් වෙන් ව ම ... ශුන්‍යත වූ විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

512\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... ශුන්‍යත ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ බැවින් වැඩු බැවින් කාමයන්ගෙන් වෙන් ව ම ... අනිමිත්ත වූ විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

513\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන්වෙන් ව ම ... ශුන්‍යතප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ මැ ලොකොත්තර කුශලධ්‍යානය කළ හෙයින් වැඩු හෙයින් කාමයන්ගෙන් වෙන් ව ම ... අප්‍රණිහිත වූ විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ. එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

514\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... ද්විතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... ශුන්‍යත වූ පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, මේ කුසලි ... ශුන්‍යත වූ (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාක යි ... ශුන්‍යත වූ (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ කුසලි ... අනිමිත්ත වූ (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාක යි, ශුන්‍යත වූ (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ කුසලි ... අප්‍රණිහිත වූ (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාක යි. එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි

ශුද්ධිකශුන්‍යත යි.
