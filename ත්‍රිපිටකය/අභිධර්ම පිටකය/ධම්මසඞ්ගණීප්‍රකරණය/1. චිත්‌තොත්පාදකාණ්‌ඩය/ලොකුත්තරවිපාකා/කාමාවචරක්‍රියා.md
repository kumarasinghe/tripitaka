# කාමාවචරක්‍රියා

577\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ රූපාලම්බන වේවයි ... යළි යමක් යමක් හෝ අරමුණු කොට ක්‍රියාමාත්‍ර වූ නො ම කුසල් වූ අකුසල් නො වූ නො ද කර්‍මවිපාක වූ සොම්නස් සහගත වූ ඥානසම්ප්‍රයුක්ත වූ මනොවිඥානධාතුව උපදනේ වේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ...අවික්ෂෙපය වෙයි ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි. ... අලොභ සඞ්ඛ්‍යාත අව්‍යාකෘතමූල යැ ... අදොස සඞ්ඛ්‍යාත අව්‍යාකෘතමූල යැ ... අමොහ සඞ්ඛ්‍යාත අව්‍යාකෘතමූල යැ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

578\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ රූපාලම්බන වේවයි ... ක්‍රියාමාත්‍ර වූ නො ම කුසල් වූ අකුසල් නො වූ නො ද කර්‍මවිපාක වූ සොම්නස් සහගත ඥානසම්ප්‍රයුක්ත වූ මනොවිඥානධාතුව සංස්කාර සහිත වැ ... සොම්නස් සහගත ඥානවිප්‍රයුක්ත ... සොම්නස් සහගත ඥානවිප්‍රයුක්ත (මනොවිඥාන ධාතුව) සංස්කාර සහිත වැ ... උපෙක්ෂා සහගත ඥානසම්ප්‍රයුක්ත ... උපෙක්ෂා සහගත ඥානසම්ප්‍රයුක්ත (මනොවිඥාන ධාතුව) සංස්කාර සහිත ව ... උපෙක්ෂා සහගත ඥානවිප්‍රයුක්ත ... රූපාලම්බන වේවයි ... ධර්‍මාලම්බන වේවයි යළි යම් යම් අරමුණක් හෝ අරමුණු කොට උපෙක්ෂා සහගත ඥානවිප්‍රයුක්ත (මනොවිඥානධාතුව) උපදනේ වේ ද, එසමයෙහි ස්පර්‍ශය වෙයි … අවික්ෂෙපය වෙයි ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි. ... අලොභ සඞ්ඛ්‍යාත අව්‍යාකෘතමූල ය ... අදොස සඞ්ඛ්‍යාත අව්‍යාකෘතමූල ය ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

කාමාවචරක්‍රියා යි.
