# ශුන්‍යතප්‍රතිපදා

515\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ බැවින් වැඩු බැවින් කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

516\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ බැවින් වැඩු බැවින් කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අනිමිත්ත වූ විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ. එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

517\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ බැවින් වැඩු බැවින් කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අප්‍රණිහිත වූ විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

518\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, විතර්‍ක විචාරයන්ගේ සන්හිඳීමෙන් ... ද්විතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, මේ කුසලි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාකයි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ කුසලි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අනිමිත්ත වූ (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාකයි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ කුසලි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අප්‍රණිහිත වූ (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාක යි. එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

519\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:
