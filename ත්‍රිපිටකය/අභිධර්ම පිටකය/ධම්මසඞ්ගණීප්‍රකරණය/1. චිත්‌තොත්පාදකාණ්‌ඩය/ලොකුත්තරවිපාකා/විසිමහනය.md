# විසිමහනය

529\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ ලොකොත්තර වූ මාර්‍ගය වඩා ද ... ලොකොත්තර වූ සතිපට්ඨානය වඩා ද ... ලොකොත්තර වූ සම්‍යක්ප්‍රධානය වඩා ද ... ලොකොත්තර වූ ඍද්ධිපාදය වඩා ද ... ලොකොත්තර වූ ඉන්ද්‍රියය වඩා ද ... ලොකොත්තර වූ බලය වඩා ද ... ලොකොත්තර වූ බොධ්‍යඞ්ග වඩා ද ... ලොකොත්තර වූ සත්‍යය වඩා ද ලොකොත්තර වූ ශමථය වඩා ද ...... ලොකොත්තර වූ ධර්‍මය වඩා ද ... ලොකොත්තර වූ ස්කන්ධය වඩා ද ... ලොකොත්තර වූ ආයතන වඩා ද ... ලොකොත්තර වූ ධාතුව වඩා ද ... ලොකොත්තර වූ ආහාරය වඩා ද ... ලොකොත්තර වූ ස්පර්‍ශය වඩා ද ... ලොකොත්තර වූ වේදනාව වඩා ද ... ලොකොත්තර වූ සංඥාව වඩා ද ... ලොකොත්තර වූ චේතනාව වඩා ද ... දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තර චිත්තය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ බැවින් වැඩු බැවින් කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද ... අනිමිත්ත වූ .... අප්‍රණිහිත වූ (ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද) එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ... අව්‍යාකෘතයෝ යි.

විසිමහතය යි.
