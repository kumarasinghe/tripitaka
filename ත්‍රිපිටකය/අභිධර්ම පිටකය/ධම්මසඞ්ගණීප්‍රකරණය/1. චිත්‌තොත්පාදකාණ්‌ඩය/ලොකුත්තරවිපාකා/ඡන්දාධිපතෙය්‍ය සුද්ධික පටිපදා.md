# ඡන්දාධිපතෙය්‍ය සුද්ධික පටිපදා

530\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තර කුශලධ්‍යානය කළ බැවින් වැඩු බැවින් කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ... අව්‍යාකෘතයෝ යි.

531\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තර කුශලධ්‍යානය කළ හෙයින් වැඩු හෙයින් කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අනිමිත්ත වූ ඡන්දය අධිපති කොට ඇති විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ... අව්‍යාකෘතයෝ යි.

532\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථම භූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තර ධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ හෙයින් වැඩු හෙයින් කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ... අව්‍යාකෘතයෝ යි.

533\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, විතර්‍ක විචාරයන්ගේ සන්හිඳීමෙන් ... ද්විතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ඡන්දය අධිපති කොට ඇති පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, මේ කුසලි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාක යි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ කුසලි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අනිමිත්ත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාක යි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ කුසලි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාක යි. එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

534\. කවර ධර්‍මයෝ අව්‍යාකෘතහ යත්:
