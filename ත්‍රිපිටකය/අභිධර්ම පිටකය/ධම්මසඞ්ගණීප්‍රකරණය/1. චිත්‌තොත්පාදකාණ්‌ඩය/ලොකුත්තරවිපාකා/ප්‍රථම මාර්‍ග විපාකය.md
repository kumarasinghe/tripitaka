# ප්‍රථම මාර්‍ග විපාකය

539\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථම භූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ හෙයින් වැඩු හෙයින් කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

540\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ හෙයින් වැඩු හෙයින් කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති අනිමිත්ත වූ ඡන්දය අධිපති කොට ඇති විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

541\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථම භූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ හෙයින් වැඩු හෙයින් කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ... අව්‍යාකෘතයෝ යි.

542\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන්... ද්විතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, මේ කුසලි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ විපාක යි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ කුසලි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අනිමිත්ත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ විපාක යි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ කුසලි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ විපාක යි. එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

543\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති ක්ෂිප්‍රාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති ... සුඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද ... ද්විතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ කුසලි ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ විපාක යි ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ කුසලි ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාභිඥ වූ අනිමිත්ත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ විපාක යි ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ කුසලි ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාභිඥ වූ අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ විපාක යි. එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

544\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශල ධ්‍යානය කළ බැවින් වැඩු බැවින් කාමයන්ගෙන් වෙන් ව ම ... අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

545\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ බැවින් වැඩු බැවින් කාමයන්ගෙන් වෙන් ව ම ... අනිමිත්ත වූ ඡන්දය අධිපති කොට ඇති විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

546\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ බැවින් වැඩු බැවින් කාමයන්ගෙන් වෙන් ව ම ... ශුන්‍යත වූ ඡන්දය අධිපති කොට ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

547\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... ද්විතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, මේ කුසලි ... අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ විපාක යි ... අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ කුසලි ... අනිමිත්ත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ විපාක යි ... අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ කුසලි ... ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), මේ විපාක යි. එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

548\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ බැවින් වැඩු බැවින් කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

549\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ බැවින් වැඩු බැවින් කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අනිමිත්ත වූ ඡන්දය අධිපති කොට ඇති විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

550\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අප්‍රණිහිත වූ ඡන්දය අධිපති කොට ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ හෙයින් වැඩු හෙයින් කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ඡන්දය අධිපති කොට ඇති විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ... අව්‍යාකෘතයෝ යි.

551\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

(ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද,) එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.
