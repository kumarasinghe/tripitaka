# සුද්ධික පටිපදා

506\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ස්‍රොතාපත්තිඵල සඞ්ඛ්‍යාත ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තර කුශලධ්‍යානය කළ බැවින් වැඩු බැවින් විපාක වූ, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අඤ්ඤින්ද්‍රිය වෙයි ... අවික්ෂෙපය වේ. මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

507\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තර කුශලධ්‍යානය කළ බැවින් වැඩු බැවින් විපාක වූ, කාමයන් ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අනිමිත්ත වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අඤ්ඤින්ද්‍රිය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

508\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එ ම ලොකොත්තරකුශලධ්‍යානය කළ බැවින් වැඩු බැවින් කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අප්‍රණිහිත වූ විපාක වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අඤ්ඤින්ද්‍රිය වෙයි … අවික්ෂෙපය වේ. මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

509\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස, ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ද්විතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, මේ කුශල යි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ශුන්‍යත වූ (ධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාක යි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ (ධ්‍යානයට පැමිණ වෙසේ ද) මේ කුශල යි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අනිමිත්ත වූ (ධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාක යි ... යි දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ (ධ්‍යානයට පැමිණ වෙසේ ද) මේ කුශල යි ... දුඃඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ අප්‍රණිහිත වූ (ධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාක යි ... එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ. මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

510\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ දෘෂ්ටිගතයන්ගේ ප්‍රහාණය පිණිස ප්‍රථමභූමියට පැමිණීම පිණිස, නෛර්‍ය්‍යාණික වූ අපචයගාමී වූ ලොකොත්තරධ්‍යානය වඩා ද, කාමයන් ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති ක්ෂිප්‍රාහිඥ වූ ... සුඛප්‍රතිපත් ඇති දන්ධාභිඥ වූ ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාහිඥ වූ ප්‍රථමධ්‍යානයට ... ද්විතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට පැමිණ වෙසේ ද ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාහිඥ වූ ප්‍රථමධ්‍යානයට ... පැමිණ වෙසේ ද, මේ කුශල යි ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාහිඥ වූ ශුන්‍යත වූ (ධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාක යි ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාහිඥ වූ (ධ්‍යානයට පැමිණ වෙසේ ද) මේ කුශල යි ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාහිඥ වූ අනිමිත්ත වූ (ධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාක යි ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාහිඥ වූ (ධ්‍යානයට පැමිණ වෙසේ ද) මේ කුශල යි ... සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාහිඥ වූ අප්‍රණිහිත වූ (ධ්‍යානයට පැමිණ වෙසේ ද) මේ විපාක යි, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.
