# දෙඅරමුණු

211\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ කසිණරූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි කාමයන්ගෙන් වෙන් ව ම ... පරිත්ත වූ පරිත්තාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

212\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ කසිණරූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි කාමයන්ගෙන් වෙන් ව ම ... අප්‍රමාණ වූ පරිත්තාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

213\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ කසිණරූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... ද්වීතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... පරිත්ත වූ පරිත්තාලම්බන වූ පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, ... අප්‍රමාණ වූ පරිත්තාලම්බන වූ (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

දෙ අරමුණු යි.
