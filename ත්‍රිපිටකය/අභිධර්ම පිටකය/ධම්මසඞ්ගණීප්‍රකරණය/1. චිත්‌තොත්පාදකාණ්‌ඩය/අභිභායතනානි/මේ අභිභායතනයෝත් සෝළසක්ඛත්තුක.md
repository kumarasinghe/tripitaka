# මේ අභිභායතනයෝත් සෝළසක්ඛත්තුක

246\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ නිල් වූ නිල් වන් වූ හුදුනිල් වූ නිල්පැහැ වූ බැහැරි රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දන්මී දක්මී කාමයන්ගෙන් වෙන් ව ම ... ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

247\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පීත වූ පීතවර්‍ණ වූ පීතනිදර්‍ශන වූ පීතප්‍රභ වූ බාහ්‍යරූපයන් දකී ද... ලෝහිතක වූ ලෝහිතකවර්‍ණ වූ ලෝහිතකනිදර්‍ශන වූ ලෝහිතකප්‍රභ වූ ... අවදාත වූ අවදාතවර්‍ණ වූ අවදාතනිදර්‍ශන වූ අවදාතප්‍රභ වූ (බාහ්‍ය) රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දන්මී දක්මී කාමයන්ගෙන් වෙන් ව ම ... ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එකල්හි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

මේ අභිභායතනයෝත් සෝළසක්ඛත්තුකයෝ යි.
