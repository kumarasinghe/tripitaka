# බ්‍රහ්මවිහාරධ්‍යාන සෝළසක්ඛත්තුකය

251\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... මෛත්‍රීසහගත ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

252\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... මෛත්‍රීසහගත ද්වීතීයධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

253\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, ප්‍රීතියගේ ද විරාගයෙන් ... මෛත්‍රීසහගත තෘතීයධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

254\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... මෛත්‍රීසහගත ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

255\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අවිතර්‍ක වූ විචාරමාත්‍ර වූ සමාධියෙන් උපන් ප්‍රීතිය හා සුඛය ඇති මෛත්‍රීසහගත ද්වීතීයධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

256\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... මෛත්‍රීසහගත වූ තෘතීයධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

257\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, ප්‍රීතියගේ ද විරාගයෙන් ... මෛත්‍රීසහගත චතුර්‍ථධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

258\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... කරුණාසහගත ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

259\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... ද්වීතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... කරුණාසහගත චතුර්‍ථධ්‍යානයට ... පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි, ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

260\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... මුදිතා සහගත ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

261\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... ද්වීතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... මුදිතාසහගත චතුර්‍ථධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

262\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, සුඛයාගේ ද ප්‍රහාණයෙන් ... උපේක්‍ෂාසහගත චතුර්‍ථධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

සතර බ්‍රහ්මවිහාරධ්‍යානික සෝළසක්ඛත්තුකය යි.
