# විමෝක්‍ෂ සෝළසක්ඛත්තුකය

248\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපධ්‍යාන ඇතියේ (බාහ්‍ය වූ ද) රූපයන් දකී ද, කාමයන්ගෙන් වෙන් ව ම ... ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

249\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්‍මයෙහි රූපසංඥා රහිත වූයේ බාහ්‍ය රූපයන් දකී ද, කාමයන්ගෙන් වෙන් ව ම ... ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

250\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, සුභ ය යි කසිණ අරමුණෙහි නිසල වූයේ වේ ද, කාමයන්ගෙන් වෙන් ව ම ... ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

මේ තුන් විමෝක්‍ෂයෝත් සෝළසක්ඛත්තුකයෝ යි.
