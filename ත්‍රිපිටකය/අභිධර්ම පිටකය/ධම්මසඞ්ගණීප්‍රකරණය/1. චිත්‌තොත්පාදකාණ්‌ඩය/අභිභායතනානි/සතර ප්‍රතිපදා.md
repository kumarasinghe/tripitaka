# සතර ප්‍රතිපදා

227\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී කාමයන්ගෙන් වෙන් ව ම ... දුඃඛ ප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

228\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්ම රූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී කාමයන්ගෙන් වෙන් ව ම ... දුඃඛ ප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

229\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්ම රූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී කාමයන්ගෙන් වෙන් ව ම ... සුඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

230\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්ම රූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී කාමයන්ගෙන් වෙන් ව ම ... සුඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

231\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්ම රූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... ද්වීතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... දුඃඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ පඤ්චමධ්‍යානයට පැමිණ වෙසේද, ... දුඃඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ ... දුඃඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ ... සුඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ ... සුඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ පඤ්චමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

සතර ප්‍රතිපදා යි.
