# අභිභායතන

204\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්ම රූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ නො වැඩූ බාහිර වූ අටවැදෑරුම් කසිණ රූපයන් දකී ද, ඒ ‍රූපයන් මැඩැගෙන දනිමී, දක්මී යි (මෙනෙහි කොට) කාමයන්ගෙන් වෙන් ව ම ... ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, ... අවික්‍ෂේපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

205\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්ම රූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ කසිණ රූපයන් දකී ද, ඒ ‍රූපයන් මැඩැගෙන දනිමී, දක්මී යි මෙනෙහි කොට විතර්‍ක විචාරයන්ගේ සන්හිඳීමෙන් ... ද්වීතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, ... අවික්‍ෂේපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.
