# අශුභධ්‍යාන සෝළසක්ඛත්තුකය

263\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... උද්ධුමාතකසංඥාසහගත වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

264\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... විනීලකසංඥාසහගත වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද ... විපුබ්බකසංඥාසහගත වූ ... විච්‍ජිද්‍දකසංඥාසහගත වූ ... වික්‍ඛායිතකසංඥාසහගත වූ ... වික්‍ඛිත්‍තකසංඥාසහගත වූ ... හතවික්‍ඛිත්‍තකසංඥාසහගත වූ ... ලෝහිතකසංඥාසහගත වූ ... පුළවකසංඥාසහගත වූ ... අට්‍ඨිකසංඥා සහගත වූ (ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද) එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

අශුභධ්‍යාන සෝළසක්ඛත්තුකය යි.
