# අනෙකුදු අට්ඨක්ඛත්තුක

235\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දන්මී දක්මී කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ පරිත්ත වූ අප්‍රමාණාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

236\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දන්මී දක්මී කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

237\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දන්මී දක්මී කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ පරිත්ත වූ අප්‍රමාණාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

238\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දන්මී දක්මී කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

239\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දන්මී දක්මී කාමයන්ගෙන් වෙන් ව ම ... සුඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ පරිත්ත වූ අප්‍රමාණාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

240\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දන්මී දක්මී කාමයන්ගෙන් වෙන් ව ම ... සුඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

241\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දන්මී දක්මී කාමයන්ගෙන් වෙන් ව ම ... සුඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ පරිත්ත වූ අප්‍රමාණාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

242\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දන්මී දක්මී කාමයන්ගෙන් වෙන් ව ම ... සුඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

243\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්ම රූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... ද්වීතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... දුඃඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ පරිත්ත වූ අප්‍රමාණාලම්බන වූ පඤ්චමධ්‍යානයට පැමිණ වෙසේද, ... දුඃඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ ... දුඃඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ පරිත්ත වූ අප්‍රමාණාලම්බන වූ ... දුඃඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ ... ‍‍‍සුඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ පරිත්ත වූ අප්‍රමාණාලම්බන වූ ... සුඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ ... සුඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ පරිත්ත වූ අප්‍රමාණාලම්බන වූ ... සුඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ පඤ්චමධ්‍යානයට පැමිණ වෙසේද, ... එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.
