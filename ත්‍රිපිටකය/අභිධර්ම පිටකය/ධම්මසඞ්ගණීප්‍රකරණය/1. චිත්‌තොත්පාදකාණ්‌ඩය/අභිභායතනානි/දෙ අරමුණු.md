# දෙ අරමුණු

232\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දන්මී දක්මී කාමයන්ගෙන් වෙන් ව ම ... පරිත්ත වූ අප්‍රමාණාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි, ... අවික්‍ෂේපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

233\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍ය රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දන්මී දක්මී කාමයන්ගෙන් වෙන් ව ම ... අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි, ... අවික්‍ෂේපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

234\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්ම රූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ අප්‍රමාණ වූ බාහ්‍යරූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... ද්වීතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... පරිත්ත වූ අප්‍රමාණාලම්බන වූ පඤ්චමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

දෙ අරමුණු යි.
