# අභිභායතන - සතර ප්‍රතිපදා

206\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්ම රූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ කසිණරූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි (මෙනෙහි කොට) කාමයන්ගෙන් වෙන් ව ම ... දුක් වූ පටිපදා ඇති දන්‍ධ වූ අභිඥා ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ, ... මේ ධර්‍මයෝ කුශලයෝ යි.

207\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්ම රූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ කසිණරූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි කාමයන්ගෙන් වෙන් ව ම ... දුක් වූ පටිපදා ඇති ක්‍ෂිප්‍රාභිඥ වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

208\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ කසිණරූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි කාමයන්ගෙන් වෙන් ව ම ... සුඛ වූ ප්‍රතිපත් ඇති දන්‍ධ වූ අභිඥා ඇති ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

209\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ කසිණරූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි කාමයන්ගෙන් වෙන් ව ම ... සුඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

210\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ කසිණරූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... ද්වීතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... දුක් වූ පටිපදා ඇති දන්‍ධ වූ අභිඥා ඇති පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, ... දුක් වූ පටිපදා ඇති ක්‍ෂීප්‍රාභිඥ වූ ... සුඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ ... සුඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

සතර ප්‍රතිපදා යි.
