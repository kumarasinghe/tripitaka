# අට්ඨක්ඛත්තුකය

214\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ කසිණරූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ පරිත්ත වූ පරිත්තාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

215\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ කසිණරූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ අප්‍රමාණ වූ පරිත්තාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

216\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ කසිණරූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි කාමයන්ගෙන් වෙන් ව ම ... දුඃඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ පරිත්ත වූ පරිත්තාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

217\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ කසිණරූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි කාමයන්ගෙන් වෙන් ව ම දුඃඛ වූ ප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ අප්‍රමාණ වූ පරිත්තාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

218\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි කාමයන්ගෙන් වෙන් ව ම ... සුඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ පරිත්ත වූ පරිත්තාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

219\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි කාමයන්ගෙන් වෙන් ව ම ... සුඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ අප්‍රමාණ වූ පරිත්තාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

220\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි කාමයන්ගෙන් වෙන් ව ම ... සුඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ පරිත්ත වූ පරිත්තාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

221\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි කාමයන්ගෙන් වෙන් ව ම ... සුඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ අප්‍රමාණ වූ පරිත්තාලම්බන වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

222\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, අධ්‍යාත්මරූපයෙහි පරිකර්‍මසංඥා රහිත වූයේ පරිත්ත වූ බාහ්‍ය වූ රූපයන් දකී ද, ඒ රූපයන් මැඩැගෙන දනිමී දක්මී යි විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... දෙවැනි ධ්‍යානයට ... තෙවැනි ධ්‍යානයට ... සතර වැනි ධ්‍යානයට ... පළමු වැනි ධ්‍යානයට ... දුඃඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ පරිත්ත වූ පරිත්තාලම්බන වූ පස්වැනි ධ්‍යානයට පැමිණ වෙසේද, ... දුඃඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ අප්‍රමාණ වූ පරිත්තාලම්බන වූ ... දුඃඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ පරිත්ත වූ පරිත්තාලම්බන වූ ... දුඃඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ අප්‍රමාණ වූ පරිත්තාලම්බන වූ ...සුඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ පරිත්ත වූ පරිත්තාලම්බන වූ ... සුඛප්‍රතිපත් ඇති දන්‍ධාභිඥ වූ අප්‍රමාණ වූ පරිත්තාලම්බන වූ ... සුඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ පරිත්ත වූ පරිත්තාලම්බන වූ ... සුඛප්‍රතිපත් ඇති ක්‍ෂීප්‍රාභිඥ වූ අප්‍රමාණ වූ පරිත්තාලම්බන වූ පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.
