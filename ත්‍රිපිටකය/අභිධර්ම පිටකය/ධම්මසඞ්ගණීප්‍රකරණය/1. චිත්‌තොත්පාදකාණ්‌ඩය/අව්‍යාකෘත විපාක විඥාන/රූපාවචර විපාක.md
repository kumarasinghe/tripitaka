# රූපාවචර විපාක

500\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ රූපභවොත්පත්තිය පිණිස මාර්‍ගය වඩා ද කාමයන්ගෙන් වෙන් ව ම ... පඨවිකසිණ සඞ්ඛ්‍යාත වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ. මේ ධර්‍මයෝ කුශලයෝ යි. එ ම රූපාවචර කුශලකර්‍මය කළ බැවින් රැස්කළ බැවින් කාමයන්ගෙන් වෙන් ව ම ... විපාක වූ පඨවිකසිණ සඞ්ඛ්‍යාත ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

501\. කවර ධර්‍මයෝ අව්‍යාකෘත වෙති යත්:

යම් කලෙකැ රූපභවොත්පත්තිය පිණිස මාර්‍ගය වඩා ද, විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් පඨවිකසිණ සඞ්ඛ්‍යාත ද්විතීයධ්‍යානයට ... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි. එම රූපාවචර කුශලකර්‍මය කළ බැවින් රැස් කළ බැවින් සුඛයාගේ ද ප්‍රහාණයෙන් ... විපාක වූ පඨවිකසිණ සඞ්ඛ්‍යාත පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ අව්‍යාකෘතයෝ යි.

රූපාවචර විපාකයෝ යි.
