# අට්‌ඨකසිණ සොළසක්‌ඛත්‌තුක වාරය

203\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... ආපොකසිණ සඞ්ඛ්‍යාත ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, ... තේජෝකසිණ … වායෝකසිණ ... නීලකසිණ … පීතකසිණ … ලොහිත කසිණ … ඔදාතකසිණ … එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වේ, ... මේ ධර්‍මයෝ කුශලයෝ යි.

අට්‌ඨකසිණ සොළසක්‌ඛත්‌තුක වාරය යි.
