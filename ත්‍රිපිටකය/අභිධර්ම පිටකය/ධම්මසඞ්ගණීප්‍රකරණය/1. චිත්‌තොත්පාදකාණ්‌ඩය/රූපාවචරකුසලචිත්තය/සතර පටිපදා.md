# සතර පටිපදා

176\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවොත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... දුක් වූ පිළිවෙත් ඇති ලැසි වූ අභිඥා ඇති, පඨවීකසිණ සඞ්ඛ්‍යාත වූ ප්‍රථමධ්‍යානය උපදවා වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වෙයි, ... මේ ධර්‍මයෝ කුශලයෝ යි.

177\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවොත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... දුක් වූ පිළිවෙත් ඇති නො ලස් වූ අභිඥා ඇති පඨවීකසිණ සඞ්ඛ්‍යාත වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වෙයි ... මේ ධර්‍මයෝ කුශලයෝ යි.

178\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... සුව වූ පිළිවෙත් ඇති ලැසි වූ අභිඥා ඇති, පඨවීකසිණ සඞ්ඛ්‍යාත වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වෙයි, ... මේ ධර්‍මයෝ කුශලයෝ යි.

179\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... සුව වූ පිළිවෙත් ඇති නො ලස් වූ අභිඥා ඇති, පඨවීකසිණ සඞ්ඛ්‍යාත වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වෙයි, ... මේ ධර්‍මයෝ කුශලයෝ යි.

180\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... ද්විතීයධ්‍යානයට .... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... දුක් වූ පිළිවෙත් ඇති ලැසි වූ අභිඥා ඇති පඨවීකසිණ සඞ්ඛ්‍යාත වූ පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද ... දුක් වූ පිළිවෙත් ඇති නො ලස් වූ අභිඥා ඇති, පඨවීකසිණ සඞ්ඛ්‍යාත ... සුව වූ පිළිවෙත් ඇති ලැසි වූ අභිඥා ඇති පඨවීකසිණ … සුව වූ පිළිවෙත් ඇති නො ලස් වූ අභිඥා ඇති පඨවීකසිණ (සඞ්ඛ්‍යාත පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද,) එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

සතර පටිපදා යි.
