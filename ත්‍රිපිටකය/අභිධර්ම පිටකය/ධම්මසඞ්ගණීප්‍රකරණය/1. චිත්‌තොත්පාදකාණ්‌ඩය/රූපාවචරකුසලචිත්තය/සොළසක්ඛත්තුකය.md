# සොළසක්ඛත්තුකය

186\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවොත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... දුක් වූ පිළිවෙත් ඇති ලස් වූ අභිඥා ඇති පරිත්ත වූ පරිත්තාලම්බන වූ පඨවීකසිණ සඞ්ඛ්‍යාත වූ ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

187\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... දුක් වූ පිළිවෙත් ඇති ලස් වූ අභිඥා ඇති, පරිත්ත වූ අප්‍රමාණාලම්බන වූ පඨවීකසිණ සඞ්ඛ්‍යාත ප්‍රථමධ්‍යානයට පැමිණ වෙසේද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වෙයි, ... මේ ධර්‍මයෝ කුශලයෝ යි.

188\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවොත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... දුක් වූ පිළිවෙත් ඇති දන්ඩාභිඥ වූ අප්‍රමණ වූ පරිත්තාලම්බන වූ පඨවීකසිණ - ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි … අවික්ෂෙපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

189\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... දුක් වූ පිළිවෙත් ඇති දන්ඩාභිඥ වූ අප්‍රමණ වූ අප්‍රමාණාලම්බන වූ පඨවීකසිණ - ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

190\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... දුක් වූ පිළිවෙත් ඇති ක්ෂිප්‍රාභිඥ වූ පරිත්ත වූ පරිත්තාලම්බන වූ පඨවීකසිණ - ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි … අවික්ෂෙපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

191\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... දුක් වූ පිළිවෙත් ඇති ක්ෂිප්‍රාභිඥ වූ පරිත්ත වූ අප්‍රමාණාලම්බන වූ පඨවීකසිණ - ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි … අවික්ෂෙපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

192\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... දුක් වූ ප්‍රතිපදා ඇති ක්ෂිප්‍රාභිඥ වූ අප්‍රමාණ වූ පරිත්තාලම්බන වූ පඨවීකසිණ - ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි … අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

193\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... දුක් වූ ප්‍රතිපත් ඇති ක්ෂිප්‍ර වූ අභිඥා ඇති අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ පඨවීකසිණ - ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි … අවික්ෂෙපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

194\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... සුඛ වූ ප්‍රතිපත් ඇති දන්ඬ වූ අභිඥා ඇති පරිත්ත වූ පරිත්තාලම්බන වූ පඨවීකසිණ - ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වේ, ... මේ ධර්‍මයෝ කුශලයෝ යි.

195\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... සුඛ වූ ප්‍රතිපත් ඇති දන්ඬ වූ අභිඥා ඇති පරිත්ත වූ අප්‍රමාණාලම්බන වූ පඨවීකසිණ - ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වේ, ... මේ ධර්‍මයෝ කුශලයෝ යි.

196\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... සුඛ වූ ප්‍රතිපත් ඇති දන්ඬ වූ අභිඥා ඇති අප්‍රමාණ වූ පරිත්තාලම්බන වූ පඨවීකසිණ - ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වේ, ... මේ ධර්‍මයෝ කුශලයෝ යි.

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් ව ම ... සුඛ වූ ප්‍රතිපත් ඇති ක්ෂිප්‍ර වූ අභිඥා ඇති, පරිත්ත වූ අප්‍රමාණාලම්බන වූ පඨවීකසිණ - ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වේ, ... මේ ධර්‍මයෝ කුශලයෝ යි.

201\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... සුඛ වූ ප්‍රතිපත් ඇති ක්ෂිප්‍ර වූ අභිඥා ඇති, අප්‍රමාණ වූ අප්‍රමණාලම්බන ඇති පඨවීකසිණ - ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

202\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... දෙවැනි දැහැන ... තෙවැනි දැහැන ... සතර වැනි දැහැන ... පළමුවැනි දැහැන ... දුක් වූ පිළිවෙත් ඇති, ලස් වූ අභිඥා ඇති, අල්ප වූ අල්පාලම්බන වූ පඨවීකසිණ - පස්වැනි දැහැන … දුක් වු පිළිවෙත් ඇති දන්ඬ වූ අභිඥා ඇති, පරිත්ත වූ අප්‍රමාණාලම්බන වූ පඨවිකසිණ … දුක් වූ පිළිවෙත් ඇති දන්ඬ වූ අභිඥා ඇති, අප්‍රමාණ වූ පරිත්තාලම්බන වූ පඨවීකසිණ … දුක් වූ පිළිවෙත් ඇති දන්ඬ වූ අභිඥා ඇති, අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ පඨවීකසිණ … දුක් වූ පිළිවෙත් ඇති නො ලස් වූ අභිඥා ඇති, පරිත්ත වූ පරිත්තාලම්බන වූ පඨවීකසිණ … දුක් වූ පිළිවෙත් ඇති නො ලස් වූ අභිඥා ඇති, පරිත්ත වූ අප්‍රමාණාලම්බන වූ පඨවීකසිණ … දුක් වූ පිළිවෙත් ඇති නො ලස් වූ අභිඥා ඇති, අප්‍රමාණ වූ පරිත්තාලම්බන වූ පඨවීකසිණ … දුක් වූ පිළිවෙත් ඇති නො ලස් වූ අභිඥා ඇති, අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ පඨවීකසිණ … සුඛ වූ ප්‍රතිපත් ඇති, දන්ඬ වූ අභිඥා ඇති, පරිත්ත වූ පරිත්තාලම්බන වූ පඨවීකසිණ … සුඛ වූ ප්‍රතිපත් ඇති, දන්ඬාභිඥ වූ පරිත්ත වූ අප්‍රමාණාලම්බන වූ පඨවීකසිණ … සුඛප්‍රතිපදා ඇති දන්ඬාභිඥ වූ අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ පඨවීකසිණ … සුඛප්‍රතිපදා ඇති ක්ෂිප්‍රාභිඥ වූ පරිත්ත වූ පරිත්තාලම්බන වූ පඨවීකසිණ … සුඛප්‍රතිපදා ඇති ක්ෂිප්‍රාභිඥ වූ පරිත්ත වූ අප්පමාණාලම්බන වූ පඨවීකසිණ … සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාභිඥ වූ අප්‍රමාණ වූ පරිත්තාලම්බන වූ පඨවීකසිණ … සුඛප්‍රතිපත් ඇති ක්ෂිප්‍රාභිඥ වූ අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ පඨවීකසිණ (පඤ්චමධ්‍යානය වඩා ද) එසමයෙහි ස්පර්‍ශය වෙයි, ... අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.
