# සතර ආරම්මණ

181\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන් ගෙන් වෙන් වැ මැ පරිත්ත (අප්‍රගුණ) වූ පරිත්තාලම්බන වූ පඨවිකසිණ සඞ්ඛ්‍යාත ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

182\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවොත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන් ගෙන් වෙන් වැ මැ ... පරිත්ත වූ අප්‍රමාණාලම්බන වූ පඨවිකසිණ සඞ්ඛ්‍යාත ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

183\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවොත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන් ගෙන් වෙන් වැ මැ ... අප්‍රමාණ වූ පරිත්තාලම්බන වූ පඨවිකසිණ සඞ්ඛ්‍යාත ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වේ. ... මේ ධර්‍මයෝ කුශලයෝ යි.

184\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවොත්පත්තිය පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... අප්‍රමාණ වූ අප්‍රමාණාලම්බන වූ පඨවිකසිණ සඞ්ඛ්‍යාත ප්‍රථමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

185\. කවර ධර්‍මයෝ කුසල්හ යත්:

යම් කලෙකැ රූපභවොත්පත්තිය පිණිස මාර්‍ගය වඩා ද, විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... ද්විතීයධ්‍යානයට .... තෘතීයධ්‍යානයට ... චතුර්‍ථධ්‍යානයට ... ප්‍රථමධ්‍යානයට ... පරිත්ත වූ පරිත්තාලම්බන වූ පඨවීකසිණ සඞ්ඛ්‍යාත පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, ... පරිත්ත වූ අප්‍රමාණාලම්බන වූ පඨවීකසිණ සඞ්ඛ්‍යාත ...අප්‍රමාණ වූ පරිත්තාලම්බන වූ පඨවීකසිණ සඞ්ඛ්‍යාත වූ ... අප්‍රමාණ වූ (ඵරණ වශයෙන්) අප්‍රමාණාලම්බන වූ පඨවීකසිණ සඞ්ඛ්‍යාත වූ පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි, … අවික්ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

සතර ආරම්මණයෝ යී.
