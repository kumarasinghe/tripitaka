# අරූපාවචරකුශල

273\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ අරූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, සර්‍වප්‍ර‍කාරයෙන් රූපසංඥාවන්ගේ ඉක්මයෑමෙන් ප්‍ර‍තිඝසංඥාවන්ගේ ප්‍ර‍හාණයෙන් නානාත්‍වසංඥාවන්ගේ නො මෙනෙහි කිරීමෙන් ආකාසානඤ්චායතනසංඥා සහගත වූ, සුඛයාගේ ද ප්‍ර‍හාණයෙන් ... හීන වූ චතුර්ථධ්‍යානයට පැමිණ වෙසේ ද ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... ඡන්‍දය අධිපති කොට ඇති ... වීර්‍ය්‍යය අධිපති කොට ඇති ... චිත්තය අධිපති කොට ඇති ... වීමංසාව අධිපති කොට ඇති ... ඡන්‍දය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීර්‍ය්‍යය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... චිත්තය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීමංසාව අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... (චතුර්ථධ්‍යානයට පැමිණ වෙසේ ද), එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

274\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ අරූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, සර්‍වප්‍ර‍කාරයෙන් ආකාසානඤ්චායතනය ඉක්මවා විඤ්ඤාණඤ්චායතන සංඥා සහගත වූ, සුඛයාගේ ද ප්‍ර‍හාණයෙන් ... හීන වූ චතුර්ථධ්‍යානයට පැමිණ වෙසේ ද ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... ඡන්‍දය අධිපති කොට ඇති ... වීර්‍ය්‍යය අධිපති කොට ඇති ... චිත්තය අධිපති කොට ඇති ... වීමංසාව අධිපති කොට ඇති ... ඡන්‍දය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීර්‍ය්‍යය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... චිත්තය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීමංසාව අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... (චතුර්ථධ්‍යානයට පැමිණ වෙසේ ද), එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

275\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ අරූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, සර්‍වප්‍ර‍කාරයෙන් විඤ්ඤාණඤ්චායතනය ඉක්මවා ආකිඤ්චඤ්ඤායතනසංඥාසහගත වූ, සුඛයාගේ ද ප්‍ර‍හාණයෙන් ... හීන වූ චතුර්ථධ්‍යානයට පැමිණ වෙසේ ද ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... ඡන්‍දය අධිපති කොට ඇති ... වීර්‍ය්‍යය අධිපති කොට ඇති ... චිත්තය අධිපති කොට ඇති ... වීමංසාව අධිපති කොට ඇති ... ඡන්‍දය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීර්‍ය්‍යය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... චිත්තය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීමංසාව අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ චතුර්ථධ්‍යානයට පැමිණ වෙසේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

276\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ අරූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, සර්‍වප්‍ර‍කාරයෙන් ආකිඤ්චඤ්ඤායතනය ඉක්මවා නේවසඤ්ඤානාසඤ්ඤායතනසංඥා සහගත වූ, සුඛයාගේ ප්‍ර‍හාණයෙන් ... හීන වූ චතුර්ථධ්‍යානයට පැමිණ වෙසේ ද ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... ඡන්‍දය අධිපති කොට ඇති ... වීර්‍ය්‍යය අධිපති කොට ඇති ... චිත්තය අධිපති කොට ඇති ... වීමංසාව අධිපති කොට ඇති ... ඡන්‍දය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීර්‍ය්‍යය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... චිත්තය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීමංසාව අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... (චතුර්ථධ්‍යානයට පැමිණ වෙසේ ද), එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.
