# කාමාවචරකුශල

269\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ සොමනස්සසහගත වූ ඥානසම්ප්‍ර‍යුක්ත වූ හීන වූ කාමාවචර කුශලචිත්තය උපදනේ වේ ද ... මධ්‍යම වූ .... ප්‍ර‍ණීත වූ ... ඡන්දය අධිපති කොට ඇති ... වීර්‍ය්‍යය අධිපති කොට ඇති ... චිත්තය අධිපති කොට ඇති ... වීමංසාව අධිපති කොට ඇති ... ඡන්‍දය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීර්‍ය්‍යය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... චිත්තය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීමංසාව අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ - (කාමාවචර කුශලචිත්තය උපදනේ වේ ද) එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

270\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ සෝමනස්සසහගත වූ ඥානසම්ප්‍ර‍යුක්ත වූ කාමාවචර කුශලචිත්තය සංස්කාර සහිත වැ උපදනේ වේ ද ... සෝමනස්සසහගත වූ ඥනවිප්‍ර‍යුක්ත වූ ... සෝමනස්සසහගත වූ ඥානවිප්‍ර‍යුක්ත වූ, සංස්කාර සහිත වැ ... උපේක්‍ෂාසහගත වූ ඥානසම්ප්‍ර‍යුක්ත වූ ... උපේක්‍ෂාසහගත වූ ඥානසම්ප්‍ර‍යුක්ත වූ, සංස්කාර සහිත වැ .. උපේක්‍ෂාසහගත ඥානවිප්‍ර‍යුක්ත ... උපේක්‍ෂාසහගත ඥානවිප්‍ර‍යුක්ත, සංස්කාර සහිත වැ හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... ඡන්‍දය අධිපති කොට ඇති ... වීර්‍ය්‍ය අධිපති කොට ඇති ... චිත්තය අධිපති කොට ඇති ... ඡන්‍දය අධිපති කොට ඇති හීන වු ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීර්‍ය්‍ය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... (කාමාවචර කුශලචිත්තයය) උපදනේ වේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

කාමාවචරකුශල යි.
