# රූපාවචරකුශල

271\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්ති පිණිස මාර්‍ගය වඩා ද, කාමයන්ගෙන් වෙන් වැ මැ ... හීන වූ පඨවිකසිණ - ප්‍ර‍ථමධ්‍යානයට පැමිණ වෙසේ ද ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... ඡන්‍දය අධිපති කොට ඇති ... වීර්‍ය්‍යය අධිපති කොට ඇති ... චිත්තය අධිපති කොට ඇති ... විමංසාව අධිපති කොට ඇති ... ඡන්‍දය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීර්‍ය්‍යය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... චිත්තය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීමංසාව අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ (ප්‍ර‍ථමධ්‍යානයට පැමිණ වෙසේ ද), එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

272\. කවර ධර්‍මයෝ කුශලහ යත්:

යම් කලෙකැ රූපභවෝත්පත්තිය පිණිස මාර්‍ගය වඩා ද, විතර්‍කවිචාරයන්ගේ සන්හිඳීමෙන් ... ද්වීතියධ්‍යානයට ... තෘතියධ්‍යානයට ... චතුර්ථධ්‍යානයට ... ප්‍ර‍ථමධ්‍යානයට ... හීන වූ පඨවිකසිණ - පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද .... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... ඡන්‍දය අධිපති කොට ඇති ... වීර්‍ය්‍යය අධිපති කොට ඇති ... චිත්තය අධිපති කොට ඇති ... විමංසාව අධිපති කොට ඇති ... ඡන්‍දය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීර්‍ය්‍යය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... චිත්තය අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ ... වීමංසාව අධිපති කොට ඇති හීන වූ ... මධ්‍යම වූ ... ප්‍ර‍ණීත වූ (පඤ්චමධ්‍යානයට පැමිණ වෙසේ ද), එසමයෙහි ස්පර්‍ශය වෙයි .... අවි‍ක්‍ෂෙපය වේ ... මේ ධර්‍මයෝ කුශලයෝ යි.

රූපාවචරකුශල යි.
