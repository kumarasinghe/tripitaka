# චතුත්ථං චිත්තං

149\. කවර දහම්හු කුසල්හ යත්:

යම් සමයෙක් හි රූපාලම්බන වේවයි, ... ධර්‍මාලම්බන වේවයි, යම් යම් අරමුණෙකැ හෝ ‍එල්බැ සොම්නස්සහගියා වූ ඥානවිප්‍රයුක්ත වූ සංස්කාර සහිත වැ කාමාවචර කුශලචිත්තය උපදනේ වේ ද, එසමයෙහි ස්පර්‍ශය වෙයි ... අවික්‍ෂේපය වෙයි ... මේ ධර්‍මයෝ කුශලයෝ යි.

චතුර්‍ථචිත්තය යි.
