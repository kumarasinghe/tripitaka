# යෝගගොච්ඡකය

1.

යෝග (පුඟුල්හු සසරැ යොදන) වූ ධර්මයෝ යැ,‍  
යෝග නො වූ ධර්මයෝ යැ,

2.

යෝගනීය ධර්මයෝ යැ,‍  
අයෝගනීය ධර්මයෝ යැ,

3.

යෝගසම්ප්‍රයුක්ත වූ ධර්මයෝ යැ,‍  
යෝගවිප්‍රයුක්ත වූ ධර්මයෝ යැ,

4.

යෝග වූ ද යෝගනීය වූ ද ධර්මයෝ යැ,‍  
යෝගනීය වූ ද යෝග නො වූ ද ධර්මයෝ යැ,

5.

යෝග වූ ද යෝගසම්ප්‍රයුක්ත වූ ද ධර්මයෝ යැ,‍  
යෝග සම්ප්‍රයුක්ත වූ ද නො ද යෝග වූ ද ධර්මයෝ යැ,

6.

යළි යෝගවිප්‍රයුක්ත වූ යෝගනීය ද වූ ධර්මයෝ යැ,‍  
(යෝග විප්‍රයුක්ත වූ) අයෝගනීය ද වූ ධර්මයෝ යි.
