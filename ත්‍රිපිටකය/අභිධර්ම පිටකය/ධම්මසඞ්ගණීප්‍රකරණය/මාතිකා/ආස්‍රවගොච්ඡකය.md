# ආස්‍රවගොච්ඡකය

1.

ආස්‍රව වූ ධර්මයෝ යැ,‍  
ආස්‍රව නො වූ ධර්මයෝ යැ,

2.

ආස්‍රව සහිත වූ ධර්මයෝ යැ,‍  
ආස්‍රව රහිත වූ ධර්මයෝ යැ,

3.

ආස්‍රවසම්ප්‍රයුක්ත වූ ධර්මයෝ යැ,‍  
ආස්‍රවවිප්‍රයුක්ත වූ ධර්මයෝ යැ,

4.

ආස්‍රව වූ ද ආස්‍රව සහිත වූ ද ධර්මයෝ යැ,‍  
ආස්‍රව සහිත වූ ද නො ද ආස්‍රව වූ ද ධර්මයෝ යැ,

5.

ආස්‍රව වූ ද ආස්‍රවසම්ප්‍රයුක්ත වූ ද ධර්මයෝ යැ,‍  
(ඛ) ආසවසම්‌පයුත්‌තා චෙව ධම්‌මා නො ච ආසවා.

6.

යළි ආස්‍රව විප්‍රයුක්ත වූ සාස්‍රව ද වූ ධර්මයෝ යැ,‍  
(ආස්‍රව විප්‍රයුක්ත වූ) අනාස්‍රව ද වූ ධර්මයෝ යි.
