# පරාමාසගොච්ඡකය

1.

පරාමර්ශ (නිත්‍යාදි වශයෙන් වරදවා ගන්නා) ධර්මයෝ යැ,‍  
පරාමර්ශ නො වූ ධර්මයෝ යැ,

2.

පරාමාෂ්ට (පරාමාසයන් විසින් අරමුණු කළ) ධර්මයෝ යැ,‍  
අපරාමාෂ්ට (පරාමාසයන් විසින් අරමුණු නො කළ) ධර්මයෝ යැ,

3.

පරාමාසසම්ප්‍රයුක්ත වූ ධර්මයෝ යැ,‍  
පරාමාසවිප්‍රයුක්ත වූ ධර්මයෝ යැ,

4.

පරාමර්ශ වූ ද පරාමාෂ්ට වූ ද ධර්මයෝ යැ,‍  
පරාමාෂ්ට වූ ද නො ද පරාමර්ශ වූ ධර්මයෝ යැ,

5.

යළි පරාමර්ශ විප්‍රයුක්ත වූ පරාමාෂ්ට ද වූ ධර්මයෝ යැ,‍  
(පරාමර්ශ විප්‍රයුක්ත වූ) අපරාමාෂ්ට ද වූ ධර්මයෝ යි.
