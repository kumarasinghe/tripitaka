# ඕඝගොච්ඡකය

1.

ඕඝ (පුඟුල්හු සසරැ ගලනා) වූ ධර්මයෝ යැ,‍  
ඕඝ නො වූ ධර්මයෝ යැ,

2.

ඕඝනීය (ඕඝයෙන් ඉක්මියයුතු) වූ ධර්මයෝ යැ,‍  
ඕඝනීය නො වූ ධර්මයෝ යැ,

3.

ඕඝසම්ප්‍රයුක්ත වූ ධර්මයෝ යැ,‍  
ඕඝවිප්‍රයුක්ත වූ ධර්මයෝ යැ,

4.

ඕඝ වූ ද ඕඝනීය වූ ද ධර්මයෝ යැ,‍  
ඕඝනීය වූ ද ඕඝ නො වූ ද ධර්මයෝ යැ,

5.

ඕඝ වූ ද ඕඝසම්ප්‍රයුක්ත වූ ද ධර්මයෝ යැ,‍  
ඕඝ සම්ප්‍රයුක්ත වූ ද නො ද ඕඝ වූ ද ධර්මයෝ යැ,

6.

යළි ඕඝ විප්‍රයුක්ත වූ ඕඝනීය ද වූ ධර්මයෝ යැ,‍  
(ඕඝ විප්‍රයුක්ත වූ) ඕඝනීය ද නො වූ ධර්මයෝ යි.
