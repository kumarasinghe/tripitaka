# චුල්ලන්තරදුකය

1.

සප්‍රත්‍යය වූ ධර්මයෝ යැ,‍  
අප්‍රත්‍යය වූ ධර්මයෝ යැ,

2.

සඞ්ඛත (ප්‍රත්‍යයන් විසින් සකස් කළ) ධර්මයෝ යැ,‍  
අසඞ්ඛත (ප්‍රත්‍යයන් විසින් සකස් නො කළ) ධර්මයෝ යැ,

3.

නිදර්ශන (දැක්ක යුතු බව) සහිත වූ ධර්මයෝ යැ,‍  
අනිදර්ශන වූ ධර්මයෝ යැ,

4.

සප්‍රතිඝ (ගැටීම් සහිත) වූ ධර්මයෝ යැ,‍  
අප්‍රතිඝ වූ ධර්මයෝ යැ,

5.

රූපී වූ ධර්මයෝ යැ,‍  
අරූපී වූ ධර්මයෝ යැ,

6.

ලෞකික (ලෝකයෙහි ඇතුළත්) වූ ධර්මයෝ යැ,‍  
ලෝකෝත්තර වූ ධර්මයෝ යැ,

7.

කිසි විඥානයකින් දතයුතු වූ ධර්මයෝ යැ,‍  
කිසි විඥානයකින් නො දතයුතු වූ ධර්මයෝ යි.
