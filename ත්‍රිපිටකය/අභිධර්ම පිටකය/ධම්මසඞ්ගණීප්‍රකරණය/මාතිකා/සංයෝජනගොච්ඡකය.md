# සංයෝජනගොච්ඡකය

1.

සංයෝජන (පුඟුල්හු සසරැ බඳනා) වූ ධර්මයෝ යැ,‍  
සංයෝජන නො වූ ධර්මයෝ යැ,

2.

සංයෝජනීය (සංයෝජනයන්ට හිත) වූ ධර්මයෝ යැ,‍  
සංයෝජනීය නො වූ ධර්මයෝ යැ,

3.

සංයෝජනසම්ප්‍රයුක්ත වූ ධර්මයෝ යැ,‍  
සංයෝජනවිප්‍රයුක්ත වූ ධර්මයෝ යැ,

4.

සංයෝජන වූ ද සංයෝජනීය වූ ද ධර්මයෝ යැ,‍  
සංයෝජනීය වූ ද නො ද සංයෝජන වූ ද ධර්මයෝ යැ,

5.

සංයෝජන වූ ද සංයෝජනසම්ප්‍රයුක්ත වූ ද ධර්මයෝ යැ,‍  
සංයෝජන සම්ප්‍රයුක්ත වූ ද නො ද සංයෝජන වූ ද ධර්මයෝ යැ,

6.

යළි සංයෝජන විප්‍රයුක්ත වූ සංයෝජනීය ද වූ ධර්මයෝ යැ,‍  
(සංයෝජන විප්‍රයුක්ත වූ) අසංයෝජනීය ද වූ ධර්මයෝ යි.
