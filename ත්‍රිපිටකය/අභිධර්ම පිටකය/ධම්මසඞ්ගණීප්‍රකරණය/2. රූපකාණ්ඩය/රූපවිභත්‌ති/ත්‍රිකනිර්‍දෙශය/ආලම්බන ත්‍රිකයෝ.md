# ආලම්බන ත්‍රිකයෝ

799\. ඒ අධ්‍යාත්මික චක්‍ඛුසම්‍ඵස්‍සයට අරමුණු නො වූ රූපය කවරේ යත්:

චක්‍ෂුරායතන ය ... කායායතන යි. මේ ඒ අධ්‍යාත්මික චක්‍ඛුසම්‍ඵස්‍සයට අරමුණු නො වූ රූප යි.

800\. ඒ බාහිර චක්‍ඛුසම්‍ඵස්‍සයට අරමුණු වූ රූපය කවරේ යත්:

රූපායතන යි. මේ ඒ බාහිර චක්‍ඛුසම්‍ඵස්‍සයට ‍අරමුණු වූ රූප යි.

801\. ඒ බාහිර චක්‍ඛුසම්‍ඵස්‍සයට අරමුණු නො වූ රූපය කවරේ යත්:

ශබ්දායතන ය ... කබලීකාරාහාර යි. මේ ඒ බාහිර චක්‍ඛුසම්‍ඵස්‍සයට ‍අරමුණු නො වූ රූප යි.

802\. ඒ ආධ්‍යාත්මික චක්‍ඛුසම්‍ඵස්‍සජවේදනාවට ... සංඥාවට ... චේතනාවට ... චක්ඛුවිඤ්ඤාණයට අරමුණු නො වූ රූපය කවරේ යත්:

චක්‍ෂුරායතන ය ... කායායතන යි. මේ ඒ ආධ්‍යාත්මික චක්‍ෂුර්විඥානයට අරමුණු නො වූ රූප යි.

803\. ඒ බාහිර චක්ඛුවිඤ්ඤාණයට අරමුණු වූ රූපය කවරේ යත්:

රූපායතන යි. මේ ඒ බාහිර චක්‍ෂුර්විඥානයට ‍අරමුණු වූ රූප යි.

804\. ඒ බාහිර චක්ඛුවිඤ්ඤාණයට අරමුණු නො වූ රූපය කවරේ යත්:

ශබ්දායතන ය ... කබලීකාරාහාර යි. මේ ඒ බාහිර චක්‍ෂුර්විඥානයට ‍අරමුණු නො වූ රූප යි.

805\. ඒ ආධ්‍යාත්මික සෝතසම්ඵස්සයට ... ඝානසම්ඵස්සයට ... ජිව්හාසම්ඵස්සයට ... කායසම්ඵස්සයට අරමුණු නො වූ රූපය කවරේ යත්:

චක්‍ෂුරායතන ය ... කායායතන යි. මේ ඒ ආධ්‍යාත්මික කායසංස්පර්‍ශයට අරමුණු නො වූ රූප යි.

806\. ඒ බාහිර කායසම්ඵස්සයට අරමුණු වූ රූපය කවරේ යත්:

ස්ප්‍රෂ්ටව්‍යායතන යි. මේ ඒ බාහිර කායසංස්පර්‍ශයට ‍අරමුණු වූ රූප යි.

807\. ඒ බාහිර කායසම්ඵස්සයට අරමුණු නො වූ රූපය කවරේ යත්:

රූපායතන ය ... කබලීකාරාහාර යි. මේ ඒ බාහිර කායසංස්පර්‍ශයට ‍අරමුණු නො වූ රූප යි.

808\. ඒ ආධ්‍යාත්මික කායසම්ඵස්සජ වේදනාවට ... සංඥාවට ... චේතනාවට ... කායවිඤ්ඤාණයට අරමුණු නො වූ රූපය කවරේ යත්:

චක්‍ෂුරායතන ය ... කායායතන යි. මේ ඒ අධ්‍යාත්මික කායවිඥානයට අරමුණු නො වූ රූප යි.

809\. ඒ බාහිර කායවිඤ්ඤාණයට අරමුණු වූ රූපය කවරේ යත්:

ස්ප්‍රෂ්ටව්‍යායතන යි. මේ ඒ බාහිර කායවිඥානයට ‍අරමුණු වූ රූප යි.

810\. ඒ බාහිර කායවිඤ්ඤාණයට අරමුණු නො වූ රූපය කවරේ යත්:

රූපායතන ය ... කබලීකාරාහාර යි. මේ ඒ බාහිර කායවිඥානයට ‍අරමුණු නො වූ රූප යි.

ආලම්බන ත්‍රිකයෝ යි.
