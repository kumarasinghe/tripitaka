# ආයතන ත්‍රිකයෝ

811\. ඒ බාහිර චක්‍ෂුරායතන නො වූ රූපය කවරේ යත්:

රූපායතන ය ... කබලීකාරාහාර යි. මේ ඒ බාහිර චක්‍ෂුරායතන නො වූ රූප යි.

812\. ඒ අධ්‍යාත්මික චක්‍ඛායතනරූපය කවරේ යත්:

යම් ඇසෙක් සතර මහාභූතයන් නිසා උපන් ප්‍රසාදය වේ ද ... තෙලෙ ‘ශූන්‍යග්‍රාම’ නමුදු වේ. මේ ඒ අධ්‍යාත්මික චක්‍ෂුරායතන රූප යි.

813\. ඒ අධ්‍යාත්මික චක්‍ඛායතන නො වූ රූපය කවරේ යත්:

ශ්‍රොත්‍රායතන ය ... කායායතන යි. මේ ඒ අධ්‍යාත්මික චක්‍ෂුරායතන නො වූ රූප යි.

814\. ඒ බාහිර සෝතායතන නො වූ ... ඝානායතන නො වූ ... ජිව්හායතන නො වූ ... කායායතන නො වූ රූපය කවරේ යත්:

රූපායතන ය ... කබලීකාරාහාර යි. මේ ඒ බාහිර කායායතන නො වූ රූප යි.

815\. ඒ අධ්‍යාත්මික කායායතනරූපය කවරේ යත්:

යම් කයෙක් සතර මහාභූතයන් නිසා උපන් ප්‍රසාදය වූයේ ද ... තෙලෙ ශූන්‍යග්‍රාම නමුදු වේ. මේ ඒ අධ්‍යාත්මික කායායතනරූප යි.

816\. ඒ අධ්‍යාත්මික කායායතන නො වූ රූපය කවරේ යත්:

චක්‍ෂුරායතන ය ... ජිව්හායතන යි. මේ ඒ අධ්‍යාත්මික කායායතන නො වූ රූප යි.

817\. ඒ අධ්‍යාත්මික රූපායතන නො වූ රූපය කවරේ යත්:

චක්‍ෂුරායතන ය ... කායායතන යි. මේ ඒ අධ්‍යාත්මික රූපායතන නො වූ රූප යි.

818\. ඒ බාහිර රූපායතනරූපය කවරේ යත්:

යම් රූපයෙක් සතර මහාභූතයන් නිසා උපන් පැහැසටහන වූයේ ද ... තෙලෙ රූපධාතු නමුදු වේ. මේ ඒ බාහිර රූපායතනරූප යි.

819\. ඒ බාහිර රූපායතන නො වූ රූපය කවරේ යත්:

ශබ්දායතන ය ... කබලීකාරාහාර යි. මේ ඒ බාහිර රූපායතන නො වූ රූප යි.

820\. ඒ අධ්‍යාත්මික ශබ්දායතන නො වූ ... ගන්‍ධායතන නො වූ ... රසායතන නො වූ ... ස්ප්‍රෂ්ටව්‍යායතන නො වූ රූපය කවරේ යත්:

චක්‍ෂුරායතන ය ... කායායතන යි. මේ ඒ අධ්‍යාත්මික ස්ප්‍රෂ්ටව්‍යායතන නො වූ රූප යි.

821\. ඒ බාහිර ඵොට්ඨබ්බායතන රූපය කවරේ යත්:

පෘථිවිධාතු ය ... තෙලෙ ඵොට්ඨබ්බධාතු නමුදු වේ. මේ ඒ බාහිර ස්ප්‍රෂ්ටව්‍යායතනරූප යි.

822\. ඒ බාහිර ඵොට්ඨබ්බායතන නො වූ රූපය කවරේ යත්:

රූපායතන ය ... කබලීකාරාහාර යි. මේ ඒ බාහිර ස්ප්‍රෂ්ටව්‍යායතන නො වූ රූප යි.

ආයතන ත්‍රිකයෝ යි.
