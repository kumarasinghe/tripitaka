# දශකය

982\. ඒ චක්ඛුන්‍ද්‍රියරූපය කවරේ යත්:

යම් ඇසෙක් සතර මහාභූතයන් නිසා උපන් ප්‍රසාදය වූයේ ... තෙල ශූන්‍යග්‍රාම නමුදු වේ. මේ ඒ චක්‍ෂුරින්‍ද්‍රියරූපයි.

983\. ඒ සෝතින්‍ද්‍රියරූපය ... ඝානින්‍ද්‍රිය ... ජිව්හින්‍ද්‍රිය ... කායින්‍ද්‍රිය ... ඉත්‍ථින්‍ද්‍රිය ... පුරිසින්‍ද්‍රිය ... ජීවිතින්‍ද්‍රියරූපය කවරේ යත්:

ඒ රූපීධර්‍මයන්ගේ ආයුෂයෙක් සිටීමෙක් යපනයෙක් යාපනයෙක් ඊර්‍ය්‍යාපථයෙක් වර්‍තනයෙක් පාලනයෙක් ජීවිතයෙක් ජීවිතේන්‍ද්‍රියයෙක් වේ ද, මේ ඒ ජීවිතේන්‍ද්‍රියරූප යි.

984\. ඒ ඉන්‍ද්‍රිය නො වූ සප්‍රතිඝරූපය කවරේ යත්:

රූපායතන ය ... ස්ප්‍රෂ්ටව්‍යායතන යි. මේ ඒ ඉන්‍ද්‍රිය නො වූ සප්‍රතිඝරූප යි.

985\. ඒ ඉන්‍ද්‍රිය නො වූ අප්‍රතිඝරූපය කවරේ යත්:

කායවිඥප්ති ය ... කවලීකාරාහාර යි. මේ ඒ ඉන්‍ද්‍රිය නො වූ අප්‍රතිඝරූප යි.

මෙසේ දශප්‍රකාරයෙකින් රූපසඞ්ග්‍රහය වේ.

දශකය යි.
