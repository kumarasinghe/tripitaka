# ආයතන දුක

698\. ඒ චක්ඛායතනරූපය කවරෙ යත්:

යම් ඇසෙක් සතර මහාභූතයන් නිසා උපන් ප්‍රසාදය වූයේ ද … තෙලෙ ශුන්‍යග්‍රාම නමුදු වේ. මේ එ චක්ෂුරායතන රූප යි.

699\. ඒ චක්ෂුරායතන නො වූ රූපය කවරෙ යත්:

ශ්‍රොතායතන ය … කවලීකාරාහාර යි. මේ ඒ චක්ෂුරායතන නො වූ රූප යි.

700\. ඒ ශ්‍රොත්‍රායතන … ඝ්‍රාණායතන ... ජිහ්වායතන … කායායතන රූපය කවරෙ යත්:

යම් කයෙක් සතරමහාභූතයන් නිසා උපන් ප්‍රසාදය වූයේ ද … තෙලෙ ශුන්‍යග්‍රාම නමුදු වේ. මේ ඒ කායායතනරූප යි.

701\. ඒ කායායතන නො වූ රූපය කවරෙ යත්:

චක්ෂුරායතන ය … කවලීකාරාහාර යි. මේ ඒ කායායතන නො වූ රූප යි.

702\. ඒ රූපායතනරූපය කවරෙ යත්:

යම් රූපයෙක් සතර මහාභූතයන් නිසා උපන් පැහැසටහන වූයේ ද … තෙල රූපධාතු නමුදු වේ. මේ ඒ රූපායතනරූප යි.

703\. ඒ රූපායතන නො වූ රූපය කවරෙ යත්:

චක්ෂුරායතන ය … කවලීකාරාහාර යි. මේ ඒ රූපායතන නො වූ රූප යි.

704\. ඒ ශබ්දායතනරූපය … ගන්ධායතන … රසායතන … ස්ප්‍රෂ්ටව්‍යායතනරූපය කවරෙ යත්:

පෘථිවිධාතු ය … තෙල ස්ප්‍රෂ්ටව්‍යධාතු නමුදු වේ. මේ ඒ ස්ප්‍රෂ්ටව්‍යායතනරූපය යි.

705\. ඒ ඵොට්ඨබ්බායතන නො වූ රූපය කවරෙ යත්:

චක්ෂුරායතන ය … කවලීකාරාහාර යි. මේ ඒ ස්ප්‍රෂ්ටව්‍යායතන නො වූ රූප යි.

(ආයතන දුක යි.)
