# ආලම්බන දුක

690\. ඒ චක්ඛුසම්ඵස්සයට ආලම්බන වූ රූපය කවරෙ යත්:

රූපායතන ය, මේ ඒ චක්ඛුසම්ඵස්සයට ආලමබන වූ රූප යි.

691\. ඒ චක්ඛුසම්ඵස්සයට ආලම්බන නො වූ රූපය කවරෙ යත්:

චක්ෂුරායතන ය … කවලීකාරාහාර යි. මේ ඒ චක්ඛුසම්ඵස්සයට ආලමබන නො වූ රූප යි.

692\. ඒ චක්ඛුසම්ඵස්සජ වෙදනාවට … සංඥාවට … චෙතනාවට … චක්ෂුර්විඥානයට ආලම්බන වූ රූපය කවරෙ යත්:

රූපායතන යි. මේ චක්ෂුර්විඥානයට ආලම්බන වූ රූප යි.

693\. ඒ චක්ෂුර්විඥානයට ආලම්බන නො වූ රූපය කවරෙ යත්:

චක්ෂුරායතන ය … කවලීකාරාහාර යි. මේ ඒ චක්ෂුර්විඥානයට ආලම්බන නො වූ රූප යි.

694\. ඒ සොතසම්ඵස්සයට ඝානසම්ඵස්සයට … ජිව්හාසම්ඵස්සයට … කායසම්ඵස්සයට ආලම්බන වූ රූපය කවරෙ යත්:

ස්ප්‍රෂ්ටව්‍යායතන යි. මේ ඒ කායසම්ඵස්සයට ආලම්බන වූ රූප යි.

695\. ඒ කායසම්ඵස්සයට ආලම්බන නො වූ රූපය කවරෙ යත්:

චක්ෂුරායතන ය … කවලීකාරාහාර යි. මේ ඒ කායසම්ඵස්සයට අරමුණු නො වූ රූප යි.

696\. ඒ කායසම්ඵස්සජවෙදනාවට … සංඥාවට … චෙතනාවට … කායවිඥානයට අරමුණු වූ රූපය කවරෙ යත්:

ස්ප්‍රෂ්ටව්‍යායතන යි. මේ කායවිඥානයට අරමුණු වූ රූප යි.

697\. ඒ කායවිඥානයට අරමුණු නො වූ රූපය කවරෙ යත්:

චක්ෂුරායතන ය … කවලීකාරාහාර යි. මේ ඒ කායවිඥානයට අරමුණු නො වූ රූප යි.

(ආලම්බන දුක යි.)
