# ධාතු දුක

706\. ඒ චක්ඛුධාතු රූපය කවරෙ යත්:

චක්ෂුරායතන යි. මේ ඒ චක්ෂූර්ධාතුරූප යි.

707\. ඒ චක්ඛුධාතු නො වූ රූපය කවරෙ යත්:

ශ්‍රොත්‍රායතන ය … කවලීකාරාහාර යි. මේ ඒ චක්ෂුර්ධාතු නො වූ රූප යි.

708\. ඒ සොතධාතුරූපය … ඝානධාතු … ජිව්හාධාතු … කායධාතුරූපය කවරෙ යත්:

කායායතන යි. මේ ඒ කායධාතු රූප යි.

709\. ඒ කායධාතු නො වූ රූපය කවරෙ යත්:

චක්ෂුරායතන ය … කවලීකාරාහාර යි. මේ ඒ කායධාතු නො වූ රූප යි.

710\. ඒ රූපධාතුරූපය කවරෙ යත්:

රූපායතන යි. මේ ඒ රූපධාතුරූප යි.

711\. ඒ රූපධාතු නො වූ රූපය කවරෙ යත්:

චක්ෂුරායතන ය … කවලීකාරාහාර යි. මේ ඒ රූපධාතු නොවූ රූප යි.

712\. ඒ ශබ්දධාතුරූපය … ගන්ධධාතු … රසධාතු … ඵොට්ඨබ්බධාතු රූපය කවරෙ යත්:

ස්ප්‍රෂ්ටව්‍යායතන යි. මේ ඒ ස්ප්‍රෂ්ටව්‍යධාතුරූප යි.

713\. ඒ ඵොට්ඨබ්බධාතු නො වූ රූපය කවරෙ යත්:

චක්ෂුරායතන ය … කවලීකාරාහාර යි. මේ ඒ ස්ප්‍රෂ්ටව්‍යධාතු නො වූ රූප යි.

(ධාතු දුක යි.)
