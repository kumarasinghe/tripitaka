# ඒකාදශකය

986\. ඒ චක්‍ඛායතන රූපය කවරේ යත්:

යම් ඇසෙක් සතර මහාභූතයන් නිසා උපන් ප්‍රසාදය වූයේ ... තෙල ශූන්‍යග්‍රාම නමුදු වේ. මේ ඒ චක්‍ෂුරායතන රූපයි.

987\. ඒ සෝතායතනරූපය ... ඝානායතන ... ජිව්හායතන ... කායායතන ... රූපායතන ... සද්දායතන ... ගන්‍ධායතන ... රසායතන ... ඵොට්ඨබ්බායතනරූපය කවරේ යත්:

පෘථිවිධාතු ය ... තෙල ස්ප්‍රෂ්ටව්‍යධාතු නමුදු වේ. මේ ඒ ස්ප්‍රෂ්ටව්‍යායතනරූප යි.

988\. ඒ අනිදස්සන අප්පටිඝ වූ ධම්මායතනපරියාපන්න රූපය කවරේ යත්:

ස්ත්‍රීන්‍ද්‍රිය ය ... කවලීකාරාහාර යි. මේ ඒ අනිදර්‍ශන අප්‍රතිඝ වූ ධර්‍මායතනපර්‍යාපන්නරූප යි.

මෙසේ ඒකාදශප්‍රකාරයෙකින් රූපසඞ්ග්‍රහය වේ.

ඒකාදශකය යි.
