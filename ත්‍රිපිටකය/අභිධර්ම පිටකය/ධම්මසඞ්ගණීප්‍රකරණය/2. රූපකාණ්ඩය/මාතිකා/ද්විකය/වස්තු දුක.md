# වස්තු දුක

චක්ඛුසම්ඵස්සයට වස්තු වූ රූපයෙක් ඇත, චක්ඛු සම්ඵස්සයට වස්තු නො වූ රූපයෙක් ඇත. චක්ඛුසම්ඵස්සජ වූ වෙදනාවට … සංඥාවට … චෙතනාවට … චක්ෂුර්විඥානයට වස්තු වූ රූපයෙක් ඇත, චක්ෂුර්විඥානයට වස්තු නො වූ රූපයෙක් ඇත.

සොතසම්ඵස්සයට … ඝානසම්ඵස්සයට … ජිව්හාසම්ඵස්සයට … කායසම්ඵස්සයට වස්තු වූ රූපයෙක් ඇත. කායසම්ඵස්සයට වස්තු නො වූ රූපයෙක් ඇත. කායසම්ඵස්සජවෙදනාවට … සංඥාවට … චෙතනාවට … කායවිඥානයට වස්තු වූ රූපයෙක් ඇත, කායවිඥානයට වස්තු නො වූ රූපයෙක් ඇති.

වස්තුදුක යි. (25)
