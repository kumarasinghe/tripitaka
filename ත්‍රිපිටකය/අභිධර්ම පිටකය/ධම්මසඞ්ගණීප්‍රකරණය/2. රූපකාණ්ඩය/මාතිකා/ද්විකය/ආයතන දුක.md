# ආයතන දුක

චක්ඛායතන රූපයෙක් ඇත, චක්ඛායතන නො වූ රූපයෙක් ඇත.

සොතායතන … ඝානායතන … ජිව්හායතන … කායායතනරූපයෙක් ඇත, කායායතන නො වූ රූපයෙක් ඇත.

රූපායතන වූ රූපයෙක් ඇත, රූපායතන නො වූ රූපයෙක් ඇත.

ශබ්දායතනරූපයෙක් ඇත, ශබ්දායතන නො වූ රූපයෙක් ඇත.

ගන්ධායතනරූපයෙක් ඇත, ගන්ධායතන නො වූ රූපයෙක් ඇත.

රසායතනරූපයෙක් ඇත, රසායතන නො වූ රූපයෙක් ඇත.

ඵොට්ඨබ්බායතනරූපයෙක් ඇත, ඵොට්ඨබ්බායතන නො වූ රූපයෙක් ඇති.

ආයතන දුක යි. (10)
