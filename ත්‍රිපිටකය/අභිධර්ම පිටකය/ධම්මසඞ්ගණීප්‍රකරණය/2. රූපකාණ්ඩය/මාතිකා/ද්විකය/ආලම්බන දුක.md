# ආලම්බන දුක

චක්ඛුසම්ඵස්සයට අරමුණු වූ රූපයෙක් ඇත, චක්ඛුසම්ඵස්සයට අරමුණු නො වූ රූපයෙක් ඇත. චක්ඛුසම්ඵස්සජවෙදනාවට … සංඥාවට … චෙතනාවට … චක්ඛුවිඤ්ඤාණයට අරමුණු වූ රූපයෙක් ඇත, චක්ඛුවිඤ්ඤාණයට අරමුණු නො වූ රූපයෙක් ඇත.

සොතසම්ඵස්සයට … ඝානසම්ඵස්සයට … ජිව්හාසම්ඵස්සයට … කායසම්ඵස්සයට අරමුණු වූ රූපයෙක් ඇත, කායසම්ඵස්සයට අරමුණු නො වූ රූපයෙක් ඇත. කායසම්ඵස්සජවෙදනාවට ... සංඥාවට … චෙතනාවට … කායවිඤ්ඤාණයට අරමුණු වූ රූපයෙක් ඇත, කායවිඤ්ඤාණයට අරමුණු නො වූ රූපයෙක් ඇති.

ආලම්බනදුක යි. (25)
