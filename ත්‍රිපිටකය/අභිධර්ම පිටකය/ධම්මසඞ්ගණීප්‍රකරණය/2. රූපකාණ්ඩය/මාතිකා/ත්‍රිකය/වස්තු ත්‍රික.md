# වස්තු ත්‍රික

යම් බාහිරරූපයෙක් වේ ද එය චක්ඛුසම්ඵස්සයට වස්තු නො වූයේ ය. යම් ආධ්‍යාත්මිකරූපයෙක් වේ ද එය චක්ඛුසම්ඵස්සයට වස්තු වූයේ ඇත, චක්ඛුසම්ඵස්සයට වස්තු නො වූයේ ඇති.

යම් බාහිරරූපයෙක් වේ ද එය චක්ඛුසම්ඵස්සජවෙදනාවට … සංඥාවට … චෙතනාවට … චක්ඛුවිඤ්ඤාණයට වස්තු නො වූයේ ය. යම් ආධ්‍යාත්මික රූපයෙක් වේ ද එය චක්ඛුවිඤ්ඤාණයට වස්තු වූයේ ඇත, චක්ඛුවිඤ්ඤාණයට වස්තු නො වූයේ ඇති.

යම් බාහිරරූපයෙක් වේ ද එය සොතසම්ඵස්සයට … ඝානසම්ඵස්සයට … ජිව්හාසම්ඵස්සයට … කායසම්ඵස්සයට වස්තු නො වූයේ ය. යම් ආධ්‍යාත්මික රූපයෙක් වේ ද එය කායසම්ඵස්සයට වස්තු වූයේ ඇත, කායසම්ඵස්සයට වස්තු නො වූයේ ඇති.

යම් බාහිරරූපයෙක් වේ ද එය කායසම්ඵස්සජ වෙදනාවට … සංඥාවට … චෙතනාවට … චක්ඛුවිඥානයට වස්තු නො වූයේ ය. යම් ආධ්‍යාත්මිකරූපයෙක් වේ ද එය කායවිඥානයට වස්තු වූයේ ඇත, කායවිඥානයට වස්තු නො වූයේ ඇති.

වස්තු ත්‍රික යි. (25)
