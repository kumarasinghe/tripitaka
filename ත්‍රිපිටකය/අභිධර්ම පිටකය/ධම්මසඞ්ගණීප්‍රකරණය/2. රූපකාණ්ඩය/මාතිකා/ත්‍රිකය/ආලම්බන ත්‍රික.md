# ආලම්බන ත්‍රික

යම් ආධ්‍යාත්මිකරූපයෙක් වේ ද එය චක්ඛුසම්ඵස්සයට අරමුණු නො වූයේ ය. යම් බාහිරරූපයෙක් වේ ද එය චක්ඛුසම්ඵස්සයට අරමුණු වූයේ ඇත, චක්ඛුසම්ඵස්සයට අරමුණු නො වූයේ ඇති.

යම් ආධ්‍යාත්මික රූපයෙක් වේ ද එය චක්ඛුසම්ඵස්සජවෙදනාවට … සංඥාවට … චෙතනාවට … චක්ඛුවිඤ්ඤාණයට අරමුණු නො වූයේ ය. යම් බාහිරරූපයෙක් වේ ද එය චක්ඛුවිඤ්ඥාණයට අරමුණු වූයේ ඇත, චක්ඛුවිඤ්ඥාණයට අරමුණු නො වූයේ ඇති.

යම් ආධ්‍යාත්මිකරූපයෙක් වේ ද එය සොතසම්ඵස්සයට … ඝානසම්ඵස්සයට ජිව්හාසම්ඵස්සයට … කායසම්ඵස්සයට අරමුණු නො වූයේ ය. යම් බාහිරරූපයෙක් වේ ද එය කායසම්ඵස්සයට අරමුණු වූයේ ඇත, කායසම්ඵස්සයට අරමුණු නො වූයේ ඇති.

යම් ආධ්‍යාත්මිකරූපයෙක් වේ ද එය කායසඵස්සජවෙදනාවට … සංඥාවට … චෙතනාවට. … කායවිඥානයට අරමුණු නො වූයේ ය. යම් බාහිරරූපයෙක් වේ ද එය කායවිඥානයට අරමුණු වූයේ ඇත, කායවිඥානයට අරමුණු නො වූයේ ඇති.

ආලම්බන ත්‍රික යි. (25)
