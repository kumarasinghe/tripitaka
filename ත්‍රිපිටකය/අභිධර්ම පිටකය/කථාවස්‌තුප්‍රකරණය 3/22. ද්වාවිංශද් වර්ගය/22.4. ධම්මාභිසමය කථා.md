# 22.4. ධම්මාභිසමය කථා

**\[අතීතභවයෙහි ශ්‍රොතාපන්නයකු මව්කුසැ වැස නික්මෙන්නා දැක ගර්‍භාශය්‍යායෙහි ධර්‍මාභිසමය (ලොකොත්තර ධර්‍මාවබොධය) ඇතැ’යි සමහර උත්තරාපථකයන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ‘ධම්මාභිසමය කථා’ එයි.\]**

1\. ස. පු: ගර්‍භාශය්‍යායෙහි (මව්කුසැ) ධර්‍මාභිසමය ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ගර්‍භාශය්‍යායෙහි ධර්‍මදෙශනා, ධර්‍මශ්‍රවණ, ධර්‍මසාකච්ඡා, පරිපෘච්ඡා, ශීලසමාදාන, ඉන්ද්‍රියයන්හි ගුප්තද්වාරතා, භොජනයෙහි මාත්‍රඥතා, පූර්‍වරාත්‍ර අපරරාත්‍රයෙහි ජාගර්‍ය්‍යානුයොග ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: ගර්‍භාශය්‍යායෙහි ධර්‍මදෙශනා, ධර්‍මශ්‍රවණ, … පූර්‍වරාත්‍රාපරරාත්‍රයෙහි ජාගර්‍ය්‍යානුයොග නැත් ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් ගර්‍භශය්‍යායෙහි ධර්‍මදෙශනා, ධර්‍මශ්‍රවණ, … පූර්‍වරාත්‍රාපරරාත්‍රයෙහි ජාගර්‍ය්‍යානුයොග නැත් නම්,

ස. පා: භවත, ‘ගර්‍භශය්‍යායෙහි ධර්‍මාභහිසමය ඇතැ’යි නො කියැයුතු මැ යැ.

2\. ස. පු: ගර්‍භශය්‍යායෙහි ධර්‍මාභිසමය ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්‍යග්දෘෂ්ටිහුගේ ඉපැත්මට පරතොඝොෂ යැ යොනිසොමනස්කාර යැ යි ප්‍රත්‍යයයෝ දෙදෙනෙක් ඇත්තාහු නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් සම්‍යග්දෘෂ්ටිහුගේ ඉපැත්මට පරතොඝොෂ යැ යොනිසොමනස්කාර යැ යි ප්‍රත්‍යයයෝ දෙදෙනෙක් ඇත් නම්,

ස. පා: භවත, ‘ගර්‍භශය්‍යායෙහි ධර්‍මාභිසමය ඇතැ’යි නො කියැයුතු මැ යැ.

3\. ස. පු: ගර්‍භශය්‍යායෙහි ධර්‍මාභිසමය ඇත් ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සුප්තයාහට ප්‍රමත්තයාහට මුෂිතස්මෘති ඇතියාහට සම්‍යක්ප්‍රඥා නැතියාහට ධර්‍මාභිසමය වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ධම්මාභිසමය කථා නිමි.
