# 20.5. මග්ග කථා

**\[“පුබ්බෙව ඛො පනස්ස කායකම්මං වචීකම්මං ආජීවො සුපරිසුද්ධො හොති - පළමු මැ ඔහුගේ කායකර්‍ම වාක්කර්‍ම ආජීව සුපරිශුද්ධ වෙයි” යන මේ සූත්‍රපාඨ ද, සම්‍යක් වචන කර්‍මාන්ත ආජීවයන්ගේ චිතත විප්‍රයුක්තතා ද නිසා නිෂ්පර්‍ය්‍යායයෙන් මාර්‍ගය පඤ්චාඞ්ගික වෙති’ යි මහිංසාසික නිකායවාසීන් සෙයින් ලබ්ධි ගත්තාහු ඇත. උන්ගේ වාද බිඳුනට ‘මග්ගකථා’ එයි\]**

1\. ස. පු: මාර්‍ගය පඤ්චාඞ්ගික වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: භාග්‍යවතුන් වහන්සේ විසින් අෂ්ටාඞ්ගිකමාර්‍ගය වදාරනලද නො: ඒ කෙසේ යැ යත්? “සම්‍යක්දෘෂ්ටි ය ... සම්‍යක්සමාධි” යි වදාරනලද නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් භාග්‍යවතුන් වහන්සේ විසින් අෂ්ටාඞ්ගික මාර්‍ගය වදාරනලද වී නම්, ඒ කෙසේ යැ යත්: ‘සම්‍යක්දෘෂ්ටි යැ … සම්‍යක් සමාධි’ යි වදාරනලද වී නම්,

ස. පා: භවත මාර්‍ගය පඤ්චාඞ්ගික යයි නො කියැයුතු මැ යැ.

2\. ස. පු: මාර්‍ගය පඤ්චාඞ්ගික වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: භාග්‍යවතුන් වහන්සේ විසින් වදාරනලද නො:

“මාර්‍ගයන් අතුරින් අෂ්ටාඞ්ගිකමාර්‍ගය උතුමි. සත්‍යයන් අතුරින් (දඃඛසත්‍යාදි) සතරපදයෝ උතුම්හ. ධර්‍මයන් අතුරින් විරාගය උතුමි. දෙපා ඇතියවුන් අතුරින් චක්ෂුස්මත් බුදුහු උතුම්හ” යි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. පා: එකරුණින් මාර්‍ගය අෂ්ටාඞ්ග ඇති වෙයි.

3\. ස. පු: සම්‍යග් වචනය මාර්‍ගාඞ්ග වෙයි, එ ද මාර්‍ග නො වන්නේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්‍යග්දෟෂ්ටිය මාර්‍ගාඞ්ග වෙයි, එ ද මාර්‍ග නො වන්නේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

4\. ස. පු: සම්‍යග් වචනය මාර්‍ගාඞ්ග වෙයි, එ ද මාර්‍ග නො වන්නේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්‍යක්සංකල්පය … සම්‍යග්ව්‍යායමය සම්‍යක්සමෘතිය … සම්‍යක්සමාධිය මාර්‍ගාඞ්ග වෙයි, එ ද මාර්‍ග නො වන්නේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

5\. ස. පු: සම්‍යග්කර්‍මාන්තය සම්‍යගාජීවය මාර්‍ගාඞ්ග වෙයි, එ ද මාර්‍ග නො වන්නේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්‍යග්දෘෂ්ටිය … සම්‍යක්සමාධිය මාර්‍ගාඞ්ග වෙයි, එ ද මාර්‍ග නො වන්නේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

6\. ස. පු: සම්‍යග්දෘෂ්ටි මාර්‍ගාඞ්ග වෙයි, එ ද මාර්‍ග වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්‍යග්වචන මාර්‍ගාඞ්ග වෙයි, එ ද මාර්‍ග වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

7\. ස. පු: සම්‍යග්දෘෂ්ටි මාර්‍ගාඞ්ග වෙයි, එ ද මාර්‍ග වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්‍යක්කර්‍මාන්ත … සම්‍යගාජීව මාර්‍ගාඞ්ග වෙයි එ ද මාර්‍ග වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

8\. ස. පු: සම්‍යක්සංකල්ප … සම්‍යග්ව්‍යායම ... සම්‍යක්ස්මෘතිය … සම්‍යක්සමාධි මාර්‍ගාඞ්ග වෙයි, එ ද මාර්‍ග වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්‍යග්වචන … සම්‍යග්කර්‍මාන්ත … සම්‍යගාජීව මාර්‍ගාඞ්ග වෙයි, එ ද මාර්‍ග වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

9\. ප. පු: ආර්‍ය්‍යඅෂ්ටාඞ්ගික මාර්‍ග වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: භාග්‍යවතුන් වහන්සේ විසින් වදාරනලද නො:

“පළමු මැ ඔහට කායකර්‍ම වාක්කර්‍ම ආජීව සුපරිශුද්ධ වෙයි. මෙසේ ඔහට මේ ආර්‍ය්‍ය අෂ්ටාඞ්ගික මාර්‍ගය භාවනායෙන් පරිපූර්‍ණ බවට යේ” යයි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ …

ප. පා: එකරුණින් පඤ්චාඞ්ගික මාර්‍ගය වේ.

10\. ස. පු: පඤ්චාඞ්ගික මාර්‍ගය වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: භාග්‍යවතුන් වහන්සේ විසින් වදාරනලද නො?

“සුභද්‍රය, යම් ධර්‍මවිනයෙකැ ආර්‍ය්‍ය අෂ්ටාඞ්ගික මාර්‍ගය නො ලැබේ ද, එහි ශ්‍රමණ ද නො ලැබෙයි. එහි ද්විතීය ශ්‍රමණ ද නො ලැබෙයි, එහි තෘතිය ශ්‍රමණ ද නො ලැබෙයි. එහි චතුර්‍ථ ශ්‍රමණ ද නො ලැබෙයි. සුභද්‍රය වැළිත් යම් ධර්‍මවිනයෙකැ ආර්‍ය්‍ය අෂ්ටාඞ්ගික මාර්‍ගය උපලබ්ධ වේ ද, එහි ශ්‍රමණ ද උපලබ්ධ ද වෙයි. එහි ද්විතීය ශ්‍රමණ ද උපලබ්ධ වෙයි. එහි තෘතිය … චතුර්‍ථ ශ්‍රමණ ද උපලබ්ධ වෙයි. සුභද්‍රය, මේ ධර්‍ම විනයෙහි ආර්‍ය්‍ය අෂ්ටාඞ්ගික මාර්‍ගය උපලබ්ධ වෙයි. සුභද්‍රය, ශ්‍රමණ (සෝවන්) මෙ සස්නෙහි මැ වෙයි. ද්විතීය ශ්‍රමණ (සෙදගැමි) ද මෙහි මැ වෙයි. තෘතිය ශ්‍රමණ (අනගැමි) ද මෙහි මැ වෙයි. චතුර්‍ථ ශ්‍රමණ (රහත්) ද මෙහි මැ වෙයි. අන්‍ය පර (දෙසැට මිසදිටු ගතුවන්ගේ) ප්‍රවාදයෝ ශ්‍රමණයන්ගෙන් ශුන්‍ය වෙති” යි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ නො?

ස. පටිඤ්‍ඤා: එසේ යැ.

ස. පා: එකරුණින් අෂ්ටාඞ්ගිකමාර්‍ග වෙයි.

මග්ග කථා නිමි.
