# 20.6. ඤාණ කථා

**\[ලෞකික ලොකොත්තර විසින් ඥානය ද්විවිධ යි. සමාපත්ති ඥානයත් දානාදි වශයෙන් පැවැති කර්‍මස්වකෘත ඥානයත් ලෞකික වෙයි. සත්‍යය පිරිසිඳුනා මාර්‍ගඥානයත් ඵලඥානයත් ලොකොත්තර වෙයි. මේ වෙනස නො ගෙන ‘සත්‍යය පිරිසිඳුනා ඥානය මැ ඥාන වෙයි, සෙසු ඥාන නැතැ’යි ද එහෙයින් මැ පෘථග්ජනහට ඥානය නැතැ’ යි ද හෙතුවාදීන් සෙයින් ලබ්ධි ගතුවන්ගේ වාද බිඳුනට ‘ඤාණකථා’ එයි\]**

1\. ස. පු: ද්වාදශවස්තුකඥානය ලොකොත්තර වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ලොකොත්තර ඥාන දොළොසෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

2\. ස. පු: ලොකොත්තරඥාන දොළොසෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සෝවන් මාර්‍ග දොළොසෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

3\. ස. පු: ශ්‍රොතාපත්තිමාර්‍ග දොළොසෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ශ්‍රොතාපත්තිඵල දොළොසෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ නො කියැයුතු යැ …

ස. අනු: සකෘදාගාමිමාර්‍ග දොළොසෙක් … අනාගාමි … අර්‍හන්මාර්‍ග දොළොසෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

4\. ස. පු: අර්‍හන්මාර්‍ග දොළොසෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අර්‍හත්ඵල දොළොසෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

5\. ප. පු: ද්වාදශවස්තුක ඥානය ලොකොත්තර වෙති යි නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: භාග්‍යවතුන් වහන්සේ විසින් වදාරන ලද නො:

“මහණෙනි, මේ (පඤ්චස්කන්ධය) දුඃඛාර්‍ය්‍යසත්‍ය යැයි පෙර නො ඇසූ දහම්හි මට චක්ෂුස පහළ වී. ඥානය පහළ වී. විද්‍යාව පහළ වී. ප්‍රඥාව පහළ වී. ආලොකය පහළ වී. මහණෙනි, ඒ මේ දුඃඛාර්‍ය්‍යසත්‍යය පරිඥා කටයුතු යැ යි මට … මහණෙනි, පරිඥා කරනලදැ’ යි මට … මහණෙනි, මේ (තෘෂ්ණාව) දුඃඛසමුදයාර්‍ය්‍ය සත්‍ය යැ යි මට … මහණෙනි, ඒ මේ දුඃඛසමුදයාර්‍ය්‍යසත්‍යය ප්‍රහාතව්‍ය යැ යි මට ... මහණෙනි, මට ප්‍රහීණ යැ යි … මහණෙනි, මේ දුඃඛනිරොධාර්‍ය්‍යසත්‍ය යැ යි මට ... මහණෙනි, ඒ මේ දුඃඛනිරොධාර්‍ය්‍යසත්‍ය සාක්ෂාත්කර්‍තව්‍ය යැ යි මට … මහණෙනි, මා විසින් සාක්ෂාත්කෘත යැ යි … මහණෙනි, මේ දුඃඛනිරොධගාමිනීප්‍රතිපදාර්‍ය්‍යසත්‍ය යැ යි මට … මහණෙනි, ඒ මේ දුඃඛනිරොධගාමිනීප්‍රතිපදාර්‍ය්‍ය සත්‍යය භාවිතව්‍ය යැයි මට … මහණෙනි, මා විසින් භාවිත යැ යි පෙර නො ඇසූ දහම්හි චක්ෂුස පහළ වී … ආලොකය පහළ වී” යයි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ නො වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ස. පා: එකරුණින් ද්වාදශවස්තුක ඥානය ලොකොත්තර වේ.

ඤාණ කථා නිමි.
