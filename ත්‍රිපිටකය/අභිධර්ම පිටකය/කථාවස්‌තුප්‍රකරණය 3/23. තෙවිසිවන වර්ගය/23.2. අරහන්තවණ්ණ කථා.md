# 23.2. අරහන්තවණ්ණ කථා

**\[සමෘද්ධ ඉරියව් ඇති සමෘද්ධ ආකල්ප ඇති පාපභික්‍ෂූන් දැක රහත්වෙසින් සිට අමනුෂ්‍යයෝ මෛථුනධර්‍මසෙවන කෙරෙති’යි සමහර උත්තරාපථකයන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ‘අරහන්තවණ්ණ කථා’ එයි\]**

1\. ස. පු: රහත්වනින් (රහත් වෙසින්) අමනුෂ්‍යයෝ මෛථුනධර්‍මසෙවන කෙරෙත්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: රහත්වතින් අමනුෂ්‍යයෝ ප්‍රාණඝාත කෙරෙත් ද … අයිනාදන් කෙරෙත් ද … මුසවා බෙණෙත් ද පිසුනුබස් බෙණෙත් ද … පරොස්බස් බෙණෙත් ද … පලප් බෙණෙත් ද … සන්‍ධිච්ඡෙද කෙරෙත් ද … තිල්ලොපහරණ (මහාවිලොප) කෙරෙත් ද … එක් ගෙයක් පැහැරගන්නා කම් කෙරෙත් ද … කඩපලාසොරකම් කෙරෙත් ද … විරඹු සොරකම් කෙරෙත් ද … ගම්පහරනා කම් කෙරෙත් ද … නියම්ගම්පහරනා කම් කෙරෙත් ද?

ස. පටි: එසේ නො කියැයුතු යැ ...

අරහන්තවණ්ණ කථා නිමි.
