# 23.1. එකාධිප්පාය කථා

**\[කාරුණ්‍යයෙන් එකාධිප්‍රාය ඇති වැ වේවයි, සසර දී එක්වම්හයි ස්ත්‍රිය හා සමග බුද්ධපූජාදිය කොට ප්‍රාර්‍ථනා තැබීම් වශයෙන් එකාධිප්‍රාය ඇති වැ වේවයි, මෙසේ ස්ත්‍රීපුරුෂ දෙදෙනා මැ එකාධිප්‍රාය ඇති වැ මෛථුන ධර්‍ම සෙවන කටයුතු යයි අන්‍ධක වෛතුල්‍යවාදීන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ‘එකාධිප්පාය කථා’ එයි.\]**

1\. ස. පු: එකාධිප්‍රායයෙන් මෛථුනධර්‍ම සෙවන කටයුතු වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: එකාධිප්‍රායයෙන් අශ්‍රමණ වියයුතු වේ ද, අභික්‍ෂු වියයුතු වේ ද, ඡින්‍නමූල වියයුතු වේ ද, පාරාජික වියයුතු වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

2\. ස. පු: එකාධිප්‍රායයෙන් මෛථුනධර්‍ම සෙවන කටයුතු වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: එකාධිප්‍රායයෙන් ප්‍රාණඝාත කළයුතු වේ ද, … අයිනාදන් කළයුතු වේ ද, මුසවා බිණියයුතු වේ ද, පිසුණුබස් බිණියයුතු වේ ද, … පරොස්බස් බිණියයුතු වේ ද … පලප් බිණියයුතු වේ ද … සන්‍ධිච්ඡෙද කළයුතු වේ ද … මහාවිලොප කළයුතු වේ ද, එකාගාරික කම් කළයුතු වේ ද … (කඬපලා සොරකම් පිණිස) මහමඟ සිටියයුතු වේ ද … පරඹුවන් කරා යායුතු වේ ද … ගම් පැහැරීම් කළයුතු වේ ද … නියම්ගම්පැහැරීම් කළයුතු වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

එකාධිප්පාය කථා නිමි.
