# 23.3. ඉස්සරියකාමකාරිතා කථා

**\[“බෝසත් ඡද්දන්තජාතකාදියෙහි පෙණෙන පරිදි ඓශ්චර්‍ය්‍යකාමකාරී බව (කර්‍මබලය නොව සිය කැමැත්තෙන් ක්‍රියා කිරීම) හෙතුකොට ගෙන තිරිසන් ගත වෙයි, විනිපාතයට යෙයි, ගර්‍භශය්‍යාවට බැසගනී, දුෂ්කරක්‍රියා කෙළේ යැ. මෙරමා නො තවනු කෙළේ යැ. අන්‍යශාස්තෘවක්හු උදෙසී”යයි අන්‍ධකනිකායවාසීන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ‘ඉස්සරියකාමකාරිතා කථා’ එයි.\]**

1\. ස. පු: බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා (අදහස ඉටුකිරීමට සිය කැමැත්තෙන් ක්‍රියා කිරීම්) හෙතුවින් විනිපාතයට (නිරයට වුවත්) යන්නේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් නිරයට යන්නේ වේ ද … සඤ්‍ජවමහනිරයට යන්නේ වේ ද … කාලසූත්‍රමහනිරයට යන්නේ වේ ද … තාපමහනිරයට යන්නේ වේ ද … ප්‍රතාපමහනිරයට යන්නේ වේ ද ... සඞ්ඝාත මහනිරයට යන්නේ වේ ද … රෞරව මහනිරයට යන්නේ වේ ද … අවීචිමහනිරයට යන්නේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

2\. ස. පු: බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් විනිපාතයට යන්නේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ‘බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් විනිපාතයට යෙති’යි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ වේ ද?

ප. පටි: නැත.

ස. ඨ: ඉදින් ‘බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් විනිපාතයට යෙති’යි වදාළ සූත්‍රාන්තයෙක් නැත්තේ මැ වී නම්,

ස. පා: භවත, ‘බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් විනිපාතයට යෙති’ යි නො කියැයුතු මැ යැ.

3\. ස. පු: බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් ගර්‍භශය්‍යාවට (ගැබහොවීමට) වදනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් නිරයෙහි උපදනේ වේ ද … තිරිසන්යොන්හි උපදනේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

4\. ස. පු: බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් ගර්‍භශය්‍යාවට වදනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: බෝසත් ඍද්ධිමත් වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

5\. ස. පු: බෝසත් ඍද්ධිමත් වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: බෝසතුන් විසින් ඡන්දිද්ධිපාද වඩනලද වේ ද … වීරියිද්ධිපාද වඩනලද වේ ද … චිත්තිද්ධිපාද වඩනලද වේ ද … වීමංසිද්ධිපාද වඩනලද වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

6\. ස. පු: බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් ගර්‍භශය්‍යාවට වදනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ‘බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් ගර්‍භශය්‍යාවට වදනේ වේ’ යයි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ වේ ද?

ප. පටි: නැත.

ස. ඨ: ඉදින් ‘බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් ගර්‍භශය්‍යාවට වදනේ වේ’යයි වදාළ සූත්‍රාන්තයෙක් නැත්තේ මැ වී නම්,

ස. පා: භවත, “බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් ගර්‍භශය්‍යාවට වදනේ වේ”යයි නො කියැයුතු මැ යැ.

7\. ස. පු: බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් දුෂ්කර ක්‍රියා කෙළේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් ‘ලොකය ශාශ්වත යැ'යි හැදහී ද, ‘ලොකය අශාශ්වත යැ’යි … ‘ලොකය අන්තවත් යැ’යි … ‘ලොකය අනන්තවත් යැ'යි … ‘එ මැ ජීව යැ එ මැ ශරීර යැ’යි … ‘ජීවය අනෙකෙකැ ශරීරය අනෙකෙකැ'යි … ‘සත්ත්‍ව මරණින් මතු වේ’ යයි, ‘සත්ත්‍ව මරණින් මතු නො වේ’ යයි … ‘සත්ත්‍ව මරණින් මතු වන්නේත් නො වන්නේත් වේ'යයි ‘සත්ත්‍ව මරණින් මතු නො මැ වන්නේත් නො ද නො වන්නේත් වේ’ යයි හැදැහී ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

8\. ස. පු: බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් දුෂ්කර ක්‍රියා කෙළේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ‘බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් දුෂ්කර ක්‍රියා කෙළේ වේ’යයි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ වේ ද?

ප. පටි: නැත.

ස. ඨ: ඉදින් ‘බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් දුෂ්කර ක්‍රියා කෙළේ වේ’යයි වදාළ සූත්‍රාන්තයෙක් නැත්තේ මැ වී නම්,

ස. පා: භවත, ‘බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් දුෂ්කර ක්‍රියා කෙළේ වේ’යයි නො කියැයුතු මැ යැ.

9\. ස. පු: බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් මෙරමා නො තවනු කෙළේ ද, අන්‍යශාස්තෘවක්හු උදෙසී ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් ‘ලොකය ශාශ්වත යැ'යි ඇදැහී ද … ‘සත්ත්‍ව මරණින් මතු නො මැ වන්නේත් නො ද වන්නේත් වේ’යයි ඇදැහී ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

10\. ස. පු: බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් අන්‍ය ශාස්තෘවක්හු උදෙසී ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ‘බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් අන්‍ය ශාස්තෘවක්හු උදෙසී’යයි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ වේ ද?

ප. පටි: නැත.

ස. ඨ: ඉදින් ‘බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් අන්‍ය ශාස්තෘවක්හු උදෙසී’යයි වදාළ සූත්‍රාන්තයෙක් නැත්තේ මැ වී නම්,

ස. පා: භවත, ‘බෝසත් ඓශ්චර්‍ය්‍යකාමකාරිතා හෙතුවින් අන්‍ය ශාස්තෘවක්හු උදෙසී’යයි නො කියැයුතු මැ යැ.

ඉස්සරියකාමකාරිතා කථා නිමි.
