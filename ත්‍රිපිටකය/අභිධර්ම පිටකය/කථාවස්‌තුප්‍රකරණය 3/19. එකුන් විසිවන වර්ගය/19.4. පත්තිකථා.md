# 19.4. පත්තිකථා

**\[ප්‍රතිලාභය ප්‍රාප්ති නමු. එ අසංස්කෘත වෙතියි පූර්‍වශෛලීය නිකායවාසීන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ‘පත්ති කථා’ එයි.\]**

1\. ස. පු: ප්‍රාප්ති අසංස්කෘත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ත්‍රාණ ලෙණ සරණ පරායණ අච්‍යුත අමෘත වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

2\. ස. පු: ප්‍රාප්ති අසංස්කෘත වෙයි, නිර්‍වාණ අසංස්කෘත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ස. පු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ත්‍රාණයෝ දෙදෙනෙක් ඇද්ද, … අන්තරිකාවෝ දෙදෙනෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ නො කියැයුතු යැ ...

4\. ස. පු: චීවරයාගේ ප්‍රාප්තිය අසංස්කෘත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: නිර්‍වාණය ... අමෘත වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

5\. ස. පු: චීවර ප්‍රාප්තිය අසංස්කෘත වෙයි, නිර්‍වාණය අසංස්කෘත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

6\. ස. පු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ත්‍රාණයෝ දෙදෙනෙක් ඇද්ද, … අන්තරිකාවෝ දෙදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

7\. ස. පු: පිණ්ඩපාත … ශයනාසන … ග්ලානප්‍රත්‍යය භෛෂජ්‍යපරිෂ්කාර ප්‍රාප්ති අසංස්කෘත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: නිර්‍වාණය … අච්‍යුත්‍යය අමෘතය වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

8\. ස. පු: ග්ලානප්‍රත්‍යය භෛෂජ්‍යපරිෂ්කාර ප්‍රාප්ති අසංස්කෘත වෙයි. නිර්‍වාණය අසංස්කෘත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

9\. ස. පු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ත්‍රාණයෝ දෙදෙනෙක් … අන්තරිකාවෝ දෙදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

10\. ස. පු: චීවරප්‍රාප්තිය අසංස්කෘත වෙයි … පිණ්ඩපාත … ශයනාසන … ග්ලානප්‍රත්‍යය භෛෂජ්‍යපරිෂ්කාර ප්‍රාප්තිය අසංස්කෘත වෙයි. නිර්‍වාණය අසංස්කෘත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අසංස්කෘතයෝ පස් දෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

11\. ස. පු: අසංස්කෘතයෝ පස් දෙනෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ත්‍රාණයෝ පස්දෙනෙක් ඇද්ද, … අන්තරිකාවෝ පස් දෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

12\. ස. පු: ප්‍රථමධ්‍යානයාගේ ප්‍රාප්තිය අසංස්කෘත වේ ද, ද්විතීයධ්‍යානයාගේ … තෘතීයධ්‍යානයාගේ චතුත්ර්‍ථධ්‍යානයාගේ … ආකාසානඤ්චායතනයාගේ … විඤ්ඤාණඤ්චායතනයාගේ … ආකිඤ්චඤ්ඤායතනයාගේ … නෙවසඤ්ඤානාසඤ්ඤායතනයාගේ … ශ්‍රොතාපත්තිමාර්‍ගයාගේ … ශ්‍රොතාපත්තිඵලයාගේ … සකෘදාගාමීමාර්‍ගයාගේ … සකෘදාගාමීඵලයාගේ ... අනාගාමීමාර්‍ගයාගේ … අනාගාමීඵලයාගේ ... අර්‍හන්මාර්‍ගයාගේ … අර්‍හත්ඵලයාගේ ප්‍රාප්තිය අසංස්කෘත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: නිර්‍වාණය … අච්‍යුත අමෘත වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

13\. ස. පු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ත්‍රාණයෝ දෙදෙනෙක් … අන්තරිකාවෝ දෙදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

14\. ස. පු: ශ්‍රොතාපත්තිමාර්‍ගප්‍රාප්තිය අසංස්කෘත වේ ද, ශ්‍රොතාපත්තිඵලප්‍රාප්තිය අසංස්කෘත වේ ද … අර්‍හත්ත්වඵල අසංස්කෘත වේ ද, නිර්‍වාණය අසංස්කෘත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අසංස්කෘතයෝ නව දෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

15\. ස. පු: අසංස්කෘතයෝ නව දෙනෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ත්‍රාණයෝ නවදෙනෙක් … අන්තරිකාවෝ නව දෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

16\. ප. පු: ‘ප්‍රාප්ති අසංස්කෘත වෙති’ යි නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: ප්‍රාප්තිය රූප, වෙදනා, සංඥා, සංස්කාර, විඥාන වේ ද?

ස. පු: එසේ නො කියැයුතු යැ …

ප. පා: එකරුණින් ‘ප්‍රාප්තිය අසංස්කෘත’ වේ.

ප්‍රාප්ති කථා නිමි.
