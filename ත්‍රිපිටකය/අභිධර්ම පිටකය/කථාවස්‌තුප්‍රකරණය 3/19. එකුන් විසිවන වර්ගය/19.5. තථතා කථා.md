# 19.5. තථතා කථා

**\[රූපාදි හැම ධර්‍මයන්ගේ රූපාදිභාව සඞ්ඛ්‍යාත තථතාවක් ඇද්ද, එ ද සංස්කෘත රූපාදින්හි අපර්‍ය්‍යාපන්න බැවින් අසංස්කෘත යයි සමහර උත්තරාපථකයන් සෙයින් ලබ්ධි ගතුවන්ගේ වාද බිඳුනට ‘තථතාකථා’ එයි.\]**

1\. ස. පු: හැම ධර්‍මයන්ගේ තථතා අසංස්කෘත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: නිර්‍වාණය ත්‍රාණය ලෙණය සරණය පරායණය අච්‍යුතය අමෘතය වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

2\. ස. පු: හැම දහමුන්ගේ තථතා අසංස්කෘත වේ ද? නිර්‍වාණය අසංස්කෘත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ස. පු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ත්‍රාණයෝ දෙදෙනෙක් … අන්තරිකාවෝ දෙදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

4\. ස. පු: රූපයාගේ රූපත්වය ඇත, රූපත්වය අසංස්කෘත නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: නිර්‍වාණය … අමෘතය වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

5\. ස. පු: රූපයාගේ රූපත්වය ඇත, රූපත්වය අසංස්කෘත නො? නිර්‍වාණය අසංස්කෘත නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

6\. ස. පු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ත්‍රාණයෝ දෙදෙනෙක් … අන්තරිකාවෝ දෙදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

7\. ස. පු: වෙදනාවගේ වෙදනාත්වය ඇත, වෙදනාත්වය … සංඥාවගේ සංඥාත්වය ඇත, සංඥාත්වය … සංස්කාරයන්ගේ සංස්කාරත්වය ඇත, සංස්කාරත්වය … විඥානයන්ගේ විඥානත්වය ඇත, විඥානත්වය අසංස්කෘත නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: නිර්‍වාණය … අමෘත්‍ය වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ......

8\. ස. පු: රූපයාගේ රූපත්වය ඇත, රූපත්වය … විඥානයාගේ විඥානත්වය ඇත, විඥානත්වය … අසංස්කෘත නො? නිර්‍වාණය අසංස්කෘත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අසංස්කෘතයෝ සදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

9\. ස. පු: අසංස්කෘතයෝ සදෙනෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ත්‍රාණයෝ සදෙනෙක් … අන්තරිකාවෝ සදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

10\. ප. පු: ‘සර්‍වධර්‍මයන්ගේ තථතාව අසංස්කෘත වෙති’ යි නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: සර්‍වධර්‍මයන්ගේ තථතාව රූප, වෙදනා, සංඥා, සංස්කාර, විඥාන වේ ද?

ස. පටි: එසේ නො කියැයුතු යැ …

ප. පා: එකරුණින් ‘සර්‍වධර්‍මයන්ගේ තථතාව අසංස්කෘත වේ'

තථතා කථා නිමි.
