# 19.6. කුසල කථා

**\[අනවද්‍ය ද කුශල යැ, ඉෂ්ටවිපාකපුණ්‍යය ද කුශල යැ, එහි ‘අනවද්‍ය’ යනු ක්ලෙශවිප්‍රයුක්ත යන අර්‍ථ යි. මේ අර්‍ථය අකුසල් තබා හැම දහම් ඇසුරු කෙරෙයි. ‘ඉෂ්ටවිපාක’ යනු ඉෂ්ටඵල නිෂ්පාදක පුණ්‍යය යන අර්‍ථ යි. මේ අර්‍ථය කුශලත්‍රිකයෙහි එන ආදිපදය මැ ඇසුරු කෙරෙයි. මේ මාර්‍ගය නො ගෙන අනවද්‍යභාව මාත්‍රයෙක් මැ ‘නිවන කුශලැ’ යි ලබ්ධි ගත්තෝ ඇත. අන්‍ධකනිකාය වාසීන් සෙයිනි. උනට ඉෂ්ටවිපාකාර්‍ථයෙන් නිර්‍වාණයාගේ කුශලභාව නැති සේ දක්වන්නට ‘කුශලකථා’ එයි.\]**

1\. ස. පු: නිර්‍වාණධාතු කුශල වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සාලම්බන වේ ද, එයට ආවර්‍තනා … ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: අනාලම්බන නො එයට ආවර්‍තනා … ප්‍රණිධි නැත් නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින්, අනාලම්බන වේ නම්, එයට ආවර්‍තනා … ප්‍රණිධි නැත් නම්,

ස. පා: භවත ‘නිර්‍වාණධාතු කුසල වෙති’යි නො කියැයුතු මැ යැ.

2\. ස. පු: අලොභය කුශල වේ ද, සාලම්බන වේ ද, එයට ආවර්‍තනා … ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: නිර්‍වාණධාතු කුශල වේ ද, සාලම්බන වේ ද, එයට ආවර්‍තනා … ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ස. පු: අද්වෙෂය කුශල … අමොහය කුශල … ශ්‍රද්ධා … වීර්‍ය්‍ය … ස්මෘති … සමාධි … ප්‍රඥා කුශල වේ ද, සාලම්බන වේ ද, එයට ආවර්‍තනා … ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: නිර්‍වාණධාතු කුශල වේ ද, සාලම්බන වේ ද, එයට ආවර්‍තනා … ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

4\. ස. පු: නිර්‍වාණධාතු කුසල අනාලම්බන වේ ද? එයට ආවර්‍තනා … ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අලොභය කුශල අනාලම්බන වේ ද? එයට ආවර්‍තනා … ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

5\. ස. පු: නිර්‍වාණධාතු කුශල අනාලම්බන වේ ද? එයට ආවර්‍තනා … ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අද්වෙෂය කුශල … අමොහය කුශල … ශ්‍රද්ධා … වීර්‍ය්‍ය … ස්මෘති … සමාධි … ප්‍රඥා කුශල අනාලම්බන වේ ද? එයට ආවර්‍තනා … ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

6\. ප. පු: ‘නිර්‍වාණධාතු කුශල වෙති’ යි නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: නිර්‍වාණධාතු අනවද්‍ය නො?

ස. පටිඤ්‍ඤා: එසේ යැ.

ස. පටි: ඉදින් නිර්‍වාණධාතු අනවද්‍ය වී නම්,

ප. පා: භවත, එකරුණින් ‘නිර්‍වාණධාතු කුශල වෙති’ යි කියැයුතු මැ යැ.

කුශල කථා නිමි.
