# 16.5. රූපං හෙතූති කථා

**\[‘හෙතු’ යනු කුසලමූලාදි හෙතුවටත් කිසි යම් ප්‍රත්‍යයටත් නමෙකි. මේ වෙනස නො ගෙන “චත්තාරො මහා භූතා හෙතු” යන වචන මාත්‍රය නිසා අවිශෙෂයෙන් ‘රූපය හෙතු’ යයි උත්තරාපථකයන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ‘රූපං හෙතූති කථා’ එයි.\]**

1\. ස. පු: රූපය හෙතු වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අලොභ හෙතු වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: අදොසහෙතු වේ ද? ... අමොහහෙතු ... ලොභහෙතු ... දොසහෙතු ... මොහහෙතු වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

2\. ස. පු: රූපය හෙතු වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: අනාලම්බන නො? එයට ආවර්‍තනා ... ප්‍රණිධි නැත් නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් අනාලම්බන වේ නම්, එයට ආවර්‍තනා ... ප්‍රණිධි නැත් නම්,

ස. පා: භවත ‘රූපය හෙතු වේ’ යයි නො කියැයුතු මැ යැ.

3\. ස. පු: අලොභහෙතු සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: (තොප මතයෙහි) රූපහෙතු සාලම්බන වේ. එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

4\. ස. පු: අදොසහෙතු ... අමොහහෙතු ... ලොභහෙතු ... දොසහෙතු ... මොහහෙතු සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: රූපහෙතු සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

5\. ස. පු: රූපහෙතු අනාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අලොභහෙතු අනාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

6\. ස. පු: රූපහෙතු අනාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අදොසහෙතු ... අමොහහෙතු ... ලොභහෙතු ... දොසහෙතු ... මොහහෙතු අනාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

7\. ප. පු: ‘රූපය හෙතු වේ’ යයි නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: මහාභූතයෝ, උපාදායරූපනට උපාදායහෙතු නො?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. ඨ: ඉදින් මහාභූතයෝ, උපාදායරූපනට උපාදායහෙතු වෙත් නම්,

ප. පා: භවත එකරුණින් ‘රූපය හෙතු වේ’ යයි කියැයුතු මැ යැ.

රූපං හෙතූති කථා නිමි.
