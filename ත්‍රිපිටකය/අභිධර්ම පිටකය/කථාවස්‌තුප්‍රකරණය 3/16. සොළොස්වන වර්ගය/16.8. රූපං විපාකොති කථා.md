# 16.8. රූපං විපාකොති කථා

**\[‘කර්‍මය කළ බැවින් උපන් චිත්තචෛතසිකයන් සෙයින් කර්‍මය කළ බැවින් උපන් රූපය ද විපාක වෙති’යි අන්‍ධක සමමිතිය නිකායවාසීන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ‘රූපං විපාකොති කථා’ එයි.\]**

1\. ස. පු: රූපය විපාක වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: රූපය සුඛවෙදනීය, දුඃඛවෙදනීය, අදුඃඛාසුඛවෙදනීය, සුඛවෙදනා සම්ප්‍රයුක්ත, දුඃඛවෙදනා සම්ප්‍රයුක්ත, අදුඃඛාසුඛවෙදනා සම්ප්‍රයුක්ත, ස්පර්‍ශසම්ප්‍රයුක්ත ... චිත්තසම්ප්‍රයුක්ත සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: සුඛවෙදනීය නො වූයේ ... දුඃඛවෙදනීය නො වූයේ ... අනාලම්බන වූයේ නො? එයට ආවර්‍තනා ... ප්‍රණිධි නැත් නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් සුඛවෙදනීය නො වී නම්, ... දුඃඛවෙදනීය නො වී නම් ... අනාලම්බන වී නම්, එයට ආවර්‍තනා ... ප්‍රණිධි නැත් නම්,

ස. පා: භවත, ‘රූපය විපාක වෙතී’ නො කියැයුතු මැ යැ.

2\. ස. පු: ස්පර්‍ශය විපාක වෙයි. ස්පර්‍ශය සුඛවෙදනීය ... දුඃඛවෙදනීය ... සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: (තොප මතයෙහි) රූපය විපාක වෙයි. රූපය සුඛවෙදනීය ... දුඃඛවෙදනීය ... සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ස. පු: (තොප මතයෙහි) රූපය විපාක වෙයි. රූපය සුඛවෙදනීය නො වූයේ ... දුඃඛවෙදනීය නො වූයේ ... අනාලම්බන වූයේ වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ස්පර්‍ශය විපාක වෙයි. ස්පර්‍ශය සුඛවෙදනීය නො වූයේ ... දුඃඛවෙදනීය නො වූයේ ... අනාලම්බන වූයේ වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

4\. ප. පු: ‘රූපය විපාක වෙතී’ නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: කර්‍මය කළ බැවින් උපන් චිත්තචෛතසික ධර්‍මයෝ විපාක නො?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. ඨ: ඉදින් කර්‍මය කළ බැවින් උපන් චිත්තචෛතසික ධර්‍ම විපාක වී නම්,

ප. පා: භවත, එකරුණින් ‘කර්‍මය කළ බැවින් උපන් රූපය විපාක වෙතී’ කියැයුතු මැ යැ.

රූපං විපාකොති කථා නිමි.
