# 16.7. රූපං කුසලාකුසලන්ති කථා

**\[“කායකම්මං වචීකම්මං කුසලස්ස අකුසලස්ස - කායකර්‍ම වාක්කර්‍ම කුසලත් අකුසලත් වෙයි” යන වචන නිසා ‘කායවාක්කර්‍ම සඞ්ඛ්‍යාත කායවිඥප්ති වාග්විඥප්ති රූප කුසලත් අකුසලත් වෙතී’ මහීශාසක සම්මිතිය නිකායවාසීන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ‘රූපං කුසලාකුසලන්ති කථා’ එයි.\]**

1\. ස. පු: රූපය කුශල වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: අනාලම්බන නො? එයට ආවර්‍තනා ... ප්‍රණිධි නැත් නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් අනාලම්බන වී නම්, එයට ආවර්‍තනා ... ප්‍රණිධි නැත් නම්,

ස. පා: භවත ‘රූපය කුශල වෙතී’ නො කියැයුතු මැ යැ.

2\. ස. පු: අලොභය කුශල සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: රූපය කුශල සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ස. පු: අද්වෙෂය කුශල ... අමොහය කුශල ... ශ්‍රද්ධාව කුශල ... වීර්‍ය්‍යය - ස්මෘතිය - සමාධි - ප්‍රඥාව - කුශල සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: රූපය කුශල සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

4\. ස. පු: රූපය කුශල අනාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අලොභය කුශල අනාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

5\. ස. පු: රූපය කුශල අනාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අද්වෙෂය කුශල ... ප්‍රඥාව කුශල අනාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

6\. ස. පු: රූපය අකුශල වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: අනාලම්බන නො, එයට ආවර්‍තනා ... ප්‍රණිධි නැත් නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් අනාලම්බන වී නම්, එයට ආවර්‍තනා ... ප්‍රණිධි නැත් නම්,

ස. පා: භවත, ‘රූපය අකුශල වෙති’ නො කියැයුතු මැ යැ.

7\. ස. පු: ලොභය අකුශල සාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි, ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: රූපය අකුශල සාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

8\. ස. පු: ද්වෙෂය අකුශලය ... මොහය ... මානය ... අනපත්‍රපාව අකුශල, සාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: රූපය අකුශල, සාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

9\. ස. පු: රූපය අකුශල, අනාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ලොභය අකුශල, අනාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

10\. ස. පු: රූපය අකුශල, අනාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ද්වෙෂය ... මොහය ... අන්පත්‍රපාව අකුශල, අනාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

11\. ප. පු: ‘රූපය කුශලත් අකුශලත් වෙතී’ නො කියයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: කායකර්‍ම වාක්කර්‍ම කුශලත් අකුශලත් වෙයි නො?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. ඨ: ඉදින් කායකර්‍ම වාක්කර්‍ම කුශලත් අකුශලත් වී නම්,

ප. පා: භවත, එකරුණින් ‘රූපය කුශලත් අකුශලත් වෙතී’ කියැයුතු මැ යැ.

රූපං කුසලාකුසලන්ති කථා නිමි.
