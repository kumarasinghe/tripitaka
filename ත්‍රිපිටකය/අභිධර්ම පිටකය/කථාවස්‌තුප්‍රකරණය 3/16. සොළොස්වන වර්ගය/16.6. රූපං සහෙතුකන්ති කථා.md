# 16.6. රූපං සහෙතුකන්ති කථා

**\[මෙහි නිදානය ද ‘රූපංහෙතූති’ කථායෙහි කී පරිදි දත යුතු.\]**

1\. ස. පු: රූපය සහෙතුක වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අලොභහෙතුයෙන් වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: අදොසහෙතුයෙන් ... අමොහහෙතුයෙන් ... ලොභහෙතුයෙන් දොසහෙතුයෙන් ... මොහහෙතුයෙන් වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

2\. ස. පු: රූපය සහෙතුක වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: අනාලම්බන නො? එයට ආවර්‍තනා ... ප්‍රණිධි නැත් නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් අනාලම්බන වේ නම්, එයට ආවර්‍තනා ... ප්‍රණිධි නැත් නම්,

ස. පා: භවත, ‘රූපය සහෙතුක වෙති’ නො කියැයුතු මැ යැ.

3\. ස. පු: අලොභය සහෙතුක සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: රූපය සහෙතුක සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

4\. ස. පු: අදොසය සහෙතුක වේ ද? ... අමොහය ... ශ්‍රද්ධාව ... වීර්‍ය්‍යය ... ස්මෘතිය ... සමාධිය ... ප්‍රඥාව ... ලොභය ද්වෙෂය ... මොහය ... මානය ... දෘෂ්ටිය ... විචිකිත්සාව ... සත්‍යානය ... ඖද්ධත්‍යය ... අහ්‍රීකය ... අනපත්‍රපාව සහෙතුක සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: රූපය සහෙතුක සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

5\. ස. පු: රූපය සහෙතුක අනාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අලොභය සහෙතුක අනාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

6\. ස. පු: රූපය සහෙතුක අනාලම්බන වූයේ, එයට ආවර්‍තනා ... ප්‍රණිධි නැත්තේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අද්වෙෂය සහෙතුක ... අනපත්‍රතාව සහෙතුක ... අනාලම්බන වූයේ, එයට ආවර්‍තනා ... ප්‍රණිධි නැත්තේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

7\. ප. පු: ‘රූපය සහෙතුක වෙති’ නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: රූපය සප්‍රත්‍යය නො?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. ඨ: ඉදින් රූපය සප්‍රත්‍යය වේ නම්,

ප. පා: භවත, එකරුණින් රූපය ‘සහෙතු වෙතී’ කියැයුතු මැ යැ.

රූපං සහෙතූකන්තී කථා නිමි.
