# 11.5. ඉද්ධිබලකථා

**\[ඍද්ධිපාදභාවනානෘශංසයාගේ අර්‍ත‍ථය නුනුවණින් ගෙන ඍද්ධිබලයෙන් සමන්වාගත වූයේ කපක් සිටුනේ යැ යි මහාසඞ්ඝික නිකාය වාසීන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ඉද්ධිබලකථා එයි.\]**

1\. ස. පු: ඍද්ධිබලයෙන් සමන්වාගත වූයේ (මහ) කපක් සිටුනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ඒ ආයු ඍද්ධිමය වේ ද, ඒ ගති ඍද්ධිමය වේ ද, ඒ ආත්මභාවප්‍රතිලාභ ඍද්ධිමය වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

2\. ස. පු: ඍද්ධිබලයෙන් සමන්වාගත වූයේ කපක් සිටුනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අතීත කල්පයක් සිටුනේ ද, අනාගත කල්පයක් සිටුනේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ස. පු: ඍද්ධිබලයෙන් සමන්වාගත වූයේ කපක් සිටුනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: දෙකපක් සිටුනේ වේ ද, තුන්කපක් සිටුනේ වේ ද, සතරකපක් සිටුනේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

4\. ස. පු: ඍද්ධිබලයෙන් සමන්වාගත වූයේ කපක් සිටුනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ජීවිතය ඇති කල්හි ජීවිතාවශේෂයෙහි සිටුනේ ද, ජීවිතය නැති කල්හි ජීවිතාවශේෂයෙහි සිටුනේ ද?

ප. පටිඤ්‍ඤා: ඉදින් දිවි ඇති කල්හි ජීවිතාවශේෂයෙහි සිටුනේ නම්,

අනු. පා: භවත ‘ඍද්ධිබලයෙන් සමන්වාගත වූයේ කපක් සිටුනේ යැ’ යි නො කියැයුතු මැ යැ.

දිවි නැති කල්හි ජීවිතාවශේෂයෙහි සිටුනේ නම්, මළේ සිටුනේ යැ, කලුරිය කෙළේ සිටුනේ යි.

5\. ස. පු: ඍද්ධිබලයෙන් සමන්වාගත වූයේ කපක් සිටුනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: උපන් ස්‍පර්‍ශය ‘නහමක් නිරුද්ධ වේව’ යි ඍද්ධියෙන් ප්‍රග්‍රහ කරන්නට (අල්ලා නවතා ගන්නට) හැකි වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: උපන් වෙදනා ... උපන් සංඥා ... උපන් චෙතනා ... උපන් සිත ... උපන් ශ්‍රද්ධා ... උපන් වීර්‍ය්‍යය ... උපන් ස්‍මෘතිය ... උපන් සමාධිය ... උපන් ප්‍රඥාව නහමක් නිරුද්ධ වේ ව යි ඍද්ධියෙන් ප්‍රග්‍රහ කරන්නට (අල්ලාගන්නට) ලැබේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

6\. ස. පු: ඍද්ධිබලයෙන් සමන්වාගත වූයේ කපක් සිටුනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ‘රූපය නිත්‍ය වේව’ යි ඍද්ධියෙන් ප්‍රග්‍රහ කරන්නට (අනිත්‍යබව නවතා ලන්නට) ලැබේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: ‘වෙදනා ... සංඥා ... සංස්කාරයෝ ... විඥානය නිත්‍ය වේව’ යි ඍද්ධියෙන් ප්‍රග්‍රහ කරන්නට ලැබේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

7\. ස. පු: ඍද්ධිබලයෙන් සමන්වාගත වූයේ කපක් සිටුනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ‘ජාතිස්වභාවය ඇති සත්ත්‍වයෝ නහමක් උපදිත්ව’ යි ඍද්ධිබලයෙන් ප්‍රග්‍රහ කරන්නට ලැබේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: ‘ජරාස්වභාව ඇති සත්ත්‍වයෝ නහමක් දිරත්ව’ යි ... ‘ව්‍යාධිස්වභාව ඇති සත්ත්‍වයෝ නහමක් ව්‍යාධිත වෙත්වයි ... මරණස්වභාව ඇති සත්ත්‍වයෝ නහමක් මියෙත්ව’යි ඍද්ධිබලයෙන් ප්‍රග්‍රහ කරන්නට ලැබේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

8\. ප. පු: ‘ඍද්ධිබලයෙන් සමන්වාගත වූයේ කපක් සිටුනේ යැ’ යි නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: භාග්‍යවතුන් වහන්සේ විසින් වදාරන ලද මැ නු:

“ආනන්දය, යම් කිසිවක්හු විසින් සතර ඍද්ධිපාදයෝ භාවිත වෙත් ද, බහුලීකෘත වෙත් ද, යෙදූ යානයක් මෙන් කරන ලද වෙත් ද, ප්‍රතිෂ්ඨා කරන ලද වෙත් ද, අධිෂ්ඨිත වෙත් ද, පරිචිත වෙත් ද, මනාකොට ප්‍රගුණ කරන ලද වෙත් ද, හේ කැමැතියේ කල්පයක් හෝ කල්පාවශෙෂයක් හෝ සිටුනේ යැ” යි වදාළ සුත්‍රාන්තයක් ඇත්තේ මැ නු?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. ආ: එකරුණින් ඍද්ධිබලයෙන් සමන්වාගත වූයේ කපක් සිටුනේ යැ.

9\. ස. පු: ඍද්ධිබලයෙන් සමන්වාගත වූයේ කපක් සිටුනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: භාග්‍යවතුන් වහන්සේ විසින් වදාරන ලද මැ නු:

“මහණෙනි, සතරධර්‍ම කෙනකුන්ට-ශ්‍රමණ වේවයි බ්‍රාහ්මණ වේවයි දෙව වේවයි මාර වේවයි බ්‍රහ්ම වේවයි ලෝකයෙහි අනෙකෙක් වේවයි ඇපවන කිසිවෙක් නැත. කවර සතර දෙනකුනට යත්: ‘ජරාස්වභාව ඇතියේ නහමක් දිරාව’ යි කියා මහණ හෝ බමුණු හෝ දෙවි හෝ මර හෝ බඹ හෝ ලොව අන් කිසිවෙක් හෝ ඇපවන කිසිවෙක් නැත. ‘ව්‍යාධිස්වභාව ඇතියේ නහමක් ව්‍යාධිත වේව’ යි ... ‘මරණස්වභාව ඇතියේ නහමක් මියේව’ යි ... සංක්‍ලේශයට හිත වූ පුනර්‍භවයට වැටෙන නිස්සාර දුඃඛවිපාක ඇති අනායෙහි ජාති ජරා මරණයට හිත පෙර කළ යම් පාපකර්‍මයෝ ඇද්ද, උන්ගේ විපාක නහමක් උපදීව’යි කියා ශ්‍රමණ හෝ බ්‍රාහ්මණ හෝ දේව හෝ මාර හෝ බ්‍රහ්ම හෝ ලෝකයෙහි අන් කිසිවෙක් හෝ ඇපවන කිසිවෙක් නැත. මහණෙනි, මේ සතර ධර්‍මයනට ඇපවන ශ්‍රමණ හෝ යැ බ්‍රාහ්මණ හෝ යැ දේව හෝ යැ මාර හෝ යැ බ්‍රහ්ම හෝ යැ ලෝකයෙහි අන් කිසිවෙක් හෝ නැති” යි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ නු?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ආ: එකරුණින් ඍද්ධිබලයෙන් සමන්වාගත වූයේ කපක් සිටුනේ යයි නො කියැයුතු යැ.

ඍද්ධිබලකථා නිමි.
