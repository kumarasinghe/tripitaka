# 11.8. අනිච්චතාකථා

**\[අනිත්‍ය වූ රූපාදීන්ගේ අනිත්‍යතා ද රූපාදීන් සෙයින් පරිනිෂ්පන්නය’ යි අන්‍ධකනිකායවාසීන් සෙයින් ලබ්ධි ගත් අන්‍යවාදීන්ගේ වාද බිඳුනට ‘අනිච්චතාකථා’ එයි.\]**

1\. ස. පු: අනිත්‍යතාව පරිනිෂ්පන්න (හෙතුසමුප්පන්න) වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ඒ අනිත්‍යතාවට තදන්‍ය පරිනිෂ්පන්න අනිත්‍යතා ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

2\. ස. පු: ඒ අනිත්‍යතාවට තදන්‍ය අනිත්‍යතාවට ද පරිනිෂ්පන්න අනිත්‍යතා ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ඒ ඒ (පරිනිෂ්පන්න) අනිත්‍යතායෙන් මැ දුඃඛයාගේ අන්තකිරියායෙක් නැද්ද, වට සිඳීමෙක් නැද්ද, අනුපාදාපරිනිර්‍වාණයෙක් නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ස. පු: ජරාව පරිනිෂ්පන්න වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ඒ ජරාවට තදන්‍ය පරිනිෂ්පන්න ජරා ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

4\. ස. පු: ඒ ජරාවට තදන්‍ය පරිනිෂ්පන්න ජරා ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: එයින් එයින් මැ දුඃඛයාගේ අන්තකිරියායෙක් නැද්ද, වට සිඳීමෙක් නැද්ද, අනුපාදා පරිනිර්‍වාණයෙක් නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ

5\. ස. පු: මරණය පරිනිෂ්පන්න වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ඒ මරණයට තදන්‍ය පරිනිෂ්පන්න මරණ ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

6\. ස. පු: ඒ මරණයට තදන්‍ය පරිනිෂ්පන්න මරණ ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: එයින් එයින් මැ දුඃඛයාගේ අන්තකිරියායෙක් නැද්ද වට සිඳීමෙක් නැද්ද, අනුපාදා පරිනිර්‍වාණයෙක් නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

7\. ස. පු: රූපය පරිනිෂ්පන්න වෙයි. රූපයට අනිත්‍යතා ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අනිත්‍යතා පරිනිෂ්පන්න වෙයි. අනිත්‍යතාවට අනිත්‍යතා ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

8\. ස. පු: රූපය පරිනිෂ්පන්න යැ. රූපයට ජරා ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ජරාව පරිනිෂ්පන්න වෙයි. ජරාවට ජරා ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

9\. ස. පු: රූපය පරිනිෂ්පන්න වෙයි. රූපයාගේ බිඳීමෙක් ඇද්ද, අතුරුදහනෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: මරණය පරිනිෂ්පන්න වෙයි. මරණයට බිඳීමෙක් ඇද්ද, අතුරුදහනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

10\. ස. පු: වෙදනා ... සංඥා ... සංස්කාර ... විඥාන පරිනිෂ්පන්න වෙයි. විඥානයට අනිත්‍යතා ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අනිත්‍යතාව පරිනිෂ්පන්න වෙයි. අනිත්‍යතාවට අනිත්‍යතා ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

11\. ස. පු: විඥානය පරිනිෂ්පන්න යැ. විඥානයට ජරා ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ජරාව පරිනිෂ්පන්න යැ. ජරාවට ජරා ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

12\. ස. පු: විඥානය පරිනිෂ්පන්න යැ. විඥානයාගේ බිඳීමෙක් ඇද්ද? අතුරුදහනෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: මරණය පරිනිෂ්පන්න වෙයි. මරණයට බිඳීමෙක් ඇද්ද, අතුරුදහනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...
