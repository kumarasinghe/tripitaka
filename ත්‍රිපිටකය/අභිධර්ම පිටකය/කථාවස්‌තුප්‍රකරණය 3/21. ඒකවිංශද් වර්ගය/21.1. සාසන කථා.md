# 21.1. සාසන කථා

**\[‘සඞ්ගීතීත්‍රය සඳහා ශාසනය නවීකෘත යැ යි ද, තථාගත ශාසනය අළුත් කරන කිසිවෙක් ඇතැ යි ද, තථාගත ශාසනය අළුත් කරන්නට ලැබේ යැ යි ද සමහර උත්තරාපථකයන් සෙයින් ලබ්ධි ගත්තාහු ඇත. උන්ගේ වාද බිඳුනට ‘සාසනකථා’ එයි.\]**

1\. ස. පු: ශාසනය නවීකෘත (අළුත් කරන ලද) වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සතිපට්ඨානය නවීකෘත වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ

2\. ස. පු: ශාසනය නවීකෘත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්‍යක් ප්‍රධානය … ඍද්ධිපාද … ඉන්ද්‍රිය … බල … බොධ්‍යඞ්ගයෝ නවීකෘත වෙත් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: පළමු අකුශලය පසු වැ කුශල කරනලද ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: පළමු සාස්‍රවය … සංයොජනියය ... ග්‍රන්‍ථනීයය ... ඔඝනීයය ... යොගනීයය ... නීවරණියය … පරාමෘෂ්ටය ... උපාදානියය ... සාංක්ලෙශිකය පසුව අසාංක්ලෙශික කරනලද වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ස. පු: තථාගතයන්ගේ සස්න අළුත් කරන කිසිවෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සතිපට්ඨාන අළුත් කරන කිසිවෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ.

ස. අනු: සම්‍යක්ප්‍රධාන … ඍද්ධිපාද … ඉන්ද්‍රිය … බල … බොධ්‍යඞ්ග අළුත් කරන කිසිවෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: පළමු අකුශලය පසුව කුශල කරන කිසිවෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: පළමු සාස්‍රවය … සාංක්ලෙශිකය පසු වැ අසාංක්ලෙශික කරන කිසිවෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

4\. ස. පු: තථාගතයන්ගේ සස්න යළි අළුත් කරන්නට ලැබේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සතිපට්ඨාන යළි අළුත් කරන්නට ලැබෙද් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: සම්‍යක්ප්‍රධාන … ඍද්ධිපාද … ඉන්ද්‍රිය … බල … බොධ්‍යඞ්ගයෝ යළි අළුත් කරන්නට ලැබෙද් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: අකුශලය පසු වැ කුශල කරන්නට ලැබේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: පළමු සාස්‍රවය … සාංක්ලෙශිකය පසු වැ අසාංක්ලෙශික කරන්නට ලැබේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

සාසන කථා නිමි.
