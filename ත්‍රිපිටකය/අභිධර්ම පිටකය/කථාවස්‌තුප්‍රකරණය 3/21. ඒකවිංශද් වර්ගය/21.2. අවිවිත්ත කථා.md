# 21.2. අවිවිත්ත කථා

**\[යම් පුද්ගලයක්හට යම් ධර්‍මයෙක් ප්‍රත්‍යුත්පන්න වේ ද, හේ ඒ ධර්‍මයෙන් පමණක් අවිවික්ත (යුක්ත) නම් වෙයි යනු ස්වසමයෙහි නිශ්චය වෙයි. පෘථග්ජන පුද්ගල විසින් ත්‍රෛධාතුකධර්‍ම අපරිඥාත (පිරිසිඳ නොදත්) හෙයින් එකක්‍ෂණයෙහි මැ හැම ත්‍රෛධාතුකධර්‍මයන්ගෙන් අවිවික්තයයි උත්තරාපථකයන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ‘අවිවිත්ත කථා’ එයි.\]**

1\. ස. පු: පෘථග්ජන පුද්ගල ත්‍රෛධාතුක ධර්‍මයෙන් අවිවික්ත (යුක්ත) වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: පෘථග්ජන පුද්ගල ත්‍රෛධාතුක ස්පර්‍ශයෙන් … ත්‍රෛධාතුක වෙදනායෙන් … සංඥායෙන් … චෙතනායෙන් … චිත්තයෙන් … ශ්‍රද්ධායෙන් … වීර්‍ය්‍යයෙන් … ස්මෘතියෙන් ... සමාධියෙන් …ත්‍රෛධාතුක ප්‍රඥායෙන් (එක්වර) අවිවික්ත වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

2\. ස. පු: පෘථග්ජන පුද්ගල ත්‍රෛධාතුකධර්‍මයෙන් අවිවික්ත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: යම් කලෙක පෘථග්ජන පුද්ගල සිවුරු දේ ද, එකල්හි ප්‍රථමධ්‍යානය උපයා වාස කෙරේ ද … ආකාසානඤ්චායතනය උපයා වාස කෙරේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: යම් කලෙකැ පෘථග්ජන පුද්ගල පිඬුවා දේ ද … සෙනසුන් දේ ද … ගිලන්පස බෙහෙසත් පිරිකර දේ ද, එකල්හි චතුර්‍ථධ්‍යාන උපයා වාස කෙරේ ද … නෙවසඤ්ඤා නාසඤ්ඤායතන උපයා වාස කෙරේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ප. පු: පෘථග්ජන පුද්ගල ත්‍රෛධාතුකධර්‍මයෙන් අවිවික්ත වෙති’ යි නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: පෘථග්ජන පුද්ගල විසින් රූපධාතුඅරූපධාතුග කර්‍මය පරිඥාත වේ ද?

ස. පටි: එසේ නො කියැයුතු යැ ...

ප. පා: එකරුණින් “පෘථග්ජන පුද්ගල ත්‍රෛධාතුක ධර්‍මයෙන් අවිවික්ත” වෙයි.

අවිවික්ත කථා නිමි.
