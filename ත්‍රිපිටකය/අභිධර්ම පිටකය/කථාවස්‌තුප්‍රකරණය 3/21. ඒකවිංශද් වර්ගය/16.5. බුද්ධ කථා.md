# 16.5. බුද්ධ කථා

**\[ඒ ඒ කාලයෙහි ශරීර ආයු ප්‍රභා ප්‍රමාණ නානාත්වය මිස සෙසු බුදුදහමින් බුදුවරුන්ගේ බුදුවරුන් කෙරෙන් හීනාතිරෙක බවෙක් නම් නැත. බුදුවරුන් කෙරෙහි අවශෙෂයෙන් හීනාතිරෙකතා ඇතැයි අන්‍ධක නිකායවාසීන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ‘බුද්ධ කථා’ එයි.\]**

1\. ස. පු: බුදුවරුන්ගේ බුදුවරුන් කෙරෙන් හීනාතිරෙකතා ඇද් ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සතිපට්ඨාන හෙයින් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: සම්‍යක්ප්‍රධාන හෙයින් වේ ද ... ඍද්ධිපාද හෙයින් ... ඉන්ද්‍රිය හෙයින් … බල හෙයින් … බොධ්‍යඞ්ග හෙයින් ... වශීභාව හෙයින් … සර්‍වඥඥානදර්‍ශන හෙයින් වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

බුද්ධ කථා නිමි.
