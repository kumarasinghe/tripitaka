# 21.4. ඉද්ධි කථා

**\[ඍද්ධිය කිසි තැනෙක සමෘද්ධ වෙයි. කිසි තැනෙකැ සමඍද්ධ නො වෙයි. අනිත්‍යාදිය නිත්‍යාදිය කිරීම එකාන්තයෙන් සිද්ධ නො වයි. සභාගසන්තති පෙරළා විසභාග සන්තති කරණය හෝ සභාගසන්තති චිරකල් පවත්වාලීම හො යම් කෙනකුන් සඳහා කරනු ලැබේ නම් ඔවුන්ගේ පුණ්‍යාදි කාරණ නිසා ‘භික්‍ෂුන් සඳහා පැන් කිරි ගී බවට පෙරැළිම සෙයින් ද, මහාධාතු නිධානයෙහි පහන් ආදිය චිරසන්තාන ප්‍රවර්‍තනය සෙයින් ද සමෘද්ධ වෙයි යනු ස්වසමයෙහි සනිටුහනි. යම් හෙයකින් ආයුෂ්මත් පිළින්දවච්ඡ තෙරණුවෝ රජුගේ ප්‍රාසාදය රන්වෙවයි ඉටුහු ද එහෙයින් ‘අධිප්පාය (අදහස ඉටුවන) ඍද්ධි ඇතැ’ යි අන්‍ධක නිකායවාසීන් සෙයින් ලබ්ධිගත්තාහු ඇත. උන්ගේ වාද බිඳුනට 'ඉද්ධි කථා' එයි.\]**

1\. ස. පු: බුදුනට හෝ ශ්‍රාවකයනට හෝ අධිප්‍රායඍද්ධි ඇද් ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ‘වෘක්‍ෂයෝ නිත්‍යපර්‍ණ ඇති වෙත්ව’ යි බුදුනට හෝ ශ්‍රාවකයනට හෝ අධිප්‍රායඍද්ධි ඇද් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: ‘වෘක්ෂයෝ නිත්‍යපුෂ්ප ඇති වෙත්ව‘යි … වෘක්ෂයෝ නිත්‍යඵල ඇති වෙත්වයි … නිත්‍යජ්‍යොත්සනා (සඳරැස්) ඇති වෙවයි … නිත්‍ය ක්‍ෂෙම ඇති වෙවයි … නිත්‍යසුභික්‍ෂ වෙවයි … නිත්‍ය ස්වස්ති වෙව’යි බුදුනට හෝ ශ්‍රාවකයනට හෝ අධිප්‍රායඍද්ධි ඇත් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

2\. ස. පු: බුදුනට හෝ ශ්‍රාවකයනට හෝ අධිප්‍රායඍද්ධි ඇද් ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ‘උපන් වෙදනා … සංඥා … චෙතනා … චිත්ත ... ශ්‍රද්ධා … වීර්‍ය්‍ය … ස්මෘති … සමාධි … ප්‍රඥා නහමක් නිරුද්ධ වෙව’යි බුදුනට හෝ ශ්‍රාවකයනට හෝ අධිප්‍රායඍද්ධි ඇද් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ස. පු: බුදුනට හෝ ශ්‍රාවකයනට හෝ අධිප්‍රායඍද්ධි ඇද් ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ‘රූපය නිත්‍ය වෙව’ යි බුදුනට හෝ ශ්‍රාවකයනට හෝ අධිප්‍රායඍද්ධි ඇද් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: වෙදනා … සංඥා … සංස්කාරයෝ … විඥානය නිත්‍ය වෙව’යි බුදුනට හෝ ශ්‍රාවකයනට හෝ අධිප්‍රායඍද්ධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

4\. ස. පු: බුදුනට හෝ ශ්‍රාවකයනට හෝ අධිප්‍රායඍද්ධි ඇද් ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ජාතිස්වභාව ඇති සත්ත්‍වයෝ නහමක් උපදිත්වයි අධිප්‍රායඍද්ධි ඇද්ද … ජරාස්වභාව ඇති සත්ත්‍වයෝ නහමක් දිරත්වයි … ව්‍යාධිස්වභාව ඇති සත්ත්‍වයෝ නහමක් ව්‍යාධි ඇති වෙත්වයි … මරණස්වභාව ඇති සත්ත්‍වයෝ නහමක් මියෙත්වයි බුදුනට හෝ ශ්‍රාවකයනට හෝ අධිප්‍රායඍද්ධි ඇද් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

5\. ප. පු: 'බුදුනට හෝ ශ්‍රාවකයනට හෝ අධිප්‍රායඍද්ධි ඇතැ'යි නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: ආයුෂ්මත් පිළින්දවච්ඡ තෙරණුවෝ මගධෙශ්වර සෙනිය බිම්බිසාර රජුගේ පහය රන් වේවයි ඉටුහු නො, පහය ස්වර්‍ණත් වී නො?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. ඨ: ඉදින් ආයුෂ්මත් පිළින්දවච්ඡ තෙරණුවෝ මගධෙශ්වර සෙනිය බිම්බිසාර රජුගේ ‘පහය රන් වේව’යි ඉටු සේක් වී නම්, පහයත් රන් වී නම්,

ප. පා: භවත, එකරුණින් බුදුනට හෝ ශ්‍රාවකයනට හෝ අධිප්‍රායඍද්ධි ඇත.

(මෙහි රජුගේ පුණ්‍යමහිමයෙන් ද පහය රන් වූ හෙයින් සාධකය නො යෙදේ.)

ඉද්ධි කථා නිමි.
