# 21.7. ධම්ම කථා

**\[රූපාදිය රූපාදි ස්වභාවයෙන් නියත වෙයි. ස්වභාවය නො පියයි. එහෙයින් ‘හැම ධර්‍මයෝ නියතයහ’ යි අන්‍ධක නිකායවාසීන් - සමහර උත්තරාපථකයන් සෙයින් ලබ්ධි ගත්තාහු ඇත. උන්ගේ වාද බිඳුනට ‘ධම්ම කථා’ එයි.\]**

1\. ස. පු: හැම දහම්හු නියත වෙත් ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: මිථ්‍යාත්‍ව වශයෙන් නියත වෙත් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: අනියත රාශියෙක් නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: අනියත රාශියෙක් ඇත් නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් අනියත රාශියෙක් ඇත් නම්,

ස. පා: භවත ‘හැම දහම්හු නියත වෙති’ යි නො කියැයුතු මැ යැ.

2\. ස. පු: හැම දහම්හු නියත වෙත් ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: භාග්‍යවතුන් වහන්සේ විසින්: මිථ්‍යාත්‍වනියතරාශි යැ සම්‍යකත්‍වනියතරාශි යැ අනියතරාශි යැ යි තුන් රාශියෙක් වදාරනලද නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් භාග්‍යවතුන් වහන්සේ විසින්: මිථ්‍යාත්‍වනියතරාශි යැ සම්‍යක්ත්‍වනියතරාශි යැ අනියතරාශි යැ යි තුන්රාශියෙක් වදාරන ලද වී නම්,

ස. පා: භවත ‘හැම ධර්‍මයෝ නියතයහ’ යි නො කියැයුතු මැ යැ.

3\. ස. පු: රූපය රූපස්වභාවයෙන් නියත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: මිථ්‍යාත්‍වනියත වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: සම්‍යක්ත්‍වනියත වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

4\. ස. පු: වෙදනා … සංඥා … සංස්කාර … විඥානය විඥාන ස්වභාවයෙන් නියත වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: මිථ්‍යාත්‍ව වශයෙන් නියත වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: සම්‍යක්ත්‍ව වශයෙන් නියත වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

5\. ප. පු: “රූපය රූප ස්වභාවයෙන් නියත වෙයි … වෙදනා සංඥා ... සංස්කාරයෝ … විඥානය විඥාන ස්වභාවයෙන් නියත වේ” යයි නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: රූප වේ ද, වෙදනා වේ ද, සංඥා වේ ද, සංස්කාර වෙත් ද, විඥාන වේ ද; වෙදනා … සංඥා … සංස්කාර … විඥාන … රූප වේ ද? වෙදනා වේ ද? සංඥා වේ ද? සංස්කාර වෙත් ද?

ස. පටි: එසේ නො කියැයුතු යැ ...

ප. පා: එකරුණින් රූපය රූපස්වභාවයෙන් නියත වෙයි. වෙදනා … සංඥා ... සංස්කාර … විඥානය විඥාන ස්වරූපයෙන් නියත වේ.

ධම්ම කථා නිමි.
