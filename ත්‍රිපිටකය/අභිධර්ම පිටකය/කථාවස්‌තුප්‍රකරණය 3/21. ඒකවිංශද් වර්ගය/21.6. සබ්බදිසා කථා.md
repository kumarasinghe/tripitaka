# 21.6. සබ්බදිසා කථා

**\[‘හැම දික්හි බුදුවරු සිටිති’ යි මහාසාංඝිකනිකායවාසීන් සෙයින් ලබ්ධි ගත්තාහු ඇත. උන්ගේ වාද බිඳුනට ‘සබ්බදිසා කථා’ එයි.\]**

1\. ස. පු: හැම දික්හි බුදුවරු සිටිත් ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: පෙරදිග බුදුහු සිටිත් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

2\. ස. පු: පෙරදිගැ බුදුහු සිටිත් ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ඒ බුදුහු කිනම්හ. කවර ජාති ඇතියහ. කවර ගොත්‍ර ඇතියහ. ඒ බුදුන්ගේ මාපියෝ කිනම්හ. ඒ බුදුන්ගේ ශ්‍රාවකයුගය කිනම් ඇතියේ යැ. ඒ බුදුන්ගේ උපස්ථායක කවරෙ යැ. ඒ බුදුහු කෙබඳු සිවුරු දරත් ද, කෙබඳු පාත්‍ර ධරත් ද, කවර ගමෙක හෝ නියම්ගමෙක හෝ නුවරෙක හෝ රටෙක හෝ ජනපදයෙක හෝ සිටිත් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: දකුණුදිගැ ... පැළදිගැ … උතුරුදිගැ … යටදිගැ බුදුහු සිටිත් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ස. පු: යට දෙස බුදුහු සිටිත් ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ඒ බුදුහු කිනම්හ … කිනම් දනවියෙකැ හෝ සිටිත් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: උඩ දෙස බුදුහු සිටිත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …

4\. ස. පු: උඩ දෙස බුදුහු සිටිත් ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සිවුමහරදයෙහි සිටිත් ද, … තව්තිසැ සිටිත් ද, ... යාම දෙව්ලොවැ සිටිත් ද, … තුසීපුර සිටිත් ද, ... නිර්‍මාණරති දෙව්ලොවැ සිටිත් ද, පරනිර්මිත වශවර්ති දෙව්ලොවැ සිටිත් ද ... බඹලොවැ සිටිත් ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

සබ්බදිසා කථා නිමි.
