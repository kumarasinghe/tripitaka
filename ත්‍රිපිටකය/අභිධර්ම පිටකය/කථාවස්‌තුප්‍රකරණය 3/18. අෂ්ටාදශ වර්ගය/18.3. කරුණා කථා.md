# 18.3. කරුණා කථා

**\[“ප්‍රිය වස්තූන්ගේ විපත්හි සරාගපුද්ගලයන්ගේ රාග වශයෙන් කරුණාප්‍රතිරූපක ප්‍රවෘත්ති දැක රාගය මැ කරුණා නම් වෙයි. එ ද භාග්‍යවතුන් වහන්සේට නැත. එයින් භාග්‍යවත් බුදුන්ට කරුණා නැතැ” යි උත්තරාපථකයන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ‘කරුණාකථා’ එයි.\]**

1\. ස. පු: භාග්‍යවත් බුදුන්ට ‘කරුණා’ නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: භාග්‍යවත් බුදුන්ට ‘මෛත්‍රී‘ නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

2\. ස. පු: භාග්‍යවත් බුදුන්ට ‘කරුණා’ නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: භාග්‍යවත් බුදුන්ට ‘මෘදුතා …උපෙක්ෂා’ නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ස. පු: භාග්‍යවත් බුදුන්ට ‘මෛත්‍රී‘ ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: භාග්‍යවත් බුදුන්ට ‘කරුණා’ ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

4\. ස. පු: භාග්‍යවත් බුදුන්ට ‘මෘදුතා … උපෙක්ෂා’ ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: භාග්‍යවත් බුදුන්ට ‘කරුණා’ ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

5\. ස. පු: භාග්‍යවත් බුදුන්ට ‘කරුණා’ නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: භාග්‍යවත්හු අකාරුණික වෙත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: භාග්‍යවත්හු කාරුණික වූවාහු ලොවට හිත වූවාහු ලොවට අනුකම්පා ඇත්තාහු ලොවට වැඩ කරන්නාහු නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් භාග්‍යවත්හු කාරුණික වූවාහු ලොවට හිත වූවාහු ලොවට අනුකම්පා ඇත්තාහු ලොවට වැඩ කරන්නාහු නම්,

ස. පා: භවත, ‘භාග්‍යවත් බුදුන්ට කරුණා නැතැ’යි නො කියැයුතු මැ යැ.

6\. ස. පු: භාග්‍යවත් බුදුන්ට ‘කරුණා’ නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: භාග්‍යවත්හු මහකුලුණු සමවත් සමවන්හු නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් භාග්‍යවත්හු මහකුලුණු සමවත් සමවන්හු නම්,

ස. පා: භවත, භාග්‍යවත් බුදුන්ට කරුණා නැතැ’යි නො කියැයුතු මැ යැ.

7\. ප. පු: භාග්‍යවත් බුදුන්ට ‘කරුණා’ ඇද්ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: බුදුහු සරාග වෙත් ද?

ස. පටි: එසේ නො කියැයුතු යැ …

ප. පා: එකරුණින් ‘භාග්‍යවත් බුදුන්ට කරුණා නැති’.

කරුණා කථා නිමි.
