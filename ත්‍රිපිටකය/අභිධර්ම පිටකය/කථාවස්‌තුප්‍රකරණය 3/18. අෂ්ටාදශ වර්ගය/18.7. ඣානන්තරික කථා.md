# 18.7. ඣානන්තරික කථා

**\[අවිතර්‍ක විචාරමාත්‍ර සමාධියට අවකාශ නො දැන “ප්‍රථමධ්‍යානයටත් ද්විතීයධ්‍යානයටත් අතුරෙහි ධ්‍යානානත්තරිකාවක් ඇතැ” යි සම්මිතිය හා සමහර අන්ධකනිකායවාසීන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ‘ඣානන්තරික කථා’ එයි.\]**

1\. ස. පු: ධ්‍යානාන්තරික (ධ්‍යානයට අතුරෙහි වූ) සමාධියෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ස්පර්‍ශාන්තරික ඇද්ද … සංඥාන්තරිකා ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

2\. ස. පු: ධ්‍යානාන්තරික සමාධියෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ද්වීතීයධ්‍යානයටත් තෘතීයධ්‍යානයටත් අතුරෙහි ධ්‍යානාන්තරික සමාධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ස. පු: ධ්‍යානාන්තරික සමාධියෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: තෘතීයධ්‍යානයටත් චතුර්‍ථධ්‍යානයටත් අතුරෙහි ධ්‍යානාන්තරික සමාධියෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ නො කියැයුතු යැ ...

4\. ස. පු: ද්වීතීයධ්‍යානයටත් තෘතීයධ්‍යානයටත් අතුරෙහි ධ්‍යානාන්තරික සමාධියෙක් නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් ද්වීතීයධ්‍යානයටත් තෘතීයධ්‍යානයටත් අතුරෙහි ධ්‍යානාන්තරික සමාධියක් නැත් නම්,

ස. පා: භවත, ධ්‍යානාන්තරික සමාධියෙක් ඇතැ’යි නො කියැයුතු මැ යැ.

5\. ස. පු: තෘතීයධ්‍යානයටත් චතුර්‍ථධ්‍යානයටත් අතුරෙහි ධ්‍යානාන්තරික සමාධියෙක් නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් තෘතීයධ්‍යානයටත් චතුර්‍ථධ්‍යානයටත් අතුරෙහි ධ්‍යානාන්තරික සමාධියක් නැත් නම්,

ස. පා: භවත, ’ධ්‍යානාන්තරික සමාධියෙක් ඇතැ’යි නො කියැයුතු මැ යැ.

6\. ස. පු: ප්‍රථමධ්‍යානයටත් ද්වීතීයධ්‍යානයටත් අතුරෙහි ධ්‍යානාන්තරික සමාධියෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ද්වීතීයධ්‍යානයටත් තෘතීයධ්‍යානයටත් අතුරෙහි ධ්‍යානාන්තරික සමාධියෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

7\. ස. පු: ප්‍රථමධ්‍යානයටත් ද්වීතීයධ්‍යානයටත් අතුරෙහි ධ්‍යානාන්තරික සමාධියෙක් ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: තෘතීයධ්‍යානයටත් චතුර්‍ථධ්‍යානයටත් අතුරෙහි ධ්‍යානාන්තරික සමාධියෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

8\. ස. පු: ද්වීතීයධ්‍යානයටත් තෘතීයධ්‍යානයටත් අතුරෙහි ධ්‍යානාන්තරික සමාධියෙක් නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

9\. ස. පු: තෘතීයධ්‍යානයටත් චතුර්‍ථධ්‍යානයටත් අතුරෙහි ධ්‍යානාන්තරික සමාධියෙක් නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ප්‍රථමධ්‍යානයටත් ද්විතීයධ්‍යානයටත් අතුරෙහි ධ්‍යානාන්තරික සමාධියෙක් නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

10\. ස. පු: අවිතර්‍ක විචාරමාත්‍ර සමාධිය ධ්‍යානාන්තරික වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සවිතර්‍ක සවිචාරසමාධිය ධ්‍යානාන්තරික වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

11\. ස. පු: අවිතර්‍ක විචාරමාත්‍ර සමාධිය ධ්‍යානාන්තරික වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අවිතර්‍ක අවිචාර සමාධිය ධ්‍යානාන්තරික වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

12\. ස. පු: සවිතර්‍ක සවිචාර සමාධිය ධ්‍යානාන්තරික නො වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අවිතර්‍ක විචාරමාත්‍ර සමාධිය ධ්‍යානාන්තරික නො වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

13\. ස. පු: අවිතර්‍ක අවිචාර සමාධිය ධ්‍යානාන්තරික නො වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අවිතර්‍ක විචාරමාත්‍ර සමාධිය ධ්‍යානාන්තරික නො වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

14\. ස. පු: ප්‍රත්‍යුත්පන්න ධ්‍යාන දෙකක්හුගේ අතුරෙහි අවිතර්‍ක විචාරමාත්‍ර සමාධියෙක් වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: අවිතර්‍ක විචාරමාත්‍ර සමාධිය පවත්නා කල්හි ප්‍රථමධ්‍යානය නිරුද්ධ වූයේ ද්විතීයධ්‍යානය ප්‍රත්‍යුත්පන්න වූයේ නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් අවිතර්‍ක විචාරමාත්‍ර සමාධිය පවත්නා කල්හි ප්‍රථමධ්‍යානය නිරුද්ධ වූයේ ද්විතීයධ්‍යානය ප්‍රත්‍යුත්පන්න වූයේ නම්,

ස. පා: භවත, ප්‍රත්‍යුත්පන්න ධ්‍යාන දෙකක්හුගේ අතුරෙහි අවිතර්‍ක විචාරමාත්‍ර සමාධිය ධ්‍යානන්තරික වෙතී’ නො කියැයුතු මැ යැ.

15\. ස. පු: අවිතර්‍ක විචාරමාත්‍ර සමාධිය ධ්‍යානාන්තරික නො වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: අවිතර්‍ක විචාරමාත්‍ර සමාධිය ප්‍රථමධ්‍යාන … ද්විතීයධ්‍යාන … තෘතීයධ්‍යාන … චතුර්‍ථධ්‍යාන වේ ද,

ස. පටි: එසේ නො කියැයුතු යැ ...

ප. අනු: එකරුණින් “අවිතර්‍කවිචාරමාත්‍රසමාධිය ධ්‍යානන්තරික” වෙයි.

16\. ස. පු: අවිතර්‍ක විචාරමාත්‍ර සමාධිය ධ්‍යානාන්තරික වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: භාග්‍යවතුන් වහන්සේ විසින් තුන් සමාධි කෙනෙක් වදාරන ලදහු මැ නු: “සවිතර්‍ක සවිචාරසමාධි යැ අවිතර්‍ක විචාරමාත්‍ර සමාධි යැ අවිතර්‍ක අවිචාරමාත්‍රසමාධි යැ” යි

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. ඨ: ඉදින් භාග්‍යවතුන් වහන්සේ විසින් “සවිතර්‍ක ... \[විචාරමාත්‍ර …\]: අවිචාරසමාධි යැ” යි තුන් සමාධි කෙනෙක් වදාරන ලද්දාහු වී නම්,

ස. පා: භවත, “අවිතර්‍ක විචාරමාත්‍ර සමාධිය ධ්‍යානාන්තරික වෙතී” නො කියැයුතු මැ යැ.

ධ්‍යානන්තරික කථා නිමි.
