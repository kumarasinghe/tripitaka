# 12.4. සළායතනකථා

**\[යම් හෙයෙකින් කර්‍මය කළ බැවින් ෂඩායතනය උපන්නේ වේ ද, එහෙයින් ‘ෂඩායතනය විපාක වේය’ යි මහාසාඞ්ඝිකයන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ‘සළායතනකථා’ එයි.\]**

1\. ස. පු: චක්‍ෂුරායතනය විපාක වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: (චක්‍ෂුරායතනය) සුඛවේදනීය දුඃඛවේදනීය ... සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: සුඛවේදනීය නො වනුයේ ... අනාලම්බන වනුයේ නො? එයට ආවර්‍තනා ... ප්‍රණිධි නැත්තේ නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

අනු. ඨ: ඉදින් සුඛවේදනීය නො වේ නම් ... අනාලම්බන වේ නම්, එයට ආවර්‍තනා ... ප්‍රණිධි නැත් නම්,

අනු. පා: භවත, ‘චක්‍ෂුරායතන විපාක වේය’ යි නො කියැයුතු මැ යැ.

2\. ස. පු: ස්‍පර්‍ශය විපාක වෙයි. ස්‍පර්‍ශය සුඛවේදනීය ... සාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: (තොප මතයෙහි) චක්‍ෂුරායතන විපාක වෙයි, චක්‍ෂුරායතනය සුඛවේදනීය වේ ද ... සාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ස. පු: චක්‍ෂුරායතනය විපාක වෙයි. චක්‍ෂුරායතනය සුඛවේදනීය නො වනුයේ ... අනාලම්බන වනුයේ වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ස්‍පර්‍ශය විපාක වෙයි. ස්‍පර්‍ශය සුඛවේදනීය නො වනුයේ වේ ද, ... අනාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

4\. ස. පු: ශ්‍රොත්‍රායතනය ... ඝ්‍රාණායතනය ... ජිහ්වායතනය ... කායායතනය විපාක වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: (කායායතනය) සුඛවේදනීය දුඃඛවේදනිය ... සාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: සුඛවේදනීය නො... අනාලම්බන නො, එයට ආවර්‍තනා ... ප්‍රණිධි නැත් නු?

ප. පටිඤ්‍ඤා: එසේ යැ.

අනු. ඨ: ඉදින් සුඛවේදනීය නො වී නම් ... අනාලම්බන වී නම්, එයට ආවර්‍තනා ... ප්‍රණිධි නැත් නම්,

අනු. පා: භවත, ‘කායායතන විපාක වේය’ යි නො කියැයුතු මැ යැ.

5\. ස. පු: ස්‍පර්‍ශය විපාක වෙයි. ස්‍පර්‍ශය සුඛවේදනීය වේ ද ... සාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: (තොප මතයෙහි) කායායතනය විපාක වෙයි, කායායතනය සුඛවේදනීය වේ ද, සාලම්බන වේ ද? එයට ආවර්‍තනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

6\. ස. පු: (තොප මතයෙහි) කායායතනය විපාක වෙයි, කායායතනය සුඛවේදනිය නො වේ ද ... අනාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ස්‍පර්‍ශය විපාක වෙයි, ස්‍පර්‍ශය සුඛවේදනිය නො වේ ද, ... අනාලම්බන වේ ද, එයට ආවර්‍තනා ... ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

7\. ප. පු: ‘සළායතනය විපාක වේය’ යි නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: සළායතනය කර්‍මය කළ බැවින් උපන්නේ නො?

ස. පටිඤ්‍ඤා: එසේ යැ.

අනු. ඨ: ඉදින් සළායතනය කර්‍මය කළ බැවින් උපන්නේ වී නම්,

අනු. පා: භවත, එකරුණින් “සළායතනය විපාක වේය” යි කියැයුතු මැ යැ. (මනායතනය විපාක වෙයි. සෙස්ස හුදෙක් කම්මසමුට්ඨානය විපාක නො වේ. එයින් අසාධකයි.

සළායතනකථා නිමි.
