# 17.3. සබ්බමිදං කම්මතොති කථා

**\[“කම්මුනා වත්තති ලොකො - ලොකය කර්‍ම හෙතුවින් වැටෙයි” යන සූත්‍ර නිසා මෙ හැම කර්‍මවර්‍ත ක්ලෙශවර්‍ත විපාකවර්‍ත කර්‍ම හෙයින් මැ වේ” යයි රාජගිරික සිදධාර්ත්‍ථික නිකායවාසීන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට ‘සබ්බමිදං කම්මතොති කථා’ එයි.\]**

1\. ස. පු: මෙ හැම කර්‍ම හෙයින් වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: කර්‍මයත් කර්‍ම හෙයින් වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

2\. ස. පු: මෙ හැම කර්‍ම හෙයින් වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: මෙ හැම පූර්‍වකෘතහෙතුයෙන් වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

3\. ස. පු: මෙ හැම කර්‍ම හෙයින් වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: මෙ හැම කර්‍මවිපාකයෙන් වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

4\. ස. පු: මෙ හැම කර්‍මවිපාක හෙයින් වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: කර්‍මවිපාකයෙන් ප්‍රාණඝාත කරන්නේ වේ ද?

ප. පටි: එසේ යැ.

ස. පු: ප්‍රාණඝාතය සඵල වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: කර්‍මවිපාක සඵල වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: කර්‍මවිපාක අඵල වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ප්‍රාණඝාතය අඵල වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

5\. ස. පු: කර්‍මවිපාක හෙයින් අයිනාදන් කරන්නේ වේ ද? මුසවා බණන්නේ වේ ද? ... පෙහෙසුන්බස් බණන්නේ වේ ද? පරොස්බස් බණන්නේ වේ ද? ... පලප්බස් බණන්නේ වේ ද? ... ගෘහසන්‍ධි බිඳුනේ වේ ද? ... ගම් වොලොගන්නේ වේ ද? ... එකාගාරික කර්‍ම කරන්නේ වේ ද? ... කඩපලාසොරකම් කරන්නේ වේ ද? ... විරඹුසොරකම් කරන්නේ වේ ද? ගම් නසනුයේ වේ ද? ... නියම්ගම නසනුයේ වේ ද? ... කර්‍මවිපාක හෙයින් දැන් දෙන්නේ වේ ද? ... සිවුරු දෙන්නේ වේ ද? ... පිඬුවා දෙන්නේ වේ ද? ... සෙනසුන් දෙන්නේ වේ ද? ගිලන්පස බෙහෙත් පිරිකර දෙන්නේ වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ගිලන්පසබෙහෙත්පිරිකර (දීම) සඵල වේ ද?

ප. පටි: එසේ යැ.

ස. අනු: කර්‍මවිපාක සඵල වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

8\. ස. පු: කර්‍මවිපාක අඵල වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: ගිලන්පසබෙහෙත්පිරිකර අඵල වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

9\. ප. පු: ’මේ හැම කර්‍ම හෙයින් වෙතී’ නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: භාග්‍යවතුන් වහන්සේ විසින් වදාරනලද මැ නු:

“ලොකය කර්‍ම හෙතුවින් පවති, ප්‍රජාව කර්‍ම හෙතුවින් පවති. යන්නා වූ රථයාගේ අකුර සෙයින් සතත්‍වයෝ කර්‍මනිබන්‍ධන කොට ඇත.

කර්‍ම හෙතුවින් කීර්තිප්‍රශංසා ලබයි. කර්‍ම හෙතුවින් ධනහානි ද වධබන්‍ධ ද වෙයි. ඒ කර්‍මනානාත්‍වය දැන කුමක් සෙයින් ලොකයා ‘කර්‍මය නැතැ’ යි කියන්නේ වේදැ?” යි වදාළ සුත්‍රාන්තයෙක් ඇත්තේ මැ නු?

ස. පටි: එසේ යැ.

ප. පු: ඒ කරුණින් ‘මෙ හැම කර්‍ම හෙතුවින් වේ.’

සබ්බමිදං කම්මතොති කථා නිමි.
