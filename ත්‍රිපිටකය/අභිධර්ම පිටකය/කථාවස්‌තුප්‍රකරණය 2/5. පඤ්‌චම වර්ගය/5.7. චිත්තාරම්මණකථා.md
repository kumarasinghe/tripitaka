# 5.7. චිත්තාරම්මණකථා

**\[‘චෙතොපරියායඤාණ යන වචනමාත්‍රය මැ ගෙන ඒ ඥානය චිත්තාරම්මණ මැ වෙයි’ යන අන්‍ධකයන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට චිත්තාරම්මණකථා එයි.\]**

1\. ස. පු: චෙතොපරියායඤාණය චිත්තාරම්මණ මැ වත් මුත් අන්‍යාලම්බන නො වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: සරාග චිත්තය ‘සරාග චිත්ත’ යයි දන්නා කිසිවෙක් ඇත්තේ නො වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

අනු. ඨ: ඉදින් සරාගචිත්තය ‘සරාගචිත්ත’ යයි දන්නා කිසිවෙක් ඇත් නම්,

අනු. පා: භවත, 'චෙතොපරියායඤාණය චිත්තාලම්බන මැ වත් මුත් අන්‍යාලම්බන නො වෙති’යි නො කියැයුතු මැ යැ.

ස. අනු: වීතරාගචිත්තය … සදොසචිත්තය … වීතදොසචිත්තය … සමොහචිත්තය … වීතමොහචිත්තය … සංක්‍ෂිප්තචිත්තය … වික්‍ෂිප්තචිත්තය … මහද්ගතචිත්තය … අමහද්ගතචිත්තය … සඋත්තරචිත්තය … අනුත්තරචිත්තය … සමාහිතචිත්තය … අසමාහිතචිත්තය … විමුක්තචිත්තය … අවිමුක්තචිත්තය … ‘අවිමුක්තචිත්ත’ යයි දන්නා කිසිවෙක් ඇත්තේ නො වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

අනු. ඨ: ඉදින් අවිමුක්තචිත්තය ‘අවිමුක්තචිත්ත’ යයි දන්නා කිසිවෙක් ඇත් නම්,

අනු. පා: භවත, 'චෙතොපරියායඤාණය චිත්තාලම්බන මැ වත් මුත් අන්‍යාලම්බන නො වෙති’යි නො කියැයුතු මැ යැ.

2\. ස. පු: ස්පර්‍ශාලම්බනඥානය ‘චෙතොපරියායඤාණ’ යයි කියැයුතු වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

අනු. ඨ: ඉදින් ස්පර්‍ශාලම්බනඥානය ‘චෙතොපරියායඤාණ’ යයි කියැයුතු වේ නම්,

අනු. පා: භවත, 'චෙතොපරියායඤාණය චිත්තාලම්බන මැ වත් මුත් අන්‍යාලම්බන නො වෙති’යි නො කියැයුතු මැ යැ.

3\. ස. පු: වෙදනාලම්බනඥානය … සංඥාලම්බනඥානය … චෙතනාලම්බනඥානය … චිත්තාලම්බනඥානය … ශ්‍රද්ධාලම්බනඥානය … වීර්‍ය්‍යාලම්බනඥානය … ස්මෘත්‍යාලම්බන ඥානය … සමාධ්‍යාලම්බනඥානය … ප්‍රඥාලම්බනඥානය … රාගාලම්බනඥානය … ද්වෙෂාලම්බනඥානය … මොහාලම්බනඥානය … අනපත්‍රපාලම්බනඥානය ‘චෙතො පරියායඤාණ’ යයි කියැයුතු වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

අනු. ඨ: ඉදින් අනපත්‍රපාලම්බනඥානය ‘චෙතොපරියායඤාණ’ යයි කියැයුතු වේ නම්,

අනු. පා: භවත, 'චෙතොපරියායඤාණය චිත්තාලම්බන වත් මුත් අන්‍යාලම්බන නො වෙති’යි නො කියැයුතු මැ යැ.

4\. ස. පු: ස්පර්‍ශාලම්බනඥානය ‘චෙතොපරියායඤාණ’ යයි නො කියැයුතු වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: (තෙල) ස්පර්‍ශපර්‍ය්‍යායඥාන වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ … (එබඳු සූත්‍රපදයක් නැතියෙන් කළ ප්‍රතික්‍ෂෙප යි.)

5\. ස. පු: වෙදනාලම්බනඥානය … සංඥාලම්බනඥානය … අනපත්‍රපාලම්බනඥානය ’චිත්තපර්‍ය්‍යායඥාන’ යයි නො කියැයුතු වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: (තෙල) අනපත්‍රපාපර්‍ය්‍යායඥාන වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

6\. ප. පු: ‘චිත්තපර්‍ය්‍යායඥානය චිත්තාලම්බන මැ වත් මුත් අන්‍යාලම්බන නොවෙති’ යි නො කියැයුතු වේ ද?

ස. පටිඤ්ඤා: එසේ යැ.

ප. අනු: (තෙල) චිත්තපර්‍ය්‍යායඥාන නො?

ස. පටිඤ්ඤා: එසේ යැ.

අනු. ඨ: ඉදින් චිත්තපර්‍ය්‍යායඥාන වේ නම්,

අනු. පා: භවත, එකරුණින් චිත්තපර්‍ය්‍යායඥාන චිත්තාලම්බන මැ වත් මුත් අන්‍යාලම්බන නො වෙති’යි කියැයුතු මැ යැ.

චිත්තාලම්බනකථා නිමි.
