# 3.3. විමුත්තිකථා

**\[පුහුන් රාග ඇති චිත්තයනට විමුක්තිප්‍රයොජන නම් නැත. මල ගත් වස්ත්‍රයෙක් සෝදනු ලබනුයේ මලයෙන් මිදෙන්නේ වේ ද, එපරිදි රාග සහිත චිත්තය රාගයෙන් මිදෙන්නේ යයි ලබ්ධි ගත් වාදීහු ඇත. අන්‍ධකනිකාය වාසීන් සෙයිනි. උන් සඳහා විමුක්තිකථා එයි.\]**

1\. ස.පු: රාග සහිත සිත මිදේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: රාග සහගත රාග සහජාත රාග සංසෘෂ්ට රාගසම්ප්‍රයුක්ත රාගසහභූ රාගානුපරිවර්තී අකුශල වූ ලෞකික වූ සාස්‍රව වූ සංයොජනයට හිත වු ග්‍රන්‍ථයෙන් ගෙතියයුතු වූ සසරවට ගැලෙන යොගයෙන් යෙදෙන නීවරණයට හිත වූ පරාමෘෂ්ට වූ උපාදානයනට හිත වූ සාංක්ලෙශික වූ සිත මිදේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

2\. ස.පු: ස්පර්‍ශ සහිත චිත්තය මිදෙයි, ස්පර්‍ශය හා චිත්තය හා දෙදෙන මිදෙත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …

3\. ස.පු: සවෙදන චිත්තය … සසංඥ චිත්තය … සචෙතන චිත්තය … සසංඥ චිත්තය මිදෙයි, සංඥාව හා චිත්තය හා දෙදෙන මිදෙත් ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: සරාග චිත්තය මිදෙයි, රාගය හා චිත්තය හා දෙදෙන මිදෙත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …

4\. ස.පු: සස්පර්‍ශ සරාග චිත්තය මිදෙයි, ස්පර්‍ශය හා චිත්තය හා දෙදෙන මිදෙත් ද?

ස. අනු: රාගය හා චිත්තය හා දෙදෙන මිදෙත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …

5\. ස. පු: සවෙදන සරාග … සසංඥ සරාග … සචෙතන සරාග … සසංඥ සරාග චිත්තය මිදෙයි, සංඥාව හා චිත්තය හා දෙදෙන මිදෙත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …

6\. ස.පු: සදොෂ චිත්තය මිදේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: දොෂ සහගත දොෂ සහජාත දොෂ සංසෘෂ්ට දොෂසහභූ දොෂානුපරිවර්තී අකුශල වූ ලෞකික වූ සාස්‍රව වූ සංයොජනයට හිත වූ ග්‍රන්‍ථයෙන් ගෙතුණු සසරවට ගැලෙන යොගයෙන් යෙදෙන නීවරණයට හිත වූ පරාමෘෂ්ට වූ උපාදානයට හිත වූ සාංක්ලෙශික වූ චිත්තය මිදේ ද?

ස. අනු: සදොෂ චිත්තය මිදෙයි, ද්වෙෂය හා චිත්තය හා දෙදෙන මිදෙත් ද?

8\. ස. පු: සවෙදන … සසංඥ … සචෙතන … සසංඥ චිත්තය මිදෙයි, සංඥාව හා චිත්තය හා දෙදෙන මිදෙත් ද?

14\. ස. පු: සස්පර්‍ශ සමොහචිත්තය මිදෙයි, ස්පර්‍ශය හා චිත්තය හා දෙදෙන මිදෙත් ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: මොහය හා චිත්තය හා දෙදෙන මිදෙත් ද?

ප. පටි: එසේ නො කියැයුතු මැ යැ …

15\. ස. පු: සවෙදන සමොහ … සසංඥ සමොහ … සචෙතන සමොහ … සසංඥ සමොහචිත්තය මිදෙයි. සංඥා හා චිත්තය හා දෙදෙන මිදෙත් ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: මොහය හා චිත්තය හා දෙදෙන මිදෙත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …

16\. ප. පු: ‘සරාග සද්වෙෂ සමොහචිත්තය මිදේ’ යයි නො කියැයුතු ද?

ස. පටිඤ්ඤා: එසේ යැ.

ප. අනු: වීතරාග වීතද්වෙෂ වීතමොහ නික්ලෙශචිත්තය මිදේ ද?

ස. පටි: එසේ නො කියැයුතු යැ …

අනු. ආ: එකරුණින් ‘සරාග සද්වෙෂ සමොහචිත්තය මිදේ’ යයි නො කියැයුතු යැ.

විමුත්තිකථා නිමි.
