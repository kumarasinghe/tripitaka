# 3.2. අරියන්තිකථා

**\[‘හුදෙක් ආස්‍රවක්‍ෂයඥානය මැ ආර්‍ය්‍ය නො වෙයි, වැළිත් කැලැ මැ කී නවබලයෝ ද ආර්‍ය්‍යයහ’යි ලබ්ධි ගත් වාදීහු ඇත. අන්‍ධකනිකායවාසීන් සෙයිනි. උන්ගේ වාද බිඳුනට ‘අරියකථා’ එයි.\]**

1\. ස. පු: ස්ථානාස්ථානයෙහි යථාභූතඥාන නම් වූ තථාගතබලය ආර්‍ය්‍ය වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: මඟ යැ, ඵල යැ, නිවනැ, සොවන්මඟ යැ, සොවත්ඵල යැ, සෙදගැමිමඟ යැ, සෙදගැමිඵල යැ, අනගැමිමඟ යැ, අනගැමිඵල යැ, රහත්මඟ යැ, රහත්ඵල යැ, සීවටන, සම්‍යක්ප්‍රධාන යැ, ඍද්ධිපාද යැ, ඉන්ද්‍රිය යැ, බල යැ, බොධ්‍යඞ්ග යැ යන මේ ආර්‍ය්‍ය වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

2\. ස. පු: ස්ථානාස්ථානයෙහි යථාභූතඥාන නම් වූ තථාගතබලය ආර්‍ය්‍ය වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ශුන්‍යතාලම්බනය ආර්‍ය්‍ය වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

3\. ස. පු: ශුන්‍යතාලම්බනය ආර්‍ය්‍ය වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ස්ථානාස්ථානත් මෙනෙහි කෙරේ ද, ශුන්‍යතාත් මෙනෙහි කෙරේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

4\. ස. පු: ස්ථානාස්ථානත් මෙනෙහි කෙරේ ද, ශුන්‍යතාත් මෙනෙහි කෙරේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ස්පර්‍ශයන් දෙදෙනකුන්ගේ චිත්තයන් දෙදෙනකුන්ගේ සමවධාන වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ.

5\. ස. පු: ස්ථානාස්ථානයෙහි යථාභූතඥාන නම් වූ තථාගතබලය ආර්‍ය්‍ය වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අනිමිත්තාලම්බනය … අප්‍රණිහිතාලම්බනය ආර්‍ය්‍ය වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

6\. ස. පු: අප්‍රණිහිතාලම්බනය ආර්‍ය්‍ය වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ස්ථානාස්ථානත් මෙනෙහි කෙරේ ද, අප්‍රණිහිතාලම්බනත් මෙනෙහි කෙරේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

7\. ස. පු: ස්ථානාස්ථානත් මෙනෙහි කෙරේ ද, අප්‍රණිහිතාලම්බනත් මෙනෙහි කෙරේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ස්පර්‍ශයන් දෙදෙනකුන්ගේ චිත්තයන් දෙදෙනකුන්ගේ සමවධාන වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

8\. ස. පු: ස්මෘතිප්‍රස්ථානයෝ ආර්‍ය්‍ය වූවාහුත් ශුන්‍යතාලම්බන වූවාහුත් වෙත් ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ස්ථානාස්ථානයෙහි යථාභූතඥාන නම් වූ තථාගතබලය ආර්‍ය්‍ය වූයේත් ශුන්‍යතාලම්බන වූයේත් වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

9\. ස. පු: සීවටන්හු ආර්‍ය්‍ය වූවාහු අනිමිත්තාලම්බන වූවාහු … අප්‍රණිහිතාලම්බන වූවාහු වෙත් ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ස්ථානාස්ථානයෙහි යථාභූතඥාන නම් වූ තථාගතබලය ආර්‍ය්‍ය වූයේත් අප්‍රණිහිතාලම්බන වූයේත් වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

10\. ස. පු: සම්‍යක්ප්‍රධානයෝ යැ, ඍද්ධිපාදයෝ යැ, ඉන්ද්‍රියයෝ යැ, බලයෝ යැ, බොධ්‍යඞ්ගයෝ යැ, යන මොහු ආර්‍ය්‍ය වූවාහුත් ශුන්‍යතාලම්බන වූවාහුත් වෙත් ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ස්ථානාස්ථානයෙහි යථාභූතඥාන නම් වූ තථාගතබලය ආර්‍ය්‍ය වූයේත් ශුන්‍යතාලම්බන වූයේත් වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

11\. ස. පු: බොධ්‍යඞ්ගයෝ ආර්‍ය්‍ය වූවාහුත් අනිමිත්තාලම්බන වූවාහුත් … අප්‍රණිහිතාලම්බන වූවාහුත් වෙත් ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ස්ථානාස්ථානයෙහි යථාභූතඥාන නම් වූ තථාගතබලය ආර්‍ය්‍ය වූයේත් අප්‍රණිහිතාලම්බන වූයේත් වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

12\. ස. පු: ස්ථානාස්ථානයෙහි යථාභූතඥාන නම් වූ තථාගතබලය ආර්‍ය්‍ය යැ, ශුන්‍යතාලම්බන යැයි නො කියැයුතු වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: සීවටන්හු ආර්‍ය්‍යයහ, ශුන්‍යතාලම්බනයහයි නො කියැයුතු වෙත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …

13\. ස. පු: ස්ථානාස්ථානයෙහි යථාභූතඥාන නම් වූ තථාගතබලය ආර්‍ය්‍ය යැ, අනිමිත්තාලම්බන යැ … අප්‍රණිහිතාලම්බන යැ යි නො කියැයුතු වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: සීවටන්හු ආර්‍ය්‍යයහ, අප්‍රණිහිතාලම්බනයහයි නො කියැයුතු වෙත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …
