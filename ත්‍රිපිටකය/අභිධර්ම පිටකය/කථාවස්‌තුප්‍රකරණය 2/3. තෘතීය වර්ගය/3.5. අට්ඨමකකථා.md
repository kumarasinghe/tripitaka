# 3.5. අට්ඨමකකථා

**\[මාර්‍ගක්‍ෂණයෙහි ක්ලෙශසමුදාචාර නැතියෙන් ‘අෂ්ටමක වූ ශ්‍රොතාපත්තිමාර්‍ගස්ථ පුද්ගලයාගේ පර්‍ය්‍යුත්‍ථානයෝ දෙදෙනෙක් ඇති’යි ලබ්ධිගත් වාදීහු ඇත. අන්‍ධක සමමිතිය නිකායවාසීන් සෙයිනි. උන්ගේ වාද බිඳුනට අෂ්ටමකකථා එයි.\]**

1\. ස. පු: අෂ්ටමකපුද්ගලයාගේ දෘෂ්ටිපර්‍ය්‍යුත්‍ථාන ප්‍රහීණ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ. (මාර්‍ගක්‍ෂණයෙහි පටන් දෘෂ්ටිඅනුත්පත්ති පැනෙනුයෙන් කළ ප්‍රතිඥා යි.)

ස. අනු: අෂ්ටමකපුද්ගලයා ශ්‍රොතාපන්න වූයේ, ශ්‍රොතාපත්තිඵලප්‍රාප්ත වූයේ ප්‍රතිලබ්ධ වූයේ අධිගත වූයේ සාක්‍ෂාත්කෘත වූයේ එයට පැමිණ වාස කෙරේ ද, නාමකයින් ස්පර්‍ශ කොට වාස කෙරේ ද? (දෘෂ්ටි නම් ශ්‍රොතාපන්නයාහට මැ ප්‍රහීණ යි. මාර්‍ගස්ථයාහට නො වෙයි. එයින් කළ අනුයොගයි.)

ප. පටි: එසේ නො කියැයුතු යැ …

2\. ස. පු: අෂ්ටමක පුද්ගලයාගේ විචිකිත්සාපර්‍ය්‍යුත්‍ථාන ප්‍රහීණ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අෂ්ටමකපුද්ගලයා ශ්‍රොතාපන්න වූයේ ශ්‍රොතාපත්තිඵලප්‍රාප්ත වූයේ … නාමකයින් ස්පර්‍ශ කොට වාස කෙරේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

3\. ස. පු: අෂ්ටමකපුද්ගලයාගේ දෘෂ්ටිපර්‍ය්‍යුත්‍ථාන ප්‍රහීණ වේ ද?

ප. පටිඤ්ඤා : එසේ යැ.

ස. අනු: අෂ්ටමකපුද්ගලයාගේ දෘෂ්ටිඅනුශය ප්‍රහීණ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ … (‘අනුශය නම් පර්‍ය්‍යුත්‍ථානයෙන් අනෙකෙකි’යි පරවාදීන්ගේ ලබ්ධි වෙයි. එයින් කළ ප්‍රතික්‍ෂෙප යි.)

4\. ස. පු: අෂ්ටමකපුද්ගලයාගේ දෘෂ්ටිපර්‍ය්‍යුත්‍ථාන ප්‍රහීණ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අෂ්ටමකපුද්ගලයාගේ විචිකිත්සානුශය ශීලව්‍රතපරාමර්‍ශානුශය ප්‍රහීණ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ … (ශීලව්‍රතපරාමර්‍ශපර්‍ය්‍යුත්‍ථාන නමින් ව්‍යවහාරයෙක් නැත. එයින් කළ ප්‍රතික්‍ෂෙප යි.)

5\. ස. පු: අෂ්ටමක පුද්ගලයාගේ විචිකිත්සාපර්‍ය්‍යුත්‍ථාන ප්‍රහීණ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අෂ්ටමක පුද්ගලයාගේ විචිකිත්සානුශය ප්‍රහීණ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

6\. ස. පු: අෂ්ටමක පුද්ගලයාගේ විචිකිත්සාපර්‍ය්‍යුත්‍ථාන ප්‍රහීණ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අෂ්ටමක පුද්ගලයාගේ දෘෂ්ටිඅනුශය ශීලව්‍රතපරාමර්‍ශානුශය ප්‍රහීණ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

7\. ස. පු: අෂ්ටමක පුද්ගලයාගේ දෘෂ්ටිඅනුශය අප්‍රහීණ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අෂ්ටමක පුද්ගලයාගේ දෘෂ්ටිපර්‍ය්‍යුත්‍ථාන අප්‍රහීණ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

8\. ස. පු: අෂ්ටමක පුද්ගලයාගේ දෘෂ්ටිඅනුශය අප්‍රහීණ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අෂ්ටමක පුද්ගලයාගේ විචිකිත්සාපර්‍ය්‍යුත්‍ථාන අප්‍රහීණ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

9\. ස. පු: අෂ්ටමක පුද්ගලයාගේ විචිකිත්සානුශය ශීලව්‍රතපරාමර්‍ශ අප්‍රහීණ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අෂ්ටමක පුද්ගලයාගේ දෘෂ්ටිපර්‍ය්‍යුත්‍ථාන අප්‍රහීණ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

10\. ස. පු: අෂ්ටමක පුද්ගලයාගේ ශීලව්‍රතපරාමර්‍ශ අප්‍රහීණ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අෂ්ටමක පුද්ගලයාගේ විචිකිත්සාපර්‍ය්‍යුත්‍ථාන අප්‍රහීණ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

11\. ස. පු: අෂ්ටමක පුද්ගලයාගේ දෘෂ්ටිපර්‍ය්‍යුත්‍ථාන ප්‍රහීණ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අෂ්ටමක පුද්ගලයා විසින් දෘෂ්ටිපර්‍ය්‍යුත්‍ථාන ප්‍රහාණය පිණිස මාර්‍ගය භාවිත වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ … (තත්ක්‍ෂණයෙහි වර්‍තමාන වශයෙන් භාවනා කරත් මිස භාවිත නො වනුයෙන් කළ ප්‍රතික්‍ෂෙප යි.)

12\. ස. පු: අෂ්ටමක පුද්ගලයාගේ දෘෂ්ටිපර්‍ය්‍යුත්‍ථාන ප්‍රහීණ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අෂ්ටමක පුද්ගලයා විසින් දෘෂ්ටිපර්‍ය්‍යුත්‍ථාන ප්‍රහාණය පිණිස සීවටන්හු භාවිත වෙත් ද? … සම්‍යක්ප්‍රධානයෝ … බොධ්‍යඞ්ගයෝ භාවිත වෙත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …
