# 9.1. ආනිසංසකථා

**\[‘ආදීනව විසින් සංස්කාරයනුදු ආනිසංස විසින් නිවනුදු දක්නහුට සංයොජනප්‍රහාණ වෙයි’ යනු ස්වසමය යි. ඒ දෙපක්‍ෂයෙහි මැ එකාංශවාද ගෙන ‘අනුසස් දකන්හුට මැ සංයොජනප්‍රහාණ වෙයි’ යනු අන්ධකනිකායවාසීන්ගේ ලබ්ධි වෙයි. උන්ගේ වාද බිඳුනට ‘ආනිසංසකථා’ එයි.\]**

1\. ස. පු: අනුසස් දක්නහුට සංයොජනයන්ගේ ප්‍රහාණ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: සංස්කාරයන් අනිත්‍යවශයෙන් මනස්කාර කරන්නහුට සංයොජනයෝ ප්‍රහීණ වන්නාහු නො?

ප. පටිඤ්ඤා: එසේ යැ.

ස. ඨ: ඉදින් සංස්කාර අනිත්‍ය වශයෙන් මනස්කාර කරන්නහුට සංයොජනයෝ ප්‍රහීණ වෙත් නම්,

ස. පා: භවත, ‘අනුසස් දක්නහුට සංයොජනයන්ගේ ප්‍රහාණ වේ’ යයි නො කියැයුතු මැ යැ.

2\. ස. පු: සංස්කාර දුඃඛ වශයෙන් … රොග වශයෙන් … ගණ්ඩ වශයෙන් … ශල්‍ය වශයෙන් … අවෘද්ධි වශයෙන් … ආබාධ වශයෙන් … අවශ වශයෙන් … (ව්‍යාධ්‍යාදීන්) විනාශ වශයෙන් … අනෙකව්‍යසනාවහ විසින් … ඊති වශයෙන් … උපද්‍රව වශයෙන් භය වශයෙන් … අනර්‍ථාවහ වශයෙන් … (ලොකධර්‍මයෙන්) චලද් වශයෙන් … (උපක්‍රමාදීන්) භිදුර වශයෙන් … අධ්‍රැව වශයෙන් … (ක්‍ෂෙම රාහිත්‍යයෙන්) අත්‍රාණ වශයෙන් … අලෙණ වශයෙන් … අසරණ වශයෙන් … අසරණීභූත වශයෙන් … රික්ත වශයෙන් … රික්ත හෙයින් මැ තුච්ඡ වශයෙන් … (වෙදකකාරකාදි රාහිත්‍යයෙන්) ශුන්‍ය වශයෙන් … අනාත්ම වශයෙන් … ආදීනව වශයෙන් … විපරිණාමධර්‍ම වශයෙන් මනස්කාර කරන්නහුට සංයොජනයෝ ප්‍රහීණ වන්නාහු නො?

ප. පටිඤ්ඤා: එසේ යැ.

ස. ඨ: ඉදින් සංස්කාරයන් විපරිණාමධර්‍ම වශයෙන් මනස්කාර කරන්නහුට සංයොජනයෝ ප්‍රහීණ වෙත් නම්,

ස. පා: භවත, ‘අනුසස් දක්නහුට සංයොජනප්‍රහාණ ඇති’ යි නො කියැයුතු මැ යැ.

3\. ස. පු: සංස්කාරයන් අනිත්‍යවශයෙන් මනස්කාර කරන්නේත් නිර්‍වාණයෙහි අනුසස් දක්නේත් වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ස්පර්‍ශයන් දෙදෙනෙකුන්ගේ … චිත්තයන් දෙදෙනෙකුන්ගේ සමවධාන වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: සංස්කාරයනුදු දුඃඛ වශයෙන් … රොග වශයෙන් … විපරිණාමධර්‍ම වශයෙන් මනස්කාර කරන්නේත් නිර්‍වාණයෙහිදු අනුසස් දක්නේත් වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

4\. ස. පු: සංස්කාරයනුදු විපරිණාමධර්‍ම වශයෙන් මනස්කාර කරන්නේත් නිර්‍වාණයෙහිදු අනුසස් දක්නේත් වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ස්පර්‍ශයන් දෙදෙනෙකුන්ගේ … චිත්තයන් දෙදෙනෙකුන්ගේ සමවධාන වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

5\. ප. පු: අනුසස් දන්නහුට සංයොජනප්‍රහාණය නො කියැයුතු වේ ද?

ස. පටිඤ්ඤා: එසේ යැ.

ප. අනු: “මහණෙනි, මෙසස්නෙහි මහණ නිවන්හි සුඛය නුවණින් දක්නේ, සුඛ සංඥා ඇති වැ, සුඛවිඳුනා සුලු වැ, නිරතුරු, අමිශ්‍ර කොට (එක්සිත් වැ), සිතින් බැසගන්මින් සන්නිට්ඨාපනය කරමින්, ප්‍රඥායෙන් බැසගන්මින් (අනුප්‍රවිෂ්ට වැ) වාස කෙරේ” යයි භාග්‍යවතුන් වහන්සේ විසින් වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ නු?

ස. පටිඤ්ඤා: එසේ යැ.

පටි. ආ: එ කරුණින් ’අනුසස් දක්නහුට සංයොජනප්‍රහාණ ඇති.

ආනිසංසකථා නිමි.
