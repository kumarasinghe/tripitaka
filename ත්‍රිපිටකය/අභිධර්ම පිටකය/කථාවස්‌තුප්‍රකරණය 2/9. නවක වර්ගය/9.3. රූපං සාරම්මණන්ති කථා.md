# 9.3. රූපං සාරම්මණන්ති කථා

**\[රූපය සප්‍රත්‍යයාර්‍ථයෙන් සාරම්මණ වෙයි. අනෙකක් අරමුණු නො කෙරේ යයි ආලම්බන ප්‍රත්‍යය වශයෙන් සාරම්මණ නො වෙයි. ඒ වෙනස නො ගෙන ‘රූපය සාරම්මණ යැ’ යි කියන උත්තරාපථකයන්ගේ වාද බිඳුනට ‘රූපං සාරම්මණන්ති කථා’ එයි.\]**

1\. ස. පු: රූපය සාලම්බන වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ඊට ආවර්‍ජනා, ආභොග, සමන්වාහාර, මනස්කාර, චෙතනා, ප්‍රාර්‍ථනා, ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: ඊට ආවර්‍ජනා, ආභොග, … ප්‍රණිධි නැත්තේ නො?

ප. පටිඤ්ඤා: එසේ යැ.

අනු. ඨ: ඊට ආවර්‍ජනා, ආභොග … ප්‍රණිධි නැත් නම්,

අනු. පා: ‘රූපය සාලම්බන යැ’ යි නො කියැයුතු මැ යැ.

2\. ස. පු: ස්පර්‍ශය සාලම්බන යැ. ඊට ආවර්‍ජනා ආභොග … ප්‍රණිධි ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: රූපය සාලම්බන යැ. ඊට ආවර්‍ජනා, ආභොග … ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

3\. ස. පු: වෙදනා … සංඥා … චෙතනා … චිත්ත … ශ්‍රද්ධා … වීර්‍ය්‍ය … ස්මෘති සමාධි … ප්‍රඥා … රාග … ද්වෙෂ … මොහ … මාන … දෘෂ්ටි … විචිකිත්සා … සත්‍යාන … ඖද්ධත්‍ය … අහ්‍රීක … අනපත්‍රපා සාලම්බන යැ. ඊට ආවර්‍ජනා ආභොග … ප්‍රණිධි ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: රූපය සාලම්බනය. ඊට ආවර්‍ජනා ආභොග … ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

4\. ස. පු: රූපය සාලම්බන යැ. ඊට ආවර්‍ජනා, ආභොග … ප්‍රණිධි නැද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ස්පර්‍ශ සාලම්බන යැ. ඊට ආවර්‍ජනා, ආභොග, … ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

5\. ස. පු: රූපය සාලම්බන යැ. ඊට ආවර්‍ජනා, ආභොග, … ප්‍රණිධි නැද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: වෙදනා … සංඥා … අනපත්‍රපා සාලම්බන යැ. ඊට ආවර්‍ජනා, ආභොග, … ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

6\. ප. පු: ‘රූපය සාලම්බන යැ’ යි නො කියැයුතු වේ ද?

ස. පටිඤ්ඤා: එසේ යැ.

ප. අනු: රූපය සප්‍රත්‍යය නො වේ ද?

ස. පටිඤ්ඤා: එසේ යැ.

අනු. ඨ: ඉදින් රූපය සප්‍රත්‍යය වේ නම්,

අනු. පා: භවත, එකරුණින් ‘රූපය සාලම්බන යැ’ යි කියැයුතු මැ යැ.

රූපං සාරම්මණන්තිකථා නිමි.
