# 10.3. පඤ්චවිඤ්ඤාණ සමඞ්ගිස්ස මග්ගකථා

**\[‘චක්ඛුනා රූපං දිස්වා න නිමිත්තග්ගාහි හොති - ඇසින් රූ දැක නිමිත්තග්‍රාහී නො වෙයි’ යන සූත්‍රය ඇසිරි කොට ‘පඤ්චවිඥානසමඞ්‍ගී පුද්ගලයාහට මාර්‍ගභාවනා ඇති’ යි ලබ්ධි ගත් මහාසඞ්‍ඝිකයන්ගේ වාද බිඳුනට ‘පඤ්චවිඤ්‍ඤාණ සමඞ්‍ගිස්‍ස මග්ගකථා’ එයි.\]**

1\. ස. පු: පඤ්චවිඥානසමඞ්‍ගීහට මාර්‍ගභාවනා ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: පඤ්චවිඥානයෝ උත්පන්නවස්තුක උත්පන්නාලම්බන වූවාහු නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

අනු. ඨ: ඉදින් පඤ්චවිඤ්‍ඤානයෝ උත්පන්නවස්තුක උත්පන්නාලම්බන වෙත් නම්,

අනු. පා: භවත, ‘පඤ්චවිඤ්‍ඤාණසමඞ්‍ගීහට මාර්‍ගභාවනා ඇති’ යි නො කියැයුතු මැ යැ.

2\. ස. පු: පඤ්චවිඤ්‍ඤාණයෝ පුරෙජාතවස්තුක පුරෙජාතාලම්බන, ආධ්‍යාත්මිකවස්තුක බාහ්‍යායලම්බන, අසම්භින්නවස්තුක අසම්භින්නාලම්බන, නානාවස්තුක නානාලම්බන වූවාහු, අන්‍යො’න්‍යගොචරවිෂය අනුභව නො කෙරෙත් නු සමන්නාහාර නැති වැ නූපදිත් නු? මනසිකාර නැති වැ නූපදිත් නු? අව්‍යාකීර්‍ණ වැ නූපදිත් නු? පෙරපසු නො වැ නූපදිත් නු? අන්‍යොන්‍යයනට සමනන්තර වැ නූපදිත් නු? ... පඤ්චවිඥානයෝ ආභොග (ප්‍රත්‍යවෙක්‍ෂා) නැත්තාහු නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

අනු. ඨ: ඉදින් පඤ්චවිඤ්‍ඤානයෝ අනාභොග වෙත් නම්,

අනු. පා: ‘පඤ්චවිඤ්‍ඤාණසමන්නාගතයහට මාර්‍ගභාවනා ඇති’ යි නො කියැයුතු මැ යැ.

3\. ස. පු: චක්‍ෂුර්විඥාන සමන්වාගතයහට මාර්‍ගභාවනා ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: චක්‍ෂුර්විඥානය ශුන්‍යතය ඇරැබැ උපදනේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

4\. ස. පු: චක්‍ෂුර්විඥානය ශුන්‍යතය ඇරැබැ උපදනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: චක්‍ෂුසත් ශුන්‍යතයත් නිසා චක්‍ෂුර්විඥානය උපදනේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

5\. ස. පු: චක්‍ෂුසත් ශුන්‍යතයත් නිසා චක්‍ෂුර්විඥානය උපදනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: “චක්‍ෂුසත් ශුන්‍යතයත් නිසා චක්‍ෂුර්විඥානය උපදනේ” යයි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ වේ ද?

ප. පටි: නැති.

ස. අනු: “චක්‍ෂුසත් රූපාලම්බනයනුත් නිසා චක්‍ෂුර්විඥානය උපදනේ” යයි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ නො?

ප. පටිඤ්‍ඤා: එසේ යැ.

අනු. ඨ: ඉදින් “චක්‍ෂුසත් රූපයනුත් නිසා චක්‍ෂුර්විඥානය උපදනේ” යයි වදාළ සූත්‍රාන්තයෙක් ඇත් නම්,

අනු. පා: භවත, “චක්‍ෂුසත් ශුන්‍යතයත් නිසා චක්‍ෂුර්විඥානය උපදනේ” යයි නො කියැයුතු මැ යැ.

6\. ස. පු: චක්‍ෂුර්විඥානසමඞ්‍ගී පුද්ගලයාහට මාර්‍ගභාවනා ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: චක්‍ෂුර්විඥානය අතීතානගත අරමුණු ඇරැබැ උපදනේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ.

7\. ස. පු: චක්‍ෂුර්විඥානසමඞ්‍ගී පුද්ගලයාහට මාර්‍ගභාවනා ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: චක්‍ෂුර්විඥානය ස්පර්‍ශය ඇරැබැ වෙදනාව ඇරැබැ සංඥාව ඇරැබැ චෙතනාව ඇරැබැ චිත්තය ඇරැබැ චක්‍ෂුස ඇරැබැ ... කය ඇරැබැ ... ශබ්දය ඇරැබැ ... ස්ප්‍රෂ්ටව්‍ය ඇරැබැ උපදනේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ.

8\. ස. පු: මනොවිඥානසමඞ්‍ගී පුද්ගලයාහට මාර්‍ගභාවනා ඇත, මනොවිඥානය ශුන්‍යතා ඇරැබැ උපදනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: චක්‍ෂුර්විඥානසමඞ්‍ගී පුද්ගලයාහට මාර්‍ගභාවනා ඇත, චක්‍ෂුර්විඥානය ශුන්‍යතාව ඇරැබැ උපදනේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

9\. ස. පු: මනොවිඥානසමඞ්‍ගී පුද්ගලයාහට මාර්‍ගභාවනා ඇත, මනොවිඥානය අතීතානාගතය ඇරැබැ උපදනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: චක්‍ෂුර්විඥානසමඞ්‍ගී පුද්ගලයාහට මාර්‍ගභාවනා ඇත, චක්‍ෂුර්විඥානය අතීතානාගතය ඇරැබැ උපදනේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

10\. ස. පු: මනොවිඥානසමඞ්‍ගී පුද්ගලයාහට මාර්‍ගභාවනා ඇත, මනොවිඥානය ස්පර්‍ශය ඇරැබැ ... වෙදනාව ඇරැබැ ... ස්ප්‍රෂ්ටව්‍යය ඇරැබැ උපදනේ වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: චක්‍ෂුර්විඥානසමඞ්‍ගී පුද්ගලයාහට මාර්‍ගභාවනා ඇත, චක්‍ෂුර්විඥානය ස්පර්‍ශය ඇරැබැ ... වෙදනාව ඇරැබැ ... ස්ප්‍රෂ්ටව්‍ය ඇරැබැ උපදනේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

11\. ප. පු: ‘පඤ්චවිඤ්‍ඤාණසමඞ්‍ගී පුද්ගලයාහට මාර්‍ගභාවනා ඇති’ යි නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: භාග්‍යවතුන් වහන්සේ විසින් වදාරනලද මැ නු:

“මහණෙනි, මෙසස්නෙහි මහණ ඇසින් රූ දැක නිමිත්තග්‍රාහී නො වෙයි. අනුව්‍යඤ්‍ජනග්‍රාහී නො වෙයි ... කනින් ශබ්ද අසා ... ඝ්‍රාණයෙන් ගන්‍ධ ආඝ්‍රාණ කොට ... දිවින් රස ආස්වාද කොට ... කයින් ස්ප්‍රෂ්ටව්‍ය ස්පර්‍ශ කොට නිමිත්තග්‍රාහී නො වෙයි, අනුව්‍යඤ්‍ජනග්‍රාහී නො වේ” යයි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ නු?

ස. පටිඤ්‍ඤා: එසේ යැ.

පටි. ආ: එ කරුණින් ‘පඤ්චවිඥානසමඞ්‍ගී පුද්ගලයාහට මාර්‍ගභාවනා ඇත.

පඤ්චවිඤ්‍ඤාණසමඞ්‍ගිස්‍ස මග්ගකථා නිමි.
