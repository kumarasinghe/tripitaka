# 10.2. රූපං මග්ගො’ති කථා

**\[‘සම්මාවාචා සම්මාකම්මන්ත සම්මාආජීව යන මේ මාර්‍ගාඞ්ගයෝ රූප වෙති’ යි ලබ්ධි ගත් මහිංසාසක සමමිතීය මහාසඞ්ඝිකයන්ගේ වාද බිඳුනට ‘රූපං මග්ගොති කථා’ එයි.\]**

1\. ස. පු: මාර්‍ගසමඞ්ගි පුද්ගලයාගේ රූපය මාර්‍ග වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සාලම්බන වේ ද, ඊට ආවර්‍ජනා, අභොග, සමන්‍වාහාර, මනසිකාර, චෙතනා, ප්‍රාර්‍ථනා, ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: අනාලම්බන නො? ඊට ආවර්‍ජනා ... ප්‍රණිධි නැත්තේ නො වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

අනු. ඨ: ඉදින් අනාලම්බන වේ නම්, ඊට ආවර්‍ජනා ... ප්‍රණිධි නැත් නම්,

අනු. පා: භවත, ‘මාර්‍ගසමඞ්ගීහුගේ රූපය මාර්‍ග යැ’ යි නො කියැයුතු මැ යැ.

2\. ස. පු: සම්මාවාචාව මාර්‍ග වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සාලම්බන වේ ද, ඇයට ආවර්‍ජනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: අනාලම්බන නො? ඇයට ආවර්‍ජනා ... ප්‍රණිධි නැත්තේ නො වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

අනු. ඨ: ඉදින් අනාලම්බන වේ නම්, ඇයට ආවර්‍ජනා ... ප්‍රණිධි නැත් නම්,

අනු. පා: භවත, ‘සම්මාවාචාව මාර්‍ග වේ යැ’ යි නො කියැයුතු මැ යැ.

3\. ස. පු: සම්මාකම්මන්තය ... සම්මාආජීවය මාර්‍ග වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සාලම්බන වේ ද, ඊට ආවර්‍ජනා ... ප්‍රණිධි ඇත්තේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

ස. අනු: අනාලම්බන නො? එයට ආවර්‍ජනා ... ප්‍රණිධි නැත්තේ නො වේ ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

අනු. ඨ: ඉදින් අනාලම්බන වේ නම්, එයට ආවර්‍ජනා ... ප්‍රණිධි නැත් නම්,

අනු. පා: භවත, ‘සම්මාආජීවය මාර්‍ග වේ යැ’ යි නො කියැයුතු මැ යැ

4\. ස. පු: සම්මාදිට්ඨි මාර්‍ග වේ ද? ඔ ද සාලම්බන වෙයි, ඇයට ආවර්‍ජනා ... ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්මාවාචා මාර්‍ග වෙයි. ඔ ද සාලම්බන වෙයි. ඇයට ආවර්‍ජනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

5\. ස. පු: සම්මාදිට්ඨිය මාර්‍ග වෙයි. ඔ ද සාලම්බන වෙයි. ඇයට ආවර්‍ජනා ... ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්මාකම්මන්තය ... සම්මාආජීවය මාර්‍ග වෙයි. එ ද සාලම්බන වෙයි. ඊට ආවර්‍ජනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

6\. ස. පු: සම්මාසඞ්කප්පය ... සම්මාවායාමය ... සම්මාසතිය ... සම්මාසමාධිය මාර්‍ග වෙයි. එ ද සාලම්බන වෙයි. ඊට ආවර්‍ජනා ... ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්මාවාචාව මාර්‍ග වෙයි. එ ද සාලම්බන වෙයි. ඇයට ආවර්‍ජනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

7\. ස. පු: සම්මාසමාධිය මාර්‍ග වෙයි. එ ද සාලම්බන වෙයි. ඊට ආවර්‍ජනා ... ප්‍රණිධි ඇද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්මාකම්මන්තය ... සම්මාආජීවය මාර්‍ග වෙයි. එ ද සාලම්බන වෙයි. එයට ආවර්‍ජනා ... ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

8\. ස. පු: සම්මාවාචාව මාර්‍ග වෙයි. ඔ ද අනාලම්බන වෙයි. ඇයට ආවර්‍ජනා ... ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්මාදිට්ඨිය මාර්‍ග වෙයි. ඔ ද අනාම්බන වෙයි. ඇයට ආවර්‍ජනා ... ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

9\. ස. පු: සම්මාවාචාව මාර්‍ග වෙයි. ඔ ද අනාලම්බන වෙයි. ඇයට ආවර්‍ජනා ... ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්මාසඞ්කප්පය ... සම්මාවායාමය ... සම්මාසති ... සම්මාසමාධිය මාර්‍ග වෙයි. එ ද අනාම්බන වෙයි. ඊට ආවර්‍ජනා ... ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

10\. ස. පු: සම්මාකම්මන්තය ... සම්මාආජීවය මාර්‍ග වෙයි. එ ද අනාලම්බන වෙයි. ඊට ආවර්‍ජනා ... ප්‍රණිධි නැද්ද?

ප. පටිඤ්‍ඤා: එසේ යැ.

ස. අනු: සම්මාදිට්ඨි ... සම්මාසඞ්කප්ප ... සම්මාවායාම ... සම්මාසති... සම්මාසමාධි මාර්‍ග වෙයි. එ ද අනාම්බන වෙයි. ඊට ආවර්‍ජනා ... ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ ...

11\. ප.පු: ‘මාර්‍ගසමඞ්ගිහුගේ රූපය මාර්‍ග වෙයි’ යනු නො කියැයුතු වේ ද?

ස. පටිඤ්‍ඤා: එසේ යැ.

ප. අනු: සම්මාවාචා සම්මාකම්මන්ත සම්මාආජීව තුන මාර්‍ග වන්නේ නො?

ස. පටිඤ්‍ඤා: එසේ යැ.
