# 6.2. පටිච්චසමුප්පාදකථා

**\[“උප්පාදා වා තථාගතානං - පෙ - ඨිතා’ ව සා ධාතු, ධම්මට්ඨිතතා, ධම්මනියාමතා” යි වදාළහෙයින් ‘ප්‍රතීත්‍යසමුත්පාදය අසංස්කෘත වෙති’ යි පූර්‍වශෛලියයන් හා මහීශාසකනිකායවාසීන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට පටිච්චසමුප්පාදකථා එයි.\]**

1\. ස. පු: ප්‍රතීත්‍යසමුත්පාදය අසංස්කෘත වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ප. පටි: එසේ නො කියැයුතු යැ …

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

3\. ස. පු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

4\. ස. පු: නිර්‍වාණයෝ දෙදෙනෙක් ඇද්ද?

ස. පටිඤ්ඤා: එසේ යැ.

ප. අනු: භාග්‍යවතුන් වහන්සේ විසින් වදාරනලද මැ නු:

“මහණෙනි, ජාතිප්‍රත්‍යයෙන් ජරාමරණ වෙයි. තථාගතයන් වහන්සේගේ උත්පාදයෙන් වේවයි ඔබගේ අනුත්පාදයෙන් වේවයි (ජරාමරණධර්‍මයනට කාරණාර්‍ථයෙන්) ධර්‍මස්ථිතතා වූ, (ජරාමරණධර්‍මයනට කාරණාර්‍ථයෙන් මැ) ධර්‍මනියාමතා වූ, ධර්‍මස්ථිති, ධර්‍මනියාම වශයෙන් මේ ජරාමරණාදිධර්‍මයනට ප්‍රත්‍යාර්ත්‍ථයෙන් ඉදම්ප්‍රත්‍යතා (ප්‍රත්‍යයවන ස්වභාව) වූ ඒ ධාතු තොමෝ සිටියා (පවතී) මැ යැ. තථාගතයන් වහන්සේ එය මොනොවට අවබොධ කරනසේක. අභිසම කරනසේක. මොනොවට අවබොධ කොට අභිසම කොට කියනසේක, දෙසනසේක, දන්වනසේක, පිහිටුවනසේක, විවරණ කරනසේක, බෙදනසේක, හෙළි කරනසේක, ‘බලව’යිදු බණනසේක: මහණෙනි, ජාතිප්‍රත්‍යයෙන් ජරාමරණ වෙයි. මහණෙනි, භවප්‍රත්‍යයෙන් ජාති වෙයි කියා යි. මහණෙනි, අවිද්‍යාප්‍රත්‍යයයෙන් සංස්කාරයෝ වෙති. තථාගතයන් වහන්සේගේ උත්පාදයෙන් වේවයි ඔබගේ අනුත්පාදයෙන් වේවයි ඒ ධර්‍මධාතු සිටියා මැ යැ … ‘බලව’යිදු බණනසේක: ‘මහණෙනි, අවිද්‍යාප්‍රත්‍යයයෙන් සංස්කාරය වෙති’ යි කියා යි. මහණෙනි, මෙසේ එහිලා යම් තථතායෙක් (ස්වභායෙක්) අවිතථතායෙක් (නොසිස් බවෙක්) අනන්‍යථාභාවයෙක් (නොවෙනස් බවෙක්), ඉදම්ප්‍රත්‍යතායෙක් ඇද්ද? මහණෙනි, මේ ප්‍රතීත්‍යසමුත්පාද යැයි කියනු ලැබේ” යයි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ නු?

ස. පටිඤ්ඤා: එසේ යැ.

ප. නිග: එකරුණින් ප්‍රතීත්‍යසමුත්පාදය අසංස්කෘත වෙයි.

16\. ස. පු: ‘අවිජ්ජාපච්වයා සඞ්ඛාරා’ යන එහි සංස්කාරධර්‍මයනට කාරණාර්ත්‍ථයෙන් ධර්‍මස්ථිතතාවූ, සංස්කාරධර්‍මයනට කාරණාර්ත්‍ථයෙන් (ඇති කල්හි මැ සම්භවය වන) ධර්‍මනියාමතා වූ යම් අවිද්‍යායෙක් ඇද්ද, ඕ (තොපමතයෙහි) අසංස්කෘත යැ. නිර්‍වාණයත් අසංස්කෘත වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

17\. ස. පු: අසංස්කෘතයෝ දෙදෙනෙක් ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ත්‍රාණයෝ දෙදෙනෙක් … අන්තරිකාවෝ දෙදෙනෙක් හෝ ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

18\. ස. පු: ‘අවිජ්ජාපච්වයා සඞ්ඛාරා' (අවිද්‍යාප්‍රත්‍යයෙන් සංස්කාර වෙති), යන එහි ධර්‍මස්ථිතතා ධර්‍මනියාමතා වූ යම් අවිද්‍යාවක් ඇද්ද, ඕ (තොප මතයෙහි) අසංස්කෘත යැ. ‘සඞ්ඛාරපච්වයා විඤ්ඤාණං’ යන ධර්‍මස්ථිතතා ධර්‍මනියාමතා වූ යම් සංස්කාරධාතුවක් ඇද්ද, ඕ (තොප මතයෙහි) අසංස්කෘත යැ. නිර්‍වාණයත් අසංස්කෘත වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අසංස්කෘතයෝ තුන් දෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

19\. ස. පු: අසංස්කෘතයෝ තුන් දෙනෙක් ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ත්‍රාණයෝ තුන් දෙනෙක් … අන්තරිකාවෝ තුන් දෙනෙක් හෝ ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

20\. ස. පු: ‘අවිජ්ජාපච්වයා සඞ්ඛාරා’ යන එහි ධර්‍මස්ථිතතා ධර්‍මනියාමතා වූ යම් අවිද්‍යාවක් ඇද්ද, ඕ අසංස්කෘත යැ. ‘සඞ්ඛාරපච්වයා විඤ්ඤාණං’ යන එහි ධර්‍මස්ථිතතා ධර්‍මනියාමතා වූ යම් සංස්කාරධාතුවක් ඇද්ද, ඔ ද අසංස්කෘත යැ … ‘ජාතිපච්චයා ජරාමරණං’ යන එහි ධර්‍මස්ථිතතා ධර්‍මනියාමතා වූ යම් ජාතියක් ඇද්ද උයිත් අසංස්කෘත යැ. නිවනත් අසංස්කෘත වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අසංස්කෘතයෝ දොළොස් දෙනෙක් ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

21\. ස. පු: අසංස්කෘතයෝ දොළොස් දෙනෙක් ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ත්‍රාණයෝ දොළොස් දෙනෙක් … ලෙණයෝ දොළොස් දෙනෙක් … අන්තරිකාවෝ දොළොස් දෙනෙක් හෝ යි?

ප. පටි: එසේ නො කියැයුතු යැ …

ප්‍රතීත්‍යසමුත්පාදකථා නිමි.
