# 8.5. රූපධාතුකථා

**\[“රූපී ධර්‍මයෝ මැ රූපධාතු නම්හ” යි අන්‍ධක නිකායවාසීන් සෙයින් ලබ්ධි ගත්තවුන්ගේ වාද බිඳුනට රූපධාතුකථා එයි.\]**

1\. ස. පු: රූපී ධර්‍මයෝ රූපධාතු වෙත් ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: රූපය වනාහි භව, ගති, සත්ත්‍වාවාස, සංසාර, යොනි, විඥානස්ථිති, ආත්මභාවප්‍රතිලාභ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: රූපොපග කර්‍ම ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: රූපොපග සත්ත්‍වයෝ ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: රූපයෙහි ලා සත්ත්‍වයෝ ජාතිවශයෙන් දනත් ද, ජරාවශයෙන් දිරත් ද, මරණවශයෙන් මියත් ද, ච්‍යුතිවශයෙන් සැව යෙත් ද, ප්‍රතිසන්‍ධිවශයෙන් උපදිත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: රූපයෙහි ලා රූප, වෙදනා, සංඥා, සංස්කාර, විඥාන ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: රූපය වනාහි පඤ්චවොකාරභව වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

2\. ස. පු: රූපධාතුව වනාහි භව, ගති … ආත්මභාවප්‍රතිලාභ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: රූපය වනාහි භව, ගති … ආත්මභාවප්‍රතිලාභ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

3\. ස. පු: රූපධාතූපග කර්‍ම ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: රූපූපග කර්‍ම ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

4\. ස. පු: රූපධාතූපග සත්ත්‍වයෝ ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: රූපොපග සත්ත්‍වයෝ ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

5\. ස. පු: රූපධාතුයෙහි ලා සත්ත්‍වයෝ ජාතිවශයෙන් දනත් ද, ජරාවශයෙන් දිරත් ද, මරණවශයෙන් මියත් ද, ච්‍යුතිවශයෙන් සැව යෙත් ද, පිළිසඳ විසින් උපදිත් ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: රූපයෙහි ලා සත්ත්‍වයෝ ජාතිවශයෙන් දනත් ද? ජරාවශයෙන් දිරත් ද, මරණවශයෙන් මියත් ද, ච්‍යුතිවශයෙන් සැව යෙත් ද, පිළිසඳ විසින් උපදිත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …

6\. ස. පු: රූපධාතුයෙහි රූප, වෙදනා, සංඥා, සංස්කාර, විඥාන ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: රූපයෙහි රූප, වෙදනා, සංඥා, සංස්කාර, විඥාන ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

7\. ස. පු: රූපධාතු පඤ්චවොකාරභව වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: රූපය පඤ්චවොකාරභව වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

8\. ස. පු: (තොප මතයෙහි) රූපී ධර්‍මයෝ රූපධාතුයහ. කාමධාතුයෙහිත් රූප ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: එ කාමධාතු මැ ඒ රූපධාතු වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

9\. ස. පු: ඒ කාමධාතුම මැ ඒ රූපධාතු වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කාමධාතුයෙන් සමන්වාගත පුද්ගල කාමභව හා රූපභව දෙකින් සමන්වාගත වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

රූපධාතුකථා නිමි.
