# 8.9. රූපං කම්මන්ති කථා

**\[මහිංසාසක සම්මිතීය නිකායවාසීහු කායවාග්විඥප්ති සඞ්ඛ්‍යාත රූපය මැ කායකර්‍ම වාක්කර්‍ම යයි ද, එ ද කුශල සමුත්‍ථාන ඇතියේ ‘කුශලකර්‍ම’ යයි ද, අකුශල සමුත්‍ථාන ඇතියේ ‘අකුශලකර්‍ම’ යයි ද ගනිති. උන්ගේ වාද බිඳුනට ‘රූපං කම්මන්ති කථා’ එයි.\]**

1\. ස. පු: කුශල චිත්තයා විසින් සමුත්‍ථිත (නගන ලද) කායකර්‍මරූප කුශල වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: සාරම්මණ වේ ද? ඒ කුශලරූපයට ආවර්‍තනා, ආභොග, සමන්වාහාර, මනසිකාර, ප්‍රාර්‍ථනා - ප්‍රණිධි සඞ්ඛ්‍යාත චෙතනා ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

2\. ස. පු: අනාරම්මණ නො වේ ද? ඒ කුශල රූපයට ආවර්‍තනා, ආභොග, සමන්වාහාර, මනසිකාර, ප්‍රාර්‍ථනා-ප්‍රණිධි සඞ්ඛ්‍යාත චෙතනා නැත්තේ නො?

ප. පටිඤ්ඤා: එසේ යැ.

අනු. ඨ: ඉදින් අනාරම්මණ වේ නම්, ඒ කුශලරූපයට ආවර්‍තනා, … ප්‍රාර්‍ථනා - ප්‍රණිධි සඞ්ඛ්‍යාත චෙතනා නැත් නම්,

අනු. පා: භවත,’කුශලචිත්තයා විසින් සමුත්‍ථිත කායකර්‍මරූප කුශල වේ ය’ යි නො කියැයුතු මැ යැ.

3\. ස. පු: කුශලචිත්තයා විසින් සමුත්‍ථිත ස්පර්‍ශ කුශල වේ ද? සාරම්මණ වේ ද? එයට ආවර්‍තනා, … ප්‍රණිධි ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කුශලචිත්තයා විසින් සමුත්‍ථිත කායකර්‍මරූප සාරම්මණකුසල වේ ද? එයට ආවර්‍තනා … ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

4\. ස. පු: කුශලචිත්තයා විසින් සමුත්‍ථිත වෙදනා … සංඥා - චෙතනා- ශ්‍රද්ධා - වීර්‍ය්‍ය - ස්මෘති - සමාධි … ප්‍රඥා - සාලම්බන කුශල වේ ද? එයට ආවර්‍තනා … ප්‍රණිධි ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කුශලචිත්තයා විසින් සමුත්‍ථිත කායකර්‍මරූප සාලම්බන කුශල වේ ද? එයට ආවර්‍තනා … ප්‍රණිධි ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

5\. ස. පු: කුශලචිත්තයා විසින් සමුත්‍ථිත කායකර්‍මරූප අනාලම්බන කුශල වේ ද? එයට ආවර්‍තනා, … ප්‍රණිධි නැද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කුශලචිත්තයා විසින් සමුත්‍ථිත ස්පර්‍ශ අනාලම්බන කුශල වේ ද? එයට ආවර්‍තනා … ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

6\. ස. පු: කුශලචිත්තයා විසින් සමුත්‍ථිත කායකර්‍මරූප අනාලම්බන කුශල වේ ද? එයට ආවර්‍තනා … ප්‍රණිධි නැද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කුශලචිත්තයා විසින් සමුත්‍ථිත වෙදනා … සංඥා - චෙතනා … ශ්‍රද්ධා … වීර්‍ය්‍ය … ස්මෘති … සමාධි … ප්‍රඥා අනාලම්බන කුශල වේ ද? එයට ආවර්‍තනා … ප්‍රණිධි නැද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

7\. ස. පු: කුශලචිත්තයා විසින් සමුත්‍ථිත කායකර්‍මරූප කුශල වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කුශලචිත්තයා විසින් සමුත්‍ථිත යම් මැ රූපයෙක් ඇත් නම්, ඒ හැම කුශල වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

8\. ස. පු: කුශලචිත්තයා විසින් සමුත්‍ථිත කායකර්‍මරූප කුශල වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කුශලචිත්තයා විසින් සමුත්‍ථිත රූපායතන කුශල වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

9\. ස. පු: කුශලචිත්තයා විසින් සමුත්‍ථිත වූ කායකර්‍මරූපය කුශල වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කුශලචිත්තයා විසින් සමුත්‍ථිත වූ ශබ්දායතනය … ගන්‍ධායතනය … රසායතනය … ස්ප්‍රෂ්ටව්‍යායතනය … පෘථිවිධාතු … ආපොධාතු … තෙජොධාතු … වායොධාතු කුශල වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

10\. ස. පු: කුශලචිත්තයා විසින් සමුත්‍ථිත වූ රූපායතනය අව්‍යාකෘත (කුශලාදි විසින් නො කියන ලද) වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කුශලචිත්තයා විසින් සමුත්‍ථිත වූ කායකර්‍මරූපය අව්‍යාකෘත වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

11\. ස. පු: කුශලචිත්තයා විසින් සමුත්‍ථිත වූ ශබ්දායතනය … ගන්‍ධායතනය … රසායතනය … ස්ප්‍රෂ්ටව්‍යායතනය … පෘථිවීධාතු … ආපොධාතු … තෙජොධාතු … වායොධාතු අව්‍යාකෘත වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කුශලචිත්තයා විසින් සමුත්‍ථිත වූ කායකර්‍මරූපය අව්‍යාකෘත වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

12\. ස. පු: කුශලචිත්තයා විසින් සමුත්‍ථිත වූ කායකර්‍මරූපය අනාලම්බන කුශල වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: භාග්‍යවතුන් වහන්සේ විසින වදාරන ලද්දේ මැ නු:

“මහණෙනි, ත්‍රිවිධ කායසඤ්චෙතනාව දුක් වූ දුඃඛවිපාක වූ අකුශල කායකර්‍ම වෙයි. මහණෙනි, චතුර්විධ වාක්සඤ්චෙතනාව දුක් වූ දුඃඛවිපාක වූ අකුශල වාක්කර්‍ම වෙයි. මහණෙනි, ත්‍රිවිධ මනඃසඤ්චෙතනාව දුක් වූ දුඃඛවිපාක වූ අකුශල මනඃකර්‍ම වෙයි. මහණෙනි, ත්‍රිවිධ කායසඤ්චෙතනාව සුඛ වූ සුඛවිපාක වූ කුශල කායකර්‍ම වෙයි. මහණෙනි, චතුර්විධ වාක්සඤ්චෙතනාව සුඛ වූ සුඛවිපාක වූ කුශලවාක්කර්‍ම වෙයි. මහණෙනි, ත්‍රිවිධ වූ මනඃසඤ්චෙතනාව සුඛ වූ සුඛවිපාක වූ කුශලමනඃකර්‍ම වේ යැ” යි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ නො?

ප. පටිඤ්ඤා: එසේ යැ.

අනු. ආ: එකරුණින් ‘රූපය කර්‍ම යැ’ යි නො කියැයතු යැ.

76\. ස. පු: රූපය කර්‍ම වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: භාග්‍යවතුන් වහන්සේ විසින වදාරන ලද්දේ මැ නො:

“ඉදින් ආනන්දය, මේ සමිද්ධි මොඝපුරුෂපාටලීපුත්‍ර පරිව්‍රාජක විසින් මෙසේ පුළුවුස්නා ලද්දේ මෙසේ පවසන්නේ යැ. (කෙසේ යැ යත්:) ඇවැත්නි, පාටලීපුත්‍රය, කයින් වචසින් මනසින් සුඛවෙදනාවට හිත සඤ්චෙතනික කර්‍ම කොට හේ සුව විඳියි. ඇවැත්නි, පාටලීපුත්‍රය, කයින් වචසින් මනසින් දුඃඛවෙදනාවට හිත සඤ්චෙතනික කර්‍ම කොට හේ දුක් විඳියි. ඇවැත්නි, පාටලීපුත්‍රය, කයින් වචසින් මනසින් උපෙක්‍ෂා වෙදනාවට හිත සඤ්චෙතනික කර්‍ම කොට හේ නො දුක් නො සුව වෙදනා විඳී යයි. ආනන්දය, මෙසේ පවසන සමිද්ධි මොඝපුරුෂ පාටලීපුත්‍ර පරිව්‍රාජකහට මොනොවට පවසන්නේ වේ යයි” වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ නො?

ප. පටිඤ්ඤා: එසේ යැ.

ප. අනු: එකරුණින් ‘රූපය කර්‍ම යැ’ යි නො කියැයතු යැ.

රූපං කම්මන්තිකථා නිමි.
