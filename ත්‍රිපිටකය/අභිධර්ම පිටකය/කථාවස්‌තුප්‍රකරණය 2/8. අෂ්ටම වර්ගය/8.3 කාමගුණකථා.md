# 8.3 කාමගුණකථා

**\[පඤ්චිමෙ භික්ඛවෙ කාමගුණා’ යි වදාළ වචන නිසා ‘පස්කම්ගුණයෝ මැ කාමධතුයහ’ යි විවාද ගත් පූර්‍වශෛලීය ආදි නිකාය වාසීන්ගේ වාද බිඳුනට කාමගුණකථා එයි.\]**

1\. ස. පු: පස්කම්ගුණ මැ කාමධාතු වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: තත්ප්‍රතිසංයුක්ත ඡන්දය ඇත්තේ නො වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

අනු. ඨ: ඉදින් තත්ප්‍රතිසංයුක්ත ඡන්දය ඇත් නම්,

අනු. පා: භවත, ‘පස්කම්ගුණ මැ කාමධාතු’යයි නො කියැයුතු මැ යැ

2\. ස. අනු: තත්ප්‍රතිසංයුක්ත රාගය, තත්ප්‍රතිසංයුක්ත ඡන්දය, තත්ප්‍රතිසංයුක්ත ඡන්දරාගය, තත්ප්‍රතිසංයුක්ත සංකල්පය, තත්ප්‍රතිසංයුක්ත රාගය, තත්ප්‍රතිසංයුක්ත සංකල්පරාගය, තත්ප්‍රතිසංයුක්ත ප්‍රීතිය, තත්ප්‍රතිසංයුක්ත සෞමනස්‍යය, තත්ප්‍රතිසංයුක්ත ප්‍රීතිසෞමනස්‍යය ඇත්තේ නො වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

අනු. ඨ: ඉදින් තත්ප්‍රතිසංයුක්ත ප්‍රීතිසෞමනස්‍යය ඇත් නම්,

අනු. පා: භවත, ‘පස්කම්ගූණ මැ කාමධාතු’යයි නො කියැයුතු මැ යැ.

3\. ස. පු: පස්කම්ගූණ මැ කාමධාතු වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: මිනිසුන්ගේ ඇස කාමධාතු නො වන්නේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: මිනිසුන්ගේ කන … මිනිසුන්ගේ නැහැය … මිනිසුන්ගේ දිව … මිනිසුන්ගේ කය … මිනිසුන්ගේ මනස කාමධාතු නො වන්නේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

4\. ස. පු: මිනිසුන්ගේ මනස කාමධාතු නො වන්නේ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: භාග්‍යවතුන් වහන්සේ විසින් වදාරනලද මැ නො:

“ලොකයෙහි මනස සවැනි කොට ඇති පඤ්චකාමගුණයෝ දන්නාලදහ. එහි ඡන්දය පහා මෙසේ දුකින් මිදේ” යයි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ නු?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: එකරුණින් ‘මිනිසුන්ගේ මනස කාමධාතු වෙති’ යි නො කියැයුතු යැ.

5\. ස. පු: පස්කම්ගුණ මැ කාමධාතු වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කාමගුණ භව, ගති, සත්ත්‍වාවාස, සංසාර, යොනි, විඥානස්ථිති, ආත්මභාවප්‍රතිලාභ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: කාමගුණොපග කර්‍ම ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: කාමගුණොපග සත්ත්‍වයෝ ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: කාමගුණයෙහි සත්ත්‍වයෝ ජාතිවශයෙන් දනත් ද, ජරාවශයෙන් දිරත් ද, මරණවශයෙන් මියත් ද, ච්‍යුතිවශයෙන් සැව යෙත් ද, ප්‍රතිසන්‍ධිවශයෙන් උපදිත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: කාමගුණයෙහි රූප, වෙදනා, සංඥා, සංස්කාර, විඥාන ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: කාමගුණ පඤ්චවොකාරභව වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: කාමගුණයෙහි සම්‍යක්සම්බුද්ධයෝ උපදිත් ද, ප්‍රත්‍යෙකසම්බුද්ධයෝ උපදිත් ද, ශ්‍රාවකයුගමය උපදී ද?

ප. පටි: එසේ නො කියැයුතු යැ …

6 .ස. පු: කාමධාතු වනාහි භව, ගති, සත්ත්‍වාවාස, සංසාර, යොනි, විඥානස්ථිති, ආත්මභාවප්‍රතිලාභ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කාමගුණ වනාහි භව, ගති, සත්ත්‍වාවාස, සංසාර, යොනි, විඥානස්ථිති, ආත්මභාවප්‍රතිලාභ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

7\. ස. පු: කාමධාතූපග කර්‍ම ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කාමගුණොපග කර්‍ම ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

8\. ස. පු: කාමධාතූපග සත්ත්‍වයෝ ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කාමගුණොපග සත්ත්‍වයෝ ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

9\. ස. පු: කාමධාතුයෙහි සත්ත්‍වයෝ ජාතිවශයෙන් දනත් ද, ජරාවශයෙන් දිරත් ද, මරණවශයෙන් මියත් ද, ච්‍යුතිවශයෙන් සැවෙත් ද, පිළිසිඳ විසින් උපදිත් ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කාමගුණයෙහි සත්ත්‍වයෝ ජාතිවශයෙන් දනත් ද, ජරාවශයෙන් දිරත් ද, මරණවශයෙන් මියත් ද, ච්‍යුතිවශයෙන් සැවෙත් ද, පිළිසඳ විසින් උපදිත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …

10\. ස. පු: කාමධාතුයෙහි රූප, වෙදනා, සංඥා, සංස්කාර, විඥාන ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කාමගුණයෙහි රූප, වෙදනා, සංඥා, සංස්කාර, විඥාන ඇද්ද?

ප. පටි: එසේ නො කියැයුතු යැ …

11\. ස. පු: කාමධාතු පඤ්චවොකාරභාව වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කාමගුණ පඤ්චවොකාරභව වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

12\. ස. පු: කාමධාතුයෙහි සම්‍යක් සම්බුද්ධයෝ උපදිත් ද? ප්‍රත්‍යෙකසම්බුද්ධයෝ උපදිත් ද? ශ්‍රාවක යුග්මය උපදනේ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කාමගුණයෙහි සම්‍යක්සම්බුද්ධයෝ උපදිත් ද, ප්‍රත්‍යෙකසම්බුද්ධයෝ උපදිත් ද, ශ්‍රාවකයුග්මය උපදනේ වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

13\. ප. පු: ‘පස්කම්ගුණයෝ මැ කාමධාතු නො වෙති’ යි නො කියැයුතු වේ ද?

ස. පටිඤ්ඤා: එසේ යැ.

ප. අනු: භාග්‍යවතුන් වහන්සේ විසින් වදාරනලද මැ නු:

“මහණෙනි, මේ කාමගුණයෝ පසෙකි. කවර පසෙක යත්: ඉෂ්ට කාන්ත මනාප ප්‍රිය ස්වරූප කාමොපසංහිත රඤ්ජනාර්‍හ චක්‍ෂුර්විඥෙය රූපයෝ යැ … ශ්‍රොත්‍රවිඥෙය ශබ්දයෝ යැ … ඝ්‍රාණවිඥෙය ගන්‍ධයෝ යැ … ජිහ්වාවිඥෙය රසයෝ යැ … ඉෂ්ට කාන්ත මනාප ප්‍රියස්වරූප කාමොපසංහිත රඤ්ජනාර්‍හ කායවිඥෙය ස්පර්‍ශයෝ යැ. මහණෙනි, මොහු යැ පඤ්චකාමගුණයෝ” යි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ නු?

ස. පටිඤ්ඤා: එසේ යැ.

පටි. ආ: එකරුණින් ‘පස්කම් ගුණයෝ මැ කාමධාතු වෙති’ යි

කාමගුණකථා නිමි.
