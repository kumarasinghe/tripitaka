# 8.11. කම්මහෙතු කථා

**\[‘යම් රහත් කෙනෙක් අතීත භවයෙහි දී රහත්හට අභ්‍යාඛ්‍යාන කළාහු වෙත් ද, ඔවහු තත් හෙතුයෙන් රහත් බැවින් පිරිහෙති’ යි ලබ්ධි ගත්තාහු ඇති. පූර්‍වශෛලීය සම්මිතීය නිකාය වාසීන් සෙයිනි. උන්ගේ වාද බිඳුනට ‘කම්මහෙතුකථා’ එයි.\]**

1\. ස. පු: කර්‍මහෙතුයෙන් රහත් රහත් බැවින් පිරිහී යේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කර්‍මහෙතුයෙන් සොවන්පුඟුල් සොවත්ඵලයෙන් පිරිහී යේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

2\. ස. පු: කර්‍මහෙතුයෙන් රහත් රහත් බැවින් පිරිහී යේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කර්‍මහෙතුයෙන් සෙදගැමි පුඟුල් … අනගැමි පුඟල් අනගැමි ඵලයෙන් පිරිහී යේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

3\. ස. පු: කර්‍මහෙතුයෙන් ශ්‍රොතාපන්න පුද්ගල ශ්‍රොතාපත්තිඵලයෙන් පිරිහී නො යේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කර්‍මහෙතුයෙන් අර්‍හත්පුද්ගල අර්‍හත්ත්‍වයෙන් පිරිහී නො යේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

4\. ස. පු: කර්‍මහෙතුයෙන් සකෘදාගාමී පුද්ගල … අනාගාමී පුද්ගල අනාගාමී ඵලයෙන් පිරිහී නො යේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: කර්‍මහෙතුයෙන් අර්‍හත් පුද්ගල අර්‍හත්ත්‍වයෙන් පිරිහී නො යේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

5\. ස. පු: කර්‍මහෙතුයෙන් අර්‍හත්පුද්ගල අර්‍හත්ත්‍වයෙන් පිරිහී යේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ප්‍රාණඝාතකර්‍මහෙතුයෙන් වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: අදත්තාදානකර්‍මහෙතුයෙන් වේ ද … කාමමිථ්‍යාචාර කර්‍ම හෙතුයෙන් වේ ද … මෘෂාවාදකර්‍මහෙතුයෙන් වේ ද, … පිශුනවචනකර්‍මහෙතුයෙන් වේ ද, … පරුෂවචනකර්‍මහෙතුයෙන් වේ ද, … සම්ඵප්‍රලාපකර්‍මහෙතුයෙන් වේ ද, … මාතෘඝාතකර්‍මහෙතුයෙන් වේ ද, … පිතෘඝාතකර්‍මහෙතුයෙන් වේ ද, … අර්‍හත්ඝාතකර්‍ම හෙතුයෙන් වේ ද … රුධිරොත්පාදකර්‍මහෙතුයෙන් වේ ද, … සඞ්ගභෙදකර්‍මහෙතුයෙන් වේ ද?

ප. පටි: එසේ නො කියැයුතු යැ …

ස. අනු: කවර කර්‍මයක්හුගේ හෙතුයෙන් වේ ද ?

ප. වි: රහත්හට අභ්‍යාඛ්‍යාන කෙරේ ද, එහෙයිනැ යි.

6\. ස. පු: රහත්හට අභ්‍යාඛ්‍යාන කර්‍ම හෙතුයෙන් අර්‍හත්පුද්ගල අර්‍හත්ත්‍වයෙන් පිරිහී යේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: යම් කෙනෙක් රහත්හට අභ්‍යාඛ්‍යාන කෙරෙත් ද, එ හැම දෙනා අර්‍හත්ත්‍වය ප්‍රත්‍යක්‍ෂ කෙරෙත් ද?

ප. පටි: එසේ නො කියැයුතු යැ …

කම්මහෙතුකථා නිමි.
