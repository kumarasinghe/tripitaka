# 2.7. චිත්තට්ඨිතිකථා

**\[සමාපත්තිචිත්ත හා භවාඞ්ගචිත්තය හා දෙදෙන අනුප්‍රබන්‍ධ විසින් වැටෙනු දැක එක් මැ සිතෙක බොහෝකල් සිටිතියි ලබ්ධි ගත් වාදීහු ඇත. අන්‍ධකයන් සෙයිනි. උන්ගේ වාද බිඳුනට චිත්තට්ඨිතිකථා එයි.\]**

1\. ස. පු: එක් සිතෙක් දවසක් සිටුනේ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: අඩ දිනක් උත්පාදක්‍ෂණයෙහිත් අඩදිනක් ව්‍යයක්‍ෂණයෙහිත් සිටුනේ වේ ද?

ප. පටි: එසේ නො කියැයුතු මැ යැ ...

2\. ස. පු: එක් සිතෙක් දෙදිනක් සිටුනේ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: දවසක් උත්පාදක්‍ෂණයෙහිත් දවසක් ව්‍යයක්‍ෂණයෙහිත් සිටුනේ වේ ද?

ප. පටි: එසේ නො කියැයුතු මැ යැ ...

3\. ස. පු: එක් සිතෙක් සතර දිනක් සිටුනේ වේ ද? ... අටදිනක් සිටුනේ වේ ද ... දස දිනක් සිටුනේ වේ ද ... විසිදිනක් සිටුනේ වේ ද ... මසක් සිටුනේ වේ ද ... දෙමසක් සිටුනේ වේ ද ... සාරමසක් සිටුනේ වේ ද ... අටමසක් සිටුනේ වේ ද ... දසමසක් සිටුනේ වේ ද ... හවුරුද්දක් සිටුනේ වේ ද ... දෑවුරුද්දක් සිටුනේ වේ ද ... සාරවසක් සිටුනේ වේ ද ... අට වසක් සිටුනේ වේ ද ... දසවසක් සිටුනේ වේ ද ... විසිවසක් සිටුනේ වේ ද ... තිස්වසක් සිටුනේ වේ ද ... සතලිස් වසක් සිටුනේ වේ ද ... පනස්වසක් සිටුනේ වේ ද ... සියක් වසක් සිටුනේ වේ ද ... දෙසියවසක් සිටුනේ වේ ද ... සිව්සියවසක් සිටුනේ වේ ද ... පන්සියවසක් සිටුනේ වේ ද ... දහස්වසක් සිටුනේ වේ ද ... දෙදහස්වසක් සිටුනේ වේ ද ... සාරදහස්වසක් සිටුනේ වේ ද ... අටදහස්වසක් සිටුනේ වේ ද ... සොළොස්දහස් වසක් සිටුනේ වේ ද ... කපක් සිටුනේ වේ ද ... දෙකපක් සිටුනේ වේ ද ... සාරකපක් සිටුනේ වේ ද ... අටකපක් සිටුනේ වේ ද ... සොළොස් කපක් සිටුනේ වේ ද ... දෙතිස්කපක් සිටුනේ වේ ද ... සූසැට කපක් සිටුනේ වේ ද ... පන්සියකපක් සිටුනේ වේ ද ... දහසක් කප් සිටුනේ වේ ද ... දෙදහස් කප් සිටුනේ වේ ද ... සාරදහස්කප් සිටුනේ වේ ද ... අටදහස්කප් සිටුනේ වේ ද ... සොළොස්දහස්කප් සිටුනේ වේ ද ... විසිදහස්කප් සිටුනේ වේ ද ... සතලිස්දහස්කප් සිටුනේ වේ ද ... සැටදහස්කප් සිටුනේ වේ ද ... සුවාසූදහස්කප් සිටුනේ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: දෙසාලිස් දහස්කප් උත්පාදක්‍ෂණයෙහිත් දෙසාලිස් දහස්කප් ව්‍යයක්‍ෂණයෙහිත් සිටුනේ වේ ද?

ප. පටි: එසේ නො කියැයුතු මැ යැ ...

4\. ස. පු: එක් සිතෙක් දවසක් කල් සිටුනේ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. පු: එක් දින බොහෝ වර ඉපිද නිරුද්ධ වන අන්‍ය ධර්‍මයෝ ඇද්ද?

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: ඒ ධර්‍මයෝ සිතට වඩා වහා පෙරැළෙනසුලු වෙත් ද?

ප. පටි: එසේ නො කියැයුතු මැ යැ ...

5\. ස. පු: ඒ ධර්‍මයෝ සිතට වඩා වහා පෙරැළෙනසුලු වෙත් ද?

ප. පටිඤ්ඤා: එසේ යැ. (චිත්තයාගේ දීර්‍ඝස්ථිතිය පිණිස ප්‍රතිඥා යි.)

ස. අනු: භාග්‍යවතුන් වහන්සේ විසින් “මහණෙනි, යමෙක් මෙසේ වහා ඉපැද නස්නාසුලු වේ නම්, සිත බඳු අන් එක ධර්‍මයකුදු මම නො දක්මි. මහණෙනි, සිත කෙතරම් වහා පෙරැළෙනසුලු වේ ද යත්: (එයට) උපමාවකුදු අතිශය දුෂ්කර වෙති”යි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ නු?

ප. පටිඤ්ඤා: එසේ යැ.

අනු. පා: එයින් මැ “ඒ දහම්හු සිතට වඩා වහා පෙරැළෙන සුල්ලහ”යි නො කියැයුතු.

ස. පු: තෙ ධම්මා චිත්තෙන ලහුපරිවත්තාති.

ප. පටිඤ්ඤා: එසේ යැ.

ස. අනු: භාග්‍යවතුන් වහන්සේ විසින් වදාරන ලද මැනු: “මහණෙනි, යම් පරිදි මකුළුයෙක් අරනැ මහවල්හි ඇවිදුනේ ශාඛාවක් ගන්නේ වේ ද, එ පියා අනෙකක් ගන්නේ වේ ද, මහණෙනි, එ පරිදි මැ ‘චිත්ත’ යයි ද ‘මන’ යයි ද ‘විඤ්ඤාණ’ යයි ද යම් දැයෙක් කියන ලද ද, එය රැය ද දහවල ද අනෙකෙක්ව මැ උපදනේ යැ, අනෙකෙක්ව මැ නිරුද්ධ වේ”යයි වදාළ සූත්‍රාන්තයෙක් ඇත්තේ මැ නු?

ප. පටිඤ්ඤා: එසේ යැ.

ස. නිග: එ කරුණින් ‘ඒ දහම්හු සිතට වඩා වහා පෙරළෙන සුල්ලහ’යි නො කියැයුතු මැ යැ.

6\. ස. පු: එක් සිතෙක් දවසක් සිටුනේ වේ ද?

ප. පටිඤ්ඤා: එසේ යැ.
