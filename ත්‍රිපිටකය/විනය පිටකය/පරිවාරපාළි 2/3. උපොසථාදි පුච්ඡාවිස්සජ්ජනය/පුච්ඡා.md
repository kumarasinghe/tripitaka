# පුච්ඡා

1\. උපොසථ කර්මයෙහි ආදිය කවරෙ යැ, මධ්‍යය කවරෙයැ, අවසානය කවරෙ යැ, පවාරණ කර්මයෙහි ආදිය කවරෙ යැ, මධ්‍යය කවරෙ යැ, අවසානය කවරෙ යැ, තජ්ජනීය කර්මයෙහි ආදිය කවරෙ යැ, මධ්‍යය කවරෙ යැ, අවසානය කවරෙ යැ, නියස්ස කර්මයෙහි ... පබ්බාජනීය කර්මයෙහි ... පටිසාරණීය කර්මයෙහි ... උක්ඛෙපනීය කර්මයෙහි ... පරිවාසදානයෙහි ... මූලායපටිකස්සන කර්මයෙහි ... මානත්ත දානයෙහි ... අබ්හාන කර්මයෙහි ... උපසම්පදා කර්මයෙහි ආදිය කවරෙ යැ. මධ්‍යය කවරෙ යැ, අවසානය කවරේ යැ යත්:

2\. තජ්ජනීය කර්මයාගේ සන්හිඳුම්හි ආදිය කවරෙ යැ, මධ්‍යය කවරෙ යැ, අවසානය කවරෙ යැ, නියස්ස කර්මයාගේ සන්හිඳුම්හි ... පබ්බාජනීය කර්මයාගේ සන්හිඳුම්හි ... පටිසාරණීය කර්මයාගේ සන්හිඳුම්හි ... උක්ඛෙපනීය කර්මයාගේ සන්හිඳුම්හි ආදිය කවරෙ යැ, මධ්‍යය කවරෙ යැ, අවසානය කවරෙ යැ යත්:

3\. සතිවිනයයෙහි ආදිය කවරෙ යැ, මධ්‍යය කවරෙ යැ. අවසානය කවරෙ යැ, අමූළ්හවිනයෙහි ... තස්සපාපිය්‍යසිකායෙහි .... තිණවත්ථාරකයෙහි ... භික්ඛුනොවාදකසම්මුතියෙහි ... තිචීවරඅවිප්පවාස සම්මුතියෙහි ... සන්ථතසම්මුතියෙහි ... රුපියඡඩ්ඩකසම්මුතියෙහි ... සාටියගාහාපකසම්මුතියෙහි ... පත්තගාහාපකසම්මුතියෙහි ... දණ්ඩසම්මුතියෙහි ... සික්කාසම්මුතියෙහි ... දණ්ඩසික්කාසම්මුතියෙහි ආදිය කවරෙ යැ, මධ්‍යය කවරෙ යැ, අවසානය කවරෙ යැ යත්:

පුච්ඡා නිමි.
