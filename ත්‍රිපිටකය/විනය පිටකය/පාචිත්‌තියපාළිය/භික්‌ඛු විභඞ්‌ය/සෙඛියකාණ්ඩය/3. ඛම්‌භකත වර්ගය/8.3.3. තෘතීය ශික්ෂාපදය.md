# 8.3.3. තෘතීය ශික්ෂාපදය

ශ්‍රාවස්තිනිදාන යි - එසමයෙහි සවග මහණහු හිස සහිත වැ පෙරෙවැ ඇතුළුගමෙහි යෙත් ...

**"හිස වසා පෙරෙවැ ඇතුළුගමෙහි නො යන්නෙමි යි හික්මියැ යුතු" යි.**

හිස වසා පෙරෙවැ ඇතුළුගමෙහි නො යායුතු යි. යමෙක් අනාදරිය පිණිස හිස වසා පෙරෙවැ ඇතුළුගමෙහි යේ නම් දුකුළා ඇවැත් වේ.

නොසිතා කරන්නහුට ... ආදිකම්මිකහට ඇවැත් නැති.

තෘතීය ශික්‍ෂාපද යි.
