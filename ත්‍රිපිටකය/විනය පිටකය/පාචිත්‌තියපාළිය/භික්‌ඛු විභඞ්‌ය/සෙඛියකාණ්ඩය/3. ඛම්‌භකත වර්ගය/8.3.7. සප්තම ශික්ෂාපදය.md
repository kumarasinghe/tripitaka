# 8.3.7. සප්තම ශික්ෂාපදය

ශ්‍රාවස්තිනිදාන යි - එසමයෙහි සවග මහණහු බැහැරැ දමනු කැමැතියන් මෙන් නො සකසා (සිහි නො එළවා) පිඬුවා පිළිගනිත් ...

**"සිහි එළවා පිඬුවා පිළිගන්නෙමි යි හික්මියැ යුතු" යි.**

සකසා පිඬුවා පිළිගතයුතු යි. යමෙක් අනාදරිය පිණිස බැහැරැ දමනු කැමැතියකු මෙන් නො සකසා පිඬුවා පිළිගනී නම් දුකුළා ඇවැත් වේ.

නොසිතා කරන්නහුට ... ආදිකම්මිකහට ඇවැත් නැති.

සප්තම ශික්‍ෂාපද යි.
