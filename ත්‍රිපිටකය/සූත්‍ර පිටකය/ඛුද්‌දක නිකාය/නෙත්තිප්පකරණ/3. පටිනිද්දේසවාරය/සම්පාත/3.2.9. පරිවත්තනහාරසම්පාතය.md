# 3.2.9. පරිවත්තනහාරසම්පාතය

එහි පරිවත්තනහාරසම්පාත කවරෙ යැ : “තස්මා රක්ඛිතචිත්තස්ස සම්මාසඞ්කප්ප ගොචරො” යනාදි ගාථා යි. ලෞකික ශමථවිදර්ශනා වැඩූ කල්හි (‘අරක්ඛිතෙන චිත්තෙන’ යනාදි ගාථාවට නොහොත් විභත්තිහාරයෙහි දැක්වූ අකුශලපක්ෂයට) ප්‍රතිපක්ෂවශයෙන් නිරෝධය ඵල යැ දුක්ඛය පරිඥාත යැ සමුදය ප්‍රහීණ යැ මාර්ගය භාවිත යි.

පරිවත්තනහාරසම්පාතය යොදනලදී.
