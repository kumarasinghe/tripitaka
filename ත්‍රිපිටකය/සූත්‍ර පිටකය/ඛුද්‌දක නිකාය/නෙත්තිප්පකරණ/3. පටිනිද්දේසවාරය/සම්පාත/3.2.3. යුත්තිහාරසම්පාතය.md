# 3.2.3. යුත්තිහාරසම්පාතය

ඒ හාරසම්පාතයන් කෙරෙහි යුක්තිහාරසම්පාතය කවරෙ යැ :

1\. (අරක්ෂිත සිත් ඇතියේ මාරවශී වැ සසර මැ සිටුනේ යැ,) එහෙයින් සතිසංවරයෙන් රක්නාලද සිත්ඇතිවන්නේ යැ නෙක්ඛම්මාදි සම්‍යක්සංකල්ප ගෝචර කොට ඇතිවන්නේ යැ (කම්මස්සකතා යථාභූතඤාණලක්ෂණ) සම්මාදිට්ඨිය පෙරටුකොට ඇතියේ (සමපනස්) උදයව්‍යය දැන රහත්මඟින් ථීනමිද්ධය නැසූ ක්ෂිණාස්‍රව මහණ සියලු දුර්ගති දුරුකරන්නේ යි.

රක්ඛිතචිත්තස්ස සම්මාසඞ්කප්පගොචරො යනු : රක්නාලද සිත් ඇතියේ සම්‍යක්සංකල්ප ගෝචර කොට ඇත්තේ වන්නේ යැ යන මේ අර්ථය යෙදෙයි, සම්‍යක්සංකල්ප ගොචර කොට ඇති පුද්ගල සම්‍යග්දෘෂ්ටි වන්නේ යැ යනු යෙදෙයි, සම්මාදිට්ඨිසංඛ්‍යාත විදර්ශනාඥානය පෙරටු ඇතිව වසනුයේ උදයව්‍යය ප්‍රතිවේධ කරන්නේ යැ යන අර්ථය යෙදෙයි, උදයව්‍යය ප්‍රතිවේධ කරනුයේ දුර්ගති සංඛ්‍යාත සියලු භව ගති දුරුකරන්නේ යැ යන අර්ථය යෙදෙයි, සියලු දුර්ගති දුරුකරනුයේ සියලු දුර්ගතිවිනිපාතභය (සසරභය ද) ඉක්මවන්නේ යැ යන අර්ථය යෙදේ යයි.

යුත්තිහාරසම්පාතය යොදන ලද්දේ යි.
