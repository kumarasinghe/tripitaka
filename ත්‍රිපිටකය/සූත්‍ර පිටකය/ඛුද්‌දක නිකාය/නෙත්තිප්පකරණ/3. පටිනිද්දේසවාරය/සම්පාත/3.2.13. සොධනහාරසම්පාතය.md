# 3.2.13. සොධනහාරසම්පාතය

එහි සොධනහාරසම්පාත කවරෙ යැ : “තස්මා රක්ඛිතචිත්තස්ස සම්මාසඞ්කප්ප ගොචරො” යනාදි ගාථා යැ. යම් දේශනායෙක දැනගනු කැමැති අර්ථ සංඛ්‍යාත ආරම්භය ශුද්ධ ද, ඒ ප්‍රශ්නය විසඳන ලද වෙයි, යළි යම් තැනෙක ආරම්භය ශුද්ධ නො වේ ද, ඒ ප්‍රශ්නය තව නො විසඳනලද්දේ වේ.

සොධනහාරසම්පාතය යොදන ලදී.
