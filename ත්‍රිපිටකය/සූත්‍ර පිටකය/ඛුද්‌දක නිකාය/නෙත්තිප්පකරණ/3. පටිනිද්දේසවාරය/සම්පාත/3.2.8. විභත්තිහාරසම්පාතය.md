# 3.2.8. විභත්තිහාරසම්පාතය

එහි විභත්තිහාරසම්පාත කවරෙ යැ : “තස්මා රක්ඛිතචිත්තස්ස සම්මාසඞ්කප්ප ගොචරො” යනාදි ගාථා යි. කුශලපක්ෂය චක්ඛුසංවරාදි කුශලපක්ෂයෙන් දැක්වියැ යුත්තේ යැ, අකුශලපක්ෂය චක්ඛුඅසංවරාදි අකුශලපක්ෂයෙන් දැක්වියැයුත්තේ යි.

විභත්තිහාරසම්පාතය යොදනලදී.
