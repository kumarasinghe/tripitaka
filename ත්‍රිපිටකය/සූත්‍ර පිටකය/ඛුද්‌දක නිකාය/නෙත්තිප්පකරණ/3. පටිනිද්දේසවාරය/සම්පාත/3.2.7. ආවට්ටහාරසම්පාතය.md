# 3.2.7. ආවට්ටහාරසම්පාතය

එහි ආවට්ටහාරසම්පාත කවරෙ යැ : “තස්මා රක්ඛිතචිත්තස්ස සම්මාසඞ්කප්ප ගොචරො” යනාදි ගාථාව නිදර්ශන යි. තස්මා රක්ඛිතචිත්තස්ස සම්මාසඞ්කප්පගොචරො යනු (නෙක්ඛම්මසඞ්කප්ප බහුල කොට ඇතියේ කසිණාදි වශයෙන් ලැබූ චිත්තෛකාග්‍රතායෙන් කෙලෙසුන්ගෙන් සිත රක්නාලද වේ නුයි) මේ සමථ යැ, සම්මාදිට්ඨිපුරෙක්ඛාරො යනු විදර්ශනා යැ (විදර්ශනාව ප්‍රඥා ප්‍රධාන හෙයිනි.) ඤත්වාන උදයබ්බයං යනු දුඃඛසත්‍යයාගේ පරිඥා යැ, ථීනමිද්ධාභිභූ භික්ඛු යනු සමුදය ප්‍රහාණ යැ, සබ්බා දුග්ගතියො ජහෙ යන මේ නිරෝධ යි. මේ චතුස්සත්‍යයෝ යි.

ආවට්ටහාරසම්පාතය යොදනලදී.
