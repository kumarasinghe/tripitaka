# 3.2.4. පදට්ඨානහාරසම්පාතය

එහි පදට්ඨානහාරසම්පාතය කවරෙ යත් : “තස්මා රක්ඛිතචිත්තස්ස සම්මාසඞ්කප්ප ගොචරො” යනාදි ගාථා නිදර්ශන යැ, තස්මා රක්ඛිතචිත්තස්ස යනු මේ ත්‍රිවිධ සුචරිතයනට පදට්ඨාන (ආසන්නකාරණ) වචන යැ. සම්මාසඞ්කප්පගොචරො යනු (සම්‍යක්සංකල්ප බහුලයහට නීවරණප්‍රහාණය සුකර නුයි) සමථයට ආසන්නකාරණ යැ, සම්මාදිට්ඨි පුරෙක්ඛාරො යනු ( දෙවැදෑරුම් සම්මාදිට්ඨියෙහි සිටියේ සසරදුක ඉක්මැවීමට උපාය වූ විදර්ශනා වඩා නුයි) විදර්ශනාවට පදට්ඨාන යැ, ඤත්වාන උදයබ්බයං යනු දර්ශනභූමියට පදට්ඨාන යැ, ථීනමිද්ධාභිභූ භික්ඛු යනු වීර්යයට පදට්ඨාන යැ, සබ්බා දුග්ගතියො ජහෙ යනු භාවනාවට පදට්ඨාන යි. (විශේෂකාරණ වන්නේ යි.)

පදට්ඨානහාරසම්පාතය යොදනලද්දේ යි.
