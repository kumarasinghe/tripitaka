# 3\. 4. ශාරීපුත්‍ර සූත්‍රය

1\. මා විසින් මෙසේ අසන ලදි. එක් සමයෙක්හි භාග්‍යවතුන් වහන්සේ සැවැත්නුවරැ දෙව්රම් නම් වූ අනේපිඬු සිටුහුගේ අරම්හි වැඩ වසන සේක. එ සමයෙහි වනාහි සැරියුත් තෙරණුවෝ භාග්‍යවතුන් වහන්සේට නොදුරෙහි පලක් බැඳ උඩුකය ඍජු කොට පිහිටුවා සමාධි අරමුණට සිහිය යොමු කොට එළවා තබා වැඩ හුන්නාහු වෙති. භාග්‍යවතුන් වහන්සේ පලක් බැඳ කය ඍජු කොට තබා අරමුණට අභිමුඛ කොට සිහිය එළවා නොදුරෙහි වැඩ හුන් සැරියුත් තෙරුන් දුටු සේක.

2\. ඉක්බිත්තෙන් භාග්‍යවතුන් වහන්සේ කරුණ දැන එ වේලෙහි මේ උදන් ඇනූ සේක.

“යම්සේ ගල්පව්වෙක් අචල වැ මොනොවට පිහිටියේ ද, එසේ ම මහණ තෙම මොහය ක්ෂීණ වීමෙන් පව්වක් මෙන් ‍නො සැලේ.”
