# (273) 3. කච්ඡප ජාතකය

424\. පිරූ බත් පාත්‍රයක් ගත්තකු මෙන් (මේ) කවරෙක් ද? (කාර්තික පූජායෙහි ලද) පූර්ණ පාත්‍රයක් අතැති බමුණකු මෙන් (මේ) කවරෙක් ද? කවර පෙදෙසක පිඬු සිඟා හැසුරුනේ ද? කවර ශ්‍රද්ධාවන්තයකු කරා එළැඹියේ ද?

425\. මම මූඪ වඳුරෙක්මි. ස්පර්ශ නො කටයුතු තැන් ස්පර්ශ කෙළෙමි. තෙපි මා මුදනු මැනැවි. මිදුණෙම් පර්වතයට යමි. තොපට යහපතක් වේවා!

426\. ඉදිබුවෝ කාෂ්‍යප ගෝත්‍රය ඇත්තෝ වෙති. වඳුරෝ කොණ්ඩඤ්ඤ ගෝත්‍ර ඇත්තෝ වෙති. කාශ්‍යපය, කොණ්ඩඤ්ඤයා මුදව, කාශ්‍යප ගෝත්‍රයෙහි උපන්හු විසින් කොණ්ඩඤ්ඤ ගෝත්‍රයෙහි උපන් මොහු හා නො සම මෛථුන කෘත්‍යයෙක් කරන ලදී.

3\. කච්ඡප ජාතක යි.
