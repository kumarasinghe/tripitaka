# 3.7. සෙලා ථෙරීගාථා

57\. ලොකයෙහි නිවනෙක් නැත, කාය විවෙකයෙන් කුමක් කෙරෙහි, කාමක්‍රීඩාරති වළඳ කර, පසු වැ අනුතාප ඇති නො වා. (මාරයාගේ වචන යි.)

58\. කාමයෝ විෂ පී සැත්හුල් වැනියහ. තෙල කාමයනට උපාදානස්කන්ධයෝ මස්ලොඹු වැනියහ. තෝ යම් කාමරතියක් කියෙහි ද ඒ දැන් මට අරතියෙකි.

59\. හැම විෂයෙහි තෘෂ්ණා නසන ලද. මොහස්කන්ධය පළන ලදි, මාරය මෙසේ දනුව. මාරය තෝ නටුයෙහි.

මෙසෙයින් සෙලා මෙහෙණ ගාථා කිවු යැ.

සෙලා ථෙරීගාථා යි.
