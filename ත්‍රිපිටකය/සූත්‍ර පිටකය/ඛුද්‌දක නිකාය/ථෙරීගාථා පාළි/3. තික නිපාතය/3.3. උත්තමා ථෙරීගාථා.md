# 3.3. උත්තමා ථෙරීගාථා

45\. නිර්වාණප්‍රාප්තියට මාර්ග වූ යම් සප්තබෝධ්‍යඞ්ග කෙනෙක් ඇද්ද ඒ හැම මා විසින් බුදුරදුන් දෙසන ලද පරිදි භාවිතයහ.

46\. ශූන්‍යතසමාපත්තිය ද අනිමිත්තසමාපත්තිය ද කැමති සේ ලබන, බුදුරදුන්ගේ ඖරස දුහීතෘ වූ මම් හැම කල්හි නිවන්හි ඇලුණුමු.

47\. දිව්‍යමය වූත් මනුෂ්‍යමය වූත් යම් වස්තු කාම කෙනෙක් ඇද්ද ඒ හැම කාමයෝ මා විසින් සමුච්ඡෙද විසින් සිඳුනා ලදහ. ජාති සංසාරය ක්ෂය කරන ලද, මින් මතු පුනර්භවයෙක් නැති.

මෙසෙයින් අන්‍ය උත්තමා නම් මෙහෙණක් ගාථා කිවු යැ.

උත්තමා ථෙරීගාථා යි.
