# 3.5. උබ්බරී ථෙරීගාථා

51\. උබ්බිරිය, තෝ ‘ම දියණියනි, ජීවන්තියෙනි’යි ආහ්වාන කෙරෙහි උබ්බිරිය (පළමු) තමා දනුව, තෙල සොහොන්හි සූවාසු දහසක් දෙනා දවන ලද, හැම දෙනා ජීවන්තිය සමාන නම් ඇත්තාහ. උන් කෙරෙහි කවරක සඳහා ශෝක කරව.

52\. යම් හෙයකින් සොවින් මඬනා ලද මා ගේ දුහිතෘශොකය බැහැර කළ සේක් ද, එහෙයින් දැක්ක නො හෙන හදවත් ඇසිරි කළ මාගේ ශොකශල්‍යය උපුළසේක් මැ යි.

53\. ඒ මම් අද උපුළ හුල් ඇති වැ තෘෂ්ණා රහිත වැ (එහෙයින් මැ කෙලෙස් පිරිනිවනින්) පිරිනිවියා සර්වඥ වූ බුදුනුදු දහමුදු සඟුනුදු සරණ කොට එළඹෙමි.

මෙසෙයින් උබ්බරී මෙහෙණ ගාථා කිවු යැ.

උබ්බරී ථෙරීගාථා යි.
