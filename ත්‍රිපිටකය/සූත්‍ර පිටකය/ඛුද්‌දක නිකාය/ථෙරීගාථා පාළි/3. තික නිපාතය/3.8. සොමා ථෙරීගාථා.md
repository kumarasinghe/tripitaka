# 3.8. සොමා ථෙරීගාථා

60\. බුද්ධාදි ඍෂින් විසින් පැමිණිය යුතු (තදන්‍යයන් විසින්) දුකසේ නිපදවාලියහෙන අර්හත්ත්ව සඞ්ඛ්‍යාත යම් පරමාශ්වාස කාරණයෙක් ඇද්ද එය දෑගිලි නුවණැති ස්ත්‍රිය විසින් පැමිණෙන්නට නො හැක්ක.

61\. සිත අර්හත්මාර්ග සමාධියෙන් මොනොවට සමාහිත කල්හි රහත් මාර්ගඥානය පවත්නා කල්හි සිව්සස් දහම් පරිඥා දි විසින් මොනොවට දක්මින් සිටිය දී ස්ත්‍රිභාවය අපට කුමක් කරන්නේ ද?

62\. හැම විෂයෙහි තෘෂ්ණා නසන ලද, මෝහස්කන්ධය පළන ලදි, මාරය මෙසේ දනුව, මාරය තෝ නටුයෙහි.

මෙසෙයින් සොමා මෙහෙණ ගාථා කිවු යැ.

සොමා ථෙරීගාථා යි.
