# 3 - 1. ගණනවාරය

ෂොඩශවස්තුක ආනාපානස්මෘති සමාධිය වඩන්නහුට සාතිරෙක දෙසියක් ඥානයෝ උපදිති. උවදුරු අරමුණු කොට පැවැති ඥානයෝ අට දෙනෙක, උපකාර අරමුණු කොට පැවැති ඥානයෝ අටදෙනෙක, උපක්ලෙශ අරමුණු කොට පැවැති ඥානයෝ අටළොසෙක, වොදානඥාන තුදුසෙක, සිහි ඇති වැ කරන්නහුගේ ඥාන දෙතිසෙක, සමාධිවශයෙන් පැවැති ඥානයෝ සූවිස්සෙක, විදර්‍ශනාවශයෙන් පැවැති ඥානයෝ දෙසැත්තෑවෙක, නිර්‍වෙදඥානයෝ අටදෙනෙක, නිර්‍වෙදයට අනුලොම් වූ ඥානයෝ අටදෙනෙක, නිර්‍වෙදයාගේ සන්හිඳීමෙහි ඥානයෝ අටදෙනෙක, විමුක්තිසුඛසම්ප්‍ර‍යුක්තඥානයෝ එක්විස්සෙකි.

ගණනවාරය නිමි.
