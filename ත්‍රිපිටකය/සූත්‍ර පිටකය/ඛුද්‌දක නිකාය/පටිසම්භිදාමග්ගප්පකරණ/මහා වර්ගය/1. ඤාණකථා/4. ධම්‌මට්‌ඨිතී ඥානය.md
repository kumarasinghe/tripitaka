# 4\. ධම්‌මට්‌ඨිතී ඥානය

කිසෙයින් ප්‍ර‍ත්‍යය පිරිසිඳ දැනීමේ ප්‍ර‍ඥාව ධර්‍මස්ථිතිඥාන වේ ද යත්:

අවිද්‍යාව සංස්කාරයන්ගේ උත්පාදයට කාරණ ද ප්‍ර‍වෘත්තියට කාරණ ද නිමිත්ත වූ කාරණ ද ආයූහන වූ කාරණ ද සංයොග වූ කාරණ ද පළිබොධ වූ කාරණ ද සමුදය වූ කාරණ ද හෙතු වූ කාරණ ද ප්‍ර‍ත්‍යය වූ කාරණ ද වෙයි. මේ නව අයුරින් අවිද්‍යාව ප්‍ර‍ත්‍යය වෙයි. සංස්කාරයෝ ප්‍ර‍ත්‍යසමුත්පන්න වෙත්. තෙල උභයධර්‍මයෝ මැ ප්‍ර‍ත්‍යයසමුත්පන්නයෝ යැ යි ප්‍ර‍ත්‍යයපරිග්‍ර‍හයෙහි ප්‍ර‍ඥාව ධර්‍මස්ථිතිඥාන යි.

අතීතකාලයෙහි ද අනාගත කාලයෙහි ද අවිද්‍යාව සංස්කාරයන්ගේ උත්පාදයට වූ කාරණ ද ප්‍ර‍වෘත්තියට කාරණ ද නිමිත්තකාරණ ද අයූහනකාරණ ද සංයොගකාරණ ද පරිබොධකාරණ ද සමුදයකාරණ ද හෙතුකාරණ ද ප්‍ර‍ත්‍යයකාරණ ද වෙයි. මේ නව අයුරින් අවිද්‍යාව ප්‍ර‍ත්‍ය වෙයි. සංස්කාරයෝ ප්‍ර‍ත්‍යයසමුත්පන්න වෙත්. තෙල උභයධර්‍මයෝ මැ ප්‍ර‍ත්‍යයසමුත්පන්නයෝ යැ යි ප්‍ර‍ත්‍යයපරිග්‍ර‍හයෙහි ප්‍ර‍ඥාව ධර්‍මස්ථිතිඥාන යි.

සංස්කාරයෝ විඥානයාගේ ... විඥානය නාමරූපයාගේ ... නාමරූපය ෂඩායතනයාගේ ... ෂඩායතනය ස්පර්‍ශයාගේ ... ස්පර්‍ශය වෙදනාවගේ ... වෙදනාව තෘෂ්ණාවගේ ... තෘෂ්ණාව උපදානයාගේ ... උපදානය භවයාගේ ... භවය ජාතිහුගේ ... ජාතිය ජරාමරණයාගේ උත්පාදයට කාරණ ද ප්‍ර‍වෘත්තියට කාරණ ද නිමිත්තකාරණ ද ආයූහනකාරණ ද සංයොගකාරණ ද පරිබොධකාරණ ද සමුදයකාරණ ද හෙතුකාරණ ද ප්‍ර‍ත්‍යයකාරණ ද වෙයි. මේ නව අයුරින් ජාතිය ප්‍ර‍ත්‍යය වෙයි, ජරාමරණ ප්‍ර‍ත්‍යයසමුත්පන්න වේ. තෙල උභයධර්‍මයෝ මැ ප්‍ර‍ත්‍යයසමුත්පන්නයෝ යැ යි ප්‍ර‍ත්‍යයපරිග්‍ර‍හයෙහි ප්‍ර‍ඥාව ධර්‍මස්ථිතිඥාන යි.

අතීතකාලයෙහි ද අනාගත කාලයෙහි ද ජාතිය ජරාමරණයාගේ උත්පාදයට කාරණ ද ප්‍ර‍වෘත්තියට කාරණ ද නිමිත්තකාරණ ද ආයූහනකාරණ ද සංයොගකාරණ ද පරිබොධකාරණ ද සමුදයකාරණ ද හෙතුකාරණ ද ප්‍ර‍ත්‍යයකාරණ ද වෙයි. මේ නව අයුරින් ජාතිය ප්‍ර‍ත්‍යය වෙයි. ජරාමරණ ප්‍ර‍ත්‍යය සමුත්පන්න වේ. තෙල උභයධර්‍මයෝ මැ ප්‍ර‍ත්‍යයසමුත්පන්නයෝ යැ’යි ප්‍ර‍ත්‍යය පරිග්‍ර‍හයෙහි ප්‍ර‍ඥාව ධර්‍මස්ථිතිඥාන යි.

අවිද්‍යාව (ජනකප්‍ර‍ත්‍යය හෙයින්) හෙතු යැ, සංස්කාරයෝ හෙතුසමුත්පන්න වෙති. තෙල උභයධර්‍මයෝ මැ හෙතු සමුත්පන්නයෝ යැ යි ප්‍ර‍ත්‍යයපරිග්‍ර‍හයෙහි ප්‍ර‍ඥාව ධර්‍මස්ථිතිඥාන යි. අතීතකාලයෙහි ද අනාගතකාලයෙහි ද අවිද්‍යාව හෙතු වෙයි. සංස්කාරයෝ හෙතුසමුත්පන්නයෝ වෙත්. තෙල උභයධර්‍මයෝ මැ හෙතු සමුත්පන්නයෝ යැ යි ප්‍ර‍ත්‍යයපරිග්‍ර‍හයෙහි ප්‍ර‍ඥාව ධර්‍මස්ථිතිඥාන යි.

සංස්කාරයෝ හෙතු වෙති. විඥානය හෙතුසමුත්පන්න වෙයි ... විඥානය හෙතු යැ නාමරූපය හෙතු සමුත්පන්න යි ... නාමරූපය හෙතු යැ ෂඩායතනය හෙතුසමුත්පන්න යි ... ෂඩායතනය හෙතු යැ ස්පර්‍ශය හෙතුසමුත්පන්න යි ... ස්පර්‍ශය හෙතු යැ වෙදනාව හෙතුසමුත්පන්න යි ... වෙදනාව හෙතු යැ තෘෂ්ණාව හෙතුසමුත්පන්න යි ... තෘෂ්ණාව හෙතු යැ උපාදානය හෙතුසමුත්පන්නයි ... උපාදානය හෙතු යැ භවය හෙතුසමුත්පන්නයි ... භවය හෙතු යැ ජාතිය හෙතුසමුත්පන්න යි ... ජාතිය හෙතු යැ ජරාමරණ හෙතුසමුත්පන්න යි. තෙල උභයධර්‍මයෝ මැ හෙතුසමුත්පන්නයෝ යැ යි ප්‍ර‍ත්‍යයපරිග්‍ර‍හයෙහි ප්‍ර‍ඥාව ධර්‍මස්ථිතිඥාන යි. අතීතකාලයෙහි දු අනාගතකාලයෙහි දු ජාතිය හෙතු යැ ජරාමරණ හෙතුසමුත්පන්න යි. තෙල උභයධර්‍මයෝ මැ හෙතුසමුත්පන්නයෝ යැ යි ප්‍ර‍ත්‍යයපරිග්‍ර‍හයෙහි ප්‍ර‍ඥාව ධර්‍මස්ථිතිඥාන යි.

අවිද්‍යාව (උපස්ථම්භක හෙයින්) ප්‍ර‍ත්‍යය වෙයි, සංස්කාරයෝ ප්‍ර‍ත්‍යයසමුත්පන්න වෙති. (ඒ ඒ ප්‍ර‍ත්‍යයෙන් උපන්) තෙල උභයධර්‍මයෝ මැ ප්‍ර‍ත්‍යයසමුත්පන්නයෝ යැ යි ප්‍ර‍ත්‍යයපරිග්‍ර‍හයෙහි ප්‍ර‍ඥාව ධර්‍මස්ථිතිඥාන යි. අතීතකාලයෙහි දු අනාගතකාලයෙහි දු අවිද්‍යාව ප්‍ර‍ත්‍යය වෙයි. සංස්කාරයෝ ප්‍ර‍ත්‍යයසමුත්පන්න වෙති. ‍තෙල උභයධර්‍මයෝ මැ ප්‍ර‍ත්‍යයසමුත්පන්නයෝ යැ යි ප්‍ර‍ත්‍යයපරිග්‍ර‍හයෙහි ප්‍ර‍ඥාව ධර්‍මස්ථිතිඥාන යි.

සංස්කාරයෝ ප්‍ර‍ත්‍යය වෙති. විඥානය ප්‍ර‍ත්‍යයසමුත්පන්න වේ ... විඥානය ප්‍ර‍ත්‍යය වෙයි. නාමරූප ප්‍ර‍ත්‍යයසමුත්පන්න වෙති ... නාමරූප ප්‍ර‍ත්‍යය වෙයි. ෂඩායතනය ප්‍ර‍ත්‍යයසමුත්පන්න වේ ... ෂඩායතන ප්‍ර‍ත්‍යය වෙයි. ස්පර්‍ශය ප්‍ර‍ත්‍යයසමුත්පන්න වෙයි … ස්පර්‍ශය ප්‍ර‍ත්‍යය වෙයි. වෙදනාව ප්‍ර‍ත්‍යයසමුත්පන්න වේ. වෙදනාව ප්‍ර‍ත්‍යය වෙයි. තෘෂ්ණාව ප්‍ර‍ත්‍යයසමුත්පන්න වේ... තෘෂ්ණාව ප්‍ර‍ත්‍යය වෙයි. උපාදානය ප්‍ර‍ත්‍යයසමුත්පන්න වේ ... උපාදානය ප්‍ර‍ත්‍යය වෙයි. භවය ප්‍ර‍ත්‍යයසමුත්පන්න වේ ... භවය ප්‍ර‍ත්‍යය වෙයි. ජාතිය ප්‍ර‍ත්‍යයසමුත්පන්න වේ. ජාතිය ප්‍ර‍ත්‍යය වෙයි. ජරාමරණ ප්‍ර‍ත්‍යසමුත්පන්න වේ. තෙල උභයධර්‍මයෝ මැ ප්‍ර‍ත්‍යයසමුත්පන්නයෝ යැ යි ප්‍ර‍ත්‍යයපරිග්‍ර‍හයෙහි ප්‍ර‍ඥාව ධර්‍මස්ථිතිඥාන යි. අතීතකාලයෙහි දු අනාගතකාලයෙහි දු ජාතිය ප්‍ර‍ත්‍යය වෙයි. ජරාමරණ ප්‍ර‍ත්‍යයසමුත්පන්න වේ. තෙල උභයධර්‍මයෝ මැ ප්‍ර‍ත්‍යයසමුත්පන්නයෝ යැ යි ප්‍ර‍ත්‍යයපරිග්‍ර‍හයෙහි ප්‍ර‍ඥාව ධර්‍මස්ථිතිඥාන යි.

පූර්‍වකර්‍මය කරන කල්හි මොහය අවිද්‍යා යැ, ආයූහනචෙතනාවෝ සංස්කාරයෝ යැ, භවනිකාන්තිය තෘෂ්ණා යැ, දෘඪග්‍ර‍හණය උපදාන යැ, චෙතනාව භවය යි, මේ පඤ්චධර්‍මයෝ පූර්‍වකර්‍මභවය කරන කල්හි පැවැත්තාහු වර්‍තමානප්‍ර‍තිසන්‍ධියට ප්‍ර‍ත්‍යය වූවාහු යැ.

වර්‍තමානභවයෙහි ප්‍ර‍තිසන්‍ධිය විඥාන යැ, (රූපාරූපධර්‍මයන්ගේ) අවක්‍රාන්තිය නාමරූප යැ, ප්‍ර‍සාදය ආයතන යැ, අරමුණු ස්පර්‍ශ කෙරෙමින් උපන්නේ ස්පර්‍ශ යැ, (විපාක) වෙදයිතය වෙදනා යි. මේ පඤ්චධර්‍මයෝ වර්‍තමානවිපාකභවයෙහි පූර්‍වකෘතකර්‍මයාගේ ප්‍ර‍ත්‍යයෙන් පවත්නාහු යි.

වර්‍තමානභවයෙහි ආයතනයන් මුහුකුරාගිය බැවින් මොහය අවිද්‍යා යැ, ආයූහනචෙතනාවෝ සංස්කාරයෝ යැ, භවනිකාන්තිය තෘෂ්ණා යැ, දෘඪග්‍ර‍හණය උපාදාන යැ, ආයූහනචෙතනාවෝ භව යැ යි. මේ පඤ්චධර්‍මයෝ වර්‍තමානකර්‍මභවයෙහි පැවැත්තාහු අනාගතභවයෙහි ප්‍ර‍තිසන්‍ධියට ප්‍ර‍ත්‍යය වෙත්.

අනාගතභවයෙහි ප්‍ර‍තිසන්‍ධි විඥාන යැ, (රූපාරූපධර්‍මයන්ගේ) අවක්‍රාන්තිය නාමරූප යැ, ප්‍ර‍සාදය ආයතන යැ, අරමුණු ස්පර්‍ශ කෙරෙමින් උපන්නේ ස්පර්‍ශ යැ, (විපාක) වෙදයිතය වෙදනා යැ යි මේ පඤ්චධර්‍මයෝ ආයතිභාවයෙහි උපප්‍රාප්තියට වර්‍තමානභවයෙහි කරන ලද කර්‍මයාගේ ප්‍ර‍ත්‍යයෙන් පවතිත්.

මෙසේ මේ සතර සංක්‍ෂෙපයන් ද තුන් කාලයන් ද ත්‍රිසන්‍ධියක් ඇති ප්‍ර‍තීත්‍යසමුත්පාදය ද විසි අයුරෙකින් (ශ්‍රැතානුසාරයෙන්) දනී නම් (නුවණින් ) දකී නම් (නුවණින් දැනගනී නම්) ප්‍ර‍තිවෙධ කෙරේ නම් ඒ දැනීම ඥාතාර්ථයෙන් ඥාන යැ ප්‍ර‍ජානනාර්ථයෙන් ප්‍ර‍ඥා යි. එහෙයින් කියනු ලැබෙයි: ‘පච්චය පරිග්ගහෙ ඤාණං ධම්මට්ඨිතිඤාණං’යි.

ධර්මස්ථිතිඥානනිර්දේශ යි.
