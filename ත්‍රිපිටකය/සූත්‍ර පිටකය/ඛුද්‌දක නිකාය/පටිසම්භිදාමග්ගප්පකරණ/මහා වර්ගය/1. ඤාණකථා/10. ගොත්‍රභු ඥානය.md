# 10\. ගොත්‍රභු ඥානය

කිසෙයින් බාහ්‍ය වූ සංස්කාර නිමිත්තෙන් නැඟී සිටුනා එයට පරාඞ්මුඛ වීමෙහි ප්‍රඥාව ගොත්‍රභූඥාන වෙයි යත්:

උත්පාදය ඉක්මවා (අභිභවනය කෙරේ) නු යි ගොත්‍රභූ නමැ. ප්‍රවෘත්තිය ඉක්මවා නු යි ගොත්‍රභූ නමැ. නිමිත්ත ඉක්මවා නු යි ගොත්‍රභූ නමැ. ආයූහනය ඉක්මවා නු යි ගොත්‍රභූ නමැ. ප්‍රතිසන්ධිය ඉක්මවා නු යි ගොත්‍රභූ නමැ. ගතිය ඉක්මවා නු යි ගොත්‍රභූ නමැ. නිර්‍වෘත්තිය ඉක්මවා නු යි ගොත්‍රභූ නමැ. උපප්‍රාප්තිය ඉක්මවා නු යි ගොත්‍රභූ නමැ. ජාතිය ඉක්මවා නු යි ගොත්‍රභූ නමැ. ජරාව ඉක්මවා නු යි ගොත්‍රභූ නමැ. ව්‍යාධිය ඉක්මවා නු යිගොත්‍රභූ නමැ. මරණය ඉක්මවා නු යි ගොත්‍රභූ නමැ. ශොකය ඉක්මවා නු යි ගොත්‍රභූ නමැ. පරිදෙවය ඉක්මවා නු යි ගොත්‍රභූ නමැ. උපායාසය ඉක්මවා නු යි ගොත්‍රභූ නමැ. වාහ්‍ය සංස්කාර නිමිත්ත ඉක්මවා නු යි ගොත්‍රභූ නමි.

අනුත්පාදයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ, අප්‍රවෘත්තිය ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ ... නිරොධ සඞ්ඛ්‍යාත නිර්‍වාණය ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමි.

උත්පාදය ඉක්මවා අනුත්පාදයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. ප්‍රවෘත්තිය ඉක්මවා අප්‍රවෘත්තියට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. නිමිත්ත ඉක්මවා අනිමිත්තට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ ... වාහ්‍ය සංස්කාරනිමිත්ත ඉක්මවා නිරොධ සඞ්ඛ්‍යාත නිර්‍වාණයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමි.

උත්පාදයෙන් නැඟීසිටී නු යි ගොත්‍රභූ නමැ. ප්‍රවෘත්තියෙන් නැඟීසිටී නු යි ගොත්‍රභූ නමැ. නිමිත්තයෙන් නැඟීසිටී නු යි ගොත්‍රභූ නමැ. කර්‍මායූහනයෙන් නැඟීසිටී නු යි ගොත්‍රභූ නමැ. ප්‍රතිසන්ධියන් නැඟීසිටී නු යි ගොත්‍රභූ නමැ. ගතියෙන් නැඟීසිටී නු යි ගොත්‍රභූ නමැ. නිර්‍වෘතියෙන් නැඟී සිටි නු යි ගොත්‍රභූ නමැ. උපප්‍රාප්තියෙන් නැඟීසිටී නු යි ගොත්‍රභූ නමැ. ජාතියෙන් නැඟීසිටී නු යි ගොත්‍රභූ නමැ. ජරායෙන් නැඟී සිටි නු යි ගොත්‍රභූ නමැ. ව්‍යාධියෙන් නැඟීසිටී නු යි ගොත්‍රභූ නමැ. මරණයෙන් නැඟීසිටී නු යි ගොත්‍රභූ නමැ. ශොකයෙන් නැඟීසිටී නු යි ගොත්‍රභූ නමැ. පරිදෙවයෙන් නැඟීසිටි නු යි ගොත්‍රභූ නමැ. උපායාසයෙන් නැඟීසිටී නු යි ගොත්‍රභූ නමැ. වාහ්‍ය සංස්කාරනිමිත්තයෙන් නැඟීසිටී නු යි ගොත්‍රභූ නමි.

අනුත්පාදයට ප්‍රස්කන්දනය කෙ‍රේ නු යී ගොත්‍රභූ නමැ. අප්‍රවෘත්තියට ප්‍රස්කන්දනය කෙ‍රේ නු යි ගොත්‍රභූ නමැ ... නිරොධ සඞ්ඛ්‍යාත නිර්‍වාණයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමි.

උත්පාදයෙන් නැඟී අනුත්පාදයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. ප්‍රවෘත්තියෙන් නැඟී අප්‍රවෘත්තියට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. නිමිත්තයෙන් නැඟී අනිමිත්තයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. ආයූනාහයෙන් නැඟී අනායූහනයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. ප්‍රතිසන්ධියෙන් නැඟී අප්‍රතිසන්ධියට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. ගතියෙන් නැඟී අගතියට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. තිර්‍වෘතියෙන් නැඟී අනිර්‍වෘතියට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. උපප්‍රාප්තියෙන් නැඟී අනුපප්‍රාප්තියට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. ජාතියෙන් නැඟී අජාතියට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. ජරායෙන් නැඟී අජරාවට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. ව්‍යාධියෙන් නැඟී අව්‍යාධියට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. මරණයෙන් නැඟී අමරණයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. ‍ශොකයෙන් නැඟී අශොකයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. උපායාසයෙන් නැඟී අනුපායාසයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ වාහ්‍යසංස්කාර නිමිත්තෙන් නැඟී නිරොධ සඞ්ඛ්‍යාත නිර්‍වාණයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමි.

උත්පාදයෙන් අපගත වේ නු යි ගොත්‍රභූ නමැ. ප්‍රවෘත්තියෙන් අපගත වේ නු යි ගොත්‍රභූ නමැ ... වාහ්‍ය සංස්කාර නිමිත්තෙන් අපගත වේ නු යි ගොත්‍රභූ නමැ.

අනුත්පාදයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ ... අප්‍රවෘත්තියට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ ... නිරොධ සඞ්ඛ්‍යාත නිර්‍වාණයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමි.

උත්පාදයෙන් අපගත වැ අනුත්පාදයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ. ප්‍රවෘත්තියෙන් අපගත වැ අප්‍රවෘත්තියට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමැ ... වාහ්‍ය සංස්කාර නිමිත්තෙන් අපගත වැ නිරොධ සඞ්ඛ්‍යාත නිර්‍වාණයට ප්‍රස්කන්දනය කෙරේ නු යි ගොත්‍රභූ නමි.

කෙ‍තෙක් ගොත්‍රභූධර්‍මයෝ ශමථවශයෙන් උපදිත් ද: කෙතෙක් ගොත්‍රභූධර්‍මයෝ විදර්‍ශනාවශයෙන් උපදිත් ද යත්: ගොත්‍රභූධර්‍මයෝ අටදෙනෙක් ශමථවශයෙන් උපදිත්. ගොත්‍රභූධර්‍මයෝ දසදෙනෙක් විදර්‍ශනා වශයෙන් උපදිත්.

කවර ගොත්‍රභූධර්‍මයෝ අටදෙනෙක් ශමථවශයෙන් උපදිත් ද යත්: ප්‍රථමධ්‍යානය ලබනු පිණිස නීවරණයන් ඉක්මවා (අභිභවනය කෙරේ) නු යි ගොත්‍රභූ යැ. ද්විතීයධ්‍යානය ලබනු පිණිස විතර්‍කවිචාරයන් ඉක්මවා නු යි ගොත්‍රභූ යැ. තෘතීයධ්‍යානය ලබනු පිණිස ප්‍රීතිය ඉක්මවා නු යි ගොත්‍රභූ යැ. චතුර්‍ථධ්‍යානය ලබනු පිණිස සුඛදුඃඛයන් ඉක්මවා නු යි ගොත්‍රභූ යැ ආකාසනඤ්චායතන සමාපත්තිය ලබනු පිණිස රූපසංඥාව ප්‍රතිඝසංඥාව නානාත්වසංඥාව ඉක්මවා නු යි ගොත්‍රභූ යැ. විඤ්ඤාණඤ්චායතනය සමාපත්තිය ලබනු පිණිස ආකාසනඤ්චායතනසංඥාව ඉක්මවා නු යි ගොත්‍රභූ යැ. ආකිඤ්චඤ්ඤායතනසමාපත්තිය ලබනු පිණිය විඤ්ඤාණඤ්චායතනසංඥාව ඉක්මවා නු යි ගොත්‍රභූ යැ. නෙවසඤ්ඤානාසඤ්ඤායතනසමාපත්තිය ලබනු පිණිය ආකිඤ්චඤ්ඤායතනසංඥාව ඉක්මවා නු යි ගොත්‍රභූ යැ. මේ ගෝත්‍රභූධර්‍මයෝ අටදෙන ශමථ වශයෙන් උපදිත්.

කවර ගොත්‍රභූධර්‍මයෝ දසදෙනෙක් විදර්‍ශනාවශයෙන් උපදිත් ද යත්: ස්‍රොතාපත්තිමාර්‍ගප්‍රතිලාභය පිණිස උත්පාදය ප්‍රවෘත්තිය නිමිත්ත ආයුහනය ප්‍රතිසන්ධිය ගතිය නිර්වෘත්තිය උපප්‍රාප්තිය ජාතිය ජරාව ව්‍යාධිය මරණය ශොකය පරිදෙවය උපායාසය වාහ්‍ය සංස්කාරනිමිත්ත ඉක්මවා නු යි ගොත්‍රභූ යැ. ස්‍රොතාපත්තිඵලසමාපත්තිය පිණිස උත්පාදය ප්‍රවෘත්තිය නිමිත්ත ආයුහනය ප්‍රතිසන්ධිය ... ඉක්මවා නු යි ගොත්‍රභූ යැ. සකෘදාගාමිමාර්‍ගප්‍රතිලාභය පිණිස ... සකෘදාගාමිඵලසමාපත්තිය පිණිස ... අනාගාමීමාර්‍ගප්‍රතිලාභය පිණිස ... අනාගාමීඵලසමාපත්තිය පිණිස ... අර්‍හන්මාර්‍ගප්‍රතිලාභය පිණිස උත්පාදය ප්‍රවෘත්තිය නිමිත්ත ආයූහනය ප්‍රතිසන්ධිය ගතිය නිර්‍වෘත්තිය උපප්‍රාප්තිය ජාතිය ජරාව ව්‍යාධිය මරණය ශොකය පරිදෙවය උපායාසය වාහ්‍ය සංස්කාරනිමිත්ත ඉක්මවා නු යි ගොත්‍රභූ යැ. අර්‍හත්ඵලසමාපත්තිය පිණිස ... ශුන්‍යතවිහාර සමාපත්තිය පිණිස ... අනිමිත්තවිහාරසමාපත්තිය පිණිස උත්පාදය ප්‍රවෘත්තිය නිමිත්ත ආයූහනය ප්‍රතිසන්ධිය ... ඉක්මවා නු යි ගොත්‍රභූ යැ. මේ ගොත්‍රභූධර්‍මයෝ දසදෙන විදර්‍ශනාවශයෙන් උපදිත්.

කෙතෙක් ගොත්‍රභූධර්‍මයෝ කුශලයෝ ද කෙතෙක් අකුශලයෝ ද කෙතෙක් අව්‍යකෘතයෝ ද යත්: ගොත්‍රභූධර්‍මයෝ පසළොස්දෙනෙක් කුශලයෝ යැ. ගොත්‍රභූධර්‍මයෝ තිදෙනෙක් අව්‍යාකෘතයෝ යැ. ගොත්‍රභූධර්‍මයෝ අකුශල වූවාහු නැති යි.

1\. නිකාන්ති සහිත බැවින් සාමිෂ යැ (නිකාන්ති රහිත බැවින්) නිරාමිෂ යැ. (නිකාන්ති ප්‍රණිධියෙන් සිටි බැවින්) ප්‍රණිහිත යැ. (තදභාවයෙන්) අප්‍රණිහිත යැ. (නිකාන්ති සංයොගයෙන්) සංයුක්ත යැ (තදභාවයෙන්) විසංයුක්ත යැ (නිකාන්තිය සිඳුනා බැවින්) ච්‍යුත්ථිත යැ (තදභාවයෙන්) අච්‍යුත්ථිත යි.

2\. ගොත්‍රභූධර්‍මයෝ අටදෙනෙක් සමාධියට ප්‍රත්‍යයෝ වෙති. දසදෙනෙක් ඥානයට ගොචර වූවාහු වෙති. අටළොස් ගොත්‍රභූධර්‍මයෝ තුන් විමොක්ෂයට ප්‍රත්‍යයෝ වෙත්.

3\. මේ අටළොස් ආකාර ප්‍රඥාවෝ යමක්හු විසින් පුරැදු කරන ලද්දාහු ද ඒ විවර්‍තයෙහි හා ව්‍යුත්ථානයෙහි නිපුණ වූයේ නානා දෘෂ්ටීන්හි නො සැලේ යයි.

එය ඥාතාර්‍ථයෙන් ඥාන යැ. ප්‍රජානනාර්‍ථයෙන් ප්‍රඥා යි. එයින් කියනු ලැබෙයි: ‘බහිද්ධා වුට්ඨානවිවට්ටනෙ පඤ්ඤා ගොත්‍රභූඤාණං’ යි.

ගොත්‍රභූඥාන නිර්දෙශ යි.
