# 72-73. සර්‍වඥතා - අනාවරණඥානයෝ

තථාගතයන් වහන්සේගේ සර්‍වතාඥානය කවරෙ යත්:

හැම සඞ්ඛතය අසඞ්ඛතය නිරවශෙෂ කොට දනී නු යි සර්‍වඥතාඥාන යැ, එහි ආවරණයෙක් නැත් නු යි අනාවරණඥාන යැ.

අතීත වූ සියල්ල දනී නු යි සර්‍වඥතාඥාන යැ, එහි ආවරණයෙක් නැත් නු යි අනාවරණඥාන යැ.

අනාගත වූ සියල්ල දනී නු යි සර්‍වඥතාඥාන යැ, එහි ආවරණයෙක් නැත් නු යි අනාවරණඥාන යැ.

ප්‍ර‍ත්‍යත්පන්න වූ සියල්ල දනී නු යි සර්‍වඥතාඥාන යැ, එහි ආවරණයෙක් නැත් නු යි අනාවරණඥාන යැ.

ඇස ද රූපයෝ ද යන තෙල සියල්ල දනී නු යි සර්‍වඥතාඥාන යැ එහි ආවරණයෙක් නැත් නු යි අනාවරණඥානයැ. කන ද ශබ්දයෝ ද ... නැහැය ද ගන්‍ධයෝ ද ... දිව ද රසයෝ ද .. කය ද ස්ප්‍ර‍ෂ්ටව්‍යයෝ ද ... මනස ද ධර්‍මයෝ ද යන තෙල සියල්ල දනී නු යි සර්‍වඥාතාඥාන යැ, එහි ආවරණයෙක් නැත් නු යි අනාවරණඥාන යි.

යම්තාක් අනිත්‍යස්වභාව දුඃඛස්වභාව අනාත්මස්වභාව වේ ද ඒ සියල්ල දනී නු යි සර්‍වඥතාඥාන යැ එහි ආවරණයෙක් නැත් නු යි අනාවරණඥාන යැ, යම්තාක් රූපයාගේ අනිත්‍යස්වභාව දුඃඛස්වභාව අනාත්මස්වභාව වේ ද ඒ සියල්ල දනී නු යි සර්‍වඥතාඥාන යැ එහි ආවරණයෙක් නැති නු යි අනාවරණඥාන යැ. යම්තාක් වේදනාවගේ ... යම්තාක් සංඥාවගේ ... යම්තාක් සංස්කාරයන්ගේ ... යම්තාක් විඥානයාගේ ... යම්තාක් චක්‍ෂුස්හුගේ ... යම්තාක් ජරාමරණයාගේ අනිත්‍යස්වභාව දුඃඛස්වභාව අනාත්මස්වභාව වේ ද ඒ සියල්ල දනී නු යි සර්‍වඥතාඥාන යැ, එහි ආවරණයෙක් නැති නු යි අනාවරණඥාන යි.

යම්තාක් අභිඥාවගේ අභිඥාස්වභාව වේ ද එ හැම දනී නු යි සර්‍වඥතාඥාන යැ එහි ආවරණයෙක් නැත් නු යි අනාවරණ ඥාන යැ. යම්තාක් පරිඥාවගේ පරිඥාස්වභාව වේ ද ... යම්තාක් ප්‍ර‍හාණයාගේ ප්‍ර‍හාණස්වභාව .. යම්තාක් භාවනාවගේ භාවනාස්වභාව ... යම්තාක් සාක්‍ෂාත් ක්‍රියාවගේ සාක්‍ෂාත්ක්‍රියාස්වභාව ...

යම්තාක් ස්කන්‍ධයන්ගේ ස්කන්‍ධස්වභාව ... යම්තාක් ධාතූන්ගේ ධාතුස්වභාව ... යම්තාක් ආයතනයන්ගේ ආයතනස්වභාව ... යම්තාක් සඞ්ඛතයන්ගේ සඞ්ඛතස්වභාව ... යම්තාක් අසඞ්ඛතයාගේ අසඞ්ඛත ස්වභාව වේ ද එ හැම දනී නු යි සර්‍වඥතාඥාන යැ, එහි ආවරණයෙක් නැත් නු යි අනාවරණඥාන යි.

යම්තාක් කුශලධර්‍මය වේ ද එ හැම දනී නු යි සර්‍වඥතාඥාන යැ එහි ආවරණයෙක් නැත් නු යි ආවරණඥාන යැ. යම්තාක් අකුශලධර්‍මය ... යම්තාක් අව්‍යාකෘතධර්‍මය ...

යම්තාක් කාමාවචරධර්‍මය ... යම්තාක් රූපවචරධර්‍මය ... යම්තාක් අරූපාවචරධර්‍මය ... යම්තාක් අපර්‍ය්‍යාපන්නධර්‍මය වේ ද එ හැම දනී නු යි සර්‍වඥතාඥාන යැ, එහි ආවරණයෙක් නැත් නු යි ආනාවරණඥාන යි.

යම්තාක් දුඃඛයාගේ දුඃඛස්වභාව ... යම්තාක් සමුදයාගේ සමුදය ස්වභාව ... යම්තාක් නිරොධයාගේ නිරොධස්වභාව ... යම්තාක් මාර්‍ගයාගේ මාර්‍ගස්වභාවය වේ ද එ හැම දනී නු යි සර්‍වඥතාඥාන යැ, එහි ආවරණයෙක් නැත් නු යි අනාවරණඥාන යි.

යම්තාක් අත්‍ථපටිසම්භිදාවගේ අත්‍ථපටිසම්භිදා ස්වභාව ... යම්තාක් ධම්මපටිසම්භිදාවගේ ධම්මපටිසම්භිදා ස්වභාව ... යම්තාක් නිරුත්තිපටිසම්භිදාවගේ නිරුත්තිපටිසම්භිදා ස්වභාව ... යම්තාක් පටිභානපටිසම්භිදාවගේ පටිභානපටිසම්භිදාස්වභාව වේ ද එ හැම දනී නු යි සර්‍වඥතාඥාන යැ, එහි ආවරණයෙක් නැති නු යි අනාවරණඥාන යි.

යම්තාක් මරුන් සහිත වූ බඹුන් සහිත වූ සදෙවක ලොකයා විසින් මහණබමුණන් සහිත වූ දෙවිමිනිසුන් සහිත ප්‍ර‍ජාව විසින් දක්නාලද රූපායතනයෙක් ශබ්දායතනයෙක් ගන්‍ධරසස්ප්‍ර‍ෂ්ටව්‍යායතනයෙක් ධර්‍මායතනයෙක් පැමිණෙන ලද රූපාදියෙක් සොයනලද රූපාදියෙක් මනසින් විමසන ලද රූපාදියෙක් වේ ද එ හැම දනී නු යි සර්‍වඥතාඥාන යැ, එහි ආවරණයෙක් නැත් නු යි අනාවරණඥාන යි.
