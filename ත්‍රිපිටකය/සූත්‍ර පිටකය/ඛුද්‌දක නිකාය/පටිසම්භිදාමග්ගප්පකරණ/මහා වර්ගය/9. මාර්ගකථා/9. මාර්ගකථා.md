# 9\. මාර්ගකථා

මග්ගො යනු: කවර අර්ත්‍ථ‍යෙකින් මාර්‍ග වේ ද යත්: ස්‍රොතාපත්තිමාර්‍ගක්‍ෂණයෙහි දර්‍ශන ස්වභාවයෙන් සම්‍යග්දෘෂ්ටිය මිථ්‍යාදෘෂ්ටිහුගේ ප්‍රහාණයට මාර්‍ග ද හේතු ද වෙයි, සහජාතධර්‍මයනට උපස්ථම්භනය පිණිස මාර්‍ග ද හේතු ද වෙයි, කෙලෙසුන් ගෙවීම පිණිස මාර්‍ග ද හේතු ද වෙයි, ප්‍රතිවේධාදිය විශෝධනය පිණිස මාර්‍ග ද හේතු ද වෙයි, චිත්තයාගේ අධිෂ්ඨානය පිණිස මාර්‍ග ද හේතු ද වෙයි, චිත්තයාගේ වෝදානය පිණිස මාර්‍ග ද හේතු ද වෙයි. විශේෂාධිගමය පිණිස මාර්‍ග ද හේතු ද වෙයි. මත්තෙහි ප්‍රතිවේධය පිණිස මාර්‍ග ද හේතු ද වෙයි, සත්‍යාභිසමය පිණිස මාර්‍ග ද හේතු ද වෙයි, නිරෝධයෙහි පිහිටුවනු පිණිස මාර්‍ග ද හේතු ද වේ.

අභිනිරෝපණ ස්වභාවයෙන් සම්‍යක්සංකල්පය මිථ්‍යාසංකල්පයාගේ ප්‍රහාණය පිණිස මාර්‍ග ද හේතු ද වෙයි, සහජාතධර්‍මයනට උපස්ථම්භනය පිණිස මාර්‍ග ද හේතු ද වෙයි, කෙලෙසුන් ගෙවීම පිණිස මාර්‍ග ද හේතු ද වෙයි, ප්‍රතිවේධාදිවිශෝධනය පිණිස මාර්‍ග ද හේතු ද වෙයි, චිත්තාධිෂ්ඨානය පිණිස මාර්‍ග ද හේතු ද වෙයි, චිත්තපාරිශුද්ධිය පිණිස මාර්‍ග ද හේතු ද වෙයි, විශේෂාධිගමය පිණිස මාර්‍ග ද හේතු ද වෙයි, මත්තෙහි ප්‍රතිවේධය පිණිස මාර්‍ග ද හේතු ද වෙයි, සත්‍යප්‍රතිවේධය පිණිස මාර්‍ග ද හේතු ද වෙයි, නිවන්හි පිහිටුවනු පිණිස මාර්‍ග ද හේතු ද වේ.

වාක්සංවර සඞ්ඛ්‍යාත පරිග්‍රහ ස්වභාවයෙන් සම්‍යග් වචනය මිථ්‍යාවචන ප්‍රහාණය පිණිස මාර්‍ග ද හේතු ද වෙයි, සහජාතධර්‍මයනට උපස්ථම්භනය පිණිස මාර්‍ග ද හේතු ද වෙයි, කෙලෙසුන් ගෙවීම පිණිස මාර්‍ග ද හේතු ද ප්‍රතිවේධාදිවිශෝධනය පිණිස මාර්‍ග ද හේතු ද වෙයි, චිත්තාධිෂ්ඨානය පිණිස මාර්‍ග ද හේතු ද වෙයි, චිත්ත පාරිශුද්ධිය පිණිස මාර්‍ග ද හේතු ද වෙයි, විශේෂාධිගමය පිණිස මාර්‍ග ද හේතු ද වෙයි, මත්තෙහි ප්‍රතිවේධය පිණිස මාර්‍ග ද හේතු ද වෙයි, සත්‍යාභිසමය පිණිස මාර්‍ග ද හේතු ද වෙයි, නිවන්හි පිහිටුවනු පිණිස මාර්‍ග ද හේතු ද වේ.

සමුත්‍ථාන ස්වභාවයෙන් සම්‍යක් කර්‍මාන්තය මිථ්‍යාකර්‍මාන්තයාගේ ප්‍රහාණය පිණිස මාර්‍ග ද හේතු ද වෙයි ... පරිශුද්ධභාව ස්වභාවයෙන් සම්‍යගාජීවය මිථ්‍යාජීවයාගේ ප්‍රහාණය පිණිස මාර්‍ග ද හේතු ද වෙයි ... ප්‍රග්‍රහ ස්වභාවයෙන් සම්‍යග්ව්‍යායාමය මිථ්‍යාව්‍යායාමයාගේ ප්‍රහාණය පිණිස මාර්‍ග ද හේතු ද වෙයි ... උපස්ථානස්වභාවයෙන් සම්‍යක්ස්මෘතිය මිථ්‍යාස්මෘතිහුගේ ප්‍රහාණය පිණිස මාර්‍ග ද හේතු ද වෙයි ... අවික්‍ෂෙපස්වභාවයෙන් සම්‍යක්සමාධිය මිථ්‍යාසමාධිහුගේ ප්‍රහාණය පිණිස මාර්‍ග ද හේතු ද වෙයි. සහජාතධර්‍මයනට උපස්ථම්භනය පිණිස මාර්‍ග ද හේතු ද වෙයි. කෙලෙසුන් ගෙවීම පිණිස මාර්‍ග ද හේතු ද වෙයි. ප්‍රතිවේධාදිවිශෝධනය පිණිස මාර්‍ග ද හේතු ද වෙයි, චිත්තාධිෂ්ඨානය පිණිස මාර්‍ග ද හේතු ද වෙයි. චිත්තවෝදානය පිණිස මාර්‍ග ද හේතු ද වෙයි, විශේෂාධිගමය පිණිස මාර්‍ග ද හේතු ද වෙයි, මත්තෙහි ප්‍රතිවේධය පිණිස මාර්‍ග ද හේතු ද වෙයි, සත්‍යාභිසමය පිණිස මාර්‍ග ද හේතු ද වෙයි, නිවන්හි පිහිටුවනු පිණිස මාර්‍ග ද හේතු ද වේ.

සකෘදාගාමි මාර්‍ගක්‍ෂණයෙහි දර්‍ශන ස්වභාවයෙන් සම්‍යග්දෘෂ්ටිය ... අවික්‍ෂෙපාර්ත්‍ථ‍යෙන් සම්‍යක් සමාධිය ඖදාරික වූ කාමරාගසංයොජනයාගේ ප්‍රතිඝසංයොජනයාගේ ඖදාරික වූ කාමරාගානුශයාගේ ප්‍රතිඝානුශයාගේ ප්‍රහාණය පිණිස මාර්‍ග ද හේතු ද වෙයි, සහජාතධර්‍මයනට උපස්ථම්භනය පිණිස මාර්‍ග ද හේතු ද වෙයි, කෙලෙසුන් ගෙවීම පිණිස මාර්‍ග ද හේතු ද වෙයි, ප්‍රතිවේධාදිවිශෝධනය පිණිස මාර්‍ග ද හේතු ද වෙයි, චිත්තාධිෂ්ඨානය පිණිස මාර්‍ග ද හේතු ද වෙයි. චිත්තවෝදානය පිණිස මාර්‍ග ද හේතු ද වෙයි, විශේෂාධිගමය පිණිස මාර්‍ග ද හේතු ද වෙයි, මත්තෙහි ප්‍රතිවේධය පිණිස මාර්‍ග ද හේතු ද වෙයි. සත්‍යාභිසමය පිණිස මාර්‍ග ද හේතු ද වෙයි, නිවන්හි පිහිටුවනු පිණිස මාර්‍ග ද හේතු ද වේ.

අනාගාමිමාර්‍ගක්‍ෂණයෙහි දර්‍ශන ස්වභාවයෙන් සම්‍යග්දෘෂ්ටිය ... අවික්‍ෂෙපස්වභාවයෙන් සම්‍යක්සමාධිය අණුහගත වූ කාමරාගසංයොජනයාගේ ප්‍රතිඝසංයොජනයාගේ අණුසහගත වූ කාමරාගානුශයාගේ ප්‍රතිඝානුශයාගේ ප්‍රහාණය පිණිස මාර්‍ග ද හේතු ද වෙයි. සහජාතධර්‍මයනට උපස්ථම්හනය පිණිස මාර්‍ග ද හේතු ද වෙයි. කෙලෙසුන් ගෙවීම පිණිස මාර්‍ග ද හේතු ද වෙයි. ප්‍රතිවේධාදිවිශෝධනය පිණිස මාර්‍ග ද හේතු ද වෙයි. චිත්තාධිෂ්ඨානය පිණිස මාර්‍ග ද හේතු ද වෙයි, චිත්තවෝදානය පිණිස මාර්‍ග ද හේතු ද වෙයි, විශේෂාධිගමය පිණිස මාර්‍ග ද හේතු ද වෙයි, මත්තෙහි ප්‍රතිවේධය පිණිස මාර්‍ග ද හේතු ද වෙයි, සත්‍යාභිසමය පිණිස මාර්‍ග ද හේතු ද වෙයි, නිවන්හි පිහිටුවනු පිණිස මාර්‍ග ද හේතු ද වේ.

අර්හන්මාර්‍ගක්‍ෂණයෙහි දර්‍ශනස්වභාවයෙන් සම්‍යග්දෘෂ්ටිය ... අවික්‍ෂෙප ස්වභාවයෙන් සම්‍යක්සමාධිය රූපරාගයාගේ අරූපරාගයාගේ මානයාගේ ඖද්ධත්‍යයාගේ අවිද්‍යාවගේ මානානුශයාගේ භවරාගානුශයාගේ අවිද්‍යානුශයාගේ ප්‍රහාණය පිණිස මාර්‍ග ද හේතු ද වෙයි, සහජාතධර්‍මයනට උපස්ථම්භනය පිණිස මාර්‍ග ද හේතු ද වෙයි, කෙලෙසුන් ගෙවීම පිණිස මාර්‍ග ද හේතු ද වෙයි. ප්‍රතිවේධාදිවිශෝධනය පිණිස මාර්‍ග ද හේතු ද වෙයි. චිත්තාධිෂ්ඨානය පිණිස මාර්‍ග ද හේතු ද වෙයි, චිත්තවෝදානය පිණිස මාර්‍ග ද හේතු ද වෙයි, විශේෂාධිගමය පිණිස මාර්‍ග ද හේතු ද වෙයි, මත්තෙහි ප්‍රතිවේධය පිණිස මාර්‍ග ද හේතු ද වෙයි, සත්‍යාභිසමය පිණිස මාර්‍ග ද හේතු ද වෙයි, නිවන්හි පිහිටුවනු පිණිස මාර්‍ග ද හේතු ද වේ.

දර්‍ශනමාර්‍ගය සම්‍යග්දෘෂ්ටි යැ, අභිනිරෝපණ (අරමුණට නැගීම්) මාර්‍ගය සම්‍යක්සංකල්ප යැ, පරිග්‍රහමාර්‍ගය සම්‍යග්වචනය යැ, සමුත්‍ථානමාර්‍ගය සම්‍යක්කර්‍මාන්ත යැ, ව්‍යවදානමාර්‍ගය සම්‍යගාජීව යැ, ප්‍රග්‍රහමාර්‍ගය සම්‍යග්ව්‍යායාම යැ, උපස්ථාන (අරමුණට එළඹසිටීම්) මාර්‍ගය සම්‍යක්සමෘති යැ, අවික්‍ෂෙපමාර්‍ගය සම්‍යක්සමාධි යි.

උපස්ථානමාර්‍ගය ස්මෘතිසම්ඛොධ්‍යඞ්ග යැ, ප්‍රවිචය (විමසීම්) මාර්‍ගය ධර්‍මවිචයසම්බොධ්‍යඞ්ග යැ, ප්‍රග්‍රහමාර්‍ගය වීර්‍ය්‍යසම්බොධ්‍යඞ්ග යැ, සඵරණමාර්‍ගය ප්‍රීතියසම්බොධ්‍යඞ්ග යැ, ව්‍යුපශමමාර්‍ගය ප්‍රශ්‍රබ්ධිසම්බොධ්‍යඞ්ග යැ, අවික්‍ෂෙපමාර්‍ගය සමාධිසම්බොධ්‍යඞ්ග යැ, පටිසඞ්ඛානමාර්‍ගය උපේක්‍ෂාසම්බොධ්‍යඞ්ග යි.

අශ්‍රද්ධායෙහි අකම්ප්‍යමාර්‍ගය ශ්‍රද්ධාබල යැ, කුසීදබැව්හි අකම්ප්‍යමාර්‍ගය වීර්‍ය්‍යබල යැ, ප්‍රමාදයෙහි අකම්ප්‍යමාර්‍ගය ස්මෘතිබල යැ, උද්ධත බැව්හි අකම්ප්‍යමාර්‍ගය සමාධිබල යැ, අවිද්‍යායෙහි අකම්ප්‍යමාර්‍ගය ප්‍රඥාබල යි.

අධිමොක්‍ෂමාර්‍ගය ශ්‍රද්ධෙන්‍ද්‍රිය යැ, ප්‍රග්‍රහමාර්‍ගය වීර්‍ය්‍යෙන්‍ද්‍රිය යැ, උපස්ථානමාර්‍ගය ස්මෘතීන්‍ද්‍රිය යැ, අවික්‍ෂෙපමාර්‍ගය සමාධීන්‍ද්‍රිය යැ, දර්‍ශනමාර්‍ගය ප්‍රඥෙන්‍ද්‍රිය යි.

අධිපතිභාවාර්ත්‍ථ‍‍‍යෙන් ඉන්‍ද්‍රිය මාර්‍ග වෙයි, අකම්ප්‍යාර්ත්‍ථ‍‍‍යෙන් බල මාර්‍ග වෙයි, නිර්‍ය්‍යාණාර්ත්‍ථ‍‍‍යෙන් බොධ්‍යඞ්ගය මාර්‍ග වෙයි, හෙත්‍ව‍ථර්ත්‍ථ‍‍‍යෙන් මාර්‍ගාඞ්ග මාර්‍ග වෙයි, උපස්ථානාර්ත්‍ථ‍යෙන් සතිපට්ඨානය මාර්‍ග වෙයි, ප්‍රදහනාර්ත්‍ථ‍‍‍යෙන් සම්‍යක්ප්‍රධානය මාර්‍ග වෙයි, සමෘද්ධ වන ස්වභාවයෙන් ඍද්ධිපාදය මාර්‍ග වෙයි, තථාර්ත්‍ථ‍‍‍යෙන් සත්‍යය මාර්‍ග වෙයි, අවි‍ක්‍ෂෙපාර්ත්‍ථ‍‍‍යෙන් ශමථය මාර්‍ග වෙයි. අනුදර්‍ශනාර්ත්‍ථ‍‍‍යෙන් විදර්‍ශනාව මාර්‍ග වෙයි, එකරසාර්ත්‍ථ‍‍‍යෙන් ශමථවිදර්‍ශනා මාර්‍ග වෙයි, අනතිවතීතාර්ත්‍ථ‍‍‍යෙන් යුගනද්ධධර්‍ම මාර්‍ග වෙයි, සංවරාර්ත්‍ථ‍‍‍යෙන් ශීලවිශුද්ධිය මාර්‍ග වෙයි, අවික්‍ෂෙපාර්ත්‍ථ‍‍‍යෙන් චිත්තවිශුද්ධිය මාර්‍ග වෙයි, දර්‍ශනාර්ත්‍ථ‍යෙන් දෘෂ්ටිවිශුද්ධිය මාර්‍ග වෙයි, මුක්ත්‍යර්ත්‍ථ‍‍‍යෙන් විමෝක්‍ෂය මාර්‍ග වෙයි, ප්‍රතිවේධාර්ත්‍ථ‍‍‍යෙන් විද්‍යාව මාර්‍ග වෙයි, පරිත්‍යාගාර්ත්‍ථ‍‍‍යෙන් විමුක්තිය මාර්‍ග වෙයි, සමුච්ඡේදාර්ත්‍ථ‍‍‍යෙන් ක්‍ෂයඥානය මාර්‍ග වෙයි, ඡන්‍දය මූලාර්ත්‍ථ‍‍‍යෙන් මාර්‍ග වෙයි, මනසිකාරය සමුත්‍ථානාර්ත්‍ථ‍‍‍යෙන් මාර්‍ග වෙයි, ස්පර්‍ශය සමවධානාර්ත්‍ථ‍‍‍යෙන් මාර්‍ග වෙයි, වේදනාව සමවසරණාර්ත්‍ථ‍‍‍යෙන් මාර්‍ග වෙයි, සමාධිය ප්‍රමුඛාර්ත්‍ථ‍‍‍යෙන් මාර්‍ග වෙයි, ස්මෘතිය අධිපතිභාවාර්ත්‍ථ‍‍‍යෙන් මාර්‍ග වෙයි, ප්‍රඥාව තදුත්තරාර්ත්‍ථ‍‍‍යෙන් මාර්‍ග වෙයි, විමුක්තිය සාරාර්ත්‍ථ‍‍‍යෙන් මාර්‍ග වෙයි, අමතොගධ සඞ්ඛ්‍යාත නි‍ර්‍වාණය පර්‍ය්‍යවසානාර්ත්‍ථ‍යෙන් මාර්‍ග වේ යයි.

මාර්ගකථා නිමි.
