# දෘෂ්ටිකථා

දෘෂ්ටි කවර යැ, දෘෂ්ටිකාරණයෝ කෙතෙකැ, දෘෂ්ටිපර්‍ය්‍යුත්‍ථානයෝ කෙතෙකැ, දෘෂ්ටිහු කෙතෙකැ, දෘෂ්ටිඅභිනිවෙශයෝ කෙතෙකැ, දෘෂ්ටිකාරණසමුද්ඝාතය කවරෙ යත්:

1\. දෘෂ්ටිය කවර යනු: අභිනාවෙශපරාමර්‍ශය දෘෂ්ටි යැ.

2\. දෘෂ්ටිකාරණයෝ කෙතෙකැ යනු: දෘෂ්ටිකාරණයෝ අටදෙනෙකි.

3\. දෘෂ්ටිපර්‍ය්‍යුත්‍ථානයෝ කෙතෙකැ යනු: දෘෂ්ටිපර්‍ය්‍යුත්‍ථානයෝ අටළොස්දෙනෙකි.

4\. දෘෂ්ටීහු කෙතෙකැ යනු: දෘෂ්ටිහු සොළොස්දෙනෙකි.

5\. දෘෂ්ට්‍යභිනිවෙශයෝ කෙතෙකැ යනු: දෘෂ්ට්‍යභිනිවෙශ තුන් සියයෙකි.

6\. දෘෂ්ටිකාරණසමුද්ඝාතය කවරෙ යනු: සෝවාන්මාර්‍ගය දෘෂ්ටිකාරණසමුද්ඝාතය වේ.

කිසෙයින් අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි වෙයි යත්:

රූපය ‘තෙල මාගේ යැ, තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි යැ. වෙදනාව ‘තෙල මාගේ යැ ... සංඥාව ‘තෙල මාගේ යැ ... සංස්කාර ‘තෙල මාගේ යැ ... විඥානය ‘තෙල මාගේ යැ ... තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි යැ.

චක්‍ෂුස ‘තෙල මාගේ යැ ... ශ්‍රොත්‍ර‍ය ‘තෙල මාගේ යැ ... ඝ්‍රාණය ‘තෙල මාගේ යැ ... ජිහ්වාව ‘තෙල මාගේ යැ ... කාය ‘තෙල මාගේ යැ ... මනස ‘තෙල මාගේ යැ, තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි යැ, රූපයන් ‘තෙල මාගේ යැ ... ශබ්දයන් ‘තෙල මාගේ යැ ... ගන්‍ධයන් ‘තෙල මාගේ යැ ... රසයන් ‘තෙල මාගේ යැ ... ස්ප්‍ර‍ෂ්ව්‍යයන් ‘තෙල මාගේ යැ ... ධර්‍මයන් ‘තෙල මාගේ යැ ... තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ‘ යි අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි යැ.

චක්‍ෂුර්විඥානය ‘තෙල මාගේ යැ ... ශ්‍රොත්‍ර‍විඥානය ‘තෙල මාගේ යැ ... ඝ්‍රාණවිඥානය ‘තෙල මාගේ යැ ... ජිහ්වාවිඥානය ‘තෙල මාගේ යැ ... කායවිඥානය ‘තෙල මාගේ යැ ... මනොවිඥානය ‘තෙල මාගේ යැ, තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි යැ.

චක්ඛුසම්ඵස්සය ‘තෙල මාගේ යැ ... සොතසම්ඵස්සය ‘තෙල මාගේ යැ ... ඝානසම්ඵස්සය ‘තෙල මාගේ යැ ... ජිව්හාසම්ඵස්සය ‘තෙල මාගේ යැ ... කායසම්ඵස්සය ‘තෙල මාගේ යැ ... මනොසම්ඵස්සය ‘තෙල මාගේ යැ, තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශ පරාමර්‍ශය දෘෂ්ටි යැ.

චක්ඛුසම්ඵස්සයෙන් උපන් වෙදනාව ‘තෙල මාගේ යැ ... සොතසම්ඵස්සයෙන් උපන් වෙදනාව ... ඝානසම්ඵස්සයෙන් උපන් වෙදනාව … ජිව්හාසම්ඵස්සයෙන් උපන් වෙදනාව ... කායසම්ඵස්සයෙන් උපන් වෙදනාව ... මනොසම්ඵස්සයෙන් උපන් වෙදනාව ... ‘තෙල මාගේ යැ, තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි යැ.

රූපසංඥාව ‘තෙල මාගේ යැ ... ශබ්දසංඥාව ‘තෙල මාගේ යැ ... ගන්‍ධසංඥාව ‘තෙල මාගේ යැ ... රසසංඥාව ‘තෙල මාගේ යැ ... ස්ප්‍ර‍ෂ්ටවය්‍යසංඥාව ‘තෙල මාගේ යැ ... ධර්‍මසංඥාව ‘තෙල මාගේ යැ, තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි යැ.

රූපසඤ්චෙතනාව ‘තෙල මාගේ යැ ... ශබ්දසඤ්චෙතනාව ‘තෙල මාගේ යැ ... ගන්‍ධසඤ්චෙතනාව ‘තෙල මාගේ යැ ... රසසඤ්චෙතනාව ‘තෙල මාගේ යැ ... ස්ප්‍ර‍ෂ්ටව්‍යසඤ්චෙතනාව ‘තෙල මාගේ යැ ... ධර්‍මසඤ්චෙතනාව ‘තෙල මාගේ යැ, තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි යැ.

රූපතෘෂ්ණාව ‘තෙල මාගේ යැ ... ශබ්දතෘෂ්ණාව ‘තෙල මාගේ යැ ... ගන්‍ධතෘෂ්ණාව ‘තෙල මාගේ යැ ... රසතෘෂ්ණාව ‘තෙල මාගේ යැ ... ස්ප්‍ර‍ෂ්ටව්‍යතෘෂ්ණාව ‘තෙල මාගේ යැ ... ධර්‍මතෘෂ්ණාව ‘තෙල මාගේ යැ ... තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි යැ.

රූපවිතර්‍කය ‘තෙල මාගේ යැ ... ශබ්දවිතර්‍කය ‘තෙල මාගේ යැ ... ගන්‍ධවිතර්‍කය ‘තෙල මාගේ යැ ... රසවිතර්‍කය ‘තෙල මාගේ යැ ... ස්ප්‍ර‍ෂ්ටව්‍යවිතර්‍කය ‘තෙල මාගේ යැ ... ධර්‍මවිතර්‍කය ‘තෙල මාගේ යැ, තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි යැ.

රූපවිචාරය ‘තෙල මාගේ යැ ... ශබ්දවිචාරය ‘තෙල මාගේ යැ ... ගන්‍ධවිචාරය ‘තෙල මාගේ යැ ... රසවිචාරය ‘තෙල මාගේ යැ ... ස්ප්‍ර‍ෂ්ටව්‍යවිචාරය ‘තෙල මාගේ යැ ... ධර්‍මවිචාරය ‘තෙල මාගේ යැ, තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි යැ.

චක්‍ෂූර්ධාතුව ‘තෙල මාගේ යැ ... රූපධාතුව ‘තෙල මාගේ යැ ... චක්‍ෂුර්විඥානධාතුව ‘තෙල මාගේ යැ ... ශ්‍රොත්‍රධාතුව ‘තෙල මාගේ යැ ... ශබ්දධාතුව ‘තෙල මාගේ යැ ... ශ්‍රොත්‍රවිඥානධාතුව ... ඝ්‍රාණධාතුව ... ගන්‍ධධාතුව ... ඝ්‍රාණවිඥානධාතුව ... ජිව්හාධාතුව ... රසධාතුව ... ජිහ්වාවිඥාන ධාතුව ... කායධාතුව ... ස්ප්‍ර‍ෂ්ටව්‍යධාතුව ... කායවිඥානධාතුව ... මනොධාතුව ‘තෙල මාගේ යැ ... ධම්මධාතුව ‘තෙල මාගේ යැ ... මනොවිඥානධාතුව ‘තෙල මාගේ යැ ... ‘තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි යැ.

චක්‍ෂුරින්‍ද්‍රියය ‘තෙල මාගේ යැ ... ශ්‍රොත්‍රෙන්‍ද්‍රියය ‘තෙල මාගේ යැ ... ඝ්‍රාණෙන්‍ද්‍රියය ‘තෙල මාගේ යැ ... ජිහ්වෙන්‍ද්‍රියය ‘තෙල මාගේ යැ ... කායෙන්‍ද්‍රියය ‘තෙල මාගේ යැ ... මනඉන්‍ද්‍රියය ‘තෙල මාගේ යැ ... ජිවිතෙන්‍ද්‍රියය ‘තෙල මාගේ යැ ... ස්ත්‍රීන්‍ද්‍රියය ‘තෙල මාගේ යැ ... පුරුෂෙන්‍ද්‍රියය ‘තෙල මාගේ යැ ... සුඛෙන්‍ද්‍රියය ‘තෙල මාගේ යැ ... දුඃඛෙන්‍ද්‍රියය ‘තෙල මාගේ යැ ... සොමනස්සින්‍ද්‍රියය ‘තෙල මාගේ යැ ... දොමනස්සින්‍ද්‍රියය ‘තෙල මාගේ යැ ... උපෙක්‍ෂෙන්‍ද්‍රියය ‘තෙල මාගේ යැ ... ශ්‍ර‍ද්ධෙන්‍ද්‍රියය ‘තෙල මාගේ යැ ... වීර්යෙන්‍ද්‍රියය ‘තෙල මාගේ යැ ... ස්මෘතීන්‍ද්‍රියය ‘තෙල මාගේ යැ ... සමාධීන්‍ද්‍රියය ‘තෙල මාගේ යැ ... ප්‍රඥෙන්‍ද්‍රියය ‘තෙල මාගේ යැ, තෙල මම වෙමි, ‘තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි යැ.

කාමධාතුව ‘තෙල මාගේ යැ ... රූපධාතුව ‘තෙල මාගේ යැ ... අරූපධාතුව ‘තෙල මාගේ යැ ... කාමභවය ‘තෙල මාගේ යැ ... රූපභවය ‘තෙල මාගේ යැ ... අරූපභවය ‘තෙල මාගේ යැ ... සංඥාභවය ‘තෙල මාගේ යැ ... අසංඥාභවය ‘තෙල මාගේ යැ ... නෛවසංඥානාසංඥාභවය ‘තෙල මාගේ යැ ... එකවොකාරභවය ‘තෙල මාගේ යැ ... චතුවොකාරභවය ‘තෙල මාගේ යැ ... පඤ්චවොකාරභවය ‘තෙල මාගේ යැ: ...

ප්‍ර‍ථමධ්‍යානය ‘තෙල මාගේ යැ ... ද්වීතියධ්‍යානය ‘තෙල මාගේ යැ ... තෘතීයධ්‍යානය ‘තෙල මාගේ යැ ... චතුර්ථධ්‍යානය ‘තෙල මාගේ යැ ... මෛත්‍රීචිත්තවිමුක්තිය ‘තෙල මාගේ යැ ... කරුණාචිත්තවිමුක්තිය ‘තෙල මාගේ යැ ... මෘදුතාචිත්තවිමුක්තිය ‘තෙල මාගේ යැ ... උපෙක්‍ෂාචිත්තවිමුක්තිය ‘තෙල මාගේ යැ ... ආකාසානඤ්චායතනසමාපත්තිය ‘තෙල මාගේ යැ ... විඤ්ඤාණඤ්චායතනසමාපත්තිය ‘තෙල මාගේ යැ ... ආකිඤ්චඤ්ඤායතනසමාපත්තිය ‘තෙල මාගේ යැ ... නෙවසඤ්ඤනාසඤ්ඤායතනසමාපත්තිය ‘තෙල මාගේ යැ, තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි යැ.

අවිද්‍යාව ‘තෙල මාගේ යැ ... සංස්කාරයන් ‘තෙල මාගේ යැ ... විඥානය ‘තෙල මාගේ යැ ... නාමරූපය ‘තෙල මාගේ යැ ... ෂඩායතනය ‘තෙල මාගේ යැ ... ස්පර්‍ශය ‘තෙල මාගේ යැ ... වෙදනාව ‘තෙල මාගේ යැ ... තෘෂ්ණාව ‘තෙල මාගේ යැ ... උපාදානය ‘තෙල මාගේ යැ ... භවය ‘තෙල මාගේ යැ ... ජාතිය ‘තෙල මාගේ යැ ... ජරාමරණය ‘තෙල මාගේ යැ, තෙල මම වෙමි, තෙල මාගේ ආත්ම යැ’යි අභිනිවෙශ පරාමර්‍ශය දෘෂ්ටි යැ.

මෙසෙයින් අභිනිවෙශපරාමර්‍ශය දෘෂ්ටි වේ.

දෘෂ්ටි කාරණයෝ අටදෙන කවර යත්:

ස්කන්‍ධයෝ ද දෘෂ්ටිකාරණයෙක, අවිද්‍යාව ද දෘෂ්ටිකාරණයෙක, ස්පර්‍ශය ද දෘෂ්ටිකාරණයෙක, සංඥාව ද දෘෂ්ටිකාරණයෙක, විතර්‍කය ද දෘෂ්ටිකාරණයෙක, අයොනිසොමනසිකාරය ද දෘෂ්ටිකාරණයෙක, පාපමිත්‍ර‍ තෙමේ ද දෘෂ්ටිකාරණයෙක, පරතොඝොෂය ද දෘෂ්ටිකාරණයෙක.

ස්කන්‍ධයෝ (ජනක) හෙතු යැ ස්කන්‍ධයෝ (උපස්ථම්භක) ප්‍ර‍ත්‍යය යැ දෘෂ්ටීන්ගේ උපැත්මට කාරණ බැවිනි. මෙසෙයින් ස්කන්‍ධයෝ ද දෘෂ්ටිකාරණයෙක.

අවිද්‍යාව හෙතු යැ අවිද්‍යාව ප්‍ර‍ත්‍යය යැ දෘෂ්ටීන්ගේ උපැත්මට කාරණ බැවිනි. මෙසෙයින් අවිද්‍යාව ද දෘෂ්ටිකාරණයෙක.

ස්පර්‍ශය හෙතු යැ ස්පර්‍ශය ප්‍ර‍ත්‍යය යැ දෘෂ්ටීන්ගේ උපැත්මට කාරණ බැවිනි. මෙසෙයින් ස්පර්‍ශය ද දෘෂ්ටිකාරණයෙක.

සංඥාව හෙතු යැ සංඥාව ප්‍ර‍ත්‍යය යැ දෘෂ්ටීන්ගේ උපැත්මට කාරණ බැවිනි. මෙසෙයින් සංඥාව ද දෘෂ්ටිකාරණයෙක.

විතර්‍කය හෙතු යැ විතර්‍කය ප්‍ර‍ත්‍යය යැ දෘෂ්ටීන්ගේ උපැත්මට කාරණ බැවිනි. මෙසෙයින් විතර්‍කය ද දෘෂ්ටිකාරණයෙක.

අයොනිසොමනසිකාරය හෙතු යැ අයොනිසොමනසිකාරය ප්‍ර‍ත්‍යය යැ දෘෂ්ටීන්ගේ උපැත්මට කාරණ බැවිනි. මෙසෙයින් අයොනිසොමනසිකාරය ද දෘෂ්ටිකාරණයෙක.

පාපමිත්‍ර හෙතු යැ පාපමිත්‍ර ප්‍ර‍ත්‍යය යැ දෘෂ්ටීන්ගේ උපැත්මට කාරණ බැවිනි, මෙසෙයින් පාපමිත්‍ර තෙමේ ද දෘෂ්ටිකාරණයෙක.

පරතොඝොෂය හෙතු යැ පරතොඝොෂය ප්‍ර‍ත්‍යයැ දෘෂ්ටීන්ගේ උපැත්මට කාරණ බැවිනි, මෙසෙයින් පරතොඝොෂය ද දෘෂ්ටිකාරණයෙකි.

මොහු අටදෙන දෘෂ්ටිකාරණයෝ යි.

අටළොස් දෘෂ්ටිපර්‍ය්‍යුත්‍ථානයෝ කවරහ යත්:

යම් දෘෂ්ටියෙක් දෘෂ්ටිගතයෙක් දෘෂ්ටිගහනයෙක් දෘෂ්ටිකාන්තාරයෙක් දෘෂ්ටි නමැති හුලෙක් දෘෂ්ටිහුගේ විරූප වූ චලනයෙක් දෘෂ්ටිසංයොජනයෙක් දෘෂ්ටිශල්‍යයෙක් දෘෂ්ටිසම්බොධයෙක් දෘෂ්ටිපරිබොධයෙක් දෘෂ්ටිබන්‍ධනයෙක් දෘෂ්ටිප්‍ර‍පාතයෙක් දෘෂ්ට්‍යනුශයයෙක් දෘෂ්ටිසන්තාපයෙක් දෘෂ්ටිපරිදාහයෙක් දෘෂ්ටිග්‍ර‍න්ථයෙක් දෘෂ්ට්‍යුපාදානයෙක් දෘෂ්ට්‍යභිනිවෙශයෙක් දෘෂ්ටිපරාමර්‍ශයෙක් වේ ද,

මොහු අටළොස් දෘෂ්ටිපර්‍ය්‍යුත්‍ථානයෝ යි.

සොළොස් දෘෂ්ටීහු කවරහ යත්:

ආස්වාදදෘෂ්ටි යැ අත්මානුදෘෂ්ටි යැ මිථ්‍යාදෘෂ්ටි යැ සත්කායදෘෂ්ටි යැ සත්කායවස්තුක වූ ශාශ්වතදෘෂ්ටි යැ සත්කාය වස්තුක වූ උච්ඡෙදදෘෂ්ටි යැ අන්තග්‍රාහික දෘෂ්ටි යැ පූර්‍වාන්තානුදෘෂ්ටි යැ අපරාන්තානුදෘෂ්ටි යැ සංයොජනිකදෘෂ්ටි යැ ‘මම’ යයි (උපන්) මානයෙන් බැඳුණු දෘෂ්ටි යැ ‘මාගේ යැ’යි (උපන්) මානයෙන් බැඳුණු දෘෂ්ටි යැ ආත්මවාදයෙන් බැඳුණු දෘෂ්ටි යැ ලොකවාදයෙන් බැඳුණු දෘෂ්ටි යැ භවදෘෂ්ටි යැ විභවදෘෂ්ටි යි.

මොහු සොලොස් දෘෂ්ටීහු වෙත්.

තුන්සියයක් දෘෂ්ට්‍යභිනිවෙශයෝ කවරහ යත්:

කිහි අයුරෙකින් ආස්වාදදෘෂ්ටියෙහි අභිනිවෙශය (බැසැ ගැනීම) වේ ද?

කිහි අයුරෙකින් ආත්මානුදෘෂ්ටියෙහි අභිනිවෙශය වේ ද?

කිහි අයුරෙකින් මිථ්‍යාදෘෂ්ටියෙහි අභිනිවෙශය වේ ද?

කිහි අයුරෙකින් සත්කායදෘෂ්ටියෙහි අභිනිවෙශය වේ ද?

කිහි අයුරෙකින් සත්කායවස්තුක ශාශ්වතදෘෂ්ටියෙහි අභිනිවෙශය වේ ද?

කිහි අයුරෙකින් සත්කායවස්තුක උච්ඡෙදදෘෂ්ටියෙහි අභිනිවෙශය වේ ද?

කිහි අයුරෙකින් අන්තග්‍රාහිකදෘෂ්ටියෙහි අභිනිවෙශය වේ ද?

කිහි අයුරෙකින් පූර්‍වාන්තානුදෘෂ්ටියෙහි අභිනිවෙශය වේ ද?

කිහි අයුරෙකින් අපරාන්තානුදෘෂ්ටියෙහි අභිනිවෙශය වේ ද?

කිහි අයුරෙකින් සංයොජනිකනුදෘෂ්ටියෙහි අභිනිවෙශය වේ ද?

කිහි අයුරෙකින් ‘මම’ යයි මානයෙන් බැඳුණු දෘෂ්ටියෙහි අභිනිවෙශය වේ ද?

කිහි අයුරෙකින් ‘මාගේ යැ’යි මානයෙන් බැඳුණු දෘෂ්ටියෙහි අභිනිවෙශය වේ ද?

කිහි අයුරෙකින් ආත්මවාද ප්‍ර‍තිසංයුක්තදෘෂ්ටියෙහි අභිනිවෙශය වේ ද?

කිහි අයුරෙකින් ලොකවාද ප්‍ර‍තිසංයුක්තදෘෂ්ටියෙහි අභිනිවෙශය වේ ද?

කිහි අයුරෙකින් භවදෘෂ්ටියෙහි අභිනිවෙශය වේ ද?

කිහි අයුරෙකින් විභවදෘෂ්ටියෙහි අභිනිවෙශය වේ ද යත්:

පන්තිස් අයුරෙකින් ආස්වාදදෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

විසි අයුරෙකින් ආත්මානුදෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

දස අයුරෙකින් මිථ්‍යාදෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

විසි අයුරෙකින් සත්කායදෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

පසළොස් අයුරෙකින් සත්කායවස්තුක ශාශ්වතදෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

පස් අයුරෙකින් සත්කායවස්තුක උච්ඡෙදදෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

පනස් අයුරෙකින් අන්තග්‍රාහිකදෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

අටළොස් අයුරෙකින් පූර්‍වාන්තදෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

සියුසාලිස් අයුරෙකින් අපරාන්තානුදෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

අටළොස් අයුරෙකින් සංයොජනිකදෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

අටළොස් අයුරෙකින් ‘මම’ යයි මානයෙන් බැදුණු දෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

අටළොස් අයුරෙකින් ‘මාගේ’යැ’යි මානයෙන් බැඳුණු දෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

විසි අයුරෙකින් ආත්මවාදප්‍ර‍තිසංයුක්තදෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

අට අයුරෙකින් ලොකවාදප්‍ර‍තිසංයුක්තදෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

එක් අයුරෙකින් භවදෘෂ්ටියෙහි අභිනිවෙශය වෙයි.

එක් අයුරෙකින් විභවදෘෂ්ටියෙහි අභිනිවෙශය වෙයි.
