# 2.3.3. සොභිත ස්ථවිරගාථා

165\. සිහි ඇති (අභිඥා) නුවණ ඇති පටන්ගන්නාලද බල හා වීර්ය්‍ය ඇති මහණකු වූ මම පන්සියක් කප් එක්රැය මෙන් සිහි කළෙමි.

166\. සතර සීවටන් ද සප්තබොධ්‍යඞ්ගයන් ද අෂ්ටමාර්ගාඞ්ගයන් ද වඩනුයෙම් පන්සියක් කප් එක්රැය මෙන් සිහි කළෙමි.

මෙසේ මේ ගාථා ආයුෂ්මත් සොභිත ස්ථවිරයන් වහන්සේ ...
