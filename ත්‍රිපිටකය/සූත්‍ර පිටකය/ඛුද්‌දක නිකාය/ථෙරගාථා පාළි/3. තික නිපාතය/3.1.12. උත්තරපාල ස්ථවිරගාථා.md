# 3.1.12. උත්තරපාල ස්ථවිරගාථා

252\. ලොවැ සම්මෝහයට හේතු වූ පස්කම්ගුණයෝ එකත්නෙන් නුවණැති වූ ආත්මාර්ථචින්තනයෙහි සමර්ථ වූ මා හෙලාලූහ.

253\. කෙලෙස් මරහුගේ ගෝචරයෙහි පිවිසියෙම් දැඩි වූ (රාග) ශල්‍යයෙන් විදුනා ලද්දෙම් මම මාරයාගේ පාශයෙන් මිදෙන්නට හැකි වීමි.

254\. මා විසින් සියලු කාමයෝ ප්‍රහීණ කරනලදහ. සියලු භවයෝ නසනලදහ. ජාති සංසාරය ක්ෂය කරනලද්දේ යැ. මින් මතු පුනර්භවයෙක් නැති.

මෙසේ මේ ගාථා ආයුෂ්මත් උත්තරපාල ස්ථවිරයන් වහන්සේ ...

උත්තරපාල ස්ථවිරගාථා යි.
