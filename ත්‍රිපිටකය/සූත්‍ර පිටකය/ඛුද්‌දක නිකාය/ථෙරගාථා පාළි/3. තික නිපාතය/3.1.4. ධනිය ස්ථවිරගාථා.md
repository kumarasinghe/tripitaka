# 3.1.4. ධනිය ස්ථවිරගාථා

228\. ශ්‍රමණභාවයෙහි අපේක්ෂා ඇත්තේ ඉදින් සුවසේ ජීවත්වන්නට කැමැත්තේ නම් සඟසතු සිවුරු පැන් - බොජුන් අවමන් නො කරන්නේ යැ.

229\. ශ්‍රමණභාවයෙහි අපේක්ෂා ඇත්තේ ඉදින් සුවසේ ජීවත්වනු කැමැත්තේ නම් මීයකු විසින් කණිනලද බිලය සෙවුනා සර්පයකු සෙයින් සෙනසුන සෙවුනේ යැ.

230\. ශ්‍රමණභාවයෙහි අපේක්ෂා ඇත්තේ ඉදින් සුවසේ ජීවත්වනු කැමැත්තේ නම් යථාලබ්ධප්‍රත්‍යයෙන් තුටු වන්නේ යැ. එක් මැ අප්‍රමාදධර්මය වඩන්නේ යි.

මෙසේ මේ ගාථා ආයුෂ්මත් ධනිය ස්ථවිරයන් වහන්සේ ...

ධනිය ස්ථවිරගාථා යි.
