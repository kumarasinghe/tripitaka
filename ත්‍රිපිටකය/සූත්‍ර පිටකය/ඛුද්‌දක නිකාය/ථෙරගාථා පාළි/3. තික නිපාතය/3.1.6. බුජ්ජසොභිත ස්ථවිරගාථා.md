# 3.1.6. බුජ්ජසොභිත ස්ථවිරගාථා

234\. පැළලුප්නුවරවැසි චිත්‍රකථා ඇති බහුශ්‍රැත වූ යම් ශ්‍රමණ කෙනෙක් වෙද්ද, ඔවුනතුරෙහි අන්‍යතර වූ මේ ආයුෂ්මත් බුජ්ජසොහිත තෙර දොරටුවෙහි සිටී.

235\. පැළලුප්නුවරවැසි චිත්‍රකථා ඇති බහුශ්‍රැත වූ යම් ශ්‍රමණ කෙනෙක් වෙත් නම් ඔවුනතුරෙහි අන්‍යතර වූ මේ ආයුෂ්මත් තෙමේ (ඍද්ධිචිත්තයෙන් උපන්) පවතින් ආයේ දොරටුවෙහි සිටී.

236\. (තදඞ්ගවිෂ්කම්භණ විසින්) මනා වූ ක්ලේශයුද්ධයෙන් ද සත්ප්‍රාය වූ ධර්මදානයෙන් ද (සමුච්ඡෙදප්‍රහාණ විසින්) ලබනලද ක්ලෙශසඞ්ග්‍රාමිවිජයයෙන් ද පුහුණු කළ මඟබඹසරින් ද මෙසේ මේ තෙමේ (පලසමවත්) සුවවිඳී.

මෙසේ මේ ගාථා ආයුෂ්මත් බුජ්ජසොභිත ස්ථවිරයන් වහන්සේ ...

බුජ්ජසොභිත ස්ථවිරගාථා යි.
