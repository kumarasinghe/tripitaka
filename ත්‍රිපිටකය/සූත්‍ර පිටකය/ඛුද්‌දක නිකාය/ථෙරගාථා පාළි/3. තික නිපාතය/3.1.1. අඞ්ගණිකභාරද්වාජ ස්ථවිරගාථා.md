# 3.1.1. අඞ්ගණිකභාරද්වාජ ස්ථවිරගාථා

219\. අනුපායයෙන් (නොමගින්) සංසාරශුද්ධිය සොයන්නෙම් වෙතෙහි ගිනි පිදුයෙමි. ශුද්ධියට මාර්ගය නො දන්නෙම් (පඤ්චතාපනාදි) අමරතපස් කෙළෙමි.

220\. ඒ නිවන් සුවය සුවසේ (සුඛප්‍රතිපදායෙන්) ලබනලදි. ශාසනධර්මයා ගේ සුධර්මත්වය (නෛර්ය්‍යාණිකභාවය) බලව. මා විසින් ත්‍රිවිද්‍යාව ලබනලද. බුදුරදුන්ගේ අවවාදය කරනලදි.

221\. පෙර (ජාතිමාත්‍රයෙන් ප්‍රඥප්ති විසින්) බ්‍රාහ්මණ වෙමි. දැන් (බහිතපාපි වන බැවින් පරමාර්ථ විසින්) බ්‍රාහ්මණ වෙමි. ත්‍රිවිද්‍යා ඇත්තෙම් වෙමි. (සෝදාහළ කෙලෙස්මල ඇති බැවින්) ස්නාතක ද වෙමි. සොත්ථිය ද වෙමි. වෙද සඞ්ඛ්‍යාත මාර්ගඥානයෙන් සංසාරමහොඝයාගේ නොහොත් චතුස්සත්‍යයාගේ පාරයට ගිය බැවින් වෙදගු ද වෙමි යි.

මෙසේ මේ ගාථා ආයුෂ්මත් අඞ්ගණිකභාරද්වාජ ස්ථවිරයන් වහන්සේ ...
