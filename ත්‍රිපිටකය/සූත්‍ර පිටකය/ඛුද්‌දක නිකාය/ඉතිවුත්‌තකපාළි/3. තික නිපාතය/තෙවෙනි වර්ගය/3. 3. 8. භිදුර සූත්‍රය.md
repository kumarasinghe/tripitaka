# 3\. 3. 8. භිදුර සූත්‍රය

77\. භාග්‍යවතුන් වහන්සේ විසින් මෙය වදාරණ ලද්දේ ය. අර්හත් වූ බුදුරදුන් විසින් වදාරණ ලද්දේ ය. මා විසින් මෙසේ අසන ලදී:

1\. මහණෙනි, මේ සිරුර බිඳෙන සුලු ය. විඤ්ඤාණය (ත්‍රෛභූමකකුශලාදි චිත්තය) නැසෙන සුලු ය. සියලු උපධීහු අනිත්‍යයහ. දුඃඛයහ පෙරළෙන ස්වභාව ඇත්තාහු ය.

භාග්‍යවත්හු මෙ කරුණ වදාළහ. එහි අර්ථය (ගාථා වශයෙන්) මෙසේ කියනු ලැබේ:

2\. “සිරුර බිඳෙන සුලු ය යි ද, සිතත් බිඳෙන සුලු ය යි ද දැන උපධීන්හි බිය දැක ඉපදීම හා මරණය ඉක්මියේ ය. (රහත්වූයේ ය.) උතුම්වූ ශාන්තියට (නිර්‍වාණයට) පැමිණ භාවනා වශයෙන් වඩනා ලද අත්බව් (ශීලසමාධිප්‍ර‍ඥා) ඇත්තේ තමාගේ පරිනිර්‍වාණ කාලය බලාපොරොත්තු වෙයි.”

මෙ කරුණ ද භාග්‍යවතුන් වහන්සේ විසින් වදාරණ ලද්දේ ය. මෙසේ මා විසින් අසන ලදි.
