# 8.11.4. - 510

(සැවැත් නිදාන යි)

මහණෙනි, රාගයාගේ පරිජානනය පිණිස ... පරික්ෂය පිණිස ... ප්‍ර‍හාණය පිණිස ... ක්ෂය පිණිස ... ව්‍යය පිණිස ... විරාගය පිණිස ... නිරෝධය පිණිස ... ත්‍යාගය පිණිස ... ප්‍ර‍තිනිස්සර්‍ගය පිණිස ... මේ අටදහම්හු බැවිය යුතුවහ.

ද්වේෂයාගේ ... මෝහයාගේ ... ක්‍රෝධයාගේ ... උපනාහයාගේ ... මක්ඛයාගේ ... පලාසයාගේ ... ඉර්ෂ්‍යාවගේ ... මාත්සර්‍ය්‍යාගේ ... මායාවගේ ... සාඨෙය්‍යයාගේ ... ථම්භයාගේ ... සාරම්භයාගේ ... මානයාගේ ... අතිමානයාගේ ... මදයාගේ ... පමාදයාගේ ... අභිජානනය පිණිස ... පරිජානනය පිණිස ... පරික්ෂය පිණිස ... ප්‍ර‍හාණය පිණිස ... ක්ෂය පිණිස ... ව්‍යය පිණිස ... විරාගය පිණිස ... නිරෝධය පිණිස ... ත්‍යාගය පිණිස ... ප්‍ර‍තිනිස්සර්‍ගය පිණිස මේ අටදහම්හු බැවිය යුතුවහ.
