# 10.3.3.3. තෘතිය සූත්‍රය

මහණෙනි, සුගත් විනය හැර අන් තැනක මෙ දසදහම්හු මහත්ඵල මහානෘසංස නො වෙත්. කවර දස දහමෙක් යත්: සම්මාදිට්ඨි ය ... සම්මාවිමුක්ති යි. මහණෙනි, සුගත් විනය හැර අන් තැන්හි මෙ දසදහම්හු මහත්ඵල මහා නෘසංස නො වෙත් යී.
