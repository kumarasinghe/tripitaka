# 10.5.3.510

මහණෙනි, ප්‍ර‍මාදයාගේ අභිඥානය පිණිස ... පරිඥානය පිණිස ... පරික්ෂය පිණිස ... ප්‍ර‍හාණය පිණිස ... ක්ෂය පිණිස ... ව්‍යය පිණිස ... විරාගය පිණිස ... නිරොධය පිණිස ... ත්‍යාගය පිණිස ... ප්‍ර‍තිනිස්සර්ජනය පිණිස ... දස දහම්හු වැඩිය යුත්තාහ යි.
