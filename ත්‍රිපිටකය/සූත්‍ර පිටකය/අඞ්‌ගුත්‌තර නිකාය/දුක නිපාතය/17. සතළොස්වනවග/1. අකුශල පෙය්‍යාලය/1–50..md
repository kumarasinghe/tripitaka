# 1–50.

මහණෙනි, මේ අකුසල් දහම්හු දෙදෙනෙක් වෙති. ... මහණෙනි, මේ කුසල් දහම්හු දෙදෙනෙක් වෙති. ... මහණෙනි, මේ සාවද්‍යධර්‍මයෝ දෙදෙනෙක් වෙති. ... මහණෙනි, මේ අනවද්‍යධර්‍මයෝ දෙදෙනෙක් වෙති. ... මහණෙනි, මේ දුක් වඩන දහම්හු දෙදෙනෙක් වෙති. ... මහණෙනි, මේ සුව වඩන දහම්හු දෙදෙනෙක් වෙති. ... මහණෙනි, මේ දුඃඛවිපාක ඇති දහම්හු දෙදෙනෙක් වෙති. … මහණෙනි, මේ සුඛ විපාක ඇති දහම්හු දෙදෙනෙක් වෙති. ... මහණෙනි, මේ දුක් සහිත ඇති දහම්හු දෙදෙනෙක් වෙති. … මහණෙනි, මේ නිදුක් දහම්හු දෙදෙනෙක් වෙති. … කවර දෙදෙනෙක යත්: අක්‍රොධය හා අනුපනාහය හා. ... අමක්ඛය හා අපලාසය හා ... අනීර්ෂය්‍යාව හා අමාත්සර්‍ය්‍යය හා ... අමායාව හා අසාඨෙය්‍ය හා ... හිරිය හා ඔත්තප්පය හා දෙදෙන වෙති. මහණෙනි, මේ දහම්හු දෙදෙන නිදුක් දහම්හු වෙති.
