# 11–300.

මහණෙනි, කරුණු දෙකක් පිණිස තථාගතයන් වහන්සේ විසින් සව්වනට පාමොක් පණවන ලදී. ... පාමොක් උදෙසුම් පණවන ලදි. ... පාමොක් තැබීම් පණවන ලදි. ... පවාරණා පණවන ලදු ... පවාරණ තැබුම් පණවන ලදී. ... තජ්ජනීය කර්‍මය පණවන ලදී. ... නියස්සකර්‍මය පණවන ලදී. ... පබ්බාජනීයකර්‍මය පණවන ලදී. ... පටිසාරණීයකර්‍මය පණවන ලදී. ... උක්ඛෙපනීයකර්‍මය පණවන ලදී. ... පරිවාසදානය පණවන ලදී. ... මූලායපටිකස්සනය පණවන ලදී. ... මානත්තදානය පණවන ලදී. ... අබ්භානය පණවන ලදී. ... ඔසාරණීයය පණවන ලදී. ... නිස්සාරණීයය පණවන ලදී. ... උපසපුව පණවන ලදී. ... ඤත්තිකර්‍මය පණවන ලදී. ... ඤත්තිදුතියකර්‍මය පණවන ලදී. ... ඤත්තිචතුත්‍ථකර්‍මය පණවන ලදී. ... නො පණවන ලද්දෙහි (සිකපද) පණවන ලදී. ... පණවන ලද්දෙහි අනුප්පඤ්ඤත්ති පණවන ලදු. ... සම්මුඛාවිනය පණවන ලදී. ... සතිවිනය පණවන ලදී. ... අමූළහවිනය පණවන ලදී. ... පටිඤ්ඤාතකරණය පණවන ලදී. යෙභුය්‍යසිකාව පණවන ලදී. ... තස්සපාපිය්‍යසිකාව පණවන ලදී. තිණවත්‍ථාරකය පණවන ලදී. ...

කවර දෙකෙක යත්: සඞ්ඝයාගේ සුෂ්ඨුභාවය පිණිස හා ... සඞ්ඝඵාසුතාව පිණිස හා ... දුසිල් පුඟුලනට නිගා පිණිස හා ... පෙශල භික්ෂුනට ඵාසුවිහාරය පිණිස හා ... ඉහාත්මයෙහි වන ආස්‍රවයන්ගේ සංවරය පිණිස හා … සම්පරායික ආස්‍රවයන්ගේ වැළහීම පිණිස හා … දෘෂ්ටධාර්මික වෙරයන්ගේ … විද්‍යායන්ගේ … භීතින්ගේ … අකුශලධර්මයන්ගේ සංවරය පිණිස හා … සම්පරායික අකුශල ධර්මයන්ගේ ප්‍රතිඝාතය පිණිස හා … ගිහියනට අනුකම්පා පිණිස හා … පාපෙච්ඡායන්ගේ පක්ෂොපච්ඡෙදය පිණිස හා … අප්‍රසන්නයන්ගේ ප්‍රසාදය පිණිස හා … විනයානුග්‍රහය පිණිස හා යී. මහණෙනි, මෙ කරුණු දෙක පිණිස තථාගතයන් වහන්සේ විසින් සව්වනට තිණවත්‍ථාරකය පණවන ලදී.
