# 11–70.

මහණෙනි, ද්වෙෂයාගේ ... මොහයාගේ ... ක්‍රොධයාගේ ... උපනාහයාගේ ... මක්ඛයාගේ ... පලාසයාගේ ... ඊර්ෂය්‍යාවගේ ... මාත්සර්‍ය්‍යයාගේ ... මායාවගේ ... ශඨභාවයාගේ ... ස්තම්භයාගේ සාරම්භයාගේ ... මානයාගේ ... අතිමානයාගේ ... මදයාගේ ... ප්‍රමාදයාගේ ... ප්‍රත්‍යක්‍ෂ කිරීම පිණිස ... පරිජානනය පිණිස ... පරික්‍ෂය පිණිස ... ප්‍රහාණය පිණිස ... ක්‍ෂය පිණිස ... ව්‍යය පිණිස ... විරාගය පිණිස ... නිරොධය පිණිස ... ත්‍යාගය පිණිස ... නිසර්‍ජනය පිණිස ... මේ ධර්‍මයෝ දෙදෙන වැඩිය යුත්තාහ.
