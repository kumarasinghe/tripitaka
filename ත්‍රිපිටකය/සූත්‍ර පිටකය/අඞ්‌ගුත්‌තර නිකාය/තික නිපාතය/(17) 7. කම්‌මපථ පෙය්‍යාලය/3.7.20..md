# 3.7.20.

\[මහණෙනි, දහම් තුණෙකින් ...\]

තෙමේත් සම්‍යග්දෘෂ්ටික වෙයි. මෙරමාත් සම්‍යග්දෘෂ්ටියෙහි සමාදන් කරවයි. සම්‍යග්දෘෂ්ටියෙහි අනුදන්නේත් වෙයි.

මහණෙනි, මේ තුන් දහමින් සමන්වාගත වූයේ ගෙණවුත් තැබුයේ යම්සේ ද, එපරිදි ස්වර්‍ගයෙහි සිටියේ වේ.
