# 3.3.3.7. ප්‍රථම අනුරුද්‌ධ සූත්‍රය\*

**\[සැවැත් නිදානයි:\]**

27\. එසඳ ආයුෂ්මත් **අනුරුද්ධ** තෙරණුවෝ භාග්‍යවතුන් වහන්සේට තෙල සැළ කළහ: වහන්ස, මම විශුද්ධ වූ මිනිසැස ඉක්ම සිටුනා දිවැසින් කාබුන් මරණින් මතුයෙහි සුවයෙන් පහව සිටි නපුරු ගති ඇති විවසව පතිත වන නිරයට එළඹ සිටුනා මාගම් භූයොභාවයෙන් දක්මි.

වහන්ස, කෙතෙක් දහමින් සමන්වාගත මාගම කාබුන් මරණින් මතුයෙහි සුවයෙන් පහ වූ නපුරු ගති ඇති විවසව පතිත වන නිරයට එළඹේ ද?

අනුරුද්ධය, දහම් තුණෙකින් සමන්වාගත වූ මාගම කාබුන් මරණින් මතුයෙහි සුවයෙන් පහව සිටි නපුරු ගති ඇති විවසව පතිත වන නිරයට එළඹෙයි.

කවර දහම් තුණෙකින් යත්:

අනුරුද්ධය, මෙ ලොව මාගම පෙරවරු වේලෙහි මසුරුමලින් මඩනා ලද සිතින් ගෙහි වෙසෙයි. මධ්‍යාහ්න වේලෙහි ඊර්‍ෂ්‍යායෙන් මැඩුනු සිතින් ගෙහි වෙසෙයි. සවස් වේලෙහි කාමරාගයෙන් මැඩුනු සිතින් ගෙහි වෙසේ.

අනුරුද්ධය, මේ තුන් දහමින් සමන්වාගත මාගම කාබුන් මරණින් මතුයෙහි සුවයෙන් පහව සිටි නපුරු ගති ඇති විවසව පතිත වන නිරයට එළඹෙයි.
