# 4.5.3.3. දෙවන බාල – පණ්ඩිත සූත්‍රය

23\. මහණෙනි, සතර කරුණෙකින් සමන්වාගත වූ බාල වූ අව්‍යක්ත වූ අසත්පුරුෂ තෙමේ ගුණ විසින් සාරා දමන ලද, ගුණ විසින් නට ආත්මයක් පරිහරකය කෙරෙයි. සාවද්‍ය ද වෙයි. නුවණැතියන් විසින් උපවාද කටයුතු ද වෙයි. බොහෝ අකුසල් ද රැස් කෙරෙයි. කවර සතරෙකින යත්:

කායදුශ්චරිතයෙන, වාක්දුශ්චරිතයෙන, මනොදුශ්චරිතයෙන, අකෘතඥතා–අකෘතවෙදිතායෙන. \[මේ සතර කරුණින් ... බොහෝ අකුසල් ද රැස් කෙරෙයි.\]

\[මහණෙනි, සතර කරුණෙකින් සමන්වාගත ... සත්පුරුෂ තෙමේ ...\]

කායසුචරිතයෙන, වාක්සුචරිතයෙන, මනඃසුචරිතයෙන, කෘතඥතා–කෘතවෙදිතායෙන. \[මේ සතර කරුණින්... බොහෝ පින් ද රැස් කෙරෙයි.\]
