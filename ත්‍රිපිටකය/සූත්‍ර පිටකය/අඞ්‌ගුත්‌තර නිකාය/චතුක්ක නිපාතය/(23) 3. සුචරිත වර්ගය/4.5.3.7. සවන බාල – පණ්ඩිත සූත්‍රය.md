# 4.5.3.7. සවන බාල – පණ්ඩිත සූත්‍රය

27\. \[මහණෙනි, සතර කරුණෙකින් සමන්වාගත ... අසත්පුරුෂ තෙමේ ...\]

අදෘෂ්ට ආලම්බනයෙහි දෘෂ්ටවාදී වෙයි. අශ්‍රැත ආලම්බනයෙහි ශ්‍රැතවාදී වෙයි. අමුත ආලම්බනයෙහි මුතවාදී වෙයි. අවිඥාත ආලම්බනයෙහි විඥාතවාදී වෙයි. \[මේ සතර කරුණින් ... බොහෝ අකුසල් ද රැස් කෙරෙයි.\]

\[මහණෙනි, මේ සතර කරුණෙකින් සමන්වාගත ... සත්පුරුෂ තෙමේ ...\]

අදෘෂ්ට ආලම්බනයෙහි අදෘෂ්ටවාදී වෙයි. අශ්‍රැත ආලම්බනයෙහි අශ්‍රැතවාදී වෙයි. අමුත ආලම්බනයෙහි අමුතවාදී වෙයි. අවිඥත ආලම්බනයෙහි අවිඥතවාදී වෙයි. \[මේ සතර කරුණින් ... බොහෝ කුසල් ද රැස් කෙරෙයි.\]
