# 4.5.3.5. සිවුවන බාල – පණ්ඩිත සූත්‍රය

25\. \[මහණෙනි, සතර කරුණෙකින් සමන්වාගත ... අසත්පුරුෂ තෙමේ ...\]

මිසදිටු ඇත්තේ වෙයි. මිථ්‍යසඬ්කල්ප ඇත්තේ වෙයි. මිථ්‍යාවචන ඇත්තේ වෙයි. මිථ්‍යා කර්මාන්ත ඇත්තේ වෙයි. \[මේ සතර කරුණින් ... බොහෝ අකුසල් ද රැස් කෙරෙයි.\]

\[මහණෙනි, සතර කරුණෙකින් සමන්වාගත ... සත්පුරුෂ තෙමේ ...\]

සම්‍යග් දෘෂ්ටික වෙයි. සම්‍යක් සඬ්කල්ප ඇතියේ වෙයි. සම්‍යග් වචන ඇතියේ වෙයි. සම්‍යක් කර්‍මාන්ත ඇතියේ වෙයි. \[මේ සතර කරුණින් ... බොහෝ කුසල් ද රැස් කෙරෙයි.\]
