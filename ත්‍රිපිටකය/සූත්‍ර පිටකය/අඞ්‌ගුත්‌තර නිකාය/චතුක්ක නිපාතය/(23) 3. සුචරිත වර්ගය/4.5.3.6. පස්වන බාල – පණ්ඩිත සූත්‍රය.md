# 4.5.3.6. පස්වන බාල – පණ්ඩිත සූත්‍රය

26\. \[මහණෙනි, සතර කරුණෙකින් සමන්වාගත ... අසත්පුරුෂ තෙමේ ...\]

මිථ්‍යා ආජීව ඇතියේ වෙයි. මිථ්‍යා ව්‍යායාම ඇතියේ වෙයි. මිථ්‍යාස්මෘති ඇතියේ වෙයි. මිථ්‍යාසමාධි ඇතියේ වෙයි. \[මේ සතර කරුණින් ... බොහෝ අකුසල් ද රැස් කෙරෙයි.\]

\[මහණෙනි, සතර කරුණෙකින් සමන්වාගත ... සත්පුරුෂ තෙමේ ...\]

සම්‍යග් ආජීව ඇතියේ වෙයි. සම්‍යග් ව්‍යායාම ඇතියේ වෙයි. සම්‍යක් ස්මෘති ඇතියේ වෙයි. සම්‍යක් සමාධි ඇතියේ වෙයි. \[මේ සතර කරුණින් ... බොහෝ කුසල් ද රැස් කෙරෙයි.\]
