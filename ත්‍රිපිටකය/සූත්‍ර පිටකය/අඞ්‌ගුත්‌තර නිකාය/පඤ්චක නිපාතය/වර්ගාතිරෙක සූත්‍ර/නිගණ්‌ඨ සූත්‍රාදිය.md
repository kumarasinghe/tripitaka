# නිගණ්‌ඨ සූත්‍රාදිය

\[සැවැත්නිදාන යි\]

9-17. මහණෙනි, පස් කරුණෙකින් සමන්වාගත නුවට ... නුවටසවු ... දුළුලු … පිරිවැජි … මාගන්දික තීර්ථක ... තෙදණ්ඩික තීර්ථක ... ආරූද්ධක තීර්ථක ... ගොතමක තීර්ථක ... දෙවධම්මික තීර්ථක ගෙණවුත් තබන ලද්දකු සෙයින් නිරයෙහි උපදනේය. කවර පසෙකින් යත්:

ප්‍රාණවධ කරනුයේ වෙයි, අයිනාදන් ගන්නේ වෙයි, අබ්‍රම්සැරි වෙයි, මුසාවා බණනුයේ වෙයි, මදප්‍රමාදකාරණ වූ රහමෙර පානය කරනුයේ වේ.

මහණෙනි, මෙ පස් කරුණෙන් සමන්වාගත දෙවධම්මික ගෙණවුත් තබන ලද්දකු සෙයින් නිරයෙහි උපදනේ ය යි.

(ආජිවකාදීන්ගේ සිල්පරීමක් නැති බැවින් ස්වර්ගපක්ෂය නො වදාරන ලද)
