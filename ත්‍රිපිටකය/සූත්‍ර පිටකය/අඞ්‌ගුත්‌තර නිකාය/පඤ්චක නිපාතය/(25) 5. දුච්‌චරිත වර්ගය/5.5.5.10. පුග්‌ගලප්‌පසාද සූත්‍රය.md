# 5.5.5.10. පුග්‌ගලප්‌පසාද සූත්‍රය

\[සැවැත්නිදාන යි\]

50\. මහණෙනි, පුද්ගලයකු කෙරෙහි උපන් ප්‍රසාදයෙහි මේ ආදීනව පසෙකි. කවර පසෙක යත්:

මහණෙනි, යම් පුදුගලයකු කෙරෙහි පුද්ගලයෙක් අතිශයින් පහන් වූයේ නම්, හේ (ප්‍රසාදවිෂය වූ පුද්ගල තෙමේ) සඞ්ඝයා යම් බඳු ආපත්තියෙකින් උක්ෂෙපනිය කර්ම කෙරේ නම් එබඳු ආපත්තියකට පැමිණියේ වෙයි. ඔහුට (පහන්වූ පුඟුල්හට) මෙබඳු සිතෙක් වෙයි: “මට මේ පුද්ගල ප්‍රිය මනාප වෙයි, හේ සඟහු විසින් උතක්ෂිප්තය” යි. භික්ෂූන් කෙරෙහි අප්‍රසාදය බහුල කොට ඇතියේ වෙයි. භික්ෂූන් කෙරෙහි අප්‍රසාද බහුල කොට ඇතියේ ම සෙසු මහණුන් භජනය නොකරයි. අන්‍ය භික්ෂූන් ඇසුරු නො කරනුයේ සද්ධර්මය නො අසයි. සද්ධර්මය නො අසනුයේ සද්ධර්මයෙන් පිරිහෙයි. මහණෙනි, පුද්ගලප්‍රසාදයෙහි මේ ප්‍රථම ආදීනවයි.

තවද මහණෙනි, යම් පුඟුලකු කෙරෙහි පුද්ගලයෙක් අතිශයින් පහන් වේ ද, යම් ඇවතෙකින් සඟහු (යමෙකු) ආසනාන්තයෙහි හිඳුවත් නම්, හේ එබඳු ඇවැතකට පැමිණෙයේ වෙයි. ඔහුට මෙබඳු සිතෙක් වෙයි: “මේ පුද්ගල මට ප්‍රිය මනාප වෙයි. හෙතෙම සඟහු විසින් අන්තයෙහි හිඳුවන ලදැ” යි භික්ෂූන් කෙරෙහි අප්‍රසාද බහුල වෙයි. භික්ෂූන් කෙරෙහි අප්‍රසාද බහුල වූයේ ම අන්‍ය භික්ෂූන් භජනය නො කෙරෙයි. අන්‍ය භික්ෂූන් භජනය නොකරන්නේ සද්ධර්මය නො අසයි. සද්ධර්මය නො අසනුයේ සද්ධර්මයෙන් පිරිහෙයි. මහණෙනි, පුද්ගලප්‍රසාදයෙහි මේ ද්විතීය ආදීනවයි.

තවද මහණෙනි, යම් පුද්ගලයකු කෙරෙහි පුද්ගලයෙක් අභිප්‍රසන්න වේ නම්, හේ අන්‍ය දිශාවකට ගියේ වෙයි. ඔහුට මෙබඳු සිතෙක් වෙයි: ‘මේ පුද්ගල මට ප්‍රිය මනාප වෙයි. හේ අන්‍ය දිශාවකට ගියේ වෙයි’... අන්‍ය භික්ෂූන් භජනය නො කරයි ... මහණෙනි, පුද්ගල ප්‍රසාදයෙහි මේ තෘතීය ආදිනවයි.

තවද මහණෙනි ... හේ ගිහිබවට පැමිණියේ වෙයි ... ඔහුට මෙබඳු සිතෙක් වෙයි: ‘මේ පුද්ගල මට ප්‍රියමනාප වෙයි. හේ ගිහි බවට පැමිණියේ වෙයි’... අන්‍ය භික්ෂූන් භජනය නො කෙරෙයි ... මහණෙනි, පුද්ගල ප්‍රසාදයෙහි මේ චතුත්ථ ආදීනවයි.

තවද මහණෙනි ... හේ කලුරිය කෙළේ වෙයි. ඔහුට මෙබඳු සිතෙක් වෙයි: “මේ පුද්ගල මට ප්‍රිය මනාප වෙයි. හේ කලුරිය කෙළේය” යි ... අන්‍ය භික්ෂූන් නො ඇසුරු කෙරෙයි. අන්‍ය භික්ෂූන් භජනය නො කරන්නේ සදහම් නො අසයි. සදහම් නො අසන්නේ සද්ධර්මයෙන් පිරිහෙයි. මහණෙනි, පුද්ගලප්‍රසාදයෙහි මේ පඤ්චම ආදීනවයි.

මහණෙනි, පුද්ගලප්‍රසාදයෙහි මේ ආදීනව පසයි.
