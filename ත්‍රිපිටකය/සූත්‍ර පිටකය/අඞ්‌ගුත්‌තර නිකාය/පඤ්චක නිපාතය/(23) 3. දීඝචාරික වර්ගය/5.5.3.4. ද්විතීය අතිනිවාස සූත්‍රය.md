# 5.5.3.4. ද්විතීය අතිනිවාස සූත්‍රය

\[සැවැත්නිදාන යි\]

24\. මහණෙනි, අතිනිවාසයෙහි මේ ආදීනව පසෙකි. කවර පසෙක යත්:

ආවාසයෙහි මසුරු වෙයි. කුලයෙහි මසුරු වෙයි. ලාභයෙහි මසුරු වෙයි. ගුණයෙහි මසුරු වෙයි. ධර්මයෙහි මසුරු වෙයි.

මහණෙනි, මේ අතිනිවාසයෙහි ආදීනව පස යි.

මහණෙනි, සමවස්ථිත නිවාසයෙහි මේ අනුසස් පසෙකි. කවර පසෙක යත්:

ආවාසයෙහි මසුරු නොවෙයි. කුලයෙහි මසුරු නොවෙයි. ලාභයෙහි මසුරු වෙයි. ගුණයෙහි මසුරු නොවෙයි. ධර්මයෙහි මසුරු නොවේ.

මහණෙනි, සමවස්ථිත නිවාසයෙහි මේ ආනිසංස පසයි.
