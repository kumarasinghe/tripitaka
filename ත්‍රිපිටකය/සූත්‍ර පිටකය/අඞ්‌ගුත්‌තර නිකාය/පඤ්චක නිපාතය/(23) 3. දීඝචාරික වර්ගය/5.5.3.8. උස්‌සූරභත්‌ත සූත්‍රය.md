# 5.5.3.8. උස්‌සූරභත්‌ත සූත්‍රය

\[සැවැත්නිදාන යි\]

28\. මහණෙනි, හිරුනැගි බොහෝ දහවල් අහර පිසින කුලයෙහි මේ දොස් පසෙකි.

සත්කාරාර්හ යම් අතිථිහූ (ආන්තුකයෝ) වෙත් නම්, ඔවුන් නිසි කල්හි නො පුදනාහ පුද පිළිගන්නා යම් දෙවතා කෙනෙක් වෙත් නම්, ඔවුන් නිසි කල්හි නො පුදනාහ. එකභක්තික වූ රාත්‍රිභෝජනයෙන් වැළකී විකාලභොජනයෙන් විරත වූ යම් මහණබමුණු කෙනෙක් වෙත් නම්, නිසි කල්හි ඔවුන් නො පුදනාහ. දස්කම්කරුපුරුෂයෝ විමුඛව කර්ම (වැඩ හැර හිඳිත්). අසමයෙහි වැළඳූ භොජනය එකෙණෙහි ඔජොරහිත වෙයි.

මහණෙනි, බොහෝ දහවල් කොට අහර පිසින කුලයෙහි මේ ආදීනව පස යි.

මහණෙනි, නිසි කල්හි අහර පිසින කුලෙහි මේ අනුසස් පසෙකි. කවර පසෙක යත්:

යම් සත්කාරාර්හ අතිථිහූ වෙත් නම්, ඔවුන් නිසි කල්හි පුදනාහ. පුද පිළිගන්නා යම් දෙවතා කෙනෙක් වෙත් නම්, ඔවුන් නිසි කල්හි පුදනාහ. එකභක්තික වූ රාත්‍රිභෝජනයෙන් වැළකී විකාලභොජනයෙන් විරත වූ යම් මහණබමුණෝ වෙත් නම්, ඔවුන් සුදුසු කල්හි පුදනාහ. දස්කම්කරුපුරුෂයෝ අවිමුඛ (උත්සුක) වූවාහු කර්ම කරන්නාහ. සුදුසු කාලයෙහි වැළඳූ අහර එකෙණෙහි ඔජස් ඇතියේ වෙයි.

මහණෙනි, සමයභක්තකුලයෙහි මේ අනුසස් පස යි.
