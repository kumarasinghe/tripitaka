# 5.3.3.9. පරිකුප්‌ප සූත්‍රය

(සැවැත්නිඳාන යි)

29\. මහණෙනි, මේ‍ අපායගාමීහු නිරයගාමීහු පරිකොප්‍ය (කිපෙන සැහැවී ඇති) වූවාහු පිළියම් නොකට හැකියාහු පස් දෙනෙකි. කවර පස් දෙනක යත්:

මව දිවියෙන් තොර කරණ ලදු වෙයි, පියා දිවියෙන් තොර කරණ ලද වෙයි, අර්‍හත් දිවියෙන් තොර කරණ ලද වෙයි, දුෂ්ටචිත්තයෙන් තථාගතයන්ගේ ලෙහෙ උපදවන ලද වෙයි, සඞ්ඝයා බිඳින ලද වේ.

මහණෙනි, මොහු පස්දෙන අපායගාමීහු නිරයගාමීහු පරිකොප්‍ය (පැරණි වණයන් වැනි) වූවාහු පිළියම් නොකට හැකියාහු වෙත් ය යි.
