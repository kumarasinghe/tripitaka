# 1.3.7.

මහණෙනි, යමක් මෙසේ නොවඩන ලද්දේ බහුල වශයෙන් පුරුදු නොකරණ ලද්දේ මහත් වූ අනර්ථය පිණිස පවත්නේ නම්, සිත බඳු අන් එක ද ධර්මයක් මම (නුවණැසින්) නො දක්මි. මහණෙනි, නො වැඩු බහුල වශයෙන් පුරුදු නොකළ සිත මහත් වූ අනර්ථය පිණිස පවත්නේ ය.
