# 1.3.5.

මහණෙනි, යමක් මෙසේ නොවඩන ලද්දේ (විදර්ශනා මාර්ග ඵල නිර්වාණයන් අරමුණු කොට) පහළ නො වූයේ මහත් වූ අනර්ථය පිණිස පවත්නේ නම්, සිත බඳු අන් එක ධර්මයකුදු මම (නුවණැසින්) නො දක්මි. මහණෙනි, නොවැඩු (විදර්ශනාදිය අරමුණු කොට) පහළ නො වූ සිත මහත් වූ අනර්ථය පිණිස පවත්නේ ය.
