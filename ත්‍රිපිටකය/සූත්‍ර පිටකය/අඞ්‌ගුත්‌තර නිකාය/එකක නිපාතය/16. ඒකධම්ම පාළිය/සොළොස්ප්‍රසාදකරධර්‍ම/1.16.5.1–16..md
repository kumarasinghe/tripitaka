# 1.16.5.1–16.

මහණෙනි, යම් ආරණ්‍යක බවෙක් වේ ද, ... පිණ්ඩපාතික බවෙක් වේ ද, ... පංසුකූලික බවෙක් වේ ද, ... තෙචීචරික බවෙක් වේ ද, ... ධර්‍මකථික බවෙක් වේ ද, ... විනයධර බවෙක් වේ ද, ... බහුශ්‍රැත බවෙක් වේ ද, ... ස්ථාවරප්‍රාප්ත බවෙක් වේ ද, ... ආකල්පසම්පත්තියෙක් වේ ද, ... පරිවාරසම්පත්තියෙක් වේ ද, ... මහපිරිවර ඇති බවෙක් වේ ද, ... කුලපුත් බවෙක් වේ ද, ... රූපසම්පන්න බවෙක් වේ ද, ... බස්කම්හි මියුරු හඬ ඇති බවෙක් වේ ද, ... අල්පෙච්ඡ බවෙක් වේ ද, ... ආරොග්‍යසම්පත්තියෙක් වේ ද, මෙය ලාභයන් අතුරෙහි ඒකාන්ත ලාභය වේ.
