# 1.16.6.143–182.

... ‍මෛත්‍රීසහගත වූ, ... කරුණාසහගත වූ, ... මෘදුසහගත වූ, ... උපේක්‍ෂාසහගත වූ, ... ශ්‍රද්ධෙන්ද්‍රිය ය වඩා ද, ... වීර්‍ය්‍යෙන්‍ද්‍රිය වඩා ද, ... ස්මෘතින්‍ද්‍රියය වඩා ද, ... සමාධීන‍්ද්‍රියය වඩා ද, ... ප්‍රඥෙන්‍ද්‍රියය වඩා ද, ... ශ්‍රද්ධාබලය වඩා ද, … වීර්‍ය්‍යබලය වඩා ද, ... ස්මෘතිබලය වඩා ද, … සමාධිබලය වඩා ද, ... ප්‍රඥාබලය වඩා ද, ... මහණෙනි, මේ මහණ නො හළ දැහැන් ඇත්තේ ය යි කියනු ලැබේ. ශාස්තෘශාසනකර වූයේ, ඔවා පිළිගෙන කරන්නේ නො සිස් වූ රට්ඨපිණ්ඩය වළඳයි. යම් කෙනෙක් එය බහුල වශයෙන් කෙරෙත් නම් කවර කථාය.
