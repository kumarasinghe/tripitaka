# 1.16.7.20–21.

මහණෙනි, එක් ම ධර්මයෙක් වඩන ලද්දේ බහුල වශයෙන් කරණ ලද්දේ ප්‍රඥා ප්‍රභෙදය පිණිස පවත්නේ ය … අනුපාදා පරිනිර්වාණය පිණිස පවත්නේ ය. කවර එක් ධර්මයෙක යත්: කාගියාසී ය යි. මහණෙනි, මේ එක් ම ධර්මය වඩනා ලද්දේ බහුල වශයෙන් කරණ ලද්දේ ප්‍රඥාප්‍රභෙදය පිණිස පවත්නේ ය. අනුපාදා පරිනිර්වාණය පිණිස පවත්නේ ය.
