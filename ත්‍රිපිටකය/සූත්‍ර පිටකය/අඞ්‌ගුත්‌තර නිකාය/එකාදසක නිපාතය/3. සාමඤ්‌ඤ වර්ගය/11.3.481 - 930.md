# 11.3.481 - 930

මහණෙනි, එකොළොස් කරුණෙකින් සමන්වාගත ගොපාල තෙම ගොමණ්ඩල පරිහරණයට වර්ධනයට භව්‍ය වෙයි. කවර එකොළොසෙකින යත්: මහණෙනි, මෙ ලොව්හි ගොපාලක තෙම රූපඥ වෙයි ...

මහණෙනි, එපරිද්දෙන්ම එකොළොස් කරුණෙකින් සමන්වාගත මහණ චක්ෂුස්හි අනිත්‍යානුදර්ශීව වාසය කරන්නට භව්‍ය වෙයි ... ප්‍ර‍තිනිඃසර්‍ගානුදර්ශීව වාසය කරනට භව්‍ය වේයයි.
