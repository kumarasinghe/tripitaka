# 11.4.11 - 170

ද්වෙෂයාගේ ... මොහයාගේ ... ක්‍රොධයාගේ ... උපනාහයාගේ ... මක්ඛයාගේ ... පළාසයාගේ ... ඊර්‍ෂ්‍යහුගේ ... මාත්සර්‍ය්‍යයාගේ ... මායාවගේ ... ශාඨ්‍යයාගේ ... ස්තම්භයාගේ ... සාරම්භයාගේ ... මානයාගේ ... අතිමානයාගේ ... මදයාගේ ... ප්‍ර‍මාදයාගේ අභිඥානය පිණිස ... පරිඥානය පිණිස ... පරීක්ෂය පිණිස ... ප්‍ර‍හාණය පිණිස ... ක්ෂය පිණිස .. ව්‍යය පිණිස ... විරාගය පිණිස ... ත්‍යාග පිණිස ... ප්‍ර‍තිනිඃසර්‍ගය පිණිස ... මේ එකොළොස් දහම්හු වැඩිය යුත්තාහ යි.

භාග්‍යවතුන් වහන්සේ මෙය වදාළ සේක. සතුටුසිත් ඇති ඒ භික්ෂූහූ භාග්‍යවතුන් වහන්සේගේ භාෂිතය සතුටින් පිළිගත්හ යි.
