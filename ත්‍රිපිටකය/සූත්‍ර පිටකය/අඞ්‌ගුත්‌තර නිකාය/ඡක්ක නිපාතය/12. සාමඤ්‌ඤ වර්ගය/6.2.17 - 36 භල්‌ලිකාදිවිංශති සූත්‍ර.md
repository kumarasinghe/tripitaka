# 6.2.17 - 36 භල්‌ලිකාදිවිංශති සූත්‍ර

\[සැවැත්නිදාන යි\]

17 – 36. මහණෙනි, කරුණු සයෙකින් සමන්වාගත භල්ලික (ද්වෙවාචික) ගෘහපති ... අනාථපිණ්ඩික සුදත්ත ගෘහපති ... මච්ඡිකාසණ්ඩිකචිත්ත ගෘහපති ... හත්ථක ආලවක ... මහානාම ශාක්‍ය ... වෛශාලික උග්ග ගෘහපති ... උග්ගත ගෘහපති ... සූරඅම්බට්ඨ ... ජීවකකොමාරභච්ච ... නකුලපිතු ගෘහපති ... තවකණ්ණික ගෘහපති ... පූරණ ගෘහපති ... ඉසිදත්ත ගෘහපති ... සන්ධාන ගෘහපති ... විජය ගෘහපති ... වජ්ජීයමහිත ගෘහපති ... මෙණ්ඩක ගෘහපති ... වාසෙට්ඨ උපාසක ... අරිට්ඨ උපාසක ... සාරග්ග උපාසක ... තථාගතයන් කෙරෙහි නිෂ්ඨාවට ගියේ අමෘතදර්ශන ඇතියේ අමෘතය සාක්ෂාත් කොට වෙසේ ය යි. කවර සයෙකින යත්:

බුදුන් කෙරෙහි අචෙත්‍යප්‍රසාදයෙන, ධර්මය කෙරෙහි අචෙත්‍ය ප්‍රසාදයෙන, සඞ්ඝයා කෙරෙහි අචෙත්‍යප්‍රසාදයෙන, ආර්‍ය්‍යශීලයෙන, ආර්‍ය්‍යඥානයෙන, ආර්‍ය්‍යවිමුක්තියෙනැයි.

මහණෙනි, මෙ කරුණු සයින් සමන්වාගත සාරග්ග උපාසක තථාගතයන් කෙරෙහි සැදැහැයෙන් නිෂ්ඨාවට ගියේ දුටු නිවන් ඇතියේ අමෘතය සාක්ෂාත් කොට වෙසේ යයි.
