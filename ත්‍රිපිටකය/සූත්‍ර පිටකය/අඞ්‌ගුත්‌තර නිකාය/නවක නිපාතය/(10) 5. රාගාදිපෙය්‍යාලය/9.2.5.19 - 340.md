# 9.2.5.19 - 340

(සැවැත්නිදාන යි)

දොසයාගේ … මොහයාගේ … කොධයාගේ … උපනාහයාගේ … මක්ඛයාගේ … පලාසයාගේ … ඉස්සාවගේ … මච්ඡරියයාගේ … මායාවගේ … සාඨෙය්‍යයාගේ … ථම්භයාගේ … සාරම්භයාගේ … මානයාගේ … අතිමානයාගේ … මදයාගේ … පමාදයාගේ … අභිඥානය පිණිස … පරික්ෂය පිණිස … ප්‍රහාණය පිණිස … ක්ෂය පිණිස … ව්‍යය පිණිස … විරාගය පිණිස … නිරොධය පිණිස … ත්‍යාගය පිණිස … පටිනිස්සග්ගය පිණිස මේ නව දහම්හු වැඩිය යුතු වෙති යී, භාග්‍යවතුන් වහන්සේ මෙය වදාළසේක. සතුටුසිත් ඇත් ඒ භික්ෂූහු භාග්‍යවතුන් වහන්සේගේ භාෂිතය සතුටින් පිළිගත්හ.
