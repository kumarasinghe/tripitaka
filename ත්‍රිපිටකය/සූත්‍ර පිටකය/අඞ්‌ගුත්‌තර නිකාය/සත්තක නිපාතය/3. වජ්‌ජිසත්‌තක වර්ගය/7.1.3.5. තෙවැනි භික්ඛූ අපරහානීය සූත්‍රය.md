# 7.1.3.5. තෙවැනි භික්ඛූ අපරහානීය සූත්‍රය

\[සැවැත්නිදාන යි\]

25\. මහණෙනි, තොපට සප්ත අපරිහානීය ධර්‍මයන් දෙසන්නෙමි. එ අසවු ... මහණෙනි, සප්ත අපරිහානීය ධර්‍මයෝ කවරහ යත්:

මහණෙනි, යම් තාක් භික්ෂූහු ශ්‍රද්ධා ඇතියාහු වෙත් ද, මහණෙනි, භික්ෂූන්ට වෘද්ධිය ම විනා පරිහානියෙක් නො කැමැති වියයුතු.

මහණෙනි, යම් තාක් භික්ෂූහු හිරි ඇතියාහු වෙත් ද ... ඔතප් ඇතියාහු වෙත් ද, බහුශ්‍රැත වෙත් ද ... රුකුළුවැර ඇති වෙත් ද ... ස්මෘතිමත් වෙත් ද ... ප්‍රඥාවත් වෙත් ද, මහණෙනි, භික්‍ෂූනට වෘද්ධිය ම විනා පරිහානියෙක් නො කැමැති විය යුතු.

මහණෙනි, යම් තාක් මෙ සප්ත අපරිහානීය ධර්‍මයෝ භික්ෂූන් කෙරෙහි සිටුනාහු ද, භික්ෂූහු දු මෙ සප්තාපරිහානීය ධර්‍මයන්හි පැණෙත් ද, මහණෙනි, භික්ෂූන්හට වෘද්ධිය ම මිස පරිහානියෙක් නො කැමැති විය යුතු යි.
