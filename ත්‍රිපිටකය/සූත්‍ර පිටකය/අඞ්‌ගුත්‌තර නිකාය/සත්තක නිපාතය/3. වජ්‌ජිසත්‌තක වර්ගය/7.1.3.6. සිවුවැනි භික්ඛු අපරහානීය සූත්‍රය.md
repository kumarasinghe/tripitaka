# 7.1.3.6. සිවුවැනි භික්ඛු අපරහානීය සූත්‍රය

\[සැවැත්නිදාන යි\]

26\. මහණෙනි, තොපට සප්ත අපරිහානීය ධර්‍මයන් දෙසන්නෙමි. ඒ අසවු ... මහණෙනි, සප්ත අපරිහානිය ධර්‍මයෝ කවරහ යත්:

මහණෙනි, යම් තාක් භික්‍ෂූහු සතිසම්බොජඣඞ්ගය වඩත් ද, මහණෙනි, භික්ෂුන්ට වෘද්ධිය ම මිස පරිහානියෙක් නො කැමති විය යුතු.

මහණෙනි, යම් තක් භික්‍ෂූහු ධම්මවිචය සම්බොජ්ඣඞ්ගය වඩත් ද ... විරිය සම්බොජ්ඣඞ්ගය වඩත් ද ... පීතිසම්බොජ්ඣඞ්ගය වඩත් ද ... පස්සද්ධි සම්බොජ්ඣඞ්ගය වඩත් ද ... සමාධිසම්බොජ්ඣඞ්ගය වඩත් ද ... උපෙක්ඛාසම්බොජ්ඣඞ්ගය වඩත් ද, මහණෙනි, භික්ෂූන්ට වෘද්ධිය ම මිස පරිහානියෙක් නො කැමති වියයුතු යයි.

මහණෙනි, යම් තාක් මෙ සප්ත අපරිහානීය ධර්‍මයෝ භික්‍ෂූන් කෙරෙහි පවත්නාහු නම්, භික්‍ෂුහු ද මෙ සප්ත අපරිහානීය ධර්‍මයන් කෙරෙහි පැණෙත් නම්, මහණෙනි, භික්ෂූන්ට වෘද්ධිය ම මිස පරිහානියෙක් නො කැමැති විය යුතු යයි.
