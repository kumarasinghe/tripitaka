# 7.1.2.2. අනුසයප්පහාන සූත්‍රය

(සැවැත් නිදාන යි)

12\. මහණෙනි, සප්තානුශයන්ගේ ප්‍ර‍හාණය සඳහා සමුච්ඡෙදය සඳහා බඹසර වසනු ලැබේ. කවර සතක්හුගේය යත්:

කාමරාගනුසයාගේ ප්‍ර‍හාණය සඳහා සමුච්ඡෙදය සඳහා බ්‍ර‍ම්සර වසනු ලැබේ. පටිඝානුසයයාගේ ... දිට්ඨානුසයයාගේ ... විචිකිච්ඡානුසයයාගේ ... මානානුසයයාගේ ... භවරාගානුසයයාගේ ... අවිජ්ජානුසයයාගේ ප්‍ර‍හාණය සඳහා සමුච්ඡෙදය සඳහා බ්‍ර‍හ්මචර්‍ය්‍යය වසනු ලැබේ.

මහණෙනි, මෙ සප්තානුශයයන්ගේ ප්‍ර‍හාණය සඳහා සමුච්ඡෙදය සඳහා බ්‍ර‍හ්මචර්‍ය්‍යය වසනු ලැබේ.

මහණෙනි, යම් හෙයෙකින් භික්ෂුහුගේ කාමරාගානුසය ප්‍ර‍හීණ වේ නම් උසුන්මුල් ඇති වේ නම්, තාලාවස්තුකෘත වේ නම්, අන්‍වභාවකෘත වේ නම් මතුයෙහි නූපදනා සැහැවි ඇතියේ නම්, ප්‍ර‍තිඝානුශය ... දෘෂ්ටානුශය ... විචිකිත්සානුශය .. මානානුශය ... භවරාගානුශය ... අවිද්‍යානුශය ප්‍ර‍හීණ වේ නම් උසුන්මුල් ඇතියේ වේ නම් තාලාවස්තුකෘත වේ නම් අන්‍වභාවකෘත වේ නම් මතුයෙහි නූපදනා සැහැවි ඇතියේ වේ නම්, මහණෙනි, අනුශය රහිත මහණ තෘෂ්ණාව සිඳලීය සංයොජන පෙරළාලී ය මැනැවින් අර්හත්ත්‍වාවබොධයෙන් වර්‍තදුඃඛය අවසන් කෙළේ යයි කියනු ලැබේ.
