# 7.1.1.9. සඤ්ඤොජනප්පහාන සූත්‍රය

\[සැවැත්නිදාන යි\]

9\. මහණෙනි, සංයෝජන සතක්හුගේ ප්‍ර‍හාණය සඳහා සමුච්ඡේදය සඳහා බඹසර වසනු ලැබේ. කවර සතක්හුගේ ය යත්:

අනුනය සංයෝජනයාගේ ප්‍ර‍හාණය සඳහා සමුච්ඡේදය සඳහා බඹසර වසනු ලැබේ. ප්‍ර‍තිඝසංයෝජනයාගේ ප්‍ර‍හාණය සඳහා ... දෘෂ්ටිසංයෝජනයාගේ ... විචිකිච්ඡාසංයෝජනයාගේ … මානසංයෝජනයාගේ … භවරාගසංයෝජනයාගේ ... අවිද්‍යාසංයෝජනයාගේ ප්‍ර‍හාණය සඳහා සමුච්ඡේදය සඳහා බ්‍ර‍ම්සර වැසේ.

මහණෙනි, මෙ සංයෝජන සත ප්‍ර‍හාණය කරණු සඳහා සමුච්ඡේදය සඳහා බ්‍ර‍හ්මචර්‍ය්‍යය වසනු ලැබේ.

මහණෙනි, යම්හෙයකින් භික්ෂුහුගේ අනුනය සංයෝජනය ප්‍ර‍හීණ වේ ද, උසුන්මුල් ඇතියේ වේ ද, තාලාවස්තුකෘත ද, අන්‍වභාවකෘත ද, මත්තෙහි නූපදනා සැහැවි ඇතියේ ද ... ප්‍ර‍තිඝසංයෝජනය ... දෘෂ්ටිසංයෝජනය ... විචිකිත්සාසංයෝජනය ... මානසංයෝජනය ... භවරාගසංයෝජනය ... අවිද්‍යාසංයෝජන ය ප්‍ර‍හීණ ද උසුන්මුල් ඇතියේ ද, තාලාවස්තුකෘත ද, අන්වභාවකෘත ද, මත්තෙහි නූපදනා සැහැවි ඇතියේ ද, මහණෙනි, එහෙයින් මේ මහණ තෘෂ්ණාව සිඳලී යයි ද, සංයෝජන පෙරළාලී යයි ද, මොනොවට මානප්‍ර‍හාණයෙන් සසරදුක් අවසන් කෙළේ ය යි ද කියනු ලැබේ.
