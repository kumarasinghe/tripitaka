# 2.3.7. අනන්ත සූත්‍රය

188\. **සැවැත්නුවර:**

එකත්පස් ව හුන් ආයුෂ්මත් **රාධ** තෙරණුවෝ භාග්‍යවතුන් වහන්සේට මෙය සැලකළහ: වහන්ස, …: “නිවන් කරා යොමු කළ සිත් ඇති වැ වෙසෙන්නෙම් නම් භාග්‍යවතුන් වහන්සේ එබඳු දහමක් සැකෙවින් මට දේශනා කරන සේක් නම් යහපතැ යි.

රාධය, යමෙක් අනාත්ම නම් එහි ලා තොපගේ ඡන්දය පහකටයුතු යැ. රාගය පහ කටයුතු යැ, ඡන්දරාගය පහකටයුතු යි. රාධය, කිමෙක් අනාත්ම ද යත්: රාධය, රූපය අනාත්ම යැ ; එහි ලා තොපගේ ඡන්දය පහකටයුතු යැ, රාගය පහකට යුතු යැ, ඡන්දරාගය පහකටයුතු යි. වේදනාව … සංඥාව … සංස්කාර … විඥානය අනාත්ම යැ. එහි ලා තොපගේ ඡන්දය පහකටයුතු යැ, රාගය පහකටයුතු යැ, ඡන්දරාගය පහකටයුතු යි. රාධය, යමෙක් අනාත්ම නම් එහි ලා තොපගේ ඡන්දය පහකට යුතු යැ, රාගය පහකටයුතු යැ, ඡන්දරාගය පහකටයුතු යි.
