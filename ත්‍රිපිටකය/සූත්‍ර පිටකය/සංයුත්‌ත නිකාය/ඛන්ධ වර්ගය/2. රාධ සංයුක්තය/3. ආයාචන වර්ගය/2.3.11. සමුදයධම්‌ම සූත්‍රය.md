# 2.3.11. සමුදයධම්‌ම සූත්‍රය

192\. **සැවැත්නුවර:**

එකත් පස් ව හුන් ආයුෂ්මත් **රාධ** තෙරණුවෝ භාග්‍යවතුන් වහන්සේට මෙය සැල කළහ: වහන්ස, …: “නිවන් කරා යොමු කළ සිතැති වැ වෙසෙම් ද, භාග්‍යවතුන් වහන්සේ එබඳු දහමක් සැකෙවින් මට දේශනා කරන සේක් නම් මැනවැ”යි.

රාධය, යමක් හටගන්නාසුලු නම් එහි ලා තොපගේ ඡන්දය පහකටයුතු යැ, රාගය පහකටයුතු යැ ඡන්දරාගය පහකටයුතු යි.

රාධය, කිමෙක් හටගන්නාසුලු වේ ද යත්: රාධය, රූපය හටගන්නාසුලු යැ. එහි ලා තොපගේ ඡන්දය පහකටයුතු යැ. රාගය පහකටයුතු යැ ඡන්දරාගය පහකටයුතු යි. වේදනාව … සංඥාව … සංස්කාර … විඥානය හටගන්නාසුලු යැ. එහි ලා තොපගේ ඡන්දය පහකටයුතු යැ, රාගය පහකටයුතු යැ. ඡන්දරාගය පහකටයුතු යි. රාධය, යමෙක් හටගන්නාසුලු නම් එහි ලා තොපගේ ඡන්දය පහකටයුතු යැ, රාගය පහකටයුතු යැ, ඡන්දරාගය පහකටයුතු යි.
