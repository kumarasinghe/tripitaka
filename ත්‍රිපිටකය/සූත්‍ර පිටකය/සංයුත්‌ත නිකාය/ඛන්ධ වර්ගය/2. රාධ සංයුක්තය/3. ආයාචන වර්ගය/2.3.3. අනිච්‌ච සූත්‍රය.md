# 2.3.3. අනිච්‌ච සූත්‍රය

184\. **සැවැත්නුවර:**

එකත්පස් ව හුන් ආයුෂ්මත් **රාධ** තෙරණුවෝ භාග්‍යවතුන් වහන්සේට මෙය සැල කළහ: වහන්ස, … නිවන් පිණිස යොමු කළ සිත් ඇති වැ වෙසෙන්නෙම් ද භාග්‍යවතුන් වහන්සේ එ බඳු දහමක් මට සැකෙවින් දේශනා කරන සේක් නම් මැනවැ”යි.

රාධය, යමෙක් අනිත්‍ය නම් එහි ලා තොපගේ ඡන්දය පහකළයුතු යැ, රාගය පහකළ යුතු යැ, ඡන්දරාගය පහකළයුතු යි. රාධය, කිමෙක් අනිත්‍ය ද යත්: රාධය, රූපය අනිත්‍ය යැ, එහි ලා තොපගේ ඡන්දය පහකළයුතු යැ, රාගය පහකළයුතු යැ, ඡන්දරාගය පහකළයුතු යි. වේදනාව … සංඥාව … සංස්කාර … විඥානය අනිත්‍ය යැ. එහි ලා තොපගේ ඡන්දය පහකළයුතු යැ, රාගය පහකළයුතු යැ, ඡන්දරාගය පහකළ යුතු යි. රාධය යමෙක් අනිත්‍ය නම් එහි ලා තොපගේ ඡන්දය පහකළයුතු යැ, රාගය පහකළයුතු යැ, ඡන්දරාගය පහකළයුතු යි.
