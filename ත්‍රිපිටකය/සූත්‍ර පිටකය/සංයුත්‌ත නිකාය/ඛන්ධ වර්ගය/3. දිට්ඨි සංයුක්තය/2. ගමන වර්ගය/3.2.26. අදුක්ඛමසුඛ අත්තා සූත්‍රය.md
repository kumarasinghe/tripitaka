# 3.2.26. අදුක්ඛමසුඛ අත්තා සූත්‍රය

249\. **සැවැත්නුවර:**

… “ආත්මය නොදුක් නොසුව ඇත්තේ මරණින් මතු නීරොග වේ දැ” යි.

වහන්ස, අපගේ දහම්හු භගවත්හු මුල් කොට ඇතියහ …

මහණෙනි, රූපය ඇති කල්හි රූපය ඇසුරු කොට රූපයට පිවිස “ආත්මය නොදුක් නොසුව ඇත්තේ මරණින් මතු නීරොග වේ ය”යි මෙබඳු දෘෂ්ටියක් උපදී. වේදනාව ඇති කල්හි … සංඥාව ඇති කල්හි … සංස්කාරයන් ඇති කල්හි … විඥානය ඇති කල්හි විඥානය ඇසුරු කොට විඥානයට පිවිස ආත්මය නොදුක් නොසුව ඇත්තේ මරණින් මතු නිරෝග වේ ය”යි මෙබඳු දෘෂ්ටියක් උපදී.

මහණෙනි, ඒ කිමැ යි හඟිවු ද? රූපය නිත්‍ය හෝ වේ ද? අනිත්‍ය හෝ වේ දැ? යි…  
වෙනස් වනසුලු නම් එය නො ඇසුරු කොට “ආත්මය නොදුක් නොසුව ඇත්තේ මරණින් මතු නීරොග වේ ය”යි මෙබඳු දෘෂ්ටියක් උපදනේ දැ? යි.

වහන්ස, මෙය නො වේ මැ යැ.

වේදනාව … සංඥාව … සංස්කාරයෝ … විඥානය නිත්‍ය හෝ වේ ද? අනිත්‍ය හෝ වේ දැ? යි … වෙනස් වනසුලු නම් එය නො ඇසුරු කොට “ආත්මය නොදුක් නොසුව ඇත්තේ මරණින් මතු නීරොග වේ ය”යි මෙ බඳු දෘෂ්ටියක් උපදනේ දැ?යි.

වහන්ස, මෙය නො වේ ම ය.

මහණෙනි, මෙසේ දුක ඇති කල්හි දුක ඇසුරු කොට දුකට පිවිස “ආත්මය නොදුක් නොසුව ඇත්තේ මරණින් මතු නීරොග වේ” යයි මෙබඳු දෘෂ්ටියක් උපදී. ...
