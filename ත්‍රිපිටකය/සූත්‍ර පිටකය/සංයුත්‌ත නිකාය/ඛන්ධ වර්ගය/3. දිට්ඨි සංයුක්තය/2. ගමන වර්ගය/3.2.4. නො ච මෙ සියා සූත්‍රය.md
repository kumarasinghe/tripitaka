# 3.2.4. නො ච මෙ සියා සූත්‍රය

227\. **සැවැත්නුවර:**

… “ඉදින් මම නො වන්නෙම් නම් පස්කඳ මට නො ද වන්නේ ය. (මතු) ද (මම) නො වන්නෙම් නම් (මතු) ද මට පස්කඳ නො වන්නේ ය යි. … මතු මට පස්කඳ නොවන්නේ ය යි. (මෙබඳු දෘෂ්ටියක් උපදී.)
