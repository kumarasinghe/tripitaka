# 3.2.3. සො අත්‌තා සූත්‍රය

226\. **සැවැත්නුවර:**

… ‘ඒ මම ආත්මයෙමි, ඒ මම ලෝකයෙමි, ඒ මම පරලොව වන්නෙමි, නිත්‍යයෙමි, ධ්‍රැවයෙමි, ශාශ්වතයෙමි, නොවෙනස් වනසුල්ලෙමි”යි …

වහන්ස, අපගේ දහම්හු භගවත්හු මුල් කොට ඇතියහ. … වෙනස් වන සුල්ලෙමි’යි (මෙබඳු දෘෂ්ටියක් උපදී.)
