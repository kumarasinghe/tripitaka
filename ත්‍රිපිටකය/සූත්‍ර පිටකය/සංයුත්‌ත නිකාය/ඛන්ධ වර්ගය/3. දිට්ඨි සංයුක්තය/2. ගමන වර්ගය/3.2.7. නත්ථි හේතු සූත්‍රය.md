# 3.2.7. නත්ථි හේතු සූත්‍රය

230\. **සැවැත්නුවර:**

… සත්ත්වයන්ගේ කෙලෙසීමට හේතුවෙක් නැත. ප්‍රත්‍යයෙක් නැත, හේතු රහිත ව ප්‍රත්‍යය රහිත ව සත්ත්වයෝ කෙලෙසෙත්, … අභිජාති සයෙහි ම සුවදුක් විඳිති”යි ... විඳිති යි (මෙබඳු දෘෂ්ටියක් උපදී)
