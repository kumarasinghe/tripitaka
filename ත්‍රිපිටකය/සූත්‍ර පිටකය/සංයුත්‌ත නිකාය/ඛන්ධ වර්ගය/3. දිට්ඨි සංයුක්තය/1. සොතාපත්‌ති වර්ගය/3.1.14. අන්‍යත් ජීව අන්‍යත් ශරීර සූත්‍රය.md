# 3.1.14. අන්‍යත් ජීව අන්‍යත් ශරීර සූත්‍රය

219\. **සැවැත්නුවර:**

මහණෙනි, කුමක් ඇති කල්හි කුමක් ඇසුරු කොට කුමකට පිවිස ජීවය අනෙකක, ශරීරය අනෙකෙකැ’යි මෙබඳු දෘෂ්ටියක් උපදී දැ? යි.

වහන්ස, අපගේ දහම්හු භාග්‍යවත්හු මුල් කොට ඇතියහ …

මහණෙනි, රූපය ඇති කල්හි … වේදනාව ඇති කල්හි … සංඥාව ඇති කල්හි … සංස්කාරයන් ඇති කල්හි … විඥානය ඇති කල්හි විඥානය ඇසුරු කොට විඥානයට පිවිස ජීවය අනෙකක, ශරීරය අනෙකෙකැ’යි මෙ බඳු දෘෂ්ටියක් උපදී.

මහණෙනි, ඒ කිමැ යි හඟිවු ද? රූපය නිත්‍ය හෝ වේ ද? අනිත්‍ය හෝ වේ දැ යි? ... එය නොඇසුරු කොට ‘ජීවය අනෙකක, ශරීරය අනෙකෙකැ’යි මෙබඳු දෘෂ්ටියෙක් උපදී දැ යි?

වහන්ස, තෙල නො වේ මැ යැ.

වේදනාව … සංඥාව … සංස්කාර … විඥානය … මේ යමෙකුත් දක්නා ලද්දේ අසනලද්දේ මුත විසින් ගන්නා ලද්දේ දන්නා ලද්දේ පැමිණෙන ලද්දේ සොයන ලද්දේ සිතින් හැසිරෙන ලද්දේ වේ නම් හෙ ද නිත්‍ය හෝ වේ ද? අනිත්‍ය හෝ වේ දැ? යි. … එය නොඇසුරු කොට ජීවය අනෙකක, ශරීරය අනෙකෙකැ’යි මෙබඳු දෘෂ්ටියෙක් උපදී දැ යි?

වහන්ස, මෙය නො වේ මැයි.

මහණෙනි, යම් කලෙක අරීිසවුහුගේ … නියත වූයේ සම්බෝධිය පිහිට කොට ඇත්තේ යි.
