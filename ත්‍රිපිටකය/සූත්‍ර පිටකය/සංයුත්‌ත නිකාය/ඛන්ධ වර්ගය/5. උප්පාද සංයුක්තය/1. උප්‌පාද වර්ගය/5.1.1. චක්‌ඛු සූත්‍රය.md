# 5.1.1. චක්‌ඛු සූත්‍රය

312\. **සැවැත්නුවර:**

මහණෙනි, ඇස පිළිබඳ යම් ඉපැදීමෙක් පැවැත්මෙක් බිහිවීමෙක් පහළවීමෙක් වේ නම් මේ දුක්හි ඉපැද්ම යැ. රෝගයන්ගේ පැවැත්ම යැ. ජරාමරණයෙහි පහළවීම යි. කන පිළිබඳ යම් … නැහැය පිළිබඳ යම් ... දිව පිළිබඳ යම් … කය පිළිබඳ යම් … සිත පිළිබඳ යම් ඉපැද්මෙක් පැවැත්මෙක් බිහි වීමෙක් පහළවීමෙක් වේ නම් මේ දුකෙහි ඉපැද්ම යැ. රෝගයන්ගේ පැවැත්ම යැ. ජරාමරණයෙහි පහළවීම යි.

මහණෙනි, ඇස පිළිබඳ යම් නිරෝධයෙක් ව්‍යුපශමයෙක් අස්තඞ්ගමයෙක් වේ නම් මේ දුකෙහි නිරෝධ යැ, රෝගයන්ගේ ව්‍යුපශම යැ, ජරාමරණයෙහි අස්තඞ්ගමය යි. කන පිළිබඳ යම් නිරෝධයෙක් … නැහැය පිළිබඳ යම් යම් නිරෝධයෙක් ... දිව පිළිබඳ යම් නිරෝධයෙක් … කය පිළිබඳ යම් නිරෝධයෙක් … සිත පිළිබඳ යම් නිරෝධයෙක් ව්‍යුපශමයෙක් අස්තඞ්ගමයෙක් වේ නම් මේ දුකෙහි නිරෝධ යැ, රෝගයන්ගේ ව්‍යුපශම යැ, ජරාමරණයෙහි අස්තඞ්ගම යි.
