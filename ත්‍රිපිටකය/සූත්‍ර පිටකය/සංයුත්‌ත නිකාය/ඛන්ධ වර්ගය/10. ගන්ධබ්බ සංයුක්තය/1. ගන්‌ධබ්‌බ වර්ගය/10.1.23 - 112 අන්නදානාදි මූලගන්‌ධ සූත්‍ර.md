# 10.1.23 - 112 අන්නදානාදි මූලගන්‌ධ සූත්‍ර

460–549. **සැවැත්නුවර:**

එකත්පස් ව හුන් මහණ, භාග්‍යවතුන් වහන්සේට තෙල සැල කෙළේ යැ: “වහන්ස, යම් හෙයකින් මෙහි එකෙක් කාබුන් මරණින් මතු හරගඳෙහි අරක්ගත් දෙවියන්ගේ සහභාවයට පැමිණේ නම් ... ඵල ගඳෙහි අරක්ගත් දෙවියන්ගේ ... සිවි ගඳෙහි අරක්ගත් දෙවියන්ගේ ... පොතු ගඳෙහි අරක්ගත් දෙවියන්ගේ ... පත් ගඳෙහි අරක්ගත් දෙවියන්ගේ ... මල් ගඳෙහි අරක්ගත් දෙවියන්ගේ ... පල ගඳෙහි අරක්ගත් දෙවියන්ගේ ... රස ගඳෙහි අරක්ගත් දෙවියන්ගේ ... ගඳ ගඳෙහි අරක්ගත් දෙවියන්ගේ සහභාවයට පැමිණේ නම් එයට හේතු ද, කිමෙක් ප්‍රත්‍යය දැ” යි.

මහණ, මෙහි එකෙක් කයින් සුසිරිත් කෙරෙයි, වදනින් සුසිරිත් කෙරෙයි, මනසින් සුසිරිත් කෙරෙයි, ඔහු විසින් “ගඳ අරක්ගත් දෙවියෝ දීර්ඝායුෂ්කයහ, වර්ණවත්හ, සුඛබහුලහ” යි අසන ලද්දේත් වෙයි.

(එහෙයින්) ඔහුට “මම කාබුන් මරණින් මතු ගඳ ගඳෙහි අරක්ගත් දෙවියන්ගේ සහභාවයට පැමිණෙන්නෙම් නම් ඉතා මැනැවැ”යි මෙබඳු සිතෙක් වෙයි. හේ අහර දෙයි. ... හේ බොන දෑ දෙයි. ... හේ වස්ත්‍ර දෙයි. ... හේ වහන් (වාහන) දෙයි. ... හේ මල් දෙයි. ... හේ ගඳ දෙයි. ... හේ විලවුන් දෙයි. ... හේ සෙනසුන් දෙයි. ... හේ වාසස්ථාන දෙයි. ... හේ පහන් උපකරණ දෙයි. හෙ කාබුන් මරණින් මතු ගඳගඳෙහි අරක්ගත් දෙවියන්ගේ සහභාවයට පැමිණේ.

මහණ, යම් හෙයකින් මෙහි එකෙක් කාබුන් මරණින් මතු ගඳගඳෙහි අරක්ගත් දෙවියන්ගේ සහභාවයට පැමිණේ නම් එයට හේතු යි, මේ ප්‍රත්‍යය යි.

(මෙසේ පිඬුවශයෙන් එක්සිය දොළසෙක් සූත්‍ර වෙයි.)
