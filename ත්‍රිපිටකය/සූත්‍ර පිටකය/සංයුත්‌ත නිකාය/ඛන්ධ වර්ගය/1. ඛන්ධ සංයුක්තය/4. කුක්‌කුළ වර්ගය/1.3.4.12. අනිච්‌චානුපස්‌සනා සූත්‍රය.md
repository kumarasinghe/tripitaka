# 1.3.4.12. අනිච්‌චානුපස්‌සනා සූත්‍රය

147\. **සැවැත්නුවර:**

මහණෙනි, යම් හෙයෙකින් රූපයෙහි අනිස අනුව දකිමින් වෙසෙන්නේ නම් වේදනායෙහි ... සංඥායෙහි ... සංස්කාරයන්හි ... විඥානයෙහි අනිස අනුව දකිමින් වෙසෙන්නේ නම් සැදැහැයෙන් පැවිදි වූ කුලපුත්හට මේ අනුදහම වෙයි.

හේ රූපයෙහි අනිස අනුව දකිමින් වෙසෙන්නේ වේදනායෙහි ... සංඥායෙහි ... සංස්කාරයන්හි ... විඥානයෙහි අනිස අනුව දකිමින් වෙසෙන්නේ රූපය පිරිසිඳ දනියි. වේදනාව ... සංඥාව ... සංස්කාර ... විඥානය පිරිසිඳ දනී. හෙ රූපය පිරිසිඳ දන්නේ වේදනාව ... සංඥාව ... සංස්කාර … විඥානය පිරිසිඳ දන්නේ රූපයෙන් මිදෙයි, වේදනාවෙන් මිදෙයි, සංඥායෙන් මිදෙයි, සංස්කාරයන්ගෙන් මිදෙයි, විඥානයෙන් මිදෙයි. ජාතියෙන්, ජරාමරණයෙන්, ශෝකයෙන්, පරිදේවයෙන්, දුකින්, දොම්නසින් උපායාසයෙන් මිදෙයි. (සියලු) දුකින් මිදේ ය’යි කියමි යි.
