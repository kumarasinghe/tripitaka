# 1.2.2.3. අභිනන්‌දන සූත්‍රය

65\. **සැවැත්නුවර:**

එකල්හි එක්තරා මහණෙක් භාග්‍යවතුන් වහන්සේ වැඩහුන් තැනට එළැඹියේ යැ. එළැඹ ... එකත්පසෙකැ හුන් ඒ මහණ භාග්‍යවතුන් වහන්සේට තෙල සැලකෙළේ ය:

“වහන්ස, ... (මම්) නිවන් කරා යොමු කළ සිතැති වැ වාස කෙරෙම් ද, මට එබඳු දහමක් සැකෙවින් දේශනා කරන සේක් නම් මැනවි” යි.

“මහණ, තෘෂ්ණාදි විසින් අභිනන්දන කරනුයේ මාරපාශයෙන් බැඳු‍ණේ වෙයි, අභිනන්දන නො කරනුයේ පාපිමත්හුගේ (මරහුගේ) පාශයෙන් මිදුණේ වෙයි.”

“භාග්‍යවතුන් වහන්ස, දන්නා ලදැ යැ, සුගතයන් වහන්ස, දන්නා ලදැ”යි.

“මහණ, මා විසින් සැකෙවින් දේශිත ධර්මයාගේ අර්ථ කෙසේ නම් තෙපි විස්තර විසින් දනිවු දැ?”

වහන්ස, රූපය, අභිනන්දන කරනුයේ මාරපාශයෙන් බැඳුණේ වෙයි, අභිනන්දන නො කරනුයේ මාරපාශයෙන් මිදුණේ වෙයි. වේදනාව ... සංඥාව ... සංස්කාර ... විඥානය අභිනන්දන කරනුයේ මාරපාශයෙන් බැඳුණේ වෙයි, අභිනන්දන නො කරනුයේ මාරපාශයෙන් මිදුණේ වෙයි. වහන්ස, භාග්‍යවතුන් වහන්සේ විසින් සැකෙවින් දේශිත ධර්මයාගේ අර්ථ මෙසෙයින් මම් විස්තර වශයෙන් දනිමි යි.

මහණ, මනා යැ මනා යැ. මහණ, මා විසින් සැකෙවින් දේශිතධර්මයාගේ අර්ථ මෙසෙයින් තෙපි විස්තර ලෙස දනිවු.

මහණ, රූපය අභිනන්දන කරනුයේ මාරපාශයෙන් බැඳුණේ වෙයි, අභිනන්දන නො කරනුයේ මාරපාශයෙන් මිදුණේ වෙයි. වේදනාව ... සංඥාව ... සංස්කාර ... විඥානය අභිනන්දන කරනුයේ මාරපාශයෙන් බැඳුනේ වෙයි, අභිනන්දන නො කරනුයේ මාරපාශයෙන් මිදුණේ වෙයි. මහණ මා විසින් සැකෙවින් දේශිත මේ ධර්මයාගේ අර්ථ මෙසෙයින් විස්තර ලෙස දතයුතු යි.

එකල්හි ඒ මහණ භාග්‍යවතුන් වහන්සේගේ භාෂිතය පිළිගෙන ... භාග්‍යවතුන් වහන්සේ පැදකුණු කොට නික්ම ගියේ යැ. ඉක්බිති ඒ මහණ එකලා වැ ගණයා කෙරෙන් වෙන් වැ ... එ මහණ රහතුන් වහන්සේ අතරැ අන්‍යතරයෙක් වූයේ යි.
