# 1.1.4.8. දෙවැනි අනුධම්‌ම සූත්‍රය

40\. **සැවැත්නුවර:**

මහණෙනි, ධර්මානුධර්ම ප්‍රතිපන්න මහණහට මේ අනුලෝමධර්ම වෙයි:

යම් හෙයෙකින් රූපයෙහි අනිත්‍යානුදර්ශී වැ (අනිත්‍ය නුවණින්, දකිමින්) වාස කෙරේ ද, වේදනායෙහි … සංඥායෙහි … සංස්කාරයන්හි … විඥානයෙහි අනිත්‍යානුදර්ශී වැ වාස කෙරේ ද, හේ රූපයෙහි අනිත්‍යානුදර්ශී වැ වාස කරනුයේ වේදනායෙහි ... සංඥායෙහි … සංස්කාරයන්හි … විඥානයෙහි අනිත්‍යානුදර්ශී වැ ව්‍යාස කරනුයේ රූපය පිරිසිඳ දනියි. වේදනාව ... සංඥාව …. සංස්කාර … විඥානය පිරිසිඳ දනියි. හේ රූපය පිරිසිඳ දන්නේ වේදනාව … සංඥාව … සංස්කාර … විඥානය පිරිසිඳ දන්නේ රූපයෙන් මිදෙයි. වේදනායෙන් මිදෙයි, සංඥායෙන් මිදෙයි, සංස්කාරයන්ගෙන් මිදෙයි, විඥානයෙන් මිදෙයි. ජාතියෙන් – ජරා මරණයෙන් – ශෝකයෙන් – පරිදේවයෙන් – දුකින් – දොම්නසින් උපායාසයෙන් මිදෙයි. පස්කඳ දුකින් මිදේ යයි කියමි යි.
