# 1.2.3.3. දෙවැනි සමුදය සූත්‍රය

75\. **සැවැත්නුවර:**

මහණෙනි, ශ්‍රැතවත් ආර්යශ්‍රාවක රූපයාගේ සමුදය ද, අස්තඞ්ගමය ද, ආස්වාදය ද, ආදීනව ද, නිශ්ශරණය ද, ඇති සැටි දනියි. වේදනාවගේ ... සංඥාවගේ ... සංස්කාරයන්ගේ ... විඥානයාගේ සමුදය ද, අස්තඞ්ගමය ද, ආස්වාදය ද, ආදීනව ද, නිශ්ශරණය ද ඇති සැටි දනී යි.
