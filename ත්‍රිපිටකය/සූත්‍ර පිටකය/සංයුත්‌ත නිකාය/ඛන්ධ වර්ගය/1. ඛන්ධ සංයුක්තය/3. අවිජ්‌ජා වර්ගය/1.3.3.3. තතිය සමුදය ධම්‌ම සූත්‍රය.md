# 1.3.3.3. තතිය සමුදය ධම්‌ම සූත්‍රය

128\. **බරණැස්නුවර:**

එකත්පස් ව හුන් ආයුෂ්මත් මහාකොට්ඨිත තෙරණුවෝ ආයුෂ්මත් සැරියුත් තෙරණුවන්ට තෙල කීහ: “ඇවැත්නි, ශාරිපුත්‍රයෙනි, විජ්ජා විජ්ජා යි කියනු ලැබේ. ඇවැත්නි විජ්ජා නම් කවර යැ? කොපමණෙකින් විජ්ජාගතත් වේ දැ”යි?

ඇවැත්නි, මෙ සස්නෙහි ශ්‍රැතවත් ආර්යශ්‍රාවක හටගන්නාසුලු රූපය හටගන්නාසුලු රූප යයි තතු සේ දනියි, වැනසෙනසුලු රූපය ... හටගන්නා-වැනසෙනසුලු රූපය හටගන්නා-වැනසෙනසුලු රූප යයි තතුසේ දනී. හටගන්නාසුලු වේදනාව ... හටගන්නාසුලු සංඥාව ... හටගන්නාසුලු සංස්කාර ... හටගන්නාසුලු විඥානය හටගන්නාසුලු විඥාන ය යි තතුසේ දනී වැනසෙනසුලු විඥානය ... හටගන්නා-වැනසෙනසුලු විඥානය හටගන්නා-වැනසෙනසුලු විඥාන ය යි තතුසේ දනී. ඇවැත්නි, මේ විජ්ජා යි කියනු ලැබෙයි. මෙපමණෙකින් විජ්ජාගතත් වේ යයි.
