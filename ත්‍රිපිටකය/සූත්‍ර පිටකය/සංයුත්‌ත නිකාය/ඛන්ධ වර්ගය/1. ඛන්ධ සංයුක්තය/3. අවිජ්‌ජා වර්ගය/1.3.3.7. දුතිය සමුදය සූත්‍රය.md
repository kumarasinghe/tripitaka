# 1.3.3.7. දුතිය සමුදය සූත්‍රය

132\. **බරණැස්නුවර:**

එකත්පස් ව හුන් ආයුෂ්මත් මහාකොට්ඨිත තෙරණුවෝ ආයුෂ්මත් සැරියුත් තෙරණුවන්ට මෙය කීහ: ඇවැත්නි, ශාරීපුත්‍රයෙනි, විද්‍යා විද්‍යා යි කියනු ලැබේ. ඇවැත්නි විද්‍යාව නම් කවර යැ? කොපමණෙකින් විද්‍යායුක්තත් වේ දැ? යි.

ඇවැත්නි, මෙහි ශ්‍රැතවත් ආර්යශ්‍රාවක තෙමේ රූපයාගේ සමුදයත් අස්තඞ්ගමයත් ආස්වාදයත් ආදීනවයත් නිස්සරණයත් තතු සේ දනී. වෙදනාවගේ ... සංඥාවගේ ... සංස්කාරයන්ගේ ... විඥානයාගේ සමුදයත් අස්තඞ්ගමයත් ආස්වාදයත් ආදීනවයත් නිස්සරණයත් තතුසේ දනී. ඇවැත්නි, මේ විද්‍යාව යයි කියනු ලැබේ. මෙතෙකින් විද්‍යායුක්තත් වේ ය යි.
