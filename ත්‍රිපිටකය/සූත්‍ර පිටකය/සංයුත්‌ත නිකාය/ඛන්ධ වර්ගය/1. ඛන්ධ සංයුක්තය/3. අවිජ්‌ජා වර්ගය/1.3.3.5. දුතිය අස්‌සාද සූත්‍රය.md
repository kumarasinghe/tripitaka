# 1.3.3.5. දුතිය අස්‌සාද සූත්‍රය

130\. **බරණැස්නුවර:**

එකත්පසෙක හුන් ආයුෂ්මත් මහාකොට්ඨිත තෙරණුවෝ ආයුෂ්මත් සැරියුත් තෙරණුවන්ට මෙය කීහ: “ඇවැත්නි, ශාරීපුත්‍රයෙනි, විජ්ජා විජ්ජා යි කියනු ලැබෙයි, ඇවැත්නි, විජ්ජා නම් කවරක්ද? කොපමණෙකින් විජ්ජාගතත් වේ දැ”යි.

ඇවැත්නි, මේ සස්නෙහි ශ්‍රැතවත් ආර්යශ්‍රාවක රූපයාගේ ආස්වාදයත් ආදීනවයත් නිශ්ශරණයත් තතුසේ දනියි. වෙදනාවේ ... සංඥාවේ ... සංස්කාරයන්ගේ ... විඥානයාගේ ආස්වාදයත් ආදීනවයත් නිශ්ශරණයත් තතුසේ දනී. ඇවැත්නි, මේ විජ්ජායි කියනු ලැබේ. මෙපමණෙකින් විජ්ජාගතත් වේ යයි.
