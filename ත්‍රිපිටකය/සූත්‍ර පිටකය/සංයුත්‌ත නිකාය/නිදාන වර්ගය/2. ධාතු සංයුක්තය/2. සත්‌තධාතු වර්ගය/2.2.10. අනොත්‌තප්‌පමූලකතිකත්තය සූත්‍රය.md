# 2.2.10. අනොත්‌තප්‌පමූලකතිකත්තය සූත්‍රය

244\. **සැවැත්නුවර-**

මහණෙනි, සත්ත්වයෝ අදහස් වශයෙන් ම සැසැඳෙත්, එකතු වෙත්. පවට බිය නැත්තෝ පවට බිය නැත්තවුන් සමග … අල්පශ්‍රැතයෝ අල්පශ්‍රැතයන් සමග … දුෂ්ප්‍රාඥයෝ දුෂ්ප්‍රාඥයන් සමග … පාපභීරුකයෝ පාපභීරුකයන් සමග … බහුශ්‍රැතයෝ බහුශ්‍රැතයන් සමග … ප්‍රාඥයෝ ප්‍රාඥයන් සමග සැසැඳෙත්, එකතු වෙත්. මහණෙනි, අයුකල්හි දු … මහණෙනි, අනාකල්හි දු … මහණෙනි, දැන් වත්මන් කල්හි දු … පවට බිය නැතියෝ … අල්පශ්‍රැතයෝ … දුෂ්ප්‍රාඥයෝ … පවට බිය ඇතියෝ … බහුශ්‍රැතයෝ … නුවණැත්තෝ නුවණැත්තවුන් සමග සැසැඳෙත්, එකතු වෙත්. (පළමුවන තිකය යි.)

මහණෙනි, සත්ත්වයෝ අදහස් වශයෙන් ම සැසැඳෙත්, එකතු වෙත්. පවට බිය නැත්තෝ පවට බිය නැත්තවුන් සමග සැසැඳෙත්, එකතු වෙත්. කුසීතයෝ කුසීතයන් සමග … දුෂ්ප්‍රාඥයෝ දුෂ්ප්‍රාඥයන් සමග … පාපභීරුකයෝ පාපභීරුකයන් සමග … පිරිපුන් වැර ඇත්තෝ පිරිපුන් වෙර ඇත්තවුන් සමග … ප්‍රාඥයෝ ප්‍රාඥයන් සමග සැසැඳෙත්, එකතු වෙත්. මහණෙනි, අයුකල්හි දු … මහණෙනි, අනාකල්හි දු … මහණෙනි, දැන් වත්මන් කල්හි දු … පවට බිය නැතියෝ … කුසීතයෝ … දුෂ්ප්‍රාඥයෝ … පාපභිරුකයෝ … පිරිපුන් වැර ඇත්තෝ … ප්‍රාඥයෝ ප්‍රාඥයන් සමග සැසැඳෙත්, එකතු වෙත්. (දෙවන තිකය යි.)

මහණෙනි, සත්ත්වයෝ අදහස් වශයෙන් ම සැසැඳෙත්, එකතු වෙත්. පවට බිය නැත්තෝ පවට බිය නැත්තවුන් සමග සැසැඳෙති, එකතු වෙති. මුළා සිහි ඇත්තෝ මුළා සිහි ඇත්තවුන් සමග … දුෂ්ප්‍රාඥයෝ දුෂ්ප්‍රාඥයන් සමග … පවට බිය ඇත්තෝ පවට බිය ඇත්තවුන් සමග … එළඹ සිටි සිහි ඇත්තෝ එළඹ සිටි සිහි ඇත්තවුන් සමග … පැණැත්තෝ පැණැත්තවුන් සමග සැසැඳෙත්, එකතු වෙත්. මහණෙනි, අයුකල්හි දු … මහණෙනි, අනාකල්හි දු … මහණෙනි, දැන් වත්මන් කල්හි දු … පවට බිය නැතියෝ … සිහි නැතියෝ … දුෂ්ප්‍රාඥයෝ … පවට බිය ඇත්තෝ … සිහි ඇත්තෝ … ප්‍රාඥයෝ ප්‍රාඥයන් සමග සැසැඳෙත්, එකතු වෙත්. (තෙවන තිකය යි.)
