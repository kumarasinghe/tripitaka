# 2.2.12. කුසීතමූලකතිකෙක සූත්‍රය

246\. **සැවැත්නුවර-**

මහණෙනි, සත්ත්වයෝ අදහස් වශයෙන් ම සැසැඳෙත්, එකතු වෙත්: කුසීතයෝ කුසීතයන් සමග සැසැඳෙත්. එකතු වෙත්. මුළා සිහි ඇත්තෝ මුළා සිහි ඇත්තවුන් සමග … නුවණ නැත්තෝ නුවණ නැතවුන් සමග … පිරිපුන් වැර ඇත්තෝ පිරිපුන් වැර ඇත්තවුන් සමග … එළඹ සිටි සිහි ඇත්තෝ එළඹ සිටි සිහි ඇත්තවුන් සමග ... පැණැතියෝ පණැත්තවුන් සමඟ සැසැඳෙති, එකතු වෙති. මහණෙනි, අතීතයෙහිදු … මහණෙනි, අනාගතයෙහි දු … මහණෙනි, දැන් වත්මන් හි දු … කුසීතයෝ … සිහි නැතියෝ … නුවණ නැතියෝ … පිරිපුන් වැර ඇතියෝ … සිහි ඇතියෝ … පණැතියෝ පණැත්තන් සමග සැසැඳෙති, එකතු වෙති.
