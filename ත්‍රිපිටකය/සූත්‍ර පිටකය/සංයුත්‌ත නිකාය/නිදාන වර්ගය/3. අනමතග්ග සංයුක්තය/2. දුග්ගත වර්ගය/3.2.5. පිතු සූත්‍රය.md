# 3.2.5. පිතු සූත්‍රය

278\. **සැවැත්නුවර-**

මහණෙනි, මෙසසර අවිදිතාග්‍රය … මහණෙනි, මේ දිගුකල යම් සතෙක් එක්සතක්හු පිය නො වූ විරූ ද, එබඳු සතෙක් සුලබ නො වේ. ඒ කවර හෙයින යත්: මහණෙනි, මෙසසර අවිදිතාග්‍රය ... මහණෙනි, එහෙයින් මිදෙන්නට නිසි ම ය. මිදෙන්නට සුදුසු මැ යි.
