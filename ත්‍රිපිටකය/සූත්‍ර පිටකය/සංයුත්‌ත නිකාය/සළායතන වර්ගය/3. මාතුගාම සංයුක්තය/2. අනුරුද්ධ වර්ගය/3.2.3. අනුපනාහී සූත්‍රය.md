# 3.2.3. අනුපනාහී සූත්‍රය

297\. මහණෙනි, දහම් පසෙකින් යුතු මාගම කාබුන් මරණින් මතු සුගති වූ සගලොවට පැමිණෙ යි. කවර පසෙකින් ද යත්: සැදැහැ ඇති වෙයි. ලජ්ජා ඇති වෙයි. බිය ඇති වෙයි. බැඳි වෙර නැති වෙයි. නුවණැති වෙයි. මහණෙනි මේ පස්දහමින් යුතු මාගම කාබුන් මරණින් මතු සුගති වූ සගලොවට පැමිණේ යි.
