# 3.3.5. අඞ්‌ග සූත්‍රය

310\. මහණෙනි, මාගමට මේ බල පසෙකි. කවර පසෙක් ද යත්: රූපබලය භොගබලය ඤාතිබලය පුත්‍රබලය සීලබලය යි. මහණෙනි, මාගම රූපබලයෙන් යුතු වූවා භොගබලයෙන් නො ද යුතු වේද, මෙසේ ඇය ඒ අඟින් නො පිරිපුන් වෙයි. මහණෙනි, යම් හෙයෙකින් මාගම රූපබලයෙනුත් භොගබලයෙනුත් යුතු වූවා වේ ද මෙසේ ඇය ඒ අඟින් පිරිපුන් වෙයි. මහණෙනි, මාගම රූපබලයෙනුත් භොගබලයෙනුත් යුතු වූවා ඤාතිබලයෙන් නො ද යුතු වූවා වේද, මෙසේ ඇය ඒ අඟින් නො පිරිපුන් වෙයි. මහණෙනි, යම් හෙයෙකින් මාගම රූපබලයෙනුත් භොගබලයෙනුත් ඤාතිබලයෙනුත් යුතු වූවා වේ ද මෙසේ ඇය ඒ අඟින් පිරිපුන් වෙයි.

මහණෙනි, මාගම රූපබලයෙනුත් භොගබලයෙනුත් ඤාතිබලයෙනුත් යුතු වූවා පුත්‍රබලයෙන් නො ද යුතු වූවා වේ ද, මෙසේ ඇය ඒ අඟින් නො පිරිපුන් වෙයි. මහණෙනි, යම් හෙයෙකින් මාගම රූපබලයෙනුත් භොගබලයෙනුත් ඤාතිබලයෙනුත් පුත්‍රබලයෙනුත් යුතු වූවා වේ ද මෙසේ ඇය ඒ අඟින් පිරිපුන් වෙයි. මහණෙනි, මාගම රූපබලයෙනුත් භොගබලයෙනුත් ඤාතිබලයෙනුත් පුත්‍රබලයෙනුත් යුතු වූවා සීලබලයෙන් නො ද යුතු වූවා වේද, මෙසේ ඇය ඒ අඟින් නො පිරිපුන් වෙයි. මහණෙනි, යම් හෙයකින් මාගම රූපබලයෙනුත් භොගබලයෙනුත් ඤාතිබලයෙනුත් පුත්‍රබලයෙනුත් සීලබලයෙනුත් යුතු වූවා වේ ද, මෙසේ ඇය ඒ අඟින් පිරිපුන් වෙයි. මහණෙනි, මේ පස මාගමගේ බල යි.
