# 3.3.4. අභිභුය්‍ය සූත්‍රය

309\. මහණෙනි, මාගමට මේ බල පසෙකි. කවර පසෙක් ද යත්: රූපබලය, භොගබලය, ඤාතිබලය, පුත්‍රබලය, සීලබලය යි. මහණෙනි, මේ මාගමගේ බල පස යි. මහණෙනි මේ බලපසින් යුතු මාගම සැමියා යටත් කොට ගෙන පවතියි.

මහණෙනි, එක්බලයකිනුදු යුතු පුරුෂ තෙමේ මාගම යටත් කොට ගෙන පවතියි. කවර එකෙකින්ද යත්: ඉසුරු බෙලෙනි. මහණෙනි, ඉසුරු බෙලෙන් මඬනා ලද මාගම රූපබලය නො ම රකී. භොගබලය නො ම රකී. ඤාතිබලය නො ම රකී. පුත්‍රබලය නො ම රකී. සීල බලය නො ම රකී යි.
