# 3.3.10. වඩ්‌ඪී සූත්‍රය

315\. මහණෙනි, වැඩුම් පසෙකින් වැඩෙන අරීසැවි තොමෝ අරීවැඩුමෙන් වැඩේ. කයෙහි හර ගන්නී ද උතුම් දෑ ගන්නී ද වෙයි. කවර පසෙකින් ද යත්: සැදැහැයෙන් වැඩේ. සීලයෙන් වැඩේ. ශ්‍රැතයෙන් වැඩේ. ත්‍යාගයෙන් වැඩේ. ප්‍රඥාවෙන් වැඩේ. මහණෙනි, මේ පස් වැඩුමෙන් වැඩෙන අරීසැවි තොමෝ අරීවැඩුමෙන් වැඩේ. කයෙහි හර ගන්නී ද උතුම් දෑ ගන්නී ද වෙයි.

මෙලෙව්හි යම් ස්ත්‍රියක් සැදැහැයෙනුත් සීලයෙනුත් ප්‍රඥාවෙනුත් ත්‍යාග–ශ්‍රැත දෙකිනුත් වැඩේ ද එබඳු සිල්වත් ඒ උවැසි තොමෝ මෙලෙව්හි ම තමාගේ හර ගනී යි.
