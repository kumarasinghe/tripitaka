# 3.3.9. විසාරදවාස සූත්‍රය

314\. මහණෙනි, දහම් පසෙකින් යුතු මාගම විසාරද ව ගෙහි වෙසේ. කවර පසෙකින් ද යත්: පණිවායෙන් වැළකුණී වෙයි. අයිනාදනින් වැළකුණී වෙයි. කාමයෙහි වරදවා හැසුරුමෙන් වැළකුණී වෙයි. මුසවායින් වැළකුණී වෙයි. රහමෙර පානයෙන් වැළකුණී වෙයි. මහණෙනි, මේ දහම් පසෙන් යුත් මාගම විසාරද ව ගෙහි වෙසේ යි.
