# 3.3.7. හේතු සූත්‍රය

312\. මහණෙනි, මාගමට මේ බල පසෙකි, කවර පසෙක් ද යත්: රූපබලය භොගබලය ඤාතිබලය පුත්‍රබලය සීලබලය යි. මහණෙනි, මාගම රූපබලය හේතු කොට හෝ භොගබලය හේතු කොට හෝ ඤාතිබලය හෙතු කොට හෝ පුත්‍රබලය හෙතුකොට හෝ කාබුන් මරණින් මතු සෝභන ගති ඇති සගලොවට නො පැමිණෙයි. මහණෙනි, මාගම සීලබලය හේතු කොට කාබුන් මරණින් මතු සෝභන ගති ඇති සගලොවට පැමිණෙයි. මහණෙනි, මේ මාගමගේ බල පස යි.
