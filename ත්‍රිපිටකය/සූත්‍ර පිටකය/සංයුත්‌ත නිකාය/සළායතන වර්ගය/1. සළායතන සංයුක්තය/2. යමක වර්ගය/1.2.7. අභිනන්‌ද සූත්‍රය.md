# 1.2.7. අභිනන්‌ද සූත්‍රය

19\. **සැවැත්නුවර–**

මහණෙනි, යමෙක් ඇස පිළිගනී නම් හෙතෙම දුක පිළිගනියි. යමෙක් දුක පිළිගනී නම් හෙතෙම දුකින් නො මිදුනේ යයි කියමි. යමෙක් කන පිළිගනී නම් හෙතෙම දුක පිළිගනියි. යමෙක් දුක පිළිගනී නම් හෙතෙම දුකින් නො මිදුනේ ය යි කියමි. යමෙක් නැහැය පිළිගනී නම් හෙතෙම දුක පිළිගනියි. යමෙක් දුක පිළිගනී නම් හෙතෙම දුකින් නො මිදුණේ ය යි කියමි. යමෙක් දිව පිළිගනී නම් හෙතෙම දුක පිළිගනියි. යමෙක් දුක පිළිගනී නම් හෙතෙම දුකින් නො මිදුනේ ය යි කියමි. යමෙක් කය පිළිගනී නම් හෙතෙම දුක පිළිගනියි. යමෙක් දුක පිළිගනි නම් හෙතෙම දුකින් නො මිදුනේ යයි කියමි. යමෙක් සිත පිළිගනි නම් හෙතෙමෙ දුක පිළිගනියි. යමෙක් දුක පිළිගනි නම් හෙතෙම දුකින් නො මිදුනේ ය යි කියමි.

මහණෙනි, යමෙක් ඇස නො පිළිගනී නම් හෙතෙම දුක නො පිළිගනියි. යමෙක් දුක නො පිළිගනී නම් හෙතෙම දුකින් මිදුනේ ය යි කියමි. යමෙක් කන නො පිළිගනී නම් හෙතෙම දුක නො පිළිගනියි. යමෙක් දුක නො පිළිගනී නම් හෙතෙම දුකින් මිදුනේ ය යි කියමි. යමෙක් නැහැය නො පිළිගනී නම් හෙතෙම දුක නො පිළිගනියි. යමෙක් දුක නො පිළිගනී නම් හෙතෙම දුකින් මිදුනේ ය යි කියමි. යමෙක් දිව නො පිළිගනී නම් හෙතෙම දුක නො පිළිගනියි. යමෙක් දුක නො පිළිගනී නම් හෙතෙම දුකින් මිදුනේ ය යි කියමි. යමෙක් කය නො පිළිගනී නම් හෙතෙම දුක නො පිළිගනියි. යමෙක් දුක නො පිළිගනී නම් හෙතෙම දුකින් මිදුනේ ය යි කියමි. යමෙක් සිත නො පිළිගනී නම් හෙතෙම දුක නො පිළිගනියි. යමෙක් දුක නො පිළිගනී නම් හෙතෙම දුකින් මිදුනේ ය යි කියමි.
