# 1.2.8. දුතිය අභිනන්‌ද සූත්‍රය

20\. **සැවැත්නුවර–**

මහණෙනි, යමෙක් රූප අභිනන්දනා කෙරේ නම් හෙතෙම දුක අභිනන්දනා කරයි. යමෙක් දුක අභිනන්දනා කෙරේ නම් හෙතෙම දුකින් නො මිදුනේ යයි කියමි. යමෙක් ශබ්ද … ගන්ධ … රස … ස්පර්ශ … ධර්ම අභිනන්දනා කෙරේ නම් හෙතෙම දුක අභිනන්දනා කරයි. යමෙක් දුක අභිනන්දනා කෙරේ නම් හෙතෙම දුකින් නො මිදුනේ ය යි කියමි.

මහණෙනි, යමෙක් රූප අභිනන්දනා නො කෙරේ නම් හෙතෙම දුක අභිනන්දනා නො කරයි. යමෙක් දුක අභිනන්දනා නො කෙරේ නම් හෙතෙම දුකින් මිදුනේ ය යි කියමි. යමෙක් ශබ්ද … ගන්ධ … රස … ස්පර්ශ … ධර්ම අභිනන්දනා නො කෙරේ නම් හෙතෙම දුක අභිනන්දනා නො කරයි. යමෙක් දුක අභිනන්දනා නො කෙරේ නම් හෙතෙම දුකින් මිදුනේ ය යි කියමි යි.
