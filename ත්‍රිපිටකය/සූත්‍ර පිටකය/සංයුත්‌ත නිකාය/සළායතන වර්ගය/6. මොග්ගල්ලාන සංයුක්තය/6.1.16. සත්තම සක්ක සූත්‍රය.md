# 6.1.16. සත්තම සක්ක සූත්‍රය

363\. ඉක්බිති සක්දෙව් රජ පන්සියයක් දෙවියන් සමග ආයුෂ්මත් මහමුගලන් තෙරණුවන් වෙත එළඹියේ ය. එළඹ ආයුෂ්මත් මහමුගලන් තෙරණුවන් වැඳ එකත්පස් ව සිටියේ ය. එකත්පස් ව සිටි සක්දෙව් රජුට ආයුෂ්මත් මහමුගලන් තෙරණුවෝ මෙය වදාළහ:

දේවානමින්ද, බුදුන් කෙරෙහි නො සැලෙන පැහැදීමෙන් යුතු වීම මැනවි. ඒ භගවත්හු මෙකරුණෙනුදු ... දෙවිමිනිසුන්ට ශාස්තෲහ, බුද්ධයහ, භගවත්හ යි. දේවානමින්ද, මෙසේ බුදුන් කෙරෙහි නො සැලෙන පැහැදීමෙන් යුතු වීම හේතු කොට මෙහි ඇතැම් සත්ත්ව කෙනෙක් කාබුන් මරණින් මතු සුගති සගලොවට පැමිණෙත්. ඔහු දිව්‍යමය ආයුෂයෙන් ... දිව්‍යමය ස්ප්‍රෂ්ටව්‍යයෙන් ද යන දශ කරුණින් සෙසු දෙවියන් යටපත් කෙරෙත්. දේවානමින්ද, දහම්හි නො සැලෙන පැහැදීමෙන් යුතු වීම මැනවි. … දේවානමින්ද, සඟුන් කෙරෙහි නො සැලෙන පැහැදීමෙන් යුතු වීම මැනවි.

දේවානමින්ද, නො කඩ වූ ... සමාධිය පිණිස පවත්නා ආර්‍ය්‍යකාන්ත ශීලයෙන් යුතු වීම මැනවි. දේවානමින්ද, මෙසේ ආර්‍ය්‍යකාන්ත ශීලයෙන් යුතු වීම හේතු කොට මෙහි ඇතැම් සත්ත්ව කෙනෙක් කාබුන් මරණින් මතු සුගති සගලොවට පැමිණෙති යි. ඔහු දිව්‍යමය ආයුෂයෙන් ද දිව්‍යමය වර්ණයෙන් ද දිව්‍යමය සුවයෙන් ද දිව්‍යමය යසසින් ද දිව්‍යමය ආධිපත්‍යයෙන් ද දිව්‍යමය රූපයෙන් ද දිව්‍යමය ශබ්දයෙන් ද දිව්‍යමය ගන්ධයෙන් ද දිව්‍යමය රසයෙන් ද දිව්‍යමය ස්ප්‍රෂ්ටව්‍යයෙන් ද යන දශ කරුණින් සෙසු දෙවියන් යටපත් කෙරෙති යි.

නිදුකාණෙනි මොග්ගල්ලානයෙනි, බුදුන් කෙරෙහි නො සැලෙන පැහැදීමෙන් යුතු වීම මැනවි. ඒ භගවත්හු මෙකරුණෙනුදු ... දෙවිමිනිසුන්ට ශාස්තෲහ, බුද්ධයහ, භගවත්හ යි. නිදුකාණෙනි මොග්ගල්ලානයෙනි, මෙසේ බුදුන් කෙරෙහි නො සැලෙන පැහැදීමෙන් යුතු වීම හේතු කොට මෙහි ඇතැම් සත්ත්ව කෙනෙක් කාබුන් මරණින් මතු සුගති සගලොවට පැමිණෙත්. ඔහු දිව්‍යමය ආයුෂයෙන් ... දිව්‍යමය ස්ප්‍රෂ්ටව්‍යයෙන් ද යන දශ කරුණෙන් සෙසු දෙවියන් යටපත් කෙරෙති.

නිදුකාණෙනි මොග්ගල්ලානයෙනි, දහම්හි නො සැලෙන පැහැදීමෙන් යුතු වීම මැනවි. භාග්‍යවතුන් වහන්සේ විසින් ධර්‍මය ස්වාඛාත ය ... නුවණැත්තන් විසින් තමතමා කෙරෙහි ලා දත යුත්තේ ය යි. නිදුකාණෙනි මොග්ගල්ලානයෙනි, මෙසේ දහම්හි නො සැලෙන පැහැදීමෙන් යුතු වීම හේතු කොට මෙහි ඇතැම් සත්ත්ව කෙනෙක් කාබුන් මරණින් මතු සුගති සගලොවට පැමිණෙත්. ඔහු දිව්‍යමය ආයුෂයෙන් ... දිව්‍යමය ස්ප්‍රෂ්ටව්‍යයෙන් ද යන දශ කරුණින් සෙසු දෙවියන් යටපත් කෙරෙත්.

නිදුකාණෙනි මොග්ගල්ලානයෙනි, සඟුන් කෙරෙහි නො සැලෙන පැහැදීමෙන් යුතු වීම මැනවි. භාග්‍යවතුන් වහන්සේගේ ශ්‍රාවක සඞ්ඝ තෙමේ සුප්‍රතිපන්න ය ... ලෝකයාට අනුත්තර පුණ්‍යක්ෂේත්‍රය යි. නිදුකාණෙනි මොග්ගල්ලානයෙනි, මෙසේ සඟුන් කෙරෙහි නො සැලෙන පැහැදීමෙන් යුතු වීම හේතු කොට මෙහි ඇතැම් සත්ත්ව කෙනෙක් කාබුන් මරණින් මතු සුගති සගලොවට පැමිණෙත්. ඔහු දිව්‍යමය ආයුෂයෙන් ද ... දිව්‍යමය ස්ප්‍රෂ්ටව්‍යයෙන් ද යන දශ කරුණෙන් සෙසු දෙවියන් යටපත් කෙරෙත්.

නිදුකාණෙනි මොග්ගල්ලානයෙනි, නො කඩ වූ ... සමාධිය පිණිස පවත්නා වූ ආර්‍ය්‍යකාන්ත ශීලයෙන් යුතු වීම මැනවි. නිදුකාණෙනි මොග්ගල්ලානයෙනි, මෙසේ ආර්‍ය්‍යකාන්ත ශීලයෙන් යුතු වීම හේතු කොට මෙහි ඇතැම් සත්ත්ව කෙනෙක් කාබුන් මරණින් මතු සුගති සග ලොවට පැමිණෙත්. ඔහු දිව්‍යමය ආයුෂයෙන් ද දිව්‍යමය වර්ණයෙන් ද දිව්‍යමය සුවයෙන් ද දිව්‍යමය යසසින් ද දිව්‍යමය ආධිපත්‍යයෙන් ද දිව්‍යමය රූපයෙන් ද දිව්‍යමය ශබ්දයෙන් ද දිව්‍යමය ගන්ධයෙන් ද දිව්‍යමය රසයෙන් ද දිව්‍යමය ස්ප්‍රෂ්ටව්‍යයෙන් ද යන දශ කරුණින් සෙසු දෙවියන් යටපත් කෙරෙති යි.
