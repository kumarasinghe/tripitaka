# 3.3.3. ලෝක සූත්‍රය

134\. සැවැත්නුවර -

එකත්පසෙක හුන් පසේනදි කොසොල් රජ තෙමේ භාග්‍යවතුන් වහන්සේට මෙය සැළ කළේ ය. වහන්ස, කෙතෙක් ලෝකධර්මයෝ උපදනාහු අහිත පිණිස දුක් පිණිස අපහසු විහරණය පිණිස උපදිද් ද?

මහරජ, ලෝදහම් තුනෙක් උපදනේ අහිත පිණිස දුක් පිණිස නො පහසු විහරණය පිණිස උපදී. කවර තුනෙක් ද? යත්: මහරජ, ලෝභ සඞ්ඛයාත ලෝකධර්මය උපදනේ අහිත පිණිස දුක් පිණිස නොපහසු විහරණය පිණිස උපදී. මහරජ, ද්වේෂසඞ්ඛයාත ලෝකධර්මය උපදනේ අහිත පිණිස දුක් පිණිස නොපහසු විහරණය පිණිස උපදී. මෝහ සඞ්ඛයාත ලෝකධර්මය උපදනේ අහිත පිණිස දුක් පිණිස නොපහසු විහරණය පිණිස උපදී. මහරජ, මේ ලෝකධර්ම තුන උපදනේ අහිත පිණිස දුක් පිණිස නො පහසු විහරණය පිණිස උපදී.

ලෝභය ද ද්වේෂය ද මෝහය ද තමා තුළ හටගත්තාහු පවිටු සිත් ඇති මිනිසා පෙළත්. සිය පලය පොතුහර රුකක් (නසන්නාක්) මෙනි.
