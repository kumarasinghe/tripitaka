# 4.3.10. දුතිය සමණබ්‍රාහ්‌මණ සූත්‍රය

1714\. මහණෙනි, යම්කිසි මහණ කෙනෙක් හෝ බමුණු කෙනෙක් හෝ චක්ඛුන්ද්‍රියය නො දනිත්ද, චක්ඛුන්ද්‍රිය සමුදය නො දනිත්ද, චක්ඛුන්ද්‍රිය නිරොධය නො දනිත්ද, චක්ඛුන්ද්‍රිය නිරොධගාමිනීපටීපදාව නො දනිත්ද, සොතින්ද්‍රිය නො දනිත්ද ... ඝානින්ද්‍රිය නො දනිත් ද ... ජිව්හින්ද්‍රිය නො දනිත්ද ... කායින්ද්‍රිය නො දනිත්ද ... මනින්ද්‍රිය නො දනිත්ද, මනින්ද්‍රිය සමුදය නො දනිත්ද, මනින්ද්‍රිය නිරොධය නො දනිත් ද, මනින්ද්‍රිය නිරොධගාමිනීපටිපදාව නො දනිත්ද ... දනිත්ද තුමූ වෙසෙසින් දැන පසක් කොට එයට පැමිණ වෙසෙති යි.
