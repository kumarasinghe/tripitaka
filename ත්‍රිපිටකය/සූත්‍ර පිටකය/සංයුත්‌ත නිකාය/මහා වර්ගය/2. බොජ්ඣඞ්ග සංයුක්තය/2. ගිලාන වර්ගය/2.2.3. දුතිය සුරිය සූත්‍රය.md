# 2.2.3. දුතිය සුරිය සූත්‍රය

559\. මහණෙනි, යම් මේ අරුණ නැගීමෙක් වේ ද උදාවන හිරුට මෙය පෙරටු ය, මෙය පෙරනිමිත්ත ය. මහණෙනි, එසෙයින් ම යම් මේ යෝනිසොමනසිකාරයෙක් වේ ද, මහණහට සත් බොජඟ ඉපදීමට මෙය පෙරටු ය, මෙය පෙරනිමිත්ත ය. මහණෙනි, සත් බොජඟ වඩන්නේය, සත් බොජඟ බහුල කරන්නේ ය යන මෙය යෝනිසොමනසිකාරයෙන් යුත් මහණහු විසින් කැමති විය යුතු. මහණෙනි, කෙසේ නම් යෝනිසෝමනසිකාරයෙන් යුත් මහණ තෙමේ සත්බොජඟ වඩා ද සත් බොජඟ බහුල කෙරේ ද යත්: මහණෙනි, මෙ මහණ තෙමේ විවේකනිස්සිත විරාගනිස්සිත නිරොධනිස්සිත වොස්සග්ගපරිණාමි සතිසම්බොජ්ඣඞ්ගය වඩයි ... උපෙක්ඛාසම්බොජ්ඣඞ්ගය වඩයි.මහණෙනි, යෝනිසොමනසිකාරයෙන් යුත් මහණ තෙමේ මෙසේ සත් බොජඟ වඩයි. සත් බොජඟ බහුල කෙරේ යි.
