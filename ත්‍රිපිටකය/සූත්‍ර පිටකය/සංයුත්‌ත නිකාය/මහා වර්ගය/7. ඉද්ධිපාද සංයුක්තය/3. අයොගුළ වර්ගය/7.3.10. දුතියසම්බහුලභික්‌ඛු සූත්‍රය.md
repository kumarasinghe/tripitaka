# 7.3.10. දුතියසම්බහුලභික්‌ඛු සූත්‍රය

3152\. එකල්හි බොහෝ භික්ෂූහු භාග්‍යවතුන් වහන්සේ වෙත එළඹියහ ... එකත්පසෙක හුන් ඒ භික්ෂූන්ට භාග්‍යවතුන් වහන්සේ මෙය වදාළ සේක: “මහණෙනි, ඉදු කවරේ ද, ඉදුපා කවරේ ද, ඉදුපා වැඩුම කවරේ ද, ඉදුපා වැඩුමට පමුණුවන පිළිවෙත කවරේ දැ”යි. වහන්ස, අපගේ දහම්හු භගවත්හු මුල් කොට ඇතියහ ...

මහණෙනි, ඉදු කවරේ ද යත්: මහණෙනි, මෙහි මහණ නන්වැදෑරුම් ඉදු කොටස් විඳී. එකෙක් ව ද බොහෝ අයුරු වේ ... බඹලොව තෙක් දු කයෙන් වසයෙහි පවත්වයි. මහණෙනි, මේ ඉදු යි කියනු ලැබේ. මහණෙනි, ඉදුපා කවරේ ද යත්: මහණෙනි, යම් මඟෙක් යම් පිළිවෙතෙක් ඉදු ලැබුම ඉදු පිළිලැබුම පිණිස පවතී ද, මහණෙනි, මේ ඉදුපා යි කියනු ලැබේ. මහණෙනි, ඉදුපා වැඩුම කවරේ ද යත්: මහණෙනි, මෙහි මහණ ඡන්දසමාධිපධානසඞ්ඛාරසමන්නාගත ඉද්ධිපාදය වඩයි. විරියසමාධි … චිත්තසමාධි … වීමංසාසමාධිපධානසඞ්ඛාරසමන්නාගත ඉද්ධිපාදය වඩයි. මහණෙනි, මේ ඉදුපා වැඩුමැ යි කියනු ලැබේ. මහණෙනි, ඉදුපා වැඩුමට පමුණුවන පිළිවෙත කවරේ ද යත්: මේ අරීඅටඟිමගමැ යි. එනම්: සම්මාදිට්ඨි … සම්මාසමාධි යි. මහණෙනි, මේ ඉදුපා වැඩුමට පමුණුවන පිළිවෙතැ යි කියනු ලැබේ යි.
