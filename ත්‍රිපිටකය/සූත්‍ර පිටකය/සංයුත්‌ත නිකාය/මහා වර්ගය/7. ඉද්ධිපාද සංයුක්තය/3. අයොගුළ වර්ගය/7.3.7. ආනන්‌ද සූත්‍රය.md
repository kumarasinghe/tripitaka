# 7.3.7. ආනන්‌ද සූත්‍රය

3149\. එකල්හි ආයුෂ්මත් **ආනන්‍ද තෙරණුවෝ** භාග්‍යවතුන් වහන්සේ වෙත එළඹියහ. එළඹ ... එකත්පස් ව හුන් ආයුෂ්මත් ආනන්‍ද තෙරණුවෝ භාග්‍යවතුන් වහන්සේට මෙය සැලකළහ: “වහන්ස, ඉද්ධි කවරේ ද ඉදධිපාද කවරේ ද ඉද්ධිපාදභාවනා කවරේ ද ඉද්ධිපාදභාවනාගාමිනීපටිපදා කවරේ දැ”යි.

ආනන්ද, මෙහි මහණ නන් වැදෑරුම් ඉදුකොටස් විඳියි: “එකෙක් ව ද බොහෝ අයුරු වේ ... බඹලොව තෙක් දු කයින් වසයෙහි පවත්වයි”. ආනන්ද, මේ ඉදු යි කියනු ලැබේ. ආනන්ද, ඉදුපා කවරේ ද යත්: ආනන්ද, යම් මඟෙක් යම් පිළිවෙතෙක් ඉදු ලැබුම ඉදු පිළිලැබුම පිණිස පවතී නම් ආනන්ද, මේ ඉදුපා යි කියනු ලැබේ. ආනන්ද, ඉදුපා වැඩුම කවරේද යත්: ආනන්ද, මෙහි මහණ ඡන්දසමාධිපධානසඞ්ඛාරසමන්නාගත ඉද්ධිපාදය වඩයි. විරියසමාධි ... චිත්තසමාධි ... වීමංසාසමාධිපධානසඞ්ඛාරසමන්නාගත ඉද්ධිපාදය වඩයි. ආනන්ද, මේ ඉදුපා වැඩුමැ යි කියනු ලැබේ. ආනන්ද, ඉදුපා වැඩුමට පමුණුවන පිළිවෙත කවරේ ද යත්: මේ අරිඅටඟි මඟ මැ යි. එනම්: සම්මාදිට්ඨි ... සම්මාසමාධි යි. ආනන්ද, මේ ඉදුපා වැඩුමට පමුණුවන පිළිවෙතැයි කියනු ලැබේ යි.
