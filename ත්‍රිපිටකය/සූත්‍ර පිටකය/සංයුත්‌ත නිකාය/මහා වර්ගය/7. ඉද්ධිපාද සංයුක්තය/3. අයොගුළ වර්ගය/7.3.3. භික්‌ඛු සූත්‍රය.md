# 7.3.3. භික්‌ඛු සූත්‍රය

3145\. මහණෙනි, මේ ඉද්ධිපාද සතරෙකි. කවර සතරෙක්ද යත්: මහණෙනි, මෙසස්නෙහි මහණ ඡන්දසමාධිපධානසඞ්ඛාරසමන්නාගත ඉද්ධිපාදය වඩයි. විරියසමාධි ... චිත්තසමාධි ... වීමංසාසමාධිපධානසඞ්ඛාරසමන්නාගත ඉද්ධිපාදය වඩයි. මහණෙනි, මේ ඉද්ධිපාද සතර යි. මහණෙනි, මේ සිවු ඉදුපා වැඩූ බැවින් බහුල කළ බැවින් මහණ ආස්‍රවයන් ක්‍ෂය කිරීමෙන් අනාස්‍රව වූ ඵලසමාධියත් ඵලප්‍රඥාවත් දිටුදැමියෙහි ම තෙමේ වෙසෙසින් දැන පසක් කොට එයට පැමිණ වෙසේ යි.
