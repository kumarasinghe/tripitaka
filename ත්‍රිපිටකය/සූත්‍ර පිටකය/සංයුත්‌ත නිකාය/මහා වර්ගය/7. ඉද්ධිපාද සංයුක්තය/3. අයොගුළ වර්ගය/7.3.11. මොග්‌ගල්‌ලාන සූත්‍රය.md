# 7.3.11. මොග්‌ගල්‌ලාන සූත්‍රය

3153\. මහණෙනි, ඒ කිමැයි හඟිවු ද? කවර දහම් වැඩූ බැවින් බහුල කළ බැවින් **මුගලන් මහණ** මෙබඳු මහත් සෘද්ධි ඇත්තේ මෙබඳු අනුභාව ඇත්තේ දැ යි. වහන්ස, අපගේ දහම්හු භගවත්හු මුල් කොට ඇතියහ. ... මහණෙනි, සිවු ඉදුපා වැඩූ බැවින් බහුල කළ බැවින් මුගලන් මහණ මෙබඳු මහත් සෘද්ධි ඇත්තේ මෙබඳු මහත් අනුභාව ඇත්තේ ය. කවර සිවු ඉදුපා ද යත්: මහණෙනි, මෙහි මුගලන් මහණ ඡන්‍දසමාධිපධානසඞ්ඛාරසමන්නාගත ඉද්ධිපාදය වඩයි. මෙසේ මාගේ ඡන්‍දය ඉතා සැඟවුණේ නො ද වන්නේය ... බැබළුම් සහිත සිත වඩයි. විරියසමාධි ... චිත්තසමාධි ... වීමංසාසමාධිපධානසඞ්ඛාරසමන්නාගත ඉද්ධිපාදය වඩයි. මෙසේ මාගේ වීමංසාව ඉතා සැඟවුණේ නො ද වන්නේය. ඉතා දැඩි කොට ගන්නා ලද්දේ නො ද වන්නේ ය ... මෙසේ විවෘත වූ හාත්පසින් නො බැඳුණු සිතින් බැබළුම් සහිත සිත වඩයි. මහණෙනි, මේ සිවු ඉදුපා වැඩූ බැවින් බහුල කළ බැවින් මුගලන් මහණ මෙබඳු මහත් සෘද්ධි ඇත්තේ ය. මෙබඳු මහත් අනුභාව ඇත්තේ ය.

මහණෙනි, මේ සිවු ඉදුපා ම වැඩූ බැවින් බහුල කළ බැවින් මුගලන් මහණ මෙසේ නන් වැදෑරුම් ඉදු කොටස් විඳී: “එකෙක් ව ද බොහෝ අයුරු වේ ... බඹලොව තෙක් දු කයින් වයයෙහි පවත්ව”යි. මහණෙනි, මේ සිවු ඉදුපා ම වැඩූ බැවින් බහුල කළ බැවින් මුගලන් මහණ ආස්‍රව ක්‍ෂය කිරීමෙන් අනාස්‍රව වූ ඵලසමාධියත් ඵලප්‍රඥාවත් දිටුදැමියෙහි ම තෙමේ වෙසෙසින් දැන පසක් කොට එයට පැමිණ වෙසේ යි.
