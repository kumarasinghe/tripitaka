# 3.3.5. බ්‍රාහ්‌මණ සූත්‍රය

1203\. **සැවැත්නුවර:**

එකල්හි එක්තරා බමුණෙක් භාග්‍යවතුන් වහන්සේ වෙත එළඹියේ ය. එළඹ භාග්‍යවතුන් වහන්සේ සමග සතුටු විය. සතුටු විය යුතු සිහි කටයුතු කතා කොට නිමවා එකත්පස් ව හුන්නේ ය. එකත්පස් ව හුන් ඒ බමුණු භාග්‍යවතුන් වහන්සේට මෙය සැල කෙළේ ය: “භවත් ගෞතමයෙනි, යම් හෙයෙකින් තථාගතයන් පිරිනිවි කල්හි සද්ධර්මය චිරස්ථිතික නො වේ නම් එයට කිමෙක් හේතු ද? කිමෙක් ප්‍රත්‍යය ද? භවත් ගෞතමයෙනි, යම් හෙයකින් තථාගතයන් පිරිනිවි කල්හි සද්ධර්මය චිරස්ථිතික වේ නම් එයට කිමෙක් හෙතු ද? කිමෙක් ප්‍රත්‍යය දැ? යි.”

බමුණ, සිවු සීවටන් නො වැඩූ බැවින් නො බහුල කළ බැවින් තථාගතයන් පිරිනිවි කල්හි සද්ධර්මය චිරස්ථිතික නො වෙයි. බමුණ සිවු සීවටන් වැඩූ බැවින් බහුල කළ බැවින් තථාගතයන් පිරිනිවි කල්හි සද්ධර්මය චිරස්ථිතික වේ. කවර සතරෙක් ද යත්: බමුණ, මෙහි මහණ කයෙහි කය අනුව දක්නාසුලු ව කෙලෙස් තවන වැර ඇති ව මනා දැනුම් ඇති ව සිහි ඇති ව කාලෙව්හි දැඩි ලොබ හා දොම්නස දුරු කොට වෙසේ. වෙදනාහි ... සිත්හි ... දහම්හි දහම් අනු ව දක්නා සුලු ව කෙලෙස් තවන වැර ඇති ව මනා දැනුම් ඇති ව සිහි ඇති ව දහම් ලෙව්හි දැඩිලොබ හා දොම්නස දුරු කොට වෙසේ. බමුණ, මේ සිවුසීවටන් නො වැඩූ බැවින් නො බහුල කළ බැවින් තථාගතයන් පිරිනිවි කල්හි සද්ධර්මය චිරස්ථිතික නො වෙයි. බමුණ, සිවු සීවටන් වැඩූ බැවින් බහුල කළ බැවින් තථාගතයන් පිරිනිවි කල්හි සද්ධර්මය චිරස්ථිතික වේ යි. මෙසේ වදාළ කල්හි ඒ බමුණු භාග්‍යවතුන් වහන්සේ ට මෙය සැලකළේ ය; “භවත් ගෞතමයන් වහන්ස, ඉතා මැනවි, භවත් ගෞතමයන් වහන්ස, ඉතා මැනවි, ... අද පටන් දිවිහිම් කොට සරණ ගිය (මා උවසුවකු කොට දරන සේක්වා)” යි.
