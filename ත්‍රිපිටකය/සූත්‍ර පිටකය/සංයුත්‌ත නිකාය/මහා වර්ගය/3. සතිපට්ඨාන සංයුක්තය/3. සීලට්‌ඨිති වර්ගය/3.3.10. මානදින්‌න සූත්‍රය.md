# 3.3.10. මානදින්‌න සූත්‍රය

1208\. එක්කලෙක ආයුෂ්මත් **ආනන්‍ද** තෙරණුවෝ **රජගහ**නුවර සමීපයෙහි **කලන්‍දකනිවාස** නම් **වේළුවනාරාමයෙහි** වැඩ වෙසෙති. එසමයෙහි මානදින්න ගැහැවි ගටගත් ආබාධ ඇත්තේ දුකට පැමිණියේ දැඩි යේ ගිලන් වූයේ වෙයි. එකල්හි මානදින්න ගැහැවි එක්තරා පුරුෂයකු ඇමතී ය ... වහන්ස, මම මෙබඳු දුක්‍වෙදනාවකිනුදු පහස්නා ලද්දෙම් කයෙහි කය අනු ව දක්නාසුලු ව කෙලෙස් තවන වැර ඇත්තෙම් මනා දනුම් ඇත්තෙම් සිහි ඇත්තෙම් කාලෙව්හි දැඩි ලොබ හා දොම්නස දුරු කොට වෙසෙමි. වෙදනාහි ... සිත්හි ... දහම්හි දහම් අනුව දක්නාසුලු ව කෙලෙස් තවන වැර ඇත්තෙම් මනාදැනුම් ඇත්තෙම් සිහි ඇත්තෙම් දහම්ලෙව්හි දැඩිලොබ හා දොම්නස දුරු කොට වෙසෙමි. වහන්ස, යම් මේ පඤ්ච ඔරම්භාගිය සංයෝජන කෙනෙක් භාග්‍යවතුන් වහන්සේ විසින් දෙසන ලදහු නම් වහන්ස, ඔවුනතර අප්‍රහීණ කිසිවක් තමා කෙරෙහි මම නො ම දක්මි යි. ගැහැවි, තොපට ලාභයෙකි. ගැහැවි, තොපට මනා ලැබීමෙකි. ගැහැවි, තොප විසින් අනගැමිඑලය පවසන ලදැ යි.
