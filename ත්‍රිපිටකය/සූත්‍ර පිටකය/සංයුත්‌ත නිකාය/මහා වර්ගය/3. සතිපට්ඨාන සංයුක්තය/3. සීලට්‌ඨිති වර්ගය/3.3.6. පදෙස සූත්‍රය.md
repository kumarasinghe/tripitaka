# 3.3.6. පදෙස සූත්‍රය

1204\. එක්කලෙක ආයුෂ්මත් **සැරියුත්** තෙරණුවෝත් ආයුෂ්මත් **මහමුගලන්** තෙරණුවෝත් ආයුෂ්මත් **අනුරුද්ධ** තෙරණුවෝත් **සාකෙත**නුවර සමීපයෙහි කණ්ටකී වනයෙහි වැඩ වෙසෙති. එකල්හි ආයුෂ්මත් සැරියුත් තෙරණුවෝත් ආයුෂ්මත් මහමුගලන් තෙරණුවෝත් සවස්කල විවේකයෙන් නැගී සිටියාහු ආයුෂ්මත් අනුරුද්ධ තෙරණුවන් වෙත එළඹියහ. එළඹ ආයුෂ්මත් අනුරුද්ධ තෙරණුවන් සමග සතුටු වූහ. සතුටු විය යුතු සිහි කට යුතු කතා කොට නිමවා එකත්පසෙක හුන්හ. එකත්පසෙක හුන් ආයුෂ්මත් සැරියුත් තෙරණුවෝ ආයුෂ්මත් අනුරුද්ධ තෙරණුවන්ට මෙය වදාළහ: “ඇවැත්නි, අනුරුද්ධයෙනි, සෙඛ ය සෙඛ ය යි කියනු ලැබේ. ඇවැත්නි, කෙතෙකින් සෙඛ වේ දැ?” යි.

ඇවැත්නි, සිවු සීවටන් පිළිබඳ කොටසක් වැඩූ බැවින් සෙඛ වෙයි. කවර සතරක ද? යත්: ඇවැත්නි, මෙහි මහණ කයෙහි කය අනු ව දක්නා සුලු ව කෙලෙස් තවන වැර ඇති ව මනා දනුම් ඇති ව සිහි ඇති ව කාලෙව්හි දැඩිලොබ හා දොම්නස දුරු කොට වෙසෙයි. වෙදනා ... සිත්හි ... දහම්හි දහම් අනු ව දක්නාසුලු ව කෙලෙස් තවන වැර ඇති ව මනා දැනුම් ඇති ව සිහි ඇති ව දහම් ලෙව්හි දැඩි ලොබ හා දොම්නස දුරු කොට වෙසෙයි. ඇවැත්නි, මේ සිවු සීවටන් පිළිබඳ කොටසක් වැඩූ බැවින් සෙඛ වේ යි.
