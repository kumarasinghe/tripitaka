# 3.3.7. සමත්‌ත සූත්‍රය

1205\. **කණ්ටකීවනයෙහි:**

ඇවැත්නි අනුරුද්ධයෙනි, අසෙඛ ය අසෙඛ යයි කියනු ලැබේ. ඇවැත්නි, කෙතෙකින් අසෙඛ වේ දැ යි. ඇවැත්නි, සිවුසීවටන් මුළුමනින් වැඩූ බැවින් අසෙඛ වේ. කවර සතරෙක් ද? යත්: ඇවැත්නි, මෙහි මහණතෙමේ කයෙහි කය අනු ව දක්නාසුලු ව කෙලෙස් තවන වැර ඇති ව මනාදනුම් ඇති ව සිහි ඇති ව කාලෙව්හි දැඩිලොබ හා දොම්නස දුරු කොට වෙසෙයි. වෙදනා ... සිත්හි ... දහම්හි දහම් අනු ව දක්නාසුලු ව කෙලෙස් තවන වැර ඇති ව මනාදැනුම් ඇති ව සිහි ඇති ව දහම්ලෙව්හි දැඩි ලොබ හා දොම්නස දුරු කොට වෙසේ. ඇවැත්නි, මේ සිවු සීවටන් මුළුමනින් වැඩූ බැවින් අසෙඛ වේ යි.
