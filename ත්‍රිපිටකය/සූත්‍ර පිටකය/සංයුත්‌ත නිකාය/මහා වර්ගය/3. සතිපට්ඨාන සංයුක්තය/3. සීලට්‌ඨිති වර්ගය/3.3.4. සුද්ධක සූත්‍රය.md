# 3.3.4. සුද්ධක සූත්‍රය

1202\. **සැවැත්නුවර:**

මහණෙනි, මේ සතිපට්ඨාන සතරෙකි. කවර සතරෙක් ද යත්: මහණෙනි, මෙහි මහණ කයෙහි කය අනු ව දක්නා සුලු ව කෙලෙස් තවන වැර ඇති ව මනා දනුම් ඇති ව සිහි ඇති ව කාලෙව්හි දැඩි ලොබ හා දොම්නස දුරු කොට වෙසෙයි. වෙදනාහි ... සිත්හි ... දහම්හි දහම් අනු ව දක්නා සුලු ව කෙලෙස් තවන වැර ඇති ව මනා දැනුම් ඇති ව සිහි ඇති ව දහම් ලෙව්හි දැඩි ලොබ හා දොම්නස දුරු කොට වෙසේ. මහණෙනි, මොහු සතර සතිපට්ඨානයෝ යි.
