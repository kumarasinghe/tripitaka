# 3.3.8. ලොක සූත්‍රය

1206\. **කණ්ටකීවනයෙහි:**

ආයුෂ්මත් **අනුරුද්ධයෝ** කවර දහම් වැඩූ බැවින් බහුල කළ බැවින් මහාභිඥ (මහනුවණැති) බවට පැමිණියෝ දැ යි. ඇවැත්නි, සිවුසීවටන් වැඩූ බැවින් බහුල කළ බැවින් මහනුවණැති බවට පැමිණියෙමි යි. කවර සතරෙක් ද යත්: ඇවැත්නි, මෙහි මම කයෙහි කය අනු ව දක්නාසුලුව කෙලෙස් තවන වැර ඇතියෙම් මනාදැනුම් ඇතියෙම් සිහි ඇතියෙම් කාලෙව්හි දැඩිලොබ හා දොම්නස දුරු කොට වෙසෙමි. වෙදනාහි ... සිත්හි ... දහම්හි දහම් අනු ව දක්නාසුලු ව කෙලෙස් තවන වැර ඇතිව මනා දැනුම් ඇතිව සිහි ඇතිව දහම්ලෙව්හි දැඩි ලොබ හා දොම්නස දුරු කොට වෙසෙමි. ඇවැත්නි, මේ සිවුසීවටන් වැඩූ බැවින් බහුල කළ බැවින් මහනුවණැති බවට පැමිණියෙමි. ඇවැත්නි, මේ සිවුසීවටන් වැඩූ බැවින් බහුල කළ බැවින් මම දහසක් ලොව වෙසෙසින් දනිමි යි.
