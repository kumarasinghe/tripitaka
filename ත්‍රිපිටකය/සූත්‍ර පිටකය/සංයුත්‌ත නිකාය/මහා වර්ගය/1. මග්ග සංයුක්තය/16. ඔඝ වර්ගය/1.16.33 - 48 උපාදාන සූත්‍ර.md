# 1.16.33 - 48 උපාදාන සූත්‍ර

419–434. මහණෙනි, මේ උපාදාන (දැඩි ගැනුම්) සතරෙකි. කවර සතරෙක යත්: කාමූපාදාන, දිට්ඨූපාදාන, සීලබ්බතූපාදාන, අත්තවාදූපාදාන යි. මහණෙනි, මේ උපාදාන සතර යි. මහණෙනි, මේ උපාදාන සතර වෙසෙසින් දැනුම පිණිස … පිරිසිඳ දැනුම පිණිස … ගෙවීම පිණිස … පැහීම පිණිස අරීඅටඟිමඟ වැඩිය යුතු. කවර අරීඅටඟිමඟෙක යත්: මහණෙනි, මෙසස්නෙහි මහණ විවෙකනිස්සිත විරාගනිස්සිත නිරොධනිස්සිත වොස්සග්ගපරිණාමී වූ සම්මාදිට්ඨිය වඩයි ... සම්මාසමාධිය වඩයි. මහණෙනි, මේ සිවු උපාදානයන් වෙසෙසින් දැනුම පිණිස … පිරිසිඳ දැනුම පිණිස … ගෙවීම පිණිස … පැහීම පිණිස මේඅරීඅටඟිමඟ වැඩිය යුත්තේ යි.
