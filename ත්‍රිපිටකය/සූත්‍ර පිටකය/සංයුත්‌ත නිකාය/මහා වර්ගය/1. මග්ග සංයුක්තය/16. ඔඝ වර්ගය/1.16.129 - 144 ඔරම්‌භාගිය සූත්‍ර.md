# 1.16.129 - 144 ඔරම්‌භාගිය සූත්‍ර

515–530. මහණෙනි, මේ ඔරම්භාගිය සංයෝජන පසෙකි. කවර පසෙක යත් සක්ඛායදිට්ඨි විචිකිච්ඡා සීලබ්බතපරාමාස කාමච්ඡන්ද ව්‍යාපාදය යි. මහණෙනි, මේ ඔරම්භාගිය සංයෝජන පස යි. මහණෙනි, මේ ඔරම්භාගිය සංයොජන පස වෙසෙසින් දැනුම පිණිස … පිරිසිඳ දැනුම පිණිස … ගෙවීම පිණිස … පැහීම පිණිස අරීඅටඟිමඟ වැඩිය යුතු. කවර අරීඅටඟිමඟෙක යත්: මහණෙනි, මෙසස්නෙහි මහණ විවෙකය ඇසුරු කළ විරාගය ඇසුරු කළ නිරොධය ඇසුරු කළ නිවනට නැමුණු සම්මාදිට්ඨිය වඩයි ... සම්මාසමාධිය වඩයි. මහණෙනි, මේ ඔරම්භාගිය සංයෝජන පස වෙසෙසින් දැනුම පිණිස … පිරිසිඳ දැනුම පිණිස … ක්‍ෂය කරනු පිණිස … ප්‍රහාණය පිණිස මේ අරීඅටඟිමඟ වැඩිය යුත්තේ යි.
