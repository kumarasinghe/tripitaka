# 1.16.81 - 96 කාමගුණ සූත්‍ර

467–482 මහණෙනි, මේ කාමගුණ පසෙකි. කවර පසෙක යත්: ඉෂ්ට වූ කාන්ත වූ මනාප වූ ප්‍රියස්වරූප වූ කාමොපසංහිත වූ රජනීය වූ ඇසින් දතයුතු රූපයෝ ය … කණින් දතයුතු ශබ්දයෝ ය ... නැහැයෙන් දතයුතු ගන්ධයෝ ය … දිවින් දතයුතු රසයෝ ය. ඉෂ්ට වූ කාන්ත වූ මනවඩන ප්‍රියස්වභාව වූ කාමයෙන් යුත් ඇලුම් කටයුතු කයින් දතයුතු ස්පර්ශයෝ යි. මහණෙනි, මොහු පස්කම් ගුණයෝ යි. මහණෙනි, මේ පස්කම් ගුණයන් වෙසෙසින් දැනුම පිණිස … පිරිසිඳ දැනුම පිණිස … ගෙවීම පිණිස … ප්‍රහාණය පිණිස අරීඅටඟිමඟ වැඩිය යුතු. කවර අරීඅටඟීමඟෙක යත්: මහණෙනි, මෙසස්නෙහි මහණ විවෙකනිස්සිත විරාගනිස්සිත නිරොධනිස්සිත වොස්සග්ගපරිණාමී කොට සම්මාදිට්ඨිය වඩයි ... සම්මාසමාධිය වඩයි. මහණෙනි, මේ පස්කම් ගුණයන් වෙසෙසින් දැනුම පිණිස … පිරිසිඳ දැනුම පිණිස … ගෙවීම පිණිස … පැහීම පිණිස අරීඅටඟීමඟ වැඩිය යුත්තේ යි.
