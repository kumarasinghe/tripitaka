# 1.16.65 - 80 අනුසය සූත්‍ර

451–466. මහණෙනි, මේ අනුසය (නිදිගත්කෙලෙස්) සතෙකි. කවර සතෙක යත්: කාමරාගානුසය පටිඝානුසය දිට්ඨානුසය විචිකිච්ඡානුසය මානානුසය භාවරාගානුසය අවිජ්ජානුසය යි. මහණෙනි, මොහු සත් අනුසයයෝ යි. මහණෙනි, මේ සත් අනුසයයන් වෙසෙසින් දැනුම පිණිස … පිරිසිඳ දැනුම පිණිස … ගෙවීම පිණිස … පැහීම පිණිස අරීඅටඟිමඟ වැඩිය යුතු. කවර අරීඅටඟිමඟෙක යත්: මහණෙනි, මෙසස්නෙහි මහණ විවෙකනිස්සිත විරාගනිස්සිත නිරොධනිස්සිත වොස්සග්ගපරිණාමී කොට සම්මාදිට්ඨිය වඩයි ... සම්මාසමාධිය වඩයි. මහණෙනි, මේ සත් අනුසයයන් වෙසෙසින් දැනුම පිණිස … පිරිසිඳ දැනුම පිණිස … ගෙවීම පිණිස … පැහීම පිණිස මේ අරීඅටඟිමඟ වැඩිය යුත්තේ යි.
