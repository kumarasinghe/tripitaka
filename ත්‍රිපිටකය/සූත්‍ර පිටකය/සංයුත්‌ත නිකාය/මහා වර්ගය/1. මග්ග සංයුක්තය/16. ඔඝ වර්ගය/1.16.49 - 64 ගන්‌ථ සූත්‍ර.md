# 1.16.49 - 64 ගන්‌ථ සූත්‍ර

435–450. මහණෙනි, මේ ගන්ථ (ගෙතීම්) සතරෙකි. කවර සතරෙක යත්: අභිජ්ඣාකායගන්ථ ව්‍යාපාදකායගන්ථ සීලබ්බතපරාමාසකායගන්ථ ඉදංසච්චාගිනිවෙසකායගන්ථ යි. මහණෙනි, මේ ගන්ථ සතර යි. මහණෙනි, මේ ගන්ථසතර වෙසෙසින් දැනුම පිණිස … පිරිසිඳ දැනුම පිණිස … ගෙවීම පිණිස .... පැහීම පිණිස අරීඅටඟීමඟ වැඩිය යුතු. කවර අරීඅටඟිමඟෙක යත්: මහණෙනි, මෙසස්නෙහි මහණ විවෙකනිස්සිත විරාගනිස්සිත නිරොධනිස්සිත වොස්සග්ගපරිණාමී කොට සම්මාදිට්ඨිය වඩයි ... සම්මාසමාධිය වඩයි. මහණෙනි, මේ සිවු ගන්ථයන් වෙසෙසින් දැනුම පිණිස … පිරිසිඳ දැනුම පිණිස … ගෙවීම පිණිස … පැහීම පිණිස මේ අරීඅටඟිමඟ වැඩිය යුත්තේ යි.
