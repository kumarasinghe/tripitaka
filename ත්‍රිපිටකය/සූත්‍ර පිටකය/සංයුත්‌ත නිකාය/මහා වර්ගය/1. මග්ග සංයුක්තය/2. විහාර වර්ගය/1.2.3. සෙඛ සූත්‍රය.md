# 1.2.3. සෙඛ සූත්‍රය

13\. **සැවැත්නුවර:**

එකල්හි එක්තරා මහණෙක් භාග්‍යවතුන් වහන්සේ වෙත එළඹියේය. එළඹ භාග්‍යවතුන් වහන්සේ වැඳ එකත් පසෙක හුන්නේය. එකත් පසෙක හුන් ඒ මහණ භාග්‍යවතුන් වහන්සේට මෙය සැලකළේ ය. වහන්ස, “සේඛය, සේඛ යයි කියනු ලැබේ” යි වහන්ස, කෙතෙකින් සේඛ වේ දෝ?” යි මෙසස්නෙහි මහන සේඛ වූ සම්මාදිට්ඨීයෙන් යුතු වූයේ වේද සේඛ වූ සම්මාසඞ්කප්පයෙන් යුතු වූයේ වේද සේඛ වූ සම්මාවාචායෙන් යුතු වූයේ වේ ද සේඛ වූ සම්මාකම්මන්තයෙන් යුතු වූයේ වේ ද, සේඛ වූ සම්මා ආජීවයෙන් යුතු වූයේ වේද සේඛ වූ සම්මාවායාමයෙන් යුතු වූයේ වේ ද සේඛ වූ සම්මාසතියෙන් යුතු වූයේ වේ ද සේඛ වූ සම්මාසමාධියෙන් යුතු වූයේ වේ ද, මෙතෙකින් මහණ සේඛ වේ යි.
