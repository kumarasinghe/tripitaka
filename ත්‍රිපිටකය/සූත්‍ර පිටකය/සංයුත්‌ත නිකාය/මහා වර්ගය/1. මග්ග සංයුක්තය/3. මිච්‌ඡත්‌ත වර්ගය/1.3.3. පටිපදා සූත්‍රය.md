# 1.3.3. පටිපදා සූත්‍රය

23\. **සැවැත්නුවර:**

මහණෙනි මිච්ඡාපටිපදාව (වැරදි පිළිවෙත) ද සම්මාපටිපදාව (නොවැරදි පිළිවෙත) ද තොපට දෙශනා කරන්නෙමි එය අසවු. මහණෙනි මිච්ඡාපටිපදාව කවර යත්: මිච්ඡාදිට්ඨි ... මිච්ඡාසමාධි යි. මහණෙනි, මේ මිච්ඡාපටිපදා ය යි කියනු ලැබේ.

මහණෙනි, සම්මාපටිපදාව කවර යත්: සම්මාදිට්ඨි ... සම්මාසමාධි යි. මහණෙනි, මේ සම්මාපටිපදාව යයි කියනු ලැබේ.
